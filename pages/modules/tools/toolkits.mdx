# 工具包


工具包是一组设计用于一起执行特定任务的工具集。它们具有便捷的加载方法。
要查看所有可用的现成工具包的完整列表，请访问[Integrations](/docs/integrations/toolkits/)。

所有工具包都公开了一个 `get_tools` 方法，该方法返回一个工具列表。
因此，您可以执行以下操作：

```python
# 初始化工具包
toolkit = ExampleTookit(...)

# 获取工具列表
tools = toolkit.get_tools()

# 创建代理
agent = create_agent_method(llm, tools, prompt)
```
