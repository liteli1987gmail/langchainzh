本指南向你展示如何在本地运行 LangGraph 应用程序。

## 前提条件

在开始之前，请确保你满足以下条件：

* [LangSmith](https://smith.langchain.com/settings) 的 API 密钥 - 可免费注册

## 1. 安装 LangGraph CLI

:::python
<CodeGroup>

</CodeGroup>
```bash pip
# Python >= 3.11 is required.
pip install -U "langgraph-cli[inmem]"
```

```bash uv
# Python >= 3.11 is required.
uv add "langgraph-cli[inmem]"
```
</CodeGroup>
:::

:::js
```shell
npm install --save-dev @langchain/langgraph-cli
```
## 2. 创建一个 LangGraph 应用

:::python
使用 [`new-langgraph-project-python` 模板](https://github.com/langchain-ai/new-langgraph-project) 创建一个新应用。该模板演示了一个单节点应用，你可以用它来扩展自己的逻辑。

:::

```shell
langgraph new path/to/your/app --template new-langgraph-project-python
```

<Tip>
**附加模板**
如果不指定模板而使用 `langgraph new`，系统会显示一个交互式菜单，允许你从可用模板列表中进行选择。
</Tip>
:::

:::js
使用 [`new-langgraph-project-js` 模板](https://github.com/langchain-ai/new-langgraphjs-project) 创建新应用。该模板演示了一个单节点应用程序，你可以用自己的逻辑进行扩展。


</fragment>

```shell
npm create langgraph
```

<Accordion title="将 LangGraph 添加到现有项目中">
如果您有包含 LangGraph agent 的现有项目，可以使用 `config` 命令自动生成 `langgraph.json` 配置文件：


</Accordion>

```shell
npm create langgraph config
```

此命令会扫描项目中的 LangGraph agents（如 `createAgent()`、`StateGraph.compile()` 或 `workflow.compile()` 模式）并生成包含所有导出 agents 的配置文件。

示例输出：

```json
{
  "node_version": "24",
  "graphs": {
    "agent": "./src/agent.ts:agent",
    "searchAgent": "./src/search.ts:searchAgent"
  },
  "env": ".env"
}
```

<Tip>
只有**导出**的代理会被包含在配置中。如果某个代理未被导出，命令会发出警告，以便你添加 `export` 关键字。
</Tip>
</Accordion>
:::

## 3. 安装依赖项

在你新的 LangGraph 应用的根目录中，以 `edit` 模式安装依赖项，以便服务器使用你的本地更改：

:::python
<CodeGroup>

</fragment>
```bash pip
cd path/to/your/app
pip install -e .
```

```bash uv
cd path/to/your/app
uv sync
```
</CodeGroup>
:::

:::js
```shell
cd path/to/your/app
npm install
```
:::

## 4. 创建一个 `.env` 文件

你会在新的 LangGraph 应用的根目录中找到 `.env.example` 文件。在你的新 LangGraph 应用的根目录中创建一个 `.env` 文件，并将 `.env.example` 文件的内容复制到其中，填写必要的 API 密钥：

:::

```bash
LANGSMITH_API_KEY=lsv2...
```

## 5. 启动 Agent 服务器

在本地启动 LangGraph API 服务器：

:::python
```shell
langgraph dev
```
:::

:::js
```shell
npx @langchain/langgraph-cli dev
```
:::

示例输出：

:::

```
INFO:langgraph_api.cli:

        Welcome to

╦  ┌─┐┌┐┌┌─┐╔═╗┬─┐┌─┐┌─┐┬ ┬
║  ├─┤││││ ┬║ ╦├┬┘├─┤├─┘├─┤
╩═╝┴ ┴┘└┘└─┘╚═╝┴└─┴ ┴┴  ┴ ┴

- 🚀 API: http://127.0.0.1:2024
- 🎨 Studio UI: https://smith.langchain.com/studio/?baseUrl=http://127.0.0.1:2024
- 📚 API Docs: http://127.0.0.1:2024/docs

This in-memory server is designed for development and testing.
For production use, please use LangSmith Deployment.
```

`langgraph dev` 命令以内存模式启动 Agent Server。此模式适用于开发和测试目的。对于生产环境使用，请部署 Agent Server 并能够访问持久化存储后端。更多信息，请参阅 [平台设置概述](/langsmith/platform-setup)。

## 6. 在 Studio 中测试您的应用程序

[Studio](/langsmith/studio) 是一个专用 UI，您可以通过它连接到 LangGraph API 服务器来可视化、交互和调试本地应用程序。访问 `langgraph dev` 命令输出中提供的 URL，在 Studio 中测试您的图：

```
>    - LangGraph Studio Web UI: https://smith.langchain.com/studio/?baseUrl=http://127.0.0.1:2024
```

对于运行在自定义主机/端口上的 Agent Server，请更新 URL 中的 `baseUrl` 查询参数。例如，如果您的服务器运行在 `http://myhost:3000`：

```
https://smith.langchain.com/studio/?baseUrl=http://myhost:3000
```

<Accordion title="Safari 兼容性">
    使用 `--tunnel` 标志与命令一起创建安全隧道，因为 Safari 在连接 localhost 服务器时有局限性：


</Accordion>

    ```shell
    langgraph dev --tunnel
    ```
</Accordion>

## 7. 测试 API

:::python
<Tabs>
    <Tab title="Python SDK (async)">
    1. 安装 LangGraph Python SDK：

</fragment>
        ```shell
        pip install langgraph-sdk
        ```
2. 发送消息给助手（无线程运行）：
        ```python
        from langgraph_sdk import get_client
        import asyncio

        client = get_client(url="http://localhost:2024")

        async def main():
            async for chunk in client.runs.stream(
                None,  # Threadless run
                "agent", # Name of assistant. Defined in langgraph.json.
                input={
                "messages": [{
                    "role": "human",
                    "content": "What is LangGraph?",
                    }],
                },
            ):
                print(f"Receiving new event of type: {chunk.event}...")
                print(chunk.data)
                print("\n\n")

        asyncio.run(main())
        ```
</Tab>
<Tab title="Python SDK（同步）">
1. 安装 LangGraph Python SDK：

</fragment>
        ```shell
        pip install langgraph-sdk
        ```
2. 发送消息给助手（无线程运行）：
        ```python
        from langgraph_sdk import get_sync_client

        client = get_sync_client(url="http://localhost:2024")

        for chunk in client.runs.stream(
            None,  # Threadless run
            "agent", # Name of assistant. Defined in langgraph.json.
            input={
                "messages": [{
                    "role": "human",
                    "content": "What is LangGraph?",
                }],
            },
            stream_mode="messages-tuple",
        ):
            print(f"Receiving new event of type: {chunk.event}...")
            print(chunk.data)
            print("\n\n")
        ```
</Tab>
    <Tab title="REST 接口">
    ```bash
    curl -s --request POST \
        --url "http://localhost:2024/runs/stream" \
        --header 'Content-Type: application/json' \
        --data "{
            \"assistant_id\": \"agent\",
            \"input\": {
                \"messages\": [
                    {
                        \"role\": \"human\",
                        \"content\": \"What is LangGraph?\"
                    }
                ]
            },
            \"stream_mode\": \"messages-tuple\"
        }"
    ```
<fragment>
    </Tab>
</Tabs>
:::

:::js
<Tabs>
    <Tab title="Javascript SDK">
    1. 安装 LangGraph JS SDK：

</fragment>
        ```shell
        npm install @langchain/langgraph-sdk
        ```
2. 向助手发送消息（无线程运行）：
      ```js
      import { Client } from "@langchain/langgraph-sdk";

      // only set the apiUrl if you changed the default port when calling langgraph dev
      const client = new Client({ apiUrl: "http://localhost:2024"});

      const streamResponse = client.runs.stream(
        null, // Threadless run
        "agent", // Assistant ID
        {
          input: {
            "messages": [
              { "role": "user", "content": "What is LangGraph?"}
            ]
          },
          streamMode: "messages-tuple",
        }
      );

      for await (const chunk of streamResponse) {
        console.log(`Receiving new event of type: ${chunk.event}...`);
        console.log(JSON.stringify(chunk.data));
        console.log("\n\n");
      }
      ```
</Tab>
<Tab title="REST 接口">

</Tab>

    ```bash
    curl -s --request POST \
        --url "http://localhost:2024/runs/stream" \
        --header 'Content-Type: application/json' \
        --data "{
            \"assistant_id\": \"agent\",
            \"input\": {
                \"messages\": [
                    {
                        \"role\": \"human\",
                        \"content\": \"What is LangGraph?\"
                    }
                ]
            },
            \"stream_mode\": \"messages-tuple\"
        }"
    ```
