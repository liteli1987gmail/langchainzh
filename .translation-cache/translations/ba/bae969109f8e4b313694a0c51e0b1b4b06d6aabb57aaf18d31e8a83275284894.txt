本页面列出了自托管 LangSmith 使用的数据库、工具和基础架构的最低支持版本。在部署或升级之前，请安装满足或超过最低要求的版本。使用低于最低要求的版本不受支持，可能导致安装或运行时故障。

<Tip>
如果您使用的是托管服务（例如 Amazon RDS、Google Cloud SQL 或 Azure Database），请选择等于或高于此处列出的最低要求的版本。请参阅链接的“外部服务”页面了解连接和身份验证详细信息。
</Tip>

## 数据存储

| 依赖项 | 最低版本 | 备注 |
|------------|------------------|-------|
| [PostgreSQL](/langsmith/self-host-external-postgres) | 14 | 操作数据的主要关系存储。LangSmith 和独立 Agent Server 部署都需要此版本。用于安装 `btree_gin`, `btree_gist`, `pgcrypto`, `citext`, `ltree`和 `pg_trgm` 扩展。 |
| [Redis](/langsmith/self-host-external-redis) | 5 | 用于队列和缓存。支持单机模式和 Redis Cluster 模式。 |
| [Valkey](/langsmith/self-host-external-redis) | 8 | 官方支持作为 Redis 的替代方案。支持单机模式和 Cluster 模式。 |
| [ClickHouse](/langsmith/self-host-external-clickhouse) | Helm chart 中指定的版本 [LangSmith Helm chart](https://github.com/langchain-ai/helm/releases) 或更高版本 | 存储追踪和反馈。ClickHouse 版本 >= 24.2 需要 LangSmith v0.6 或更高版本。不支持降级。 |

<Warning>
**不支持 Redis < 5 和 PostgreSQL < 14。** 指向旧版 Redis 或 PostgreSQL 实例的 LangSmith 安装可能会启动失败或运行异常。在安装或升级 LangSmith 之前，请先升级您的数据存储。
</Warning>

## 计算与编排

| 依赖项 | 最低版本 | 备注 |
|------------|------------------|-------|
| [Kubernetes](/langsmith/kubernetes) | 上游 Kubernetes 发布周期支持的任何版本 [上游 Kubernetes 发布周期](https://kubernetes.io/releases/) | LangSmith 定期在 GKE、EKS、AKS、Minikube 和 Kind 上进行测试。|
| [OpenShift](/langsmith/kubernetes) | 4.14 | 作为 LangSmith 的 Kubernetes 发行版受支持。|
| [Helm](https://helm.sh/docs/intro/install/) | 3 | 用于安装和升级 LangSmith Helm chart。|
| Docker | 与 [Docker Compose v2](https://docs.docker.com/compose/) | Docker 独立部署的 Agent Server 需要。|

## LangSmith chart 和平台

| 依赖项 | 最低版本 | 备注 |
|------------|------------------|-------|
| [LangSmith Helm chart](https://github.com/langchain-ai/helm/releases) | 最新稳定版 | 我们建议绑定到最新的稳定 chart 版本。参阅 [自托管变更日志](/langsmith/self-hosted-changelog) 查看每个版本的升级说明。|
| 出站流量至 `https://beacon.langchain.com` | LangSmith 0.9.0 或更高版本 | 除非在 [离线模式](/langsmith/self-host-egress). |

## 中运行，否则需要许可证验证和使用量报告

- PostgreSQL `>= 14`：请参阅 [连接到外部 PostgreSQL 数据库](/langsmith/self-host-external-postgres#requirements).
- Redis `>= 5` 和 Valkey `8`：请参阅 [连接到外部 Redis 或 Valkey 数据库](/langsmith/self-host-external-redis#requirements).
- ClickHouse：使用 Helm chart 中指定的版本 [LangSmith Helm chart](https://github.com/langchain-ai/helm/releases) 或更高版本：请参阅 [连接到外部 ClickHouse 数据库](/langsmith/self-host-external-clickhouse#requirements).
- Kubernetes 集群先决条件：请参阅 [在 Kubernetes 上自托管 LangSmith](/langsmith/kubernetes#prerequisites).

如果您不确定当前运行的版本，请联系您的数据库管理员或参阅您的云提供商的控制台。如果您需要在升级 LangSmith 之前升级依赖项，请参阅您的供应商的文档。
