## 概述

`InMemoryByteStore` 是 `ByteStore` 的非持久化实现，将所有内容存储在 Python 字典中。它适用于演示以及不需要在 Python 进程生命周期之外持久化的场景。

### 安装

LangChain `InMemoryByteStore` 集成位于 `langchain-core` 包中：

```python
pip install -qU langchain-core
```

## 实例化

现在您可以实例化您的字节存储：

```python
from langchain_core.stores import InMemoryByteStore

kv_store = InMemoryByteStore()
```

## 用法

你可以使用 `mset` 方法像这样设置键下的数据：

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

您也可以使用 `mdelete` 方法删除数据：

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```
