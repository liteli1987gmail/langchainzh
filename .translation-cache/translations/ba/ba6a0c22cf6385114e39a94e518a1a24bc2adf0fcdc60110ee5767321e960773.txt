## 概述

 `InMemoryByteStore` 是一个非持久化的 `ByteStore` 实现，它将所有内容存储在 Python 字典中。它适用于演示和在 Python 进程生命周期内不需要持久化的场景。

### 安装

LangChain `InMemoryByteStore` 集成位于 `langchain-core` package:

```python
pip install -qU langchain-core
```

## 实例化

现在您可以实例化您的字节存储：

```python
from langchain_core.stores import InMemoryByteStore

kv_store = InMemoryByteStore()
```

## 用法

您可以使用以下方式通过键设置数据 `mset` method:

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

您可以使用以下方式删除数据 `mdelete` method:

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```
