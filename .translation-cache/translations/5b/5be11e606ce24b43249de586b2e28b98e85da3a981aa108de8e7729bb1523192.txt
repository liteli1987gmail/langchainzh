<Note>
**私人测试版：** LLM Gateway 处于私人[测试版](/langsmith/release-stages)阶段。API 和功能可能会随着迭代而发生变化。请注册[等待列表](https://www.langchain.com/langsmith-llm-gateway-waitlist)以获取访问权限。
</Note>

LLM Gateway 是一个代理，位于您的智能体（或任何 LLM 客户端）与它们调用的 LLM 提供商之间。无需每个客户端在本地存储提供商 API 密钥，密钥只需在 LangSmith 中存储一次，作为提供商密钥。客户端使用 [LangSmith API 密钥](/langsmith/create-account-api-key) 进行身份验证。

当请求通过网关时，它会：

1. 使用 LangSmith API 密钥对调用者进行身份验证。
1. 从您工作区的提供商密钥中解析实际的提供商 API 密钥。
1. 评估治理策略（消费限制、PII 删除、密钥删除）。
1. 将请求代理到上游提供商。
1. 将调用追踪到 LangSmith。

每个通过网关代理的调用都作为[追踪](/langsmith/observability-concepts#traces)出现在 LangSmith 中。当策略触发时，事件会流入 [LangSmith Engine](/langsmith/engine) 进行分类——您可以从被阻止的请求跳转到触发它的追踪再到修复，全部在一个产品中完成。

## 功能可用性

| 功能 | 描述 |
| --- | --- |
| **消费限制** | 在[组织](/langsmith/administration-overview#organizations)、[工作区](/langsmith/administration-overview#workspaces)、[API 密钥](/langsmith/administration-overview#api-keys)或用户级别执行硬性阻止。当达到上限时，调用者会收到带有可操作错误消息的 402 响应。 |
| **消费可见性** | 按工作区、用户、API 密钥实时汇总成本。通过自定义图表可以获得按模型的可见性。 |
| **PII 删除** | 在请求到达模型之前，检测并删除姓名、地点、国籍、宗教、政治派别和年龄。 |
| **密钥删除** | 检测并删除美国电话号码、美国社会安全号码、API 密钥、令牌和凭证。覆盖 AWS、GitHub、GitLab、OpenAI、Anthropic、GCP、Azure、Slack、Datadog、PyPI、npm、私钥和 LangSmith 令牌。 |
| **LangSmith Engine 集成** | 策略违规会在 LangSmith Engine 中显示为问题。从违规可以点击进入生成它的追踪。 |
| **审计日志** | 网关配置和网关调用的管理变更会被记录。 |
| **追踪** | 每个通过网关代理的调用都会出现在与您智能体其他追踪相同的 LangSmith 工作区中。通过网关路由不会分散您的可观察性。 |

## 支持的提供商

| 提供商 | 网关路径 | 密钥名称 |
| --- | --- | --- |
| Anthropic | `/anthropic` | `ANTHROPIC_API_KEY` |
| AWS Bedrock | `/bedrock` | `AWS_BEARER_TOKEN_BEDROCK` |
| Baseten | `/baseten` | `BASETEN_API_KEY` |
| Fireworks | `/fireworks` | `FIREWORKS_API_KEY` |
| Google Gemini | `/gemini` | `GOOGLE_API_KEY` |
| Google Vertex AI | `/vertex` | `VERTEX_SERVICE_ACCOUNT_JSON` |
| OpenAI | `/openai` | `OPENAI_API_KEY` |

## 已知限制

- **Claude Desktop：** 不支持市场插件。聊天选项卡不可见，但 Cowork 可以正常工作。
- **Codex Desktop：** 不支持市场插件。
- **Cursor：** IDE 集成尚不可用。
- **ChatGPT Desktop / Gemini Desktop：** 不可配置。
