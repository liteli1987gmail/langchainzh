LangSmith 允许您直接在 UI 中运行评估。[**Playground**](/langsmith/prompt-engineering-concepts#playground) 允许您通过一系列输入来测试提示词或模型配置，以查看它在不同上下文或场景中的评分情况，而无需编写任何代码。

在运行评估之前，您需要有一个[现有数据集](/langsmith/evaluation-concepts#datasets)。了解如何[从 UI 创建数据集](/langsmith/manage-datasets-in-application#create-a-dataset-and-add-examples)。

要在 Studio 中运行评估，请参阅[在 Studio 中对数据集运行实验](/langsmith/observability-studio#run-experiments-over-a-dataset)。如果您更喜欢在代码中运行实验，请参阅[使用 SDK 运行评估](/langsmith/evaluate-llm-application)。

![Playground experiment](/langsmith/images/playground-experiment.gif)

<Callout type="info" icon="feather">
Playground 中提供 [**Chat**](/langsmith/chat) 功能，帮助您在运行评估之前优化提示词。
</Callout>

## 在 Playground 中创建实验

1. **点击侧边栏中的 Playground**。
2. **添加提示词**：选择现有保存的提示词或创建新提示词。
3. **从** **测试数据集** **下拉菜单中选择数据集**

   * 请注意，数据集中的键必须与提示词的输入变量相匹配。例如，在上面的视频中，所选数据集的输入键为"blog"，它与提示词的输入变量正确匹配。
   * Playground 中最多允许 15 个输入变量。

4. **点击** **Start** **或 CMD+Enter 启动实验**。这将在数据集中的所有示例上运行提示词，并在数据集详情页面中为该实验创建一个条目。我建议在启动实验之前先将提示词提交到提示词中心，以便在后续审查实验时能够轻松引用。
5. **点击** **View full experiment** **查看完整结果**。这将带您进入实验详情页面，您可以在其中查看实验结果。

当实验运行时，实验表中的 **Progress** 列和实验视图中的进度条会实时跟踪完成情况，包括已完成和已评估的运行次数。有关更多信息，请参阅[跟踪实验进度](/langsmith/analyze-an-experiment#track-experiment-progress)。

## 向实验添加评估分数

通过添加评估器来根据特定标准评估您的实验。使用 **+Evaluator** 按钮在 Playground 中添加 LLM-as-a-judge 或自定义代码评估器。

要了解有关通过 UI 添加评估器的更多信息，请访问[如何定义 LLM-as-a-judge 评估器](/langsmith/llm-as-judge)。
