文档加载器提供了一个**标准接口**，用于从不同来源（如 Slack、Notion 或 Google Drive）读取数据并导入到 LangChain 的 @[Document] 格式中。
这确保了无论数据来源如何，都可以一致地处理数据。

所有文档加载器都实现了 @[`BaseLoader`] 接口。

<Warning>
社区文档加载器由用户贡献且未经核实。LangChain 不会审查或认可这些集成；使用它们需自担风险。
</Warning>

## 接口

每个文档加载器可以定义自己的参数，但它们共享一个通用 API：

* `load()` – 一次加载所有文档。
* `lazy_load()` – 延迟加载文档流，适用于大型数据集。

```python
from langchain_docling.loader import DoclingLoader

FILE_PATH = "https://arxiv.org/pdf/2408.09869"

loader = DoclingLoader(file_path=FILE_PATH)

# Load all documents
documents = loader.load()

# For large datasets, lazily load documents
for document in loader.lazy_load():
    print(document)
```
