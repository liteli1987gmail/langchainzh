>[Qdrant](https://qdrant.tech/documentation/) (读作：quadrant) 是一个向量相似度搜索引擎。
> 它提供了一个生产级别的服务，并带有便捷的 API，用于存储、搜索和管理
> points - 带有额外载荷的向量。 `Qdrant` 专为扩展的过滤支持而设计。


## 安装和设置

安装 Python 合作伙伴包：

<CodeGroup>
```bash pip
pip install langchain-qdrant
```

```bash uv
uv add langchain-qdrant
```
</CodeGroup>

## 嵌入模型

### FastEmbedSparse

```python
from langchain_qdrant import FastEmbedSparse
```

### SparseEmbeddings

```python
from langchain_qdrant import SparseEmbeddings
```

## 向量存储

存在一个围绕 `Qdrant` 索引的封装器，允许您将其用作向量存储，
无论是用于语义搜索还是示例选择。

导入此向量存储：
```python
from langchain_qdrant import QdrantVectorStore
```

有关 Qdrant 封装器的更详细演示，请参阅 [本笔记本](/oss/integrations/vectorstores/qdrant)
