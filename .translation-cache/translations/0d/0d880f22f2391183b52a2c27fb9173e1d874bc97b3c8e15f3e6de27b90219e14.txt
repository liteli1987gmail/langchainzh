该 `NomicEmbeddings` 类使用 Nomic AI API 为给定文本生成嵌入。

## 设置

要使用 Nomic API，您需要 [注册一个 Nomic 账户并创建一个 API 密钥](https://atlas.nomic.ai/).

您首先需要安装 [`@langchain/nomic`](https://www.npmjs.com/package/@langchain/nomic) package:

<Tip>
参见 [本节了解安装 LangChain 包的一般说明](/oss/langchain/install).
</Tip>

```bash npm
npm install @langchain/nomic @langchain/core
```

## 用法

```typescript
import { NomicEmbeddings } from "@langchain/nomic";

/* Embed queries */
const nomicEmbeddings = new NomicEmbeddings();
const res = await nomicEmbeddings.embedQuery("Hello world");
console.log(res);
/* Embed documents */
const documentRes = await nomicEmbeddings.embedDocuments([
  "Hello world",
  "Bye bye",
]);
console.log(documentRes);
```
