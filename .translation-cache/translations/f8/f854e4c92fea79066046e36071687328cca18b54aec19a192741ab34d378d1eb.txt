## 法律与合规

#### *LangSmith 及其区域实例遵守哪些隐私和数据保护框架？*

LangSmith 遵守《通用数据保护条例》(GDPR) 以及适用于 LangSmith 服务的其他法律法规。我们还获得了 SOC 2 Type 2 认证，并符合 HIPAA 标准。您可以在 [trust.langchain.com](https://trust.langchain.com) 申请了解更多关于我们安全策略和状况的信息。如果您想与我们签署数据处理附录 (DPA)，请通过 [support.langchain.com](https://support.langchain.com) 联系支持团队。请注意，我们仅与企业计划客户签订业务关联协议 (BAA)。

#### *我的公司不在某个区域内，我仍然可以在那里托管我的数据吗？*

是的，您可以在支持的区域实例中托管您的 LangSmith 数据，与您的位置无关。

#### *你们在欧盟有我们可以签订合同的法人实体吗？*

目前我们没有在欧盟设立用于客户签约的法人实体。

#### *如果我选择特定区域，是否有不同的法律条款？*

所有支持的云区域条款相同。

## 功能

#### *如何使用特定的区域实例？*

请按照[账户和 API 密钥设置指南](/langsmith/create-account-api-key)创建账户和 API 密钥。请务必在区域下拉菜单中选择正确的区域。

#### *云托管 LangSmith 区域之间是否存在功能差异？*

根据功能不同，向各区域发布可能存在轻微延迟。除此之外，受支持的云区域在功能上保持一致。

#### *一个组织可以在不同区域拥有工作区吗？*

LangSmith 目前不支持此功能，但如果您有兴趣，请通过 [support.langchain.com](https://support.langchain.com) 联系支持团队并分享您的用例。

#### *我可以跨区域连接组织并共享账单吗？*

LangSmith 目前不支持此功能，但如果您有兴趣，请通过 [support.langchain.com](https://support.langchain.com) 联系支持团队并分享您的用例。

#### *哪些数据会存储在我选择的区域？*

请参阅[云架构参考](/langsmith/cloud#cloud-architecture-and-scalability)了解更多详情。

#### *如何查看我的组织所在区域？*

检查您的 URL - [https://smith.langchain.com](https://smith.langchain.com) 上的组织位于 GCP US，[https://eu.smith.langchain.com](https://eu.smith.langchain.com) 上的组织位于 GCP EU，[https://apac.smith.langchain.com](https://apac.smith.langchain.com) 上的组织位于 GCP APAC，[https://aws.smith.langchain.com](https://aws.smith.langchain.com) 上的组织位于 AWS US。

#### *我可以在区域之间切换我的组织吗？*

我们目前不支持区域间迁移，但如果您对此功能感兴趣，请通过 [support.langchain.com](https://support.langchain.com) 联系支持团队。

## 计划与定价

#### *区域实例是否适用于所有 LangSmith 计划？*

是的，您可以在所有计划（包括免费计划）上注册使用受支持的区域实例。

#### *各区域的定价是否不同？*

所有受支持的云区域定价相同。

#### *使用区域实例时使用哪种货币付款？*

所有 LangSmith 计划均以美元付款。
