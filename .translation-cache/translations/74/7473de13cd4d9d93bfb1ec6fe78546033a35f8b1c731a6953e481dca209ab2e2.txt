## 设置

要访问 Ollama 嵌入模型，请按照[这些说明](https://github.com/ollama/ollama)安装 Ollama，然后安装 `@langchain/ollama` 集成包。

### 凭证

如果您想获取模型调用的自动追踪，还可以通过取消注释下方内容来设置您的 [LangSmith](/langsmith/observability) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain OllamaEmbeddings 集成位于 `@langchain/ollama` 包中：

<CodeGroup>
```bash npm
npm install @langchain/ollama @langchain/core
```
```bash yarn
yarn add @langchain/ollama @langchain/core
```
```bash pnpm
pnpm add @langchain/ollama @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化模型对象并嵌入文本：

```typescript
import { OllamaEmbeddings } from "@langchain/ollama";

const embeddings = new OllamaEmbeddings({
  model: "mxbai-embed-large", // Default value
  baseUrl: "http://localhost:11434", // Default value
});
```

## 索引与检索

嵌入模型常用于检索增强生成（RAG）流程中，既作为索引数据的一部分，也用于后续的检索。更多详细说明，请参阅 [**Learn** 选项卡](/oss/learn/)下的 RAG 教程。

下面，查看如何使用上面初始化的 `embeddings` 对象来索引和检索数据。在此示例中，我们将使用演示版 [`MemoryVectorStore`](/oss/integrations/vectorstores/memory) 来索引和检索示例文档。

```typescript
// Create a vector store with a sample text
import { MemoryVectorStore } from "@langchain/classic/vectorstores/memory";

const text = "LangChain is the framework for building context-aware reasoning applications";

const vectorstore = await MemoryVectorStore.fromDocuments(
  [{ pageContent: text, metadata: {} }],
  embeddings,
);

// Use the vector store as a retriever that returns a single document
const retriever = vectorstore.asRetriever(1);

// Retrieve the most similar text
const retrievedDocuments = await retriever.invoke("What is LangChain?");

retrievedDocuments[0].pageContent;
```

```text
LangChain is the framework for building context-aware reasoning applications
```

## 直接使用

在底层，向量存储和检索器实现会调用 `embeddings.embedDocument(...)` 和 `embeddings.embedQuery(...)` 来分别为 `fromDocuments` 和检索器的 `invoke` 操作中使用的文本创建嵌入。

你可以直接调用这些方法来获取嵌入，以满足你的使用场景。

### 嵌入单个文本

你可以使用 `embedQuery` 来嵌入搜索查询。这会生成一个针对该查询的向量表示：

```typescript
const singleVector = await embeddings.embedQuery(text);

console.log(singleVector.slice(0, 100));
```

### 嵌入多个文本

您可以使用 `embedDocuments` 嵌入多个文本进行索引。 此方法使用的内部实现可能（但不一定）与嵌入查询的内部实现不同：

```typescript
const text2 = "LangGraph is a library for building stateful, multi-actor applications with LLMs";

const vectors = await embeddings.embedDocuments([text, text2]);

console.log(vectors[0].slice(0, 100));
console.log(vectors[1].slice(0, 100));
```

Ollama [模型参数](https://github.com/jmorganca/ollama/blob/main/docs/modelfile.md#valid-parameters-and-values) 也受支持：

```typescript
import { OllamaEmbeddings } from "@langchain/ollama";

const embeddingsCustomParams = new OllamaEmbeddings({
  requestOptions: {
    useMmap: true, // use_mmap 1
    numThread: 6, // num_thread 6
    numGpu: 1, // num_gpu 1
  },
});
```
