>[Weaviate](https://weaviate.io/) 是一个开源向量数据库。它允许您存储数据对象和来自
>您喜欢的 ML 模型的向量嵌入，并可无缝扩展到数十亿个数据对象。


什么是 `Weaviate`？
- Weaviate 是一种开源数据库，属于向量搜索引擎类型。
- Weaviate 允许您以类属性的方式存储 JSON 文档，同时附加机器学习向量来表示这些文档在向量空间中的位置。
- Weaviate 可以独立使用（即自带向量），也可以与各种模块一起使用，这些模块可以为您进行向量化处理并扩展核心功能。
- Weaviate 提供 GraphQL-API，方便您轻松访问数据。
- 我们的目标是帮助您将向量搜索设置投入生产环境，实现毫秒级查询（请查看我们的[开源基准测试](https://weaviate.io/developers/weaviate/current/benchmarks/)来了解 Weaviate 是否适合您的用例）。
- 可以在[基础知识入门指南](https://weaviate.io/developers/weaviate/current/core-knowledge/basics.html)中用不到五分钟的时间了解 Weaviate。

**Weaviate 详细介绍：**

`Weaviate` 是一个低延迟向量搜索引擎，开箱即用地支持不同媒体类型（文本、图像等）。它提供语义搜索、问答提取、分类、可定制模型（PyTorch/TensorFlow/Keras）等功能。Weaviate 从零开始用 Go 构建，同时存储对象和向量，允许将向量搜索与结构化过滤和云原生数据库的容错能力相结合。所有这些都可以通过 GraphQL、REST 和各种客户端编程语言访问。

## 安装和设置

安装 Python SDK：

<CodeGroup>

</CodeGroup>
```bash pip
pip install langchain-weaviate
```

```bash uv
uv add langchain-weaviate
```
</CodeGroup>

## 向量存储

存在一个围绕 `Weaviate` 索引的封装，允许你将其用作向量存储，无论是用于语义搜索还是示例选择。

导入此向量存储：
```python
from langchain_weaviate import WeaviateVectorStore
```

更详细的 Weaviate 封装器演练，请参见[此 notebook](/oss/integrations/vectorstores/weaviate)
