[工具](/oss/langchain/tools) 是设计为由模型调用的实用程序：其输入旨在由模型生成，其输出旨在传递回模型。

工具包是一组旨在一起使用的工具。

## 搜索

下表显示了以各种形式执行在线搜索的工具：

| 工具/工具包 | 免费/付费 | 返回数据 |
|-------------|-----------|-------------|
| [cloro](/oss/integrations/tools/cloro) | 付费 | URL、摘要、标题、答案 |
| [Exa Search](/oss/integrations/tools/exa_search) | 每月 1000 次免费搜索 | URL、作者、标题、发布日期 |
| [Google Search](/oss/integrations/tools/google_search) | 付费 | URL、摘要、标题 |
| [Linkup Search](/oss/integrations/tools/linkup_search) | 每月 2000 次免费搜索 | URL、内容、来源 |
| [Nia Toolkit](/oss/integrations/tools/nia) | 有免费层级 | 代码、文档、元数据、来源 |
| [Nimble Search](/oss/integrations/tools/nimble_search) | 有免费试用 | URL、内容、标题 |
| [Parallel Search](/oss/integrations/tools/parallel_search) | 付费 | URL、标题、摘要 |
| [Perplexity Search](/oss/integrations/tools/perplexity_search) | 付费（每月有免费层级） | URL、标题、摘要、日期、最后更新 |
| [Tavily Search](/oss/integrations/tools/tavily_search) | 每月 1000 次免费搜索 | URL、内容、标题、图片、答案 |
| [Apify](/oss/integrations/tools/apify_actors) | 免费层级，按使用量付费（因 Actor 而异） | Actor 输出（因 Actor 而异） |
| [You.com Search](/oss/integrations/tools/you) | 注册送 $100 积分 | URL、标题、页面内容 |

| [GOAT](/oss/integrations/tools/goat) | 免费 | 创建和接收付款、购买实物商品、进行投资等。 |
