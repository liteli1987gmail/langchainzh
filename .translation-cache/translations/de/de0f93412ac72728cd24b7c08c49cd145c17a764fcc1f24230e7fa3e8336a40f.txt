## 概述

[Graph RAG](https://datastax.github.io/graph-rag/) 提供了一个检索器接口
该接口结合了 **非结构化** 向量相似性搜索与 **结构化**
元数据属性的遍历。这使得基于图的检索能够覆盖 **现有的**
向量存储。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-graph-retriever
```

```bash uv
uv add langchain-graph-retriever
```
</CodeGroup>

## 检索器

```python
from langchain_graph_retriever import GraphRetriever
```

更多信息，请参阅 [Graph RAG 集成指南](/oss/integrations/retrievers/graph_rag).
