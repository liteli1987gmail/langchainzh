> 来自 [UnDatasIO](https://undatas.io) 的 `undatasio` 包可以从 PDF 等原始源文档中提取干净的文本。
> 本页介绍如何在 LangChain 中使用 `undatasio` 生态系统。

## 安装和设置

- 使用以下命令安装 Python SDK  
  `pip install undatasio`  
  以及  
  `pip install langchain-undatasio`  
  以使用 `UnDatasIOLoader` 并通过 UnDatasIO API 进行远程分区。
  您需要一个 API 密钥，您可以在 [undatas.io](https://undatas.io) 免费生成。

- 无需本地系统依赖；所有处理都在云端运行。

## 数据加载器

`UnDatasIO` 的主要用途是通过**文档加载器**实现。

### UnDatasIOLoader

请参阅[使用示例](/oss/integrations/document_loaders/undatasio)了解单文件解析和延迟加载。

```python
from langchain_undatasio import UnDatasIOLoader
```
