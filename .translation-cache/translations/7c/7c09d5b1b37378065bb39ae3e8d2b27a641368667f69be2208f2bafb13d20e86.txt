> 此 `undatasio` 包从
> [UnDatasIO](https://undatas.io) 从原始源文档（如）中提取干净的文本
> PDFs.
> 本页面介绍如何在 `undatasio`
> LangChain 生态系统内使用

## 安装和设置

- 使用以下命令安装 Python SDK：  
  `pip install undatasio`  
  以及  
  `pip install langchain-undatasio`  
  来使用 `UnDatasIOLoader` 并通过 UnDatasIO API 进行远程分区。
  您需要 API 密钥，可以免费获取：  
  [undatas.io](https://undatas.io).

- 无需本地系统依赖；所有处理都在云端运行。

## 数据加载器

的主要用途是通过 `UnDatasIO` 文档加载器 **UnDatasIOLoader**.

### 进行操作

请参阅 [使用示例](/oss/integrations/document_loaders/undatasio) 了解单文件解析和延迟加载。

```python
from langchain_undatasio import UnDatasIOLoader
```
