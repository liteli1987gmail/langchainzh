与各种 DeFi 协议交互。非托管。工具返回 *未签名交易*。该工具包构建在通用 DeFi API ([Compass API](https://api.compasslabs.ai/)) 之上，允许代理执行以下金融操作：

- **代币兑换** 在 Uniswap 和 Aerodrome 上
- **借贷** or **借款** 使用 Aave 协议上的资产
- **提供流动性** 在 Aerodrome 和 Uniswap 上
- **转移资金** 在钱包之间。
- 查询余额、投资组合和 **监控仓位**.

## 概览

### 集成详情

| 类                    | 包             | 可序列化 | JS 支持 |                                         版本                                         |
|:-------------------------|:--------------------| :---: | :---: |:----------------------------------------------------------------------------------------------:|
| `LangchainCompassToolkit`  | `langchain-compass` | ❌ | ❌ | ![PyPI - 版本](https://img.shields.io/pypi/v/langchain-compass?style=flat-square&label=%20) |

### 工具特性

以下是此工具包提供的工具示例（每日更新）：

- `aave_supply`：向 Aave 存入资产以赚取利息。
- `aave_borrow`：使用抵押品从 Aave 借入资产。
- `uniswap_swap_sell_exactly`：在 Uniswap 上兑换特定数量的代币。
- `generic_portfolio_get`：获取钱包以 USD 计价的投资组合和代币余额。
- `generic_transfer_erc20`：在地址之间转移 ERC20 代币。

## 设置

我们将：

1. 安装 langchain 包
2. 导入并实例化工具包
3. 使用以下方式将工具传递给您的代理 `toolkit.get_tools()`

### 安装

此工具包位于 `langchain-compass` package:

```python
pip install -qU langchain-compass
```

#### 环境设置

要运行这些示例，请确保 LangChain 可以访问 LLM 服务。例如，如果您使用的是 GPT-4o，请创建一个 `.env` 文件，内容包含：

```plaintext
# .env file
OPENAI_API_KEY=<your_openai_api_key_here>
```

### 实例化

现在我们可以实例化工具包：

```python
from langchain_compass.toolkits import LangchainCompassToolkit

toolkit = LangchainCompassToolkit(compass_api_key=None)
```

### 工具

查看 [可用工具](#tool-features):

```python
tools = toolkit.get_tools()
for tool in tools:
    print(tool.name)
```

```

...
```

## 调用

以编程方式调用单个工具：

```python
tool_name = "generic_ens_get"
tool = next(tool for tool in tools if tool.name == tool_name)
tool.invoke({"ens_name": "vitalik.eth", "chain": "ethereum:mainnet"})
```

## 在代理中使用

我们需要 LLM 或聊天模型：

```python
from dotenv import load_dotenv
from langchain_openai import ChatOpenAI

load_dotenv()

llm = ChatOpenAI(model="gpt-5.5")
```

使用工具初始化代理：

```python
from langchain.agents import create_agent


tools = toolkit.get_tools()
agent_executor = create_agent(llm, tools)
```

示例用法：

```python
example_query = "please set an allowance on Uniswap of 10 WETH for vitalic.eth."  # spelt wrong intentionally

stream = agent_executor.stream_events(
    {"messages": [("user", example_query)]},
    version="v3",
)
for snapshot in stream.values:
    snapshot["messages"][-1].pretty_print()
```
