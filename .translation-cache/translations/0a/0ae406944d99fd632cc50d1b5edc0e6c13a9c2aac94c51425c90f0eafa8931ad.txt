Dell 是一家全球性科技公司，提供一系列硬件、软件和
服务，包括 AI 解决方案。他们的 AI 产品组合包括专门构建的
AI 工作负载基础设施，包括针对 AI 数据管理优化的 Dell PowerScale 存储系统
。

## PowerScale

Dell [PowerScale](https://www.dell.com/en-us/shop/powerscale-family/sf/powerscale) is
一个企业级横向扩展存储系统，托管行业领先的 OneFS 文件系统
可本地托管或部署在云中。

### 安装和设置

<CodeGroup>
```bash pip
pip install powerscale-rag-connector
```

```bash uv
uv add powerscale-rag-connector
```
