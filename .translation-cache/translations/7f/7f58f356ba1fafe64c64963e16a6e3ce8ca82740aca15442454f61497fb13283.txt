[Zotero](https://www.zotero.org/) 是一个开源的参考文献管理系统，用于管理书目数据和相关研究材料。您可以通过以下方式连接到您的个人文献库以及共享的群组文献库： [API](https://www.zotero.org/support/dev/web_api/v3/start)。此检索器实现使用 [PyZotero](https://github.com/urschrei/pyzotero) 来访问文献库。

## 安装

```bash
pip install pyzotero
```

## 检索器

请参阅 [使用示例](/oss/integrations/retrievers/zotero).

```python
from langchain_zotero_retriever.retrievers import ZoteroRetriever
```
