## 设置

要访问 Groq 模型，您需要创建一个 Groq 账户，获取一个 API 密钥，并安装 `langchain-groq` 集成包。

### 凭据

前往 [Groq 控制台](https://console.groq.com/login?utm_source=langchain&utm_content=chat_page) 注册 Groq 并生成 API 密钥。完成后，设置 GROQ_API_KEY 环境变量：

```python
import getpass
import os

if "GROQ_API_KEY" not in os.environ:
    os.environ["GROQ_API_KEY"] = getpass.getpass("Enter your Groq API key: ")
```

要启用模型调用的自动追踪，请设置您的 [LangSmith](/langsmith/observability) API 密钥：

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

LangChain Groq 集成位于 `langchain-groq` package:

```python
pip install -qU langchain-groq
```

## 实例化

现在我们可以实例化模型对象并生成聊天补全。

<Note>
**推理格式**

如果您选择设置 `reasoning_format`，您必须确保您使用的模型支持它。您可以在 [Groq 文档](https://console.groq.com/docs/reasoning).

</Note>

```python
from langchain_groq import ChatGroq

llm = ChatGroq(
    model="qwen/qwen3-32b",
    temperature=0,
    max_tokens=None,
    reasoning_format="parsed",
    timeout=None,
    max_retries=2,
    # other params...
)
```

## 视觉

Groq 支持特定模型的视觉功能，允许您随文本提示一起发送图像。

<Note>
**支持视觉的模型**

- `meta-llama/llama-4-scout-17b-16e-instruct`
- `meta-llama/llama-4-maverick-17b-128e-instruct`

有关支持视觉功能的最新模型列表，请查看 [Groq 文档](https://console.groq.com/docs/vision).
</Note>

```python
from langchain_groq import ChatGroq
from langchain.messages import HumanMessage

llm = ChatGroq(model="meta-llama/llama-4-scout-17b-16e-instruct")

message = HumanMessage(
    content=[
        {"type": "text", "text": "Describe this image in detail."},
        {
            "type": "image_url",
            "image_url": {"url": "https://example.com/image.jpg"},
        },
    ]
)

response = llm.invoke([message])
print(response.content)
```

<Tip>
**图片 URL 要求**

Groq 直接从 URL 获取图片。请确保您的图片 URL：
- 可公开访问（无需身份验证）
- 直接返回图片（无重定向）
- 最大图片大小：每次请求 20MB
</Tip>
