所有与 `Naver`相关的功能，包括 HyperCLOVA X 模型，特别是可通过 `Naver Cloud` [CLOVA Studio](https://clovastudio.ncloud.com/).

> [Naver](https://navercorp.com/) 是一家全球科技公司，拥有尖端技术和多元化的业务组合，包括搜索、商务、金融科技、内容、云和 AI。

> [Naver Cloud](https://www.navercloudcorp.com/lang/en/) 是 Naver 的云计算部门，是领先的云服务提供商，通过其 [Naver Cloud Platform (NCP)](https://www.ncloud.com/).

请参阅 [NCP 用户指南](https://guide.ncloud-docs.com/docs/clovastudio-overview) 了解更多详细说明（也有韩语版本）。

## 安装和设置

- 通过 [获取 CLOVA Studio API 密钥](https://api.ncloud-docs.com/docs/ai-naver-clovastudio-summary#API%ED%82%A4) 并将其设置为环境变量（`CLOVASTUDIO_API_KEY`).


Naver 集成存在于两个包中：

- `langchain-naver`：一个专用于 Naver 的集成包。
- `langchain-naver-community`：一个社区维护的包，Naver 或 LangChain 官方不进行维护。

<CodeGroup>
```bash pip
pip install -U langchain-naver
# pip install -U langchain-naver-community // Install to use Naver Search tool.
```

```bash uv
uv add langchain-naver
# uv add langchain-naver-community // Install to use Naver Search tool.
```
</CodeGroup>

> **（注意）** 通过 `langchain-community`的 Naver 集成 [第三方集成](https://python.langchain.com/docs/concepts/architecture/#langchain-community)已过时。
> - **使用 `langchain-naver` ，因为新功能应仅通过此包实现**.
> - 如果你正在使用 `langchain-community` （已过时）并且获得了旧版 API 密钥（不以 `nv-*` 前缀开头），你应将其设置为 `NCP_CLOVASTUDIO_API_KEY`，可能还需要通过 [创建你的应用](https://guide.ncloud-docs.com/docs/en/clovastudio-playground01#create-test-app) 并将其设置为 `NCP_APIGW_API_KEY`.

## 工具

### Naver 搜索

Naver 搜索集成允许你的 LangChain 应用从 Naver 搜索引擎获取信息。这对于韩语查询和获取韩国主题的最新信息特别有用。

要使用 Naver 搜索工具，你需要：

1. 登录 [Naver 开发者门户](https://developers.naver.com/main/)
2. 创建新应用并启用搜索 API
3. 获取你的 **NAVER_CLIENT_ID** 和 **NAVER_CLIENT_SECRET** 从"应用列表"部分
4. 在你的应用中将其设置为环境变量

```python
from langchain_naver_community.tool import NaverSearchResults
from langchain_naver_community.utils import NaverSearchAPIWrapper

# Set up the search wrapper
search = NaverSearchAPIWrapper()

# Create a tool
tool = NaverSearchResults(api_wrapper=search)
```

查看 [使用示例](/oss/integrations/tools/naver_search) 了解更多详情。

### 专业搜索工具

该包还为不同类型的内容提供专业搜索工具：

```python
from langchain_naver_community.tool import NaverNewsSearch  # For news articles
from langchain_naver_community.tool import NaverBlogSearch  # For blog posts
from langchain_naver_community.tool import NaverImageSearch  # For images
```

这些工具都可以在 LangChain 代理中使用，以提供更有针对性的搜索功能。
