所有与 `Naver` 相关的功能，包括 HyperCLOVA X 模型，特别是可通过 `Naver Cloud` [CLOVA Studio](https://clovastudio.ncloud.com/) 访问的模型。

> [Naver](https://navercorp.com/) 是一家全球科技公司，拥有尖端技术和多元化的业务组合，涵盖搜索、电商、金融科技、内容、云和 AI 领域。

> [Naver Cloud](https://www.navercloudcorp.com/lang/en/) 是 Naver 的云计算部门，是领先的云服务提供商，通过其 [Naver Cloud Platform (NCP)](https://www.ncloud.com/) 为企业提供全面的云服务套件。

请参阅 [NCP 用户指南](https://guide.ncloud-docs.com/docs/clovastudio-overview) 获取更详细的说明（也有韩语版本）。

## 安装和设置

- 通过[颁发](https://api.ncloud-docs.com/docs/ai-naver-clovastudio-summary#API%ED%82%A4)获得 CLOVA Studio API Key，并将其设置为环境变量 (`CLOVASTUDIO_API_KEY`)。

Naver 集成位于两个包中：

- `langchain-naver`：专用的 Naver 集成包。
- `langchain-naver-community`：由社区维护的包，由 Naver 或 LangChain 官方维护。

<CodeGroup>

</CodeGroup>
```bash pip
pip install -U langchain-naver
# pip install -U langchain-naver-community // Install to use Naver Search tool.
```

```bash uv
uv add langchain-naver
# uv add langchain-naver-community // Install to use Naver Search tool.
```
</CodeGroup>

> **（注）** 通过 `langchain-community`（第三方集成集合）集成的 Naver 已过时。
> - **请改用 `langchain-naver`**，因为新功能应仅通过此包实现。
> - 如果您正在使用 `langchain-community`（已过时）且拥有旧版 API Key（不以 `nv-*` 前缀开头），则应将其设置为 `NCP_CLOVASTUDIO_API_KEY`，并可能需要通过创建应用来获取额外的 API Gateway API Key，然后将其设置为 `NCP_APIGW_API_KEY`。

## 工具

### Naver 搜索

Naver Search 集成允许您的 LangChain 应用程序从 Naver 的搜索引擎检索信息。这对于韩语查询和获取有关韩国主题的最新信息特别有用。

要使用 Naver Search 工具，您需要：

1. 登录 [Naver Developers 门户](https://developers.naver.com/main/)
2. 创建新应用程序并启用 Search API
3. 从"应用程序列表"部分获取您的 **NAVER_CLIENT_ID** 和 **NAVER_CLIENT_SECRET**
4. 在您的应用程序中将这些设置为环境变量

```python
from langchain_naver_community.tool import NaverSearchResults
from langchain_naver_community.utils import NaverSearchAPIWrapper

# Set up the search wrapper
search = NaverSearchAPIWrapper()

# Create a tool
tool = NaverSearchResults(api_wrapper=search)
```

请参阅[使用示例](/oss/integrations/tools/naver_search)了解更多详情。

### 专用搜索工具

该包还针对不同类型的内容提供专用搜索工具：

```python
from langchain_naver_community.tool import NaverNewsSearch  # For news articles
from langchain_naver_community.tool import NaverBlogSearch  # For blog posts
from langchain_naver_community.tool import NaverImageSearch  # For images
```

这些工具中的每一个都可以在 LangChain 代理中使用，以提供更有针对性的搜索能力。
