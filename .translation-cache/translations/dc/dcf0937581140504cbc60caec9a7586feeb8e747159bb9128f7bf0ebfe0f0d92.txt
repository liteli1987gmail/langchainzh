## 设置

要访问 OpenAI 模型，您需要创建一个 OpenAI 账户，获取 API 密钥，并安装 `langchain-openai` 集成包。

### 凭据

前往 [platform.openai.com](https://platform.openai.com) 注册 OpenAI 并生成 API 密钥。完成此操作后，设置 OPENAI_API_KEY 环境变量：

```python
import getpass
import os

if "OPENAI_API_KEY" not in os.environ:
    os.environ["OPENAI_API_KEY"] = getpass.getpass("Enter your OpenAI API key: ")
```

如果你正在通过代理或服务模拟器路由请求，可以通过环境变量设置 base URL，而不是传递 `base_url`。解析顺序（首次匹配优先）：

1. 显式 `base_url`（或 `openai_api_base`）关键字参数。
2. `OPENAI_API_BASE` — 在 LangChain 初始化时读取。
3. `OPENAI_BASE_URL` — 由底层 `openai` SDK 客户端读取。

要启用模型调用的自动化追踪，请设置你的 [LangSmith](/langsmith/observability) API 密钥：

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

LangChain OpenAI 集成位于 `langchain-openai` 包中：

```python
pip install -qU langchain-openai
```

如果需要指定您的组织 ID，可以使用下面的单元格。但是，如果您仅属于单个组织或打算使用默认组织，则不需要指定。您可以在 [OpenAI API 密钥页面](https://platform.openai.com/account/api-keys) 上查看您的默认组织。

要指定您的组织，可以使用以下代码：

```python
OPENAI_ORGANIZATION = getpass()

os.environ["OPENAI_ORGANIZATION"] = OPENAI_ORGANIZATION
```

## 实例化

现在我们可以实例化模型对象并生成聊天补全：

```python
from langchain_openai import OpenAI

llm = OpenAI()
```

## 调用

```python
llm.invoke("Hello how are you?")
```

```text
'\n\nI am an AI and do not have emotions like humans do, so I am always functioning at my optimal level. Thank you for asking! How can I assist you today?'
```

## 链式调用

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template("How to say {input} in {output_language}:\n")

chain = prompt | llm
chain.invoke(
    {
        "output_language": "German",
        "input": "I love programming.",
    }
)
```

```text
'\nIch liebe Programmieren.'
```

## 使用代理

如果您在显式代理之后，可以指定 http_client 来通过

```python
pip install httpx

import httpx

openai = OpenAI(
    model_name="gpt-3.5-turbo-instruct",
    http_client=httpx.Client(proxies="http://proxy.yourcompany.com:8080"),
)
```
