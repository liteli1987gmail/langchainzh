客户正在转向 [SambaNova](https://sambanova.ai/)，以快速部署最先进的 AI 能力，获得竞争优势。我们专为 AI 构建的企业级 AI 平台是下一代 AI 计算的技术支柱。我们为 AI 基础模型提供算力支持，释放数据中蕴含的宝贵商业洞察。

SambaNova RDU 专为 AI 设计，采用革命性的数据流架构构建。这种设计使 RDU 在处理这些工作负载时比 GPU 效率高得多，因为它消除了冗余的内存访问，而内存访问是 GPU 功能的固有限制。这种内置效率是使 RDU 能够以占用空间的一小部分实现比 GPU 高得多的性能的特点之一。

在架构之上，我们开发了一些平台，使企业和开发者能够充分利用 RDU 处理器和开源模型。

## API 密钥
将您的 API 密钥设置为环境变量：

如果您是 SambaCloud 用户，请申请 [API 密钥](http://cloud.sambanova.ai/apis?utm_source=langchain&utm_medium=external&utm_campaign=cloud_signup) 并将其设置为环境变量：

```bash
export SAMBANOVA_API_KEY="your-sambacloud-api-key-here"
```

或者，如果您是 SambaStack 用户，请将您的基础 URL 和 API 密钥设为环境变量：

```bash
export SAMBANOVA_API_BASE="your-sambastack-envirronment-base-url-here"
export SAMBANOVA_API_KEY="your-sambastack-api-key-here"
```
