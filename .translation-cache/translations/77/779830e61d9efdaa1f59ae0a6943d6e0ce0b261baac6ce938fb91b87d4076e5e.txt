## 概述

[Amazon Kendra](https://aws.amazon.com/kendra/) 是亚马逊网络服务 (AWS) 提供的智能搜索服务。
它利用先进的自然语言处理 (NLP) 和机器学习算法，使组织内各种数据源的强大搜索能力成为可能。
Kendra 旨在帮助用户快速准确地找到所需信息，提高生产力和决策能力。

使用 Kendra，用户可以搜索各种类型的内容，包括文档、常见问题、知识库、手册和网站。
它支持多种语言，可以理解复杂查询、同义词和上下文含义，以提供高度相关的搜索结果。

这将帮助您开始 `AWSKendraRetriever` [检索](/oss/langchain/retrieval)。有关所有 `AWSKendraRetriever` 功能和配置的详细文档，请前往 [API 参考](https://reference.langchain.com/javascript/langchain-aws/AmazonKendraRetriever).

## 设置

您需要拥有一个 AWS 账户和 Amazon Kendra 实例才能开始。参阅此 [教程](https://docs.aws.amazon.com/kendra/latest/dg/getting-started.html) 从 AWS 获取更多信息。

如果您想从单个查询获取自动跟踪，还可以设置您的 [LangSmith](/langsmith/observability) API 密钥，方法是取消注释下面的内容：

```typescript
// process.env.LANGSMITH_API_KEY = "<YOUR API KEY HERE>";
// process.env.LANGSMITH_TRACING = "true";
```

### 安装

此检索器位于 `@langchain/aws` package:

<CodeGroup>
```bash npm
npm install @langchain/aws @langchain/core
```
```bash yarn
yarn add @langchain/aws @langchain/core
```
```bash pnpm
pnpm add @langchain/aws @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化我们的检索器：

```typescript
import { AmazonKendraRetriever } from "@langchain/aws";

const retriever = new AmazonKendraRetriever({
  topK: 10,
  indexId: "YOUR_INDEX_ID",
  region: "us-east-2", // Your region
  clientOptions: {
    credentials: {
      accessKeyId: "YOUR_ACCESS_KEY_ID",
      secretAccessKey: "YOUR_SECRET_ACCESS_KEY",
    },
  },
});
```

## 用法

```typescript
const query = "..."

await retriever.invoke(query);
```
