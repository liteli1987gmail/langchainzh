## 安装和设置

 `Parallel` 集成位于其独立的 [合作伙伴包](https://pypi.org/project/langchain-parallel/):

<CodeGroup>
    ```bash pip
    pip install -U langchain-parallel
    ```
    ```bash uv
    uv add langchain-parallel
    ```
</CodeGroup>

设置 `PARALLEL_API_KEY` 环境变量为您的 Parallel API 密钥。请在 [platform.parallel.ai](https://platform.parallel.ai) 注册以获取密钥。
