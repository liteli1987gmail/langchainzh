## 使用我们的 MCP 服务器

我们的文档包含一个内置的 **Model Context Protocol (MCP) 服务器**，可让 AI 应用程序实时查询最新文档。

LangChain 文档 MCP 服务器可从以下地址获取：

```txt
https://docs.langchain.com/mcp
```

连接后，您可以向 AI 助手提问有关 LangChain、LangGraph 和 LangSmith 的问题，它将搜索我们的文档以提供准确、最新的答案。

### 连接 Claude Code

如果您正在使用 Claude Code，请在终端中运行此命令以将服务器添加到当前项目：

```bash
claude mcp add --transport http docs-langchain https://docs.langchain.com/mcp
```

<Note>
    **项目（本地）作用域**

    上述命令仅将 MCP 服务器添加到您当前的项目/工作目录。要将 MCP 服务器全局添加并在所有项目中访问它，请通过在命令中添加 `--scope user` 来添加用户作用域：

</Note>

    ```bash
    claude mcp add --transport http docs-langchain --scope user https://docs.langchain.com/mcp
    ```
</Note>

### 连接 Claude Desktop

1. 打开 Claude Desktop
2. 前往设置 > 连接器
3. 添加我们的 MCP 服务器 URL：`https://docs.langchain.com/mcp`

### 连接 Codex CLI

如果您使用的是 OpenAI Codex CLI，请在终端中运行此命令以全局添加服务器：

```sh
codex mcp add langchain-docs --url https://docs.langchain.com/mcp
```

### 连接到 Cursor

将以下内容添加到您的 MCP 设置配置文件中：

```json
{
  "mcpServers": {
    "docs-langchain": {
      "url": "https://docs.langchain.com/mcp"
    }
  }
}
```

### 连接 VS Code

将以下内容添加到您的 MCP 设置配置文件中：

```json
{
  "servers": {
    "docs-langchain": {
      "url": "https://docs.langchain.com/mcp"
    }
  }
}
```

### 连接 Antigravity

将以下内容添加到您的 MCP 设置配置文件中：

```json
{
  "mcpServers": {
    "docs-langchain": {
      "serverUrl": "https://docs.langchain.com/mcp"
    }
  }
}
```
