[Featherless AI](https://featherless.ai/) 是一个无服务器 AI 推理平台，提供对超过 4300+ 开源模型的访问。我们的目标是让所有 AI 模型都能用于无服务器推理。我们通过 API 提供对不断扩展的开源权重模型的推理服务。

# 安装和设置
`pip install langchain-featherless-ai`
1. 在以下网址注册账户 [Featherless](https://featherless.ai/register)
2. 订阅计划并从以下页面获取您的 API 密钥 [API 密钥](https://featherless.ai/account/api-keys)
3. 将您的 API 密钥设置为环境变量(`FEATHERLESSAI_API_KEY`)

Visit our model catalog for an overview of all our models: https://featherless.ai/models
