<Info>
本页仅适用于未在离线（气隙）模式下运行且使用自托管 LangSmith 实例（版本 0.9.0 或更高版本）的客户。
</Info>

自托管 LangSmith 实例在本地存储所有信息，绝不向您的网络外部发送敏感信息。但是，除非您在离线模式下运行，否则 LangSmith 需要访问 `https://beacon.langchain.com` 的出口流量，用于以下用途：

- **账单遥测** — 许可证验证和订阅/使用量报告（必需）
- **运营遥测** — 用于支持诊断的日志、指标和追踪（可选，可禁用）
- **使用量遥测** — 用于产品洞察的匿名化使用量快照（可选，可禁用）

<Warning>
**需要访问 `https://beacon.langchain.com` 的出口流量。** 如需静态 IP 地址，请参阅[允许列出 IP 部分](/langsmith/cloud#allowlisting-ip-addresses)。
</Warning>

## 账单遥测

账单遥测对于未在离线模式下运行的自托管 LangSmith 实例是**必需的**。这包括许可证验证和订阅/使用量报告。

<Info>
账单遥测**无法禁用**。如果您需要完全不使用任何出口流量，请联系您的客户团队咨询离线（气隙）许可证。
</Info>

### 功能说明

- **许可证验证**：在启动时和此后定期验证您的 LangSmith 许可证密钥。
- **订阅/使用量报告**：根据订单中的授权报告平台使用量指标，用于计费。

### 我们收集的内容

- 许可证密钥验证请求
- 聚合使用量计数（追踪数量、分配的席位、使用中的席位）
- 组织和工作区标识符

### 示例载荷

#### 许可证验证

**端点：** `POST beacon.langchain.com/v1/beacon/verify`

**请求：**

```json
{
  "license": "<YOUR_LICENSE_KEY>"
}
```

**响应：**

```json
{
  "token": "Valid JWT" //Short-lived JWT token to avoid repeated license checks
}
```

## 运营遥测

自 **0.11** 版本起，LangSmith 部署默认发送运营遥测数据。此遥测数据可帮助 LangChain 团队为自托管实例提供主动支持并加快故障排查。

<Info>
运营遥测与计费遥测是**分开的**。您可以在计费遥测保持激活状态的同时禁用运营遥测。
</Info>

### 功能说明

- 支持为自托管实例提供主动支持和加快故障排查
- 协助进行性能调优
- 根据实际使用模式帮助确定改进的优先级

### 我们收集的内容

- **请求元数据**：匿名化的请求数量、大小和持续时间
- **数据库指标**：查询持续时间、错误率和性能计数器
- **运营追踪**：高延迟或失败请求的计时和错误信息（这些**不是**客户追踪——它们是关于 LangSmith 实例本身运行状况的追踪）
- **日志消息**：仅包括警告和错误日志消息

<Info>
我们不收集实际的有效负载内容、数据库记录或任何可识别您的最终用户或客户的数据。所有遥测数据均与组织和部署相关联，但不会与个人用户关联。我们**不收集 PII**（个人身份信息），任何形式都不收集。
</Info>

### 如何禁用

您可以通过在 `langsmith_config.yaml` 文件中设置以下值来禁用运营遥测：

```yaml
config:
  telemetry:
    logs: false
    metrics: false
    traces: false
```

你也可以通过仅将特定值设置为 `false` 来禁用单个遥测类型。

<Warning>
禁用操作遥测会停止导出本节中描述的日志、指标和追踪。这**不会**禁用计费遥测（许可证验证和订阅/使用量报告）。
</Warning>

## 使用情况遥测

使用情况遥测收集您 LangSmith 实例的使用情况指标的匿名快照。此数据有助于 LangChain 了解平台采用模式并为产品开发决策提供信息。

<Info>
使用情况遥测**默认启用**，可以禁用。与计费遥测不同，您可以完全控制是否发送这些快照到 LangChain。
</Info>

### 功能

- 定期捕获聚合的使用情况指标
- 提供功能采用和平台增长的洞察
- 帮助 LangChain 根据实际使用情况优先改进和新功能

### 我们收集的内容

- **平台指标**：工作区、项目、实验、数据集、评估器和其他平台资源的计数
- **功能使用**：运行规则、标注队列、提示词和提示词相关活动的计数
- **用户**：已注册用户的总数以及过去 30 天内活跃的个人访问令牌 (PAT) 数量
- **时间戳**：快照的时间范围（UTC 格式的起始/结束时间戳）

<Info>
所有指标**仅为聚合计数**。不收集任何个人资源数据、标识符或使用模式。我们不会收集任何可用于识别您的最终用户或客户的信息。
</Info>

### 如何禁用

您可以通过在部署配置中设置以下环境变量来禁用使用遥测：

```yaml
PHONE_HOME_USAGE_REPORTING_ENABLED: false
```

将以下内容添加到 Helm 配置的 `commonEnv` 部分，以永久禁用使用情况遥测报告。

<Warning>
禁用使用情况遥测**不会**影响计费或运营遥测。许可证验证和订阅/使用情况报告将继续正常运行。
</Warning>

## 我们的承诺

LangChain 不会在计费或运营遥测中存储任何敏感信息。收集的任何数据都不会与第三方共享。日志消息仅过滤包含错误严重级别，我们不会捕获可能包含敏感应用数据的日志消息。如果您对发送的数据有任何疑虑，请禁用遥测和/或联系您的客户团队。
