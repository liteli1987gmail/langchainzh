> [FMP-Data](https://pypi.org/project/fmp-data/) 是一个 Python 包，用于连接
> Financial Data Prep API。它简化了访问生产级数据的方式。


## 安装和设置

获取 `FMP Data` API 密钥，请
访问 [此页面](https://site.financialmodelingprep.com/pricing-plans?couponCode=mehdi).
 并将其设置为环境变量（`FMP_API_KEY`).

然后，安装 [langchain-fmp-data](https://pypi.org/project/langchain-fmp-data/).

## 工具

查看 [示例](https://github.com/MehdiZare/langchain-fmp-data/tree/main/docs).

```python
from langchain_fmp_data import FMPDataTool, FMPDataToolkit
```
