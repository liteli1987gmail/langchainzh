[SingleStore](https://singlestore.com/) 是一个高性能分布式 SQL 数据库，专为在[云](https://www.singlestore.com/cloud/)和本地环境中卓越运行而设计。它提供多功能特性集、无缝部署选项和卓越性能。

此集成提供以下组件以利用 SingleStore 的能力：

- **`SingleStoreLoader`**：直接从 SingleStore 数据库表加载文档。
- **`SingleStoreSemanticCache`**：将 SingleStore 用作语义缓存，以便高效存储和检索嵌入向量。
- **`SingleStoreChatMessageHistory`**：在 SingleStore 中存储和检索聊天消息历史。
- **`SingleStoreVectorStore`**：存储文档嵌入向量并执行快速向量和全文搜索。

这些组件实现了高效的文档存储、嵌入向量管理以及高级搜索功能，结合全文和向量搜索以实现快速准确的查询。

```python
from langchain_singlestore import (
    SingleStoreChatMessageHistory,
    SingleStoreLoader,
    SingleStoreSemanticCache,
    SingleStoreVectorStore,
)
```
