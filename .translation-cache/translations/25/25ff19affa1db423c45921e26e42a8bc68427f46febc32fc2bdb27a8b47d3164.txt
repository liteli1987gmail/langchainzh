>[KoNLPy](https://konlpy.org/) 是用于韩语自然语言处理（NLP）的 Python 包。


## 安装和设置

你需要安装 `konlpy` Python 包。

<CodeGroup>
```bash pip
pip install konlpy
```

```bash uv
uv add konlpy
```
</CodeGroup>

## 文本分割器


```python
from langchain_text_splitters import KonlpyTextSplitter
```
