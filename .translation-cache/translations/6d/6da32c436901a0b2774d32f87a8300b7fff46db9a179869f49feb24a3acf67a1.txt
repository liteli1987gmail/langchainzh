LangSmith REST API 提供对 LangSmith 平台功能的编程访问，包括追踪、数据集、实验、标注等功能。

在侧边栏的 **LangSmith API** 部分浏览完整的 API 参考。

## 身份验证

每次请求时传递 `X-Api-Key` 请求头，其值应为有效的 [LangSmith API 密钥](/langsmith/create-account-api-key).

```shell
curl --request GET \
  --url https://api.smith.langchain.com/api/v1/workspaces \
  --header 'X-Api-Key: LANGSMITH_API_KEY'
```
