[Bright Data](https://brightdata.com) 是一个网页数据平台，提供用于网页抓取、SERP 收集和访问地理限制内容的工具。

Bright Data 使开发者能够从网站中提取结构化数据、执行搜索引擎查询，以及访问可能被阻止或受地理限制的内容。该平台旨在帮助克服常见的网页抓取挑战，包括反机器人系统、CAPTCHA 和 IP 封锁。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-brightdata
```

```bash uv
uv add langchain-brightdata
```

</CodeGroup>

您需要设置 Bright Data API 密钥：

在 [Bright Data](https://brightdata.com/?utm_source=tech-partner&utm_medium=link&utm_campaign=langchain&hs_signup=1) 注册并从您的账户设置中获取 API 密钥。在下面的示例中，用您的实际 API 密钥替换 `"your-api-key"`：

```python
import os
os.environ["BRIGHT_DATA_API_KEY"] = "your-api-key"
```

或者你可以在初始化工具时直接传递它：

```python
from langchain_brightdata import BrightDataSERP

tool = BrightDataSERP(bright_data_api_key="your-api-key")
```

## 工具

Bright Data 集成提供以下工具：

- [BrightDataSERP](/oss/integrations/tools/brightdata_serp) - 通过地理位置定位和自定义区域支持收集搜索引擎结果
- [BrightDataUnlocker](/oss/integrations/tools/brightdata_unlocker) - 访问任何可能受地理限制或机器人保护限制的公共网站
- [BrightDataWebScraperAPI](/oss/integrations/tools/brightdata-webscraperapi) - 从 44 个热门网站域中提取结构化数据，包括 Amazon、LinkedIn、Instagram、TikTok 等
