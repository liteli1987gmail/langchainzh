>[Robocorp](https://robocorp.com/) 帮助构建和运行可无缝部署在任何地方、任意规模的 Python 工作器


## 安装和设置

您需要安装 `langchain-robocorp` python 包：

<CodeGroup>
```bash pip
pip install langchain-robocorp
```

```bash uv
uv add langchain-robocorp
```
</CodeGroup>

您需要一个运行中的 `Action Server` 来与您的代理应用程序通信。
请参阅 [Robocorp 快速入门](https://github.com/robocorp/robocorp#quickstart) 了解如何设置 Action Server 并创建您的 Actions。

您可以使用 Action Server 引导一个新项目 `new` command.

```bash
action-server new
cd ./your-project-name
action-server start
```
