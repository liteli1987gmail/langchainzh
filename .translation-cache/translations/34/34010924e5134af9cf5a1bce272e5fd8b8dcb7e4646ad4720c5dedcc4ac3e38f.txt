>[Robocorp](https://robocorp.com/) 帮助构建和运行可无缝在任何地方、任何规模运行的 Python 工作器。


## 安装和设置

您需要安装 `langchain-robocorp` Python 包：

<CodeGroup>

</CodeGroup>
```bash pip
pip install langchain-robocorp
```

```bash uv
uv add langchain-robocorp
```
</CodeGroup>

你需要运行一个 `Action Server` 实例，以便从你的代理应用程序与之通信。
请参阅 [Robocorp Quickstart](https://github.com/robocorp/robocorp#quickstart) 了解如何设置 Action Server 并创建你的 Actions。

你可以使用 Action Server 的 `new` 命令来引导创建新项目。

```bash
action-server new
cd ./your-project-name
action-server start
```

from langchain_robocorp.toolkits import ActionServerRequestTool
