[SurrealDB](https://surrealdb.com) 是一个统一的、多模型的数据库，专为 AI 系统设计。它将结构化和非结构化数据（包括向量搜索、图遍历、关系查询、全文搜索、文档存储和时间序列数据）组合到一个符合 ACID 标准的引擎中，可从 3 MB 的边缘二进制文件扩展到云端 PB 级集群。通过消除对多个专业存储的需求，SurrealDB 简化了架构，降低了延迟，并确保了 AI 工作负载的一致性。

**为什么 SurrealDB 对 GenAI 系统至关重要**
- **一个引擎兼顾存储和内存：** 在一个系统中结合持久存储和快速、代理友好的内存，为代理提供所需的所有数据，无需同步多个系统。
- **代理的单跳内存：** 在单个查询中运行向量搜索、图遍历、语义连接和事务写入，为 LLM 代理提供快速、一致的内存访问，无需将关系、图和向量数据库拼接在一起。
- **原地推理和实时更新：** SurrealDB 使代理能够在数据旁边运行推理并接收毫秒级的新鲜更新，这对于实时推理和协作至关重要。
- **版本化、持久化上下文：** SurrealDB 支持时间旅行查询和版本化记录，让代理可以审计或"重放"过去的状态，以实现一致且可解释的推理。
- **即插即用的代理内存：** 将 AI 内存作为一个原生概念公开，使 SurrealDB 易于作为 AI 框架的即插即用后端使用。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-surrealdb
```

```bash uv
uv add langchain-surrealdb
```
</CodeGroup>
