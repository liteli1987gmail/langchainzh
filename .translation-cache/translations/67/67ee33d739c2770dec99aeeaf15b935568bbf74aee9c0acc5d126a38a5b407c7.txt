## 设置

要访问 ChatAbso 模型，您需要创建一个 OpenAI 账户，获取 API 密钥，然后安装 `langchain-abso` 集成包。

### 凭据

- 待办事项：用相关信息更新。

前往（待办事项：链接）注册 ChatAbso 并生成 API 密钥。完成此操作后，设置 ABSO_API_KEY 环境变量：

```python
import getpass
import os

if not os.getenv("OPENAI_API_KEY"):
    os.environ["OPENAI_API_KEY"] = getpass.getpass("Enter your OpenAI API key: ")
```

### 安装

LangChain ChatAbso 集成位于 `langchain-abso` package:

```python
pip install -qU langchain-abso
```

## 实例化

现在我们可以实例化模型对象并生成聊天补全：

```python
from langchain_abso import ChatAbso

llm = ChatAbso(fast_model="gpt-5.5", slow_model="o3-mini")
```

## 调用

```python
messages = [
    (
        "system",
        "You are a helpful assistant that translates English to French. Translate the user sentence.",
    ),
    ("human", "I love programming."),
]
ai_msg = llm.invoke(messages)
ai_msg
```

```python
print(ai_msg.content)
```
