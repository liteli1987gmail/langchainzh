[Shale Protocol](https://shaleprotocol.com) 为开源 LLM 提供生产级推理 API。它是一个即插即用的 API，托管在高度可扩展的 GPU 云基础设施上。

我们的免费层级每天每个密钥支持最多 1000 次请求，因为我们希望消除任何人在使用 LLM 构建 genAI 应用时的门槛。

借助 Shale Protocol，开发者和研究人员可以免费创建应用并探索开源 LLM 的能力。

本页介绍如何将 Shale-Serve API 与 LangChain 结合使用。

截至 2023 年 6 月，该 API 默认支持 Vicuna-13B。我们将在未来版本中支持更多 LLM，例如 Falcon-40B。

## 如何使用

### 1. 在 https://shaleprotocol.com 上找到我们的 Discord 链接。通过 Discord 上的"Shale bot"生成 API 密钥。无需信用卡，无免费试用。这是一个永久免费层级，每个 API 密钥每天 1000 次限制

### 2. 使用 https://shale.live/v1 作为 OpenAI API 的直接替代

例如

```python
from langchain_openai import OpenAI
from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser

import os
os.environ['OPENAI_API_BASE'] = "https://shale.live/v1"
os.environ['OPENAI_API_KEY'] = "ENTER YOUR API KEY"

llm = OpenAI()

template = """Question: {question}

# Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)


llm_chain = prompt | llm | StrOutputParser()

question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"

llm_chain.invoke(question)

```
