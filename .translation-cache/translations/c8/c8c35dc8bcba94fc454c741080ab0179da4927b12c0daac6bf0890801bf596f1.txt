## 什么是 Amazon MemoryDB？

MemoryDB 与 Redis OSS 兼容，Redis OSS 是一个流行的开源数据存储，使您能够使用与当前已使用的相同灵活且友好的 Redis OSS 数据结构、API 和命令快速构建应用程序。使用 MemoryDB，所有数据都存储在内存中，使您能够实现微秒级读取和单位数毫秒级写入延迟以及高吞吐量。MemoryDB 还使用多可用区事务日志跨多个可用区（AZ）持久存储数据，以实现快速故障转移、数据库恢复和节点重启。

## MemoryDB 的向量搜索

MemoryDB 的向量搜索扩展了 MemoryDB 的功能。向量搜索可与现有 MemoryDB 功能结合使用。不使用向量搜索的应用程序不受其存在的影响。向量搜索在 MemoryDB 可用的所有区域都可用。您可以使用现有的 MemoryDB 数据或 Redis OSS API 来构建机器学习和生成式 AI 用例，例如检索增强生成、异常检测、文档检索和实时推荐。

* Redis 哈希中多个字段的索引和 `JSON`
* 向量相似性搜索（使用 `HNSW` （ANN）或 `FLAT` （KNN））
* 向量范围搜索（例如，查找查询向量半径范围内的所有向量）
* 增量索引，无性能损失

## 设置

### 安装 Redis Python 客户端

`Redis-py` 是一个可用于连接到 MemoryDB 的 Python 客户端

```python
pip install -qU  redis langchain-aws
```

```python
from langchain_aws.embeddings import BedrockEmbeddings

embeddings = BedrockEmbeddings()
```

### MemoryDB 连接

有效的 Redis URL 架构：

1. `redis://`  - 连接到 Redis 集群，未加密
2. `rediss://` - 连接到 Redis 集群，使用 TLS 加密

有关其他连接参数的更多信息，请参阅 [redis-py 文档](https://redis-py.readthedocs.io/en/stable/connections.html).

### 创建 MemoryDB 向量存储

可以使用以下方法初始化 InMemoryVectorStore 实例
* `InMemoryVectorStore.__init__` - 直接初始化
* `InMemoryVectorStore.from_documents` - 从列表初始化 `LangChain.docstore.Document` 对象
* `InMemoryVectorStore.from_texts` - 从文本列表初始化（可选带元数据）
* `InMemoryVectorStore.from_existing_index` - 从现有 MemoryDB 索引初始化

```python
from langchain_aws.vectorstores.inmemorydb import InMemoryVectorStore

vds = InMemoryVectorStore.from_texts(
    embeddings,
    redis_url="rediss://cluster_endpoint:6379/ssl=True ssl_cert_reqs=none",
)
```

```python

```

```text

```

## 查询

根据您的用例，有多种方式查询 `InMemoryVectorStore`  实现：

* `similarity_search`：查找与给定向量最相似的向量。
* `similarity_search_with_score`：查找与给定向量最相似的向量并返回向量距离
* `similarity_search_limit_score`：查找与给定向量最相似的向量并将结果数量限制为 `score_threshold`
* `similarity_search_with_relevance_scores`：查找与给定向量最相似的向量并返回向量相似度
* `max_marginal_relevance_search`：在查找与给定向量最相似向量的同时优化多样性

```python
results = vds.similarity_search("foo")
print(results[0].page_content)
```

```text
foo
```

```python
# with scores (distances)
results = vds.similarity_search_with_score("foo", k=5)
for result in results:
    print(f"Content: {result[0].page_content} --- Score: {result[1]}")
```

```text
Content: foo --- Score: 0.0
Content: foo --- Score: 0.0
Content: foo --- Score: 0.0
Content: bar --- Score: 0.1566
Content: bar --- Score: 0.1566
```

```python
# limit the vector distance that can be returned
results = vds.similarity_search_with_score("foo", k=5, distance_threshold=0.1)
for result in results:
    print(f"Content: {result[0].page_content} --- Score: {result[1]}")
```


```python
results = vds.similarity_search_with_relevance_scores("foo", k=5)
```

## MemoryDB 作为检索器

这里我们介绍使用向量存储作为检索器的不同选项。

我们可以使用三种不同的搜索方法进行检索。默认情况下，它将使用语义相似度。

```python
query = "foo"
results = vds.similarity_search_with_score(query, k=3, return_metadata=True)

for result in results:
    print("Content:", result[0].page_content, " --- Score: ", result[1])
```

```text
Content: foo  --- Score:  0.0
Content: foo  --- Score:  0.0
Content: foo  --- Score:  0.0
```

```python
retriever = vds.as_retriever(search_type="similarity", search_kwargs={"k": 4})
```

```python
docs = retriever.invoke(query)
```

## 删除索引

要删除您的条目，您必须通过其键来定位它们。

```python
# delete the indices too
InMemoryVectorStore.drop_index(
    index_name="users", delete_documents=True, redis_url="redis://localhost:6379"
)
InMemoryVectorStore.drop_index(
    index_name="users_modified",
    delete_documents=True,
    redis_url="redis://localhost:6379",
)
```

```text
True
```
