---
title: 安装 LangGraph
sidebarTitle: 安装
---

要安装基础 LangGraph 包：

:::python
<CodeGroup>
```bash pip
pip install -U langgraph
```

```bash uv
uv add langgraph
```
</CodeGroup>
:::

:::js
<CodeGroup>
```bash npm
npm install @langchain/langgraph @langchain/core
```

```bash pnpm
pnpm add @langchain/langgraph @langchain/core
```

```bash yarn
yarn add @langchain/langgraph @langchain/core
```

```bash bun
bun add @langchain/langgraph @langchain/core
```
</CodeGroup>
:::

要使用 LangGraph，你通常需要访问 LLM 并定义工具。
你可以用任何你认为合适的方式来做这件事。

一种方式（我们将在文档中使用的方式）是使用 [LangChain](/oss/langchain/overview)。

安装 LangChain：

:::python
<CodeGroup>
```bash pip
pip install -U langchain
# Requires Python 3.10+
```

```bash uv
uv add langchain
# Requires Python 3.10+
```
</CodeGroup>
:::

:::js
<CodeGroup>

</fragment>
```bash npm
npm install langchain
```

```bash pnpm
pnpm add langchain
```

```bash yarn
yarn add langchain
```

```bash bun
bun add langchain
```
</CodeGroup>
:::

要使用特定的 LLM 提供商包，你需要单独安装它们。

请参阅[集成](/oss/integrations/providers/overview)页面获取提供商特定的安装说明。

</fragment>
