title: 安装 LangGraph
sidebarTitle: 安装
---


安装基础 LangGraph 包：

:::python
<CodeGroup>
```bash pip
pip install -U langgraph
```

```bash uv
uv add langgraph
```
</CodeGroup>
:::

:::js
<CodeGroup>
```bash npm
npm install @langchain/langgraph @langchain/core
```

```bash pnpm
pnpm add @langchain/langgraph @langchain/core
```

```bash yarn
yarn add @langchain/langgraph @langchain/core
```

```bash bun
bun add @langchain/langgraph @langchain/core
```
</CodeGroup>
:::

要使用 LangGraph，通常需要访问 LLM 并定义工具。
您可以按照自己的方式完成此操作。

一种方法（在文档中我们将使用）是使用 [LangChain](/oss/langchain/overview).

安装 LangChain：

:::python
<CodeGroup>
```bash pip
pip install -U langchain
# Requires Python 3.10+
```

```bash uv
uv add langchain
# Requires Python 3.10+
```
</CodeGroup>
:::

:::js
<CodeGroup>
```bash npm
npm install langchain
```

```bash pnpm
pnpm add langchain
```

```bash yarn
yarn add langchain
```

```bash bun
bun add langchain
```
</CodeGroup>
:::

要使用特定的 LLM 提供商包，您需要单独进行安装。

请参阅 [集成](/oss/integrations/providers/overview) 页面获取提供商特定的安装说明。
