### 连接到您的代理

Agent Chat UI 可以连接到[本地](/oss/langchain/studio)和[已部署的](/oss/langchain/deploy)代理。

启动 Agent Chat UI 后，您需要对其进行配置以连接到您的代理：

1. **Graph ID**：输入您的图名称（在 `langgraph.json` 文件的 `graphs` 下查找）
2. **Deployment URL**：您的代理服务器的端点（例如，本地开发时为 `http://localhost:2024`，或您已部署代理的 URL）
3. **LangSmith API key（可选）**：添加您的 LangSmith API key（如果您使用的是本地代理服务器，则不需要）

配置完成后，Agent Chat UI 将自动获取并显示您代理中的任何中断线程。

<Tip>
  Agent Chat UI 开箱即用地支持渲染工具调用和工具结果消息。如需自定义显示哪些消息，请参阅[隐藏聊天中的消息](https://github.com/langchain-ai/agent-chat-ui?tab=readme-ov-file#hiding-messages-in-the-chat)。
</Tip>
