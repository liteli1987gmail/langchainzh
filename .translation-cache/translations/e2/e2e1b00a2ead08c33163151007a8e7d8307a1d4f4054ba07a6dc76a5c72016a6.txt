[Fireworks AI](https://fireworks.ai/) 是一个用于运行和定制模型的 AI 推理平台。有关 Fireworks 提供服务的所有模型列表，请参阅 [Fireworks 文档](https://fireworks.ai/models)。

本文档将帮助您开始使用 LangChain 与 Fireworks 补全模型（LLM）。有关 `Fireworks` 功能配置选项的详细文档，请参阅 [API 参考](https://reference.langchain.com/javascript/langchain-fireworks/Fireworks)。

## 设置

要使用 Fireworks 模型，您需要创建一个 Fireworks 账户、获取 API 密钥，并安装 `@langchain/fireworks` 集成包。

### 凭据

访问 [fireworks.ai](https://fireworks.ai/) 注册 Fireworks 并生成 API 密钥。完成此操作后，设置 `FIREWORKS_API_KEY` 环境变量：

```bash
export FIREWORKS_API_KEY="your-api-key"
```

如果你想获取模型调用的自动追踪，你也可以通过取消下面的注释来设置你的 [LangSmith](/langsmith/observability) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain Fireworks 集成位于 `@langchain/fireworks` 包中：

<CodeGroup>
```bash npm
npm install @langchain/fireworks @langchain/core
```
```bash yarn
yarn add @langchain/fireworks @langchain/core
```
```bash pnpm
pnpm add @langchain/fireworks @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化模型对象并生成文本补全：

```typescript
import { Fireworks } from "@langchain/fireworks"

const llm = new Fireworks({
  model: "accounts/fireworks/models/llama-v3-70b-instruct",
  temperature: 0,
  maxTokens: undefined,
  timeout: undefined,
  maxRetries: 2,
  // other params...
})
```
