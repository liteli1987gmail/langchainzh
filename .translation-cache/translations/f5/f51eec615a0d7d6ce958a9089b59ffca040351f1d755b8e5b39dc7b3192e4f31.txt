[AgentMail](https://agentmail.to) 是一个为 AI 代理构建的收件箱即 API 平台——配置、发送、回复、草拟和入站 Webhook 均可通过单一 HTTP API 获取。`langchain-agentmail` 包将 AgentMail SDK 包装为 LangChain 工具，外加一个文档加载器和一个基于收件箱的检索器。

## 安装和设置

<CodeGroup>

</fragment>
```bash pip
pip install langchain-agentmail
```

```bash uv
uv add langchain-agentmail
```
</CodeGroup>

从 [agentmail.to](https://agentmail.to) 获取您的 API 密钥，并将其设置为环境变量：

```bash
export AGENTMAIL_API_KEY="your-api-key"
```
