> [VDMS](https://github.com/IntelLabs/vdms/blob/master/README.md) 是一种存储解决方案，用于高效访问大型"视觉"数据，旨在通过搜索存储为图形的视觉元数据来查找相关视觉数据，从而实现云规模，并为视觉数据启用机器友好的增强功能以加快访问速度。

## 安装和设置

### 安装客户端

<CodeGroup>

</fragment>
```bash pip
pip install langchain-vdms
```

```bash uv
uv add langchain-vdms
```
</CodeGroup>

### 安装数据库

开始使用 VDMS 有两种方式：

1. 通过 Docker 在本地机器上安装 VDMS
    ```bash
        docker run -d -p 55555:55555 intellabs/vdms:latest
    ```

2. 直接在本地计算机上安装 VDMS。请参阅
[安装说明](https://github.com/IntelLabs/vdms/blob/master/INSTALL.md)。

## VectorStore

导入此向量存储：

```python
from langchain_vdms import VDMS
from langchain_vdms.vectorstores import VDMS
```
导入 VDMS Client 连接器：

```python
from langchain_vdms.vectorstores import VDMS_Client
```

有关 VDMS 包装器的更详细演练，请参阅 [VDMS](/oss/integrations/vectorstores/vdms)。
