>[LambdaDB](https://lambdadb.ai/) 是一个用于 RAG 和 AI 代理的无服务器 AI 数据库。在灵活的文档模型上统一全文、多向量和混合搜索。立即处理无限持久内存和大规模并发——成本仅为传统方案的 1/10。

## 安装与设置

安装 Python SDK：

<CodeGroup>

</fragment>
    ```bash pip
    pip install langchain-lambdadb
    ```

    ```bash uv
    uv add langchain-lambdadb
    ```
</CodeGroup>

您还需要一个 LambdaDB 账户，包含：
- 项目 URL
- 项目 API 密钥

在以下网址获取您的凭据：[lambdadb.ai](https://lambdadb.ai)。

## 向量存储

存在一个围绕 `LambdaDB` 集合的封装，允许您将其用作向量存储，用于语义搜索、RAG 和其他 AI 应用。

导入此向量存储：
```python
from langchain_lambdadb import LambdaDBVectorStore
```

### 快速入门

```python
from langchain_lambdadb import LambdaDBVectorStore
from langchain_openai import OpenAIEmbeddings
from lambdadb import LambdaDB
import os

# Initialize client
client = LambdaDB(
    server_url=os.environ["LAMBDADB_SERVER_URL"],
    project_api_key=os.environ["LAMBDADB_API_KEY"]
)

# Create vector store with existing collection
vector_store = LambdaDBVectorStore(
    client=client,
    collection_name="my_collection",  # Must exist beforehand
    embedding=OpenAIEmbeddings()
)
```

有关 LambdaDB 包装器的更详细分步指南，请参阅[完整文档](/oss/integrations/vectorstores/lambdadb)。
