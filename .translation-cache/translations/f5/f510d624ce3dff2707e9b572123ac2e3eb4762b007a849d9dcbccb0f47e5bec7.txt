## 概述

 `LocalFileStore` 是的持久化实现 `ByteStore` 它将所有内容存储在您选择的文件夹中。如果您使用单台机器并且可以接受文件的添加或删除，这将非常有用。

### 安装

LangChain `LocalFileStore` 集成位于 `langchain` package:

```python
pip install -qU langchain-classic
```

## 实例化

现在我们可以实例化字节存储：

```python
from pathlib import Path

from langchain_classic.storage import LocalFileStore

root_path = Path.cwd() / "data"  # can also be a path set by a string

kv_store = LocalFileStore(root_path)
```

## 用法

您可以使用以下方式设置数据： `mset` method:

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

您可以在以下位置看到创建的文件： `data` folder:

```python
!ls {root_path}
```

```text
key1 key2
```

您可以使用以下方式删除数据： `mdelete` method:

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```
