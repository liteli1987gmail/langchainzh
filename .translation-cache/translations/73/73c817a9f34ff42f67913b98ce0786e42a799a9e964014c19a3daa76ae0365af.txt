[Scrapeless](https://scrapeless.com) 提供灵活、功能丰富的数据采集服务，支持广泛的参数自定义和多格式导出。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-scrapeless
```

```bash uv
uv add langchain-scrapeless
```
</CodeGroup>

您需要设置您的 Scrapeless API 密钥：

```python
import os
os.environ["SCRAPELESS_API_KEY"] = "your-api-key"
```

## 工具

Scrapeless 集成提供多种工具：

- [ScrapelessDeepSerpGoogleSearchTool](/oss/integrations/tools/scrapeless_scraping_api) - 支持全面提取各类 Google SERP 数据。
- [ScrapelessDeepSerpGoogleTrendsTool](/oss/integrations/tools/scrapeless_scraping_api) - 获取 Google 关键词趋势数据，包括随时间变化的热门程度、区域关注度及相关搜索词。
- [ScrapelessUniversalScrapingTool](/oss/integrations/tools/scrapeless_universal_scraping) - 访问并提取通常会阻止爬虫的 JS 渲染网站数据。
- [ScrapelessCrawlerCrawlTool](/oss/integrations/tools/scrapeless_crawl) - 爬取网站及其链接页面以提取全面的数据。
- [ScrapelessCrawlerScrapeTool](/oss/integrations/tools/scrapeless_crawl) - 从单个网页中提取信息。
