>[Perplexity](https://www.perplexity.ai/pro) 是最强大的互联网搜索方式，提供无限 Pro Search、升级的 AI 模型、无限文件上传、图像生成和 API 额度。
>
> 您可以查看[可用模型列表](https://docs.perplexity.ai/docs/model-cards)。

## 安装和设置

安装 Perplexity x LangChain 集成包：

<CodeGroup>

</CodeGroup>
```bash pip
pip install langchain-perplexity
```

```bash uv
uv add langchain-perplexity
```
</CodeGroup>

从 [Perplexity API 密钥仪表板](https://www.perplexity.ai/account/api/keys) 获取您的 API 密钥，并将其设置为 `PERPLEXITY_API_KEY`（或 `PPLX_API_KEY`）环境变量。查看 [Perplexity 快速入门指南](https://docs.perplexity.ai/docs/getting-started) 了解更多详情。

## 聊天模型

查看各种 [Perplexity 使用示例](/oss/integrations/chat/perplexity)。

```python
from langchain_perplexity import ChatPerplexity
```

`ChatPerplexity` 也可以针对 [Perplexity Agent API](https://docs.perplexity.ai/api-reference/agent-api)，方法是传递 `use_responses_api=True`（或传递 `tools=[{"type": "web_search"}]`，会自动启用它）。参见聊天页面上的 [Agent API 支持](/oss/integrations/chat/perplexity#agent-api-support-use_responses_api) 了解详情和示例。

## 检索器

您可以使用 [`PerplexitySearchRetriever`](/oss/integrations/retrievers/perplexity_search) 从 [Perplexity Search API](https://docs.perplexity.ai/docs/search/quickstart) 获取网络搜索结果，作为 @[`Document`] 对象在标准检索管道中使用。

参见[使用示例](/oss/integrations/retrievers/perplexity_search)。

```python
from langchain_perplexity import PerplexitySearchRetriever
```

## 工具

您可以使用 Perplexity 作为智能体工具，为您的智能体提供 Perplexity Search API 的访问权限。

请参阅[使用示例](/oss/integrations/tools/perplexity_search)。

### PerplexitySearchResults

用于查询 Perplexity Search API 并返回 JSON 数组结果的工具（包含 title、URL、snippet、date、last updated）。

```python
from langchain_perplexity import PerplexitySearchResults
```

## 嵌入模型

您可以使用 [`PerplexityEmbeddings`](/oss/integrations/embeddings/perplexity) 通过 [Perplexity 嵌入向量 API](https://docs.perplexity.ai/api-reference/embeddings-post) 来生成嵌入向量。

查看[使用示例](/oss/integrations/embeddings/perplexity)。

```python
from langchain_perplexity import PerplexityEmbeddings
```

## Deep Agents Code

在 [Deep Agents Code](/oss/deepagents/code/providers) 中将 Perplexity 作为一流提供商使用。

使用 Perplexity extra 安装 Deep Agents Code：

<CodeGroup>
```bash uv
uv tool install 'deepagents-code[perplexity]'
```
</CodeGroup>

然后设置 `PERPLEXITY_API_KEY`，在交互式 `/model` 切换器或配置中使用 `perplexity:` 前缀来引用 Perplexity 模型（例如 `perplexity:sonar-pro`）。

## 组件参考

| 类 | 抽象 | 导入路径 | 描述 |
|-------|-------------|-------------|-------------|
| `ChatPerplexity` | 聊天模型 | `from langchain_perplexity import ChatPerplexity` | 封装 Perplexity API 的聊天模型，用于获取有依据的聊天补全。 |
| `PerplexitySearchRetriever` | 检索器 | `from langchain_perplexity import PerplexitySearchRetriever` | 从 Perplexity Search API 返回 `Document` 对象的检索器。 |
| `PerplexitySearchResults` | 工具 | `from langchain_perplexity import PerplexitySearchResults` | 以 JSON 数组形式为代理返回 Perplexity Search API 结果的工具。 |
| `PerplexityEmbeddings` | 嵌入 | `from langchain_perplexity import PerplexityEmbeddings` | 封装 Perplexity Embeddings API 的嵌入模型。 |
