[Dappier](https://dappier.com) 将任何 LLM 或您的 Agentic AI 连接到来自可信来源的实时、已获授权的专有数据，使您的 AI 成为任何领域的专家。我们的专业模型包括实时网络搜索、新闻、体育、金融股票市场数据、加密数据以及来自优质出版商的独家内容。在 [marketplace.dappier.com](https://marketplace.dappier.com) 的市场上探索各种数据模型。

[Dappier](https://dappier.com) 提供丰富的、可直接用于提示的、上下文相关的数据字符串，针对与 LangChain 的无缝集成进行了优化。无论您是在构建对话式 AI、推荐引擎还是智能搜索，Dappier 与 LLM 无关的 RAG 模型都能确保您的 AI 访问经过验证的最新数据——无需承担构建和管理自己的检索管道的复杂性。

## 安装和设置

安装 `langchain-dappier` 并设置环境变量 `DAPPIER_API_KEY`。

<CodeGroup>
```bash pip
pip install -U langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```

```bash uv
uv add langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```
</CodeGroup>

我们还需要设置 Dappier API 凭据，可以访问
[Dappier 网站](https://platform.dappier.com/profile/api-keys)生成。

我们可以在以下位置找到支持的数据模型
[Dappier 市场](https://platform.dappier.com/marketplace)。
