[Dappier](https://dappier.com) 连接任何 LLM 或您的 Agentic AI 到
来自可信来源的实时、已获授权、专有数据，
让您的 AI 成为任何领域的专家。我们的专业模型包括
实时网络搜索、新闻、体育、金融股票市场数据，
加密数据，以及来自优质出版商的独家内容。在我们的市场探索
各种数据模型，网址为
[marketplace.dappier.com](https://marketplace.dappier.com).

[Dappier](https://dappier.com) 提供丰富的、可直接用于提示的、
上下文相关的数据字符串，优化以实现无缝集成
与 LangChain。无论您是在构建对话式 AI、推荐
引擎还是智能搜索，Dappier 的 LLM 无关性 RAG 模型确保
您的 AI 能够访问经过验证的最新数据——无需复杂的
构建和管理您自己的检索管道。

## 安装和设置

安装 `langchain-dappier` 并设置环境变量
`DAPPIER_API_KEY`.

<CodeGroup>
```bash pip
pip install -U langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```

```bash uv
uv add langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```
</CodeGroup>

我们还需要设置 Dappier API 凭证，可以在以下位置生成
网站 [Dappier 网站。](https://platform.dappier.com/profile/api-keys).

我们可以通过访问以下方式找到支持的数据模型
[Dappier 市场。](https://platform.dappier.com/marketplace)
