# 示例 Markdown 文档

## 简介

欢迎阅读本示例 Markdown 文档。Markdown 是一种轻量级标记语言，用于格式化文本。它广泛用于文档、readme 文件等场景。

## 功能特性

### 标题

Markdown 支持多级标题：

- **标题 1**：`# 标题 1`
- **标题 2**：`## 标题 2`
- **标题 3**：`### 标题 3`

### 列表

#### 无序列表

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

#### 有序列表

1. 第一个项目
2. 第二个项目
3. 第三个项目
