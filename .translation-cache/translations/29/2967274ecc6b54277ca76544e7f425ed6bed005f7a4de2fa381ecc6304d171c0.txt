长期记忆允许您的代理在不同的对话和会话中存储和回忆信息。
与[短期记忆](/oss/langchain/short-term-memory)（其范围限定在单个线程中）不同，长期记忆跨线程持久化，可以随时被召回。

长期记忆构建于 [LangGraph 存储](/oss/langgraph/stores) 之上，将数据保存为 JSON 文档，按命名空间和键组织。

## 记忆存储

LangGraph 将长期记忆存储为 [store](/oss/langgraph/stores) 中的 JSON 文档。

每个记忆都按照自定义的 `namespace`（类似于文件夹）和唯一的 `key`（类似于文件名）进行组织。Namespace 通常包含用户 ID 或组织 ID 或其他标签，以便更轻松地组织信息。

此结构支持记忆的层级组织。然后通过内容过滤器支持跨 namespace 搜索。
