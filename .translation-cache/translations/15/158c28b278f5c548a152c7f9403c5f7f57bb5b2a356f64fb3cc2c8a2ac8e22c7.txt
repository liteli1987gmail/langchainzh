此 [@decodo/langchain-ts](https://www.npmjs.com/package/@decodo/langchain-ts) 包使开发者能够在其 LangChain 应用程序中使用 Decodo 的 Web Scraper API。

Web Scraper API 功能特点：

- 便捷的网络数据访问。简化从网站和在线资源获取信息的过程。
- 地理位置灵活性。无论区域限制均可访问内容。
- 可靠的抓取技术。采用先进的技术避免被检测和屏蔽。

## 功能特点

The @decodo/langchain-ts plugin features:

- **网页抓取**：抓取任意 URL 并获取 Markdown 内容
- **Google 搜索**：搜索 Google 并获取结构化结果
- **Amazon 搜索**：搜索 Amazon 并获取结构化产品数据
- **地理位置选择** 用于位置敏感内容
- **JavaScript 渲染** 用于需要真正浏览器的目标
- **Markdown 输出** 用于高效连接 LLM

## 安装

```bash
npm install @decodo/langchain-ts
```

## 快速开始

要使用本项目中的工具，您需要订阅 [Decodo Advanced Web Scraping API](https://help.decodo.com/docs/web-scraping-api-core-and-advanced-plans) 。仪表板上提供免费试用。 [仪表板](https://dashboard.decodo.com/).

## 示例

每个工具的一套代理示例。

### 通用

```typescript
import dotenv from "dotenv";
import { ChatOpenAI } from "@langchain/openai";
import { createAgent } from "@langchain/agents";
import { DecodoUniversalTool } from "@decodo/langchain-ts";

dotenv.config();

const main = async () => {
  const username = process.env.SCRAPER_API_USERNAME!;
  const password = process.env.SCRAPER_API_PASSWORD!;

  const decodoUniversalTool = new DecodoUniversalTool({ username, password });

  const model = new ChatOpenAI({
    model: "gpt-5.4-mini",
  });

  const agent = createAgent({
    llm: model,
    tools: [decodoUniversalTool],
  });

  const result = await agent.invoke({
    messages: [
      {
        role: "user",
        content:
          "scrape the wikipedia NBA 2025 season page and tell me who won in 2025?",
      },
    ],
  });

  console.log(result.messages[result.messages.length - 1].content);
};

if (require.main === module) {
  main();
}
```

更多示例，请参阅 [@decodo/langchain-ts](https://github.com/Decodo/decodo-langchain-ts) GitHub 上的内容。

## 配置

所有工具都接受 `DecodoConfig` object:

```typescript
type DecodoConfig = {
  username: string; // Your Web Advanced product username
  password: string; // Your Web Advanced product password
};
```
