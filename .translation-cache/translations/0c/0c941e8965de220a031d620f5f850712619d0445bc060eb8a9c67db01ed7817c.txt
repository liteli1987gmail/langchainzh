## 设置

要使用（模块_NAME) middleware, you'll need to create a/an (MODULE_名称）账户，请获取 API 密钥并安装 `langchain-(MODULE_NAME)` 集成包。

### 凭证

```python Set API key icon="key"
import getpass
import os

if "(MODULE_NAME)_API_KEY" not in os.environ:
    os.environ["(MODULE_NAME)_API_KEY"] = getpass.getpass("Enter your (MODULE_NAME) API key: ")
```

设置 LangSmith 以实现可观测性和 <Tooltip tip="Log each step of a model's execution to debug and improve it">追踪</Tooltip>也很有帮助（但非必需）。要启用自动追踪，请设置您的 [LangSmith](/langsmith/observability) API 密钥：

```python Enable tracing icon="flask"

os.environ["LANGSMITH_TRACING"] = "true"
```

## 与 Agent 配合使用

按顺序将中间件传递给 @[`create_agent`] or @[`create_deep_agent`]。您可以将提供商中间件与 [内置中间件](/oss/langchain/middleware/built-in).

```python Agent with middleware icon="robot"
from langchain.agents import create_agent

agent = create_agent(
    model="claude-sonnet-4-6",
    tools=[...],
    middleware=[middleware],
)

agent.invoke({"messages": [{"role": "user", "content": "INSERT QUERY"}]})
```

<Tip>
    中间件在 [Agent 循环](/oss/langchain/middleware/overview#the-agent-loop)中运行。如需自定义行为，请参阅 [自定义中间件](/oss/langchain/middleware/custom).
</Tip>
