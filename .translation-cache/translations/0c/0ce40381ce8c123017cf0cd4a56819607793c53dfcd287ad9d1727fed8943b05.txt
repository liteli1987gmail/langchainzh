[Deeplake](https://www.deeplake.ai/) 是一个专为 AI 和深度学习优化的数据库
applications.


## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-deeplake
```

```bash uv
uv add langchain-deeplake
```
</CodeGroup>
