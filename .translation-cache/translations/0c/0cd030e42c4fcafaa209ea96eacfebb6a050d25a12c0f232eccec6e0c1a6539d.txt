本页涵盖所有与 [Google Gemini](https://ai.google.dev/gemini-api/docs), [Google Cloud](https://cloud.google.com/) 及其他 Google 产品（如 Google Maps、YouTube 和 [更多](#other-google-products)).

<Note>
    **统一 SDK 和包整合**

    As of `langchain-google-genai` 4.0.0 起，此包使用统一的 [`google-genai`](https://googleapis.github.io/python-genai/) SDK，现支持 **Gemini Developer API 和 Vertex AI** backends.

    该 `langchain-google-vertexai` 包继续支持 Vertex AI 平台特定功能（Model Garden、Vector Search、评估服务等）。

    阅读 [完整公告和迁移指南](https://github.com/langchain-ai/langchain-google/discussions/1422).
</Note>

不确定使用哪个包？

<AccordionGroup>
    <Accordion title="Google Generative AI (Gemini API & Vertex AI)">
        通过以下方式访问 Google Gemini 模型： **[Gemini Developer API](https://ai.google.dev/)** or **[Vertex AI](https://cloud.google.com/vertex-ai)**。后端会根据您的配置自动选择。

        - **Gemini Developer API**：使用 API 密钥快速设置，适合个人开发者和快速原型开发
        - **Vertex AI**：企业级功能，与 Google Cloud 集成（需要 GCP 项目）

        使用 `langchain-google-genai` 用于聊天模型、LLM 和嵌入模型的包。

        [参见集成。](#google-generative-ai)
    </Accordion>
    <Accordion title="Google Cloud (Vertex AI Platform Services)">
        访问 Vertex AI 平台特定服务，包括 Gemini 模型以外的 Model Garden（Llama、Mistral、Anthropic）、评估服务以及专用视觉模型。

        使用 `langchain-google-vertexai` 包用于平台服务和特定包（例如、 `langchain-google-community`, `langchain-google-cloud-sql-pg`）用于其他云服务，如数据库和存储。

        [参见集成。](#google-cloud)
    </Accordion>
</AccordionGroup>

请参阅 Google 关于 [从 Gemini API 迁移到 Vertex AI](https://ai.google.dev/gemini-api/docs/migrate-to-cloud) 的指南，了解更多差异详情。
