本页涵盖 LangChain 与 [Google Gemini](https://ai.google.dev/gemini-api/docs)、[Google Cloud](https://cloud.google.com/) 以及其他 Google 产品（如 Google Maps、YouTube 和[更多](#other-google-products)）的所有集成。

<Note>
    **统一 SDK 和包整合**

    从 `langchain-google-genai` 4.0.0 版本开始，此包使用统一的 [`google-genai`](https://googleapis.github.io/python-genai/) SDK，现已支持**Gemini Developer API 和 Vertex AI** 两种后端。

    `langchain-google-vertexai` 包仍然受支持，用于 Vertex AI 平台特定功能（Model Garden、Vector Search、评估服务等）。

    阅读[完整公告和迁移指南](https://github.com/langchain-ai/langchain-google/discussions/1422)。
</Note>

不确定使用哪个包？

<AccordionGroup>
    <Accordion title="Google Generative AI（Gemini API 和 Vertex AI）">
        通过 **[Gemini Developer API](https://ai.google.dev/)** 或 **[Vertex AI](https://cloud.google.com/vertex-ai)** 访问 Google Gemini 模型。后端会根据您的配置自动选择。

        - **Gemini Developer API**：使用 API 密钥快速设置，适合个人开发者和快速原型开发
        - **Vertex AI**：企业级功能，集成 Google Cloud（需要 GCP 项目）

        使用 `langchain-google-genai` 包进行聊天模型、LLM 和嵌入。

        [查看集成。](#google-generative-ai)
    </Accordion>
    <Accordion title="Google Cloud（Vertex AI 平台服务）">
        访问 Vertex AI 平台特定服务，包括 Model Garden（Llama、Mistral、Anthropic）、评估服务和专门的视觉模型。

        使用 `langchain-google-vertexai` 包访问平台服务，并使用特定包（如 `langchain-google-community`、`langchain-google-cloud-sql-pg`）访问其他云服务（如数据库和存储）。

        [查看集成。](#google-cloud)
    </Accordion>
</AccordionGroup>

有关差异的更多详细信息，请参阅 Google 的[从 Gemini API 迁移到 Vertex AI 指南](https://ai.google.dev/gemini-api/docs/migrate-to-cloud)。
