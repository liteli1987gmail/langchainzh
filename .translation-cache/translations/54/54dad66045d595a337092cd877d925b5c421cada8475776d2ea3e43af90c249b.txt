[Vercel 沙盒](https://vercel.com/docs/vercel-sandbox) 提供临时的隔离 Linux 环境来运行不受信任的代码。参见 [Vercel 沙盒文档](https://vercel.com/docs/vercel-sandbox) 了解注册、身份验证和平台详情。

## 安装

<CodeGroup>
```bash pip
pip install langchain-vercel-sandbox
```

```bash uv
uv add langchain-vercel-sandbox
```
</CodeGroup>

## 身份验证

Vercel SDK 从环境读取凭据。设置以下变量，或在 [OIDC](https://vercel.com/docs/oidc) 上运行 Vercel 时使用：

```bash
export VERCEL_TOKEN="your-token"
export VERCEL_PROJECT_ID="your-project-id"
export VERCEL_TEAM_ID="your-team-id"
```

## 创建沙盒后端

在 Python 中，您使用提供程序 SDK 创建沙盒，然后使用 [deepagents 后端](/oss/deepagents/backends).

```python
from vercel.sandbox import Sandbox

from langchain_vercel_sandbox import VercelSandbox

sandbox = Sandbox.create()
backend = VercelSandbox(sandbox=sandbox)

try:
    result = backend.execute("echo hello")
    print(result.output)
finally:
    sandbox.stop()
```

## 与 Deep Agents 结合使用

```python
from vercel.sandbox import Sandbox
from langchain_anthropic import ChatAnthropic

from deepagents import create_deep_agent
from langchain_vercel_sandbox import VercelSandbox

sandbox = Sandbox.create()
backend = VercelSandbox(sandbox=sandbox)

agent = create_deep_agent(
    model=ChatAnthropic(model="claude-sonnet-4-20250514"),
    system_prompt="You are a coding assistant with sandbox access.",
    backend=backend,
)

try:
    result = agent.invoke(
        {"messages": [{"role": "user", "content": "Create a small Python project and run tests"}]}
    )
finally:
    sandbox.stop()
```

## 清理

完成后请务必停止沙盒以避免持续占用资源。

另请参见： [沙盒](/oss/deepagents/sandboxes).
