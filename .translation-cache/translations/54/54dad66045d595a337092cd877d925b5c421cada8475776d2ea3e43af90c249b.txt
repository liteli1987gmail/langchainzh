[Vercel Sandbox](https://vercel.com/docs/vercel-sandbox) 提供短暂的、隔离的 Linux 环境，用于运行不受信任的代码。有关注册、认证和平台详情，请参阅 [Vercel Sandbox 文档](https://vercel.com/docs/vercel-sandbox)。

## 安装

<CodeGroup>
```bash pip
pip install langchain-vercel-sandbox
```

```bash uv
uv add langchain-vercel-sandbox
```
</CodeGroup>

## 身份验证

Vercel SDK 从环境变量中读取凭据。请设置以下环境变量，或在 Vercel 上运行时使用 [OIDC](https://vercel.com/docs/oidc)：

```bash
export VERCEL_TOKEN="your-token"
export VERCEL_PROJECT_ID="your-project-id"
export VERCEL_TEAM_ID="your-team-id"
```

## 创建沙盒后端

在 Python 中，使用 provider SDK 创建沙盒，然后使用 [deepagents 后端](/oss/deepagents/backends) 将其包装。

```python
from vercel.sandbox import Sandbox

from langchain_vercel_sandbox import VercelSandbox

sandbox = Sandbox.create()
backend = VercelSandbox(sandbox=sandbox)

try:
    result = backend.execute("echo hello")
    print(result.output)
finally:
    sandbox.stop()
```

## 与 Deep Agents 配合使用

```python
from vercel.sandbox import Sandbox
from langchain_anthropic import ChatAnthropic

from deepagents import create_deep_agent
from langchain_vercel_sandbox import VercelSandbox

sandbox = Sandbox.create()
backend = VercelSandbox(sandbox=sandbox)

agent = create_deep_agent(
    model=ChatAnthropic(model="claude-sonnet-4-20250514"),
    system_prompt="You are a coding assistant with sandbox access.",
    backend=backend,
)

try:
    result = agent.invoke(
        {"messages": [{"role": "user", "content": "Create a small Python project and run tests"}]}
    )
finally:
    sandbox.stop()
```

## 清理

完成后请务必停止沙盒，以避免持续占用资源。

另请参阅：[沙盒](/oss/deepagents/sandboxes)。
