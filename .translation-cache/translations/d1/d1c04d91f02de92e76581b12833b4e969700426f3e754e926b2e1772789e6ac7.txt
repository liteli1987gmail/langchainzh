> [Docling](https://github.com/DS4SD/docling) 可以将 PDF、DOCX、PPTX、HTML 等格式解析为丰富的统一表示，包括文档布局、表格等，使其可用于 RAG 等生成式 AI 工作流程。
>
> 此集成通过 `DoclingLoader` 文档加载器提供 Docling 的功能。

## 安装和设置

只需从您的包管理器安装 `langchain-docling`，例如 pip：

<CodeGroup>
```bash pip
pip install langchain-docling
```

```bash uv
uv add langchain-docling
```
## 文档加载器

`langchain-docling` 中的 `DoclingLoader` 类将 Docling 无缝集成到 LangChain 中，使您能够：
- 轻松快速地在 LLM 应用程序中使用各种文档类型，以及
- 利用 Docling 的丰富表示进行高级、文档原生的 grounding。

基本用法如下：

</CodeGroup>

```python
from langchain_docling import DoclingLoader

FILE_PATH = ["https://arxiv.org/pdf/2408.09869"]  # Docling Technical Report

loader = DoclingLoader(file_path=FILE_PATH)

docs = loader.load()
```

有关端到端使用示例，请查看
[此示例](/oss/integrations/document_loaders/docling)。
