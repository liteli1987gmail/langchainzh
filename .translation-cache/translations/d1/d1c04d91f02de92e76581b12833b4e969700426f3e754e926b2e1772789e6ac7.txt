> [Docling](https://github.com/DS4SD/docling) 将 PDF、DOCX、PPTX、HTML 和其他格式解析为丰富的统一表示，包括文档布局、表格等，使它们可以用于 RAG 等生成式 AI 工作流。
>
> 此集成通过以下方式提供 Docling 的功能 `DoclingLoader` 文档加载器。

## 安装和设置

只需安装 `langchain-docling` 从您的包管理器，例如 pip：

<CodeGroup>
```bash pip
pip install langchain-docling
```

```bash uv
uv add langchain-docling
```
</CodeGroup>

## 文档加载器

 `DoclingLoader` 类在 `langchain-docling` 中无缝集成 Docling 到
LangChain，使您能够：
- 轻松快速地在 LLM 应用程序中使用各种文档类型，以及
- 利用 Docling 的丰富表示进行高级的、文档原生的语义匹配。

基本用法如下：

```python
from langchain_docling import DoclingLoader

FILE_PATH = ["https://arxiv.org/pdf/2408.09869"]  # Docling Technical Report

loader = DoclingLoader(file_path=FILE_PATH)

docs = loader.load()
```

如需端到端用法，请参阅
[此示例](/oss/integrations/document_loaders/docling).
