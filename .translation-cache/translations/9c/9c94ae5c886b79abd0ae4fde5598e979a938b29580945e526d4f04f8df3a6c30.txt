>[Mistral AI](https://docs.mistral.ai/api/) 是一个提供强大开源模型托管服务的平台。


## 安装和设置

与 API 通信需要一个有效的 [API 密钥](https://console.mistral.ai/users/api-keys/)。

您还需要 `langchain-mistralai` 包：

<CodeGroup>

</CodeGroup>
```bash pip
pip install langchain-mistralai
```

```bash uv
uv add langchain-mistralai
```
</CodeGroup>

## 聊天模型

### ChatMistralAI

查看[使用示例](/oss/integrations/chat/mistralai)。

```python
from langchain_mistralai.chat_models import ChatMistralAI
```

## 嵌入模型

### MistralAIEmbeddings

请参阅[使用示例](/oss/integrations/embeddings/mistralai)。

```python
from langchain_mistralai import MistralAIEmbeddings
```
