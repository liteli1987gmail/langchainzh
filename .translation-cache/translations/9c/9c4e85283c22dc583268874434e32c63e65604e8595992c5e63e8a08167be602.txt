## 设置

要访问 Pipeshift 模型，您需要创建一个 Pipeshift 账户，获取 API 密钥，并安装 `langchain-pipeshift` 集成包。

### 凭据

前往 [Pipeshift](https://dashboard.pipeshift.com) 注册 Pipeshift 并生成 API 密钥。完成后，设置 `PIPESHIFT_API_KEY` 环境变量：

```python
import getpass
import os

if not os.getenv("PIPESHIFT_API_KEY"):
    os.environ["PIPESHIFT_API_KEY"] = getpass.getpass("Enter your Pipeshift API key: ")
```

如果您想获取模型调用的自动追踪功能，您还可以通过取消下面的注释来设置您的 [LangSmith](/langsmith/observability) API 密钥：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
```

### 安装

LangChain Pipeshift 集成位于 `langchain-pipeshift` 包中：

```python
pip install -qU langchain-pipeshift
```

## 实例化

现在我们可以实例化模型对象并生成聊天补全：

```python
from langchain_pipeshift import Pipeshift

llm = Pipeshift(
    model="meta-llama/Meta-Llama-3.1-8B-Instruct",
    temperature=0,
    max_tokens=512,
)
```
