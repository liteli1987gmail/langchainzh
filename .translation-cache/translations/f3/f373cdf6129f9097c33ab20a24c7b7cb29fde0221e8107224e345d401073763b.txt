>[Apify](https://apify.com) 是一个用于网页抓取和数据提取的云平台，
>它提供了一个包含一千多个现成应用（称为 *Actors*）的[生态系统](https://apify.com/store)，
>可用于各种抓取、爬取和提取场景。

## 概述

Apify 提供对数千个预构建工具（Actors）的访问，用于网页抓取、数据提取和自动化。
该平台负责基础设施管理，让您可以专注于数据提取逻辑。

## 何时使用 Apify

- **访问数千个预构建 Actors**，用于各种平台（社交媒体、电子商务、搜索引擎等）
- **超越简单搜索的自定义网页抓取和自动化工作流**
- **灵活的 Actor 生态系统** - 运行 Apify Store 中的任何 Actor

此集成使您能够在 `Apify` 平台上运行 Actors，并将其结果加载到 LangChain 中，
以便用来自网络的文档和数据填充您的向量索引，例如，从包含文档、博客或知识库的网站生成答案。

## 安装和设置

- 使用以下命令安装 LangChain Apify Python 包：

<CodeGroup>

</CodeGroup>
```bash pip
pip install langchain-apify
```

```bash uv
uv add langchain-apify
```
</CodeGroup>

- 获取你的 [Apify API token](https://console.apify.com/account/integrations)，并将其设置为环境变量 (`APIFY_TOKEN`) 或在构造函数中作为 `apify_api_token` 传递。

## 工具

你可以使用 `ApifyActorsTool` 来让代理使用 Apify Actors。

```python
from langchain_apify import ApifyActorsTool
```

有关示例用法以及在 [Apify LangGraph agent Actor 模板](https://apify.com/templates/python-langgraph) 中使用 LangGraph 构建工具调用代理的完整示例，请参阅[此 notebook](/oss/integrations/tools/apify_actors)。

有关此工具的更多信息，请访问 [Apify 集成文档](https://docs.apify.com/platform/integrations/langgraph)。

## 包装器

您可以使用 `ApifyWrapper` 在 Apify 平台上运行 Actor。

```python
from langchain_apify import ApifyWrapper
```

有关如何使用此包装器的更多信息，请参阅 [Apify 集成文档](https://docs.apify.com/platform/integrations/langchain)。

## 使用场景

- **网页抓取**：从网站、社交媒体、电商网站提取数据
- **搜索引擎结果**：抓取 Google、Bing 等搜索引擎
- **数据收集**：为分析和 ML 管道收集结构化数据
- **内容聚合**：为 RAG 应用从多个来源收集内容

## 文档加载器

您还可以使用我们的 `ApifyDatasetLoader` 从 Apify 数据集获取数据。

```python
from langchain_apify import ApifyDatasetLoader
```

有关此加载器的更详细教程，请参阅[此笔记本](/oss/integrations/document_loaders/apify_dataset)。

## 定价

Apify 采用按使用量或按事件计费模式，并提供免费套餐。定价因 Actor 而异：

- 部分 Actor 是免费的（您只需支付平台计算单元费用）
- 其他 Actor 按结果或事件收费
- **按事件付费 (PPE) 定价**：许多 Actor 支持 [PPE 定价](https://docs.apify.com/platform/actors/publishing/monetize/pay-per-event)，这在您希望获得可预测的基于使用量的代理部署成本时非常有用
- 详情请参阅 [Apify 定价](https://apify.com/pricing)

此集成的源代码可在 [LangChain Apify 仓库](https://github.com/apify/langchain-apify) 中找到。

## MCP 服务器

不确定使用哪个 Actor 或需要哪些参数？
Apify 提供了 [MCP（模型上下文协议）服务器](https://mcp.apify.com)，可帮助您发现可用的 Actor、探索其输入模式并了解参数要求。

通过 HTTP 连接到 Apify MCP 服务器时，请将您的 Apify 令牌包含在请求头中：

```text
Authorization: Bearer <APIFY_TOKEN>
```
