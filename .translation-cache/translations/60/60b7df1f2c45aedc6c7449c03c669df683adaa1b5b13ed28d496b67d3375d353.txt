---
title: 如何自定义 Dockerfile
sidebarTitle: 自定义 Dockerfile
---
用户可以在从父 LangGraph 镜像导入之后，向 Dockerfile 中添加一组额外的行。要实现这一点，您只需修改 `langgraph.json` 文件，通过 `dockerfile_lines` 键传入您想要运行的命令。例如，如果我们想要在图中使用 `Pillow`，您需要添加以下依赖：

```
{
    "dependencies": ["."],
    "graphs": {
        "openai_agent": "./openai_agent.py:agent",
    },
    "env": "./.env",
    "dockerfile_lines": [
        "RUN apt-get update && apt-get install -y libjpeg-dev zlib1g-dev libpng-dev",
        "RUN pip install Pillow"
    ]
}
```

这将安装使用 Pillow 所需的系统包（如果我们使用 `jpeg` 或 `png` 图像格式）。
