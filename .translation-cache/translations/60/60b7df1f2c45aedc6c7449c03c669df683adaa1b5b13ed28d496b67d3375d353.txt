---
title: 如何自定义 Dockerfile
sidebarTitle: 自定义 Dockerfile
---
用户可以添加一个额外的行数组，添加到 Dockerfile 中，这些行位于从父 LangGraph 镜像导入之后。要实现这一点，您只需修改您的 `langgraph.json` 文件，通过将您想要运行的命令传递给 `dockerfile_lines` 键。例如，如果我们想使用 `Pillow` 在我们的图中，您需要添加以下依赖项：

```
{
    "dependencies": ["."],
    "graphs": {
        "openai_agent": "./openai_agent.py:agent",
    },
    "env": "./.env",
    "dockerfile_lines": [
        "RUN apt-get update && apt-get install -y libjpeg-dev zlib1g-dev libpng-dev",
        "RUN pip install Pillow"
    ]
}
```

这将安装使用 Pillow 所需的系统包（如果我们正在处理 `jpeg` or `png` 图像格式）。
