该 [Claude Agent SDK](https://platform.claude.com/docs/en/agent-sdk/overview) 是一个用于使用 Claude 构建代理应用程序的 SDK。LangSmith 提供与 Claude Agent SDK 的原生集成，可自动追踪您的代理执行、工具调用和与 Claude 模型的交互。

## 安装

安装 Claude Agent SDK 的 LangSmith 集成

<InstallClaudeAgentSdk/>

## 设置

设置您的 [API 密钥](/langsmith/create-account-api-key):

<SetupClaudeAgentSdk/>

您可以在 LangSmith UI 中的 [LangSmith UI](https://smith.langchain.com) 下的 **设置**.

要获取 Anthropic API 密钥，请参阅 [Claude 控制台](https://claude.ai/login).

## 快速入门

要在 Claude Agent SDK 应用程序中启用 LangSmith 追踪，请在应用程序开始时调用 `configure_claude_agent_sdk()` ：

<ExampleQuickstart/>

配置完成后，所有 Claude Agent SDK 操作都将自动追踪到 LangSmith，包括：

-   代理查询和响应
-   工具调用和结果
-   Claude 模型交互
-   MCP 服务器操作
