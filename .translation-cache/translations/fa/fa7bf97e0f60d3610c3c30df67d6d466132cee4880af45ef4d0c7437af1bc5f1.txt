[You.com](https://you.com) 是一个 AI 生产力平台，提供专为 LLM 应用设计的实时网络搜索和内容提取 API。

## 安装和设置

<CodeGroup>
    ```bash npm
    npm install @youdotcom-oss/langchain
    ```

    ```bash yarn
    yarn add @youdotcom-oss/langchain
    ```

    ```bash pnpm
    pnpm add @youdotcom-oss/langchain
    ```
</CodeGroup>

设置您的 You.com API 密钥。在以下位置获取您的 API 密钥：[you.com/platform](https://you.com/platform)。

```typescript
process.env.YDC_API_KEY = "your-api-key";
```

## 工具

此包提供三个 `DynamicStructuredTool` 实例：`youSearch`、`youResearch` 和 `youContents`。请参阅[使用示例](/oss/integrations/tools/youdotcom)。

```typescript
import { youSearch, youResearch, youContents } from "@youdotcom-oss/langchain";
```
