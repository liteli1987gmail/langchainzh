<Update label="June 15-19, 2026" rss={{ title: "2026-06-15 - Fleet product update" }}>

## 新功能

- [Fleet 工具](/langsmith/fleet/tools) 现在包括自托管用户的 Salesforce OAuth 提供程序设置，这样您就可以端到端配置提供程序。
- Agent 共享已重新设计，围绕两个选择展开：谁能使用和谁能编辑一个 agent，以及一个“发布为模板”选项，允许其他人派生自己的可编辑副本。
- Fleet agent 现在会在有人在环中断暂停时向原始线程（如 Slack）发布通知，并附上返回 agent 聊天的链接。
- 您现在可以通过您自己的回调 URL 完成 Fleet 集成 OAuth，这样无界面设置就可以在不使用 LangSmith UI 的情况下完成身份验证。
- Agent 卡片现在显示 agent 所有者。
- 新的第一方 [模板](/langsmith/fleet/templates)，品牌文案撰写员和申请人筛选，已在画廊中可用。

## 修复

- 在 agent 聊天中切换线程现在会立即清除上一个线程，并显示加载状态，而不是过时的消息。
- 该 [技能](/langsmith/fleet/skills) 列表现在会在某个技能加载失败时优雅降级，因此其余技能仍会显示。

</Update>

<Update label="June 8-12, 2026" rss={{ title: "2026-06-08 - Fleet product update" }}>

## 新功能

- [模板](/langsmith/fleet/templates) 现在显示“由 Fleet 提供”以及 Fleet 标志，因此精选模板符合 Fleet 品牌。

## 修复

- Fleet 列出线程端点现在返回 `items` 而不是 `threads`，因此响应结构与 API 其他部分一致。
- Fleet 线程请求现在会在大型响应可能触发 5xx 时返回更清晰的错误，因此长列表会优雅地失败。

</Update>

<Update label="June 1-5, 2026" rss={{ title: "2026-06-01 - LangSmith Fleet update" }}>

## 新功能

- [技能](/langsmith/fleet/skills) 加载更快：技能列表先获取轻量级元数据，只有在您打开某个技能时才加载文件内容。
- Agent 创建菜单添加了一个 [模板](/langsmith/fleet/templates) entry.
- 该 [远程 MCP](/langsmith/fleet/remote-mcp-servers) 授权屏幕现在显示连接应用程序的名称、标志、主页、条款和隐私链接，而不是其原始客户端 ID。
- [Slack 集成](/langsmith/fleet/slack-app) 在 AWS 和 APAC 区域可用。

## 修复

- [计划（cron）执行](/langsmith/fleet/schedules) 已恢复用于企业 Fleet agent。
- 长时间运行的 agent 运行和 agent-builder 生成不再在 60 秒后被切断。
- Gmail 读取邮件 [工具](/langsmith/fleet/tools) 现在在您使用某个条件搜索已发送邮件时返回结果。 `in:sent` query.
- 改进了 agent 编辑器中长工具箱、技能和子 agent 列表的滚动体验，webhook 对话框现在在视口内滚动。

</Update>

<Update label="March 16-20, 2026" rss={{ title: "2026-03-16 - Fleet product update" }}>

## 新功能

- Agent Builder 现已 [LangSmith Fleet](/langsmith/fleet)。新名称体现了 Fleet 专注于为您的整个团队构建和管理智能体的定位：创建智能体、共享智能体、管理其任务，以及控制智能体访问和身份。所有现有的智能体、配置、集成、方案和合同将继续正常运行，无需您采取任何行动。

</Update>

<Update label="February 16-20, 2026" rss={{ title: "2026-02-16 - Fleet product update" }}>

## 新功能

- 一个中央聊天智能体连接到您的所有工作区 [工具](/langsmith/fleet/tools)，包括 Slack、Gmail、Linear 和 MCP 服务器，让您在无需先设置专用智能体的情况下提问和执行操作。
- 一键将有用的对话转化为可重复运行的智能体，无需提示工程或条件逻辑。
- 直接在聊天中上传文件，包括 CSV、图片、文档和样式指南，让智能体立即处理。
- 中央工具注册表让工作区管理员连接 [工具](/langsmith/fleet/tools)，管理身份验证，并控制整个组织的访问权限。

</Update>

<Update label="October 27-31, 2025" rss={{ title: "2025-10-27 - Fleet product update" }}>

## 新功能
