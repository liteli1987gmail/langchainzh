<Update label="June 15-19, 2026" rss={{ title: "2026-06-15 - Fleet 产品更新" }}>

## 新功能

- [Fleet 工具](/langsmith/fleet/tools) 现在包括面向自托管用户的 Salesforce OAuth 提供商设置，让您可以端到端配置提供商。
- 智能体共享现已重新设计，提供两个选择：谁可以使用和谁可以编辑智能体，另外还有"发布为模板"选项，允许其他人创建自己的可编辑副本。
- Fleet 智能体现在可以在暂停于人工介入中断时，向原始线程（如 Slack）发布通知，并附带返回智能体聊天的链接。
- 您现在可以通过自己的回调 URL 完成 Fleet 集成的 OAuth，让无头设置无需 LangSmith UI 即可完成身份验证。
- 智能体卡片现在显示智能体所有者。
- 画廊中提供新的第一方[模板](/langsmith/fleet/templates)，品牌文案撰写员和申请人筛选。

## 修复

- 在智能体聊天中切换线程现在会立即清除上一个线程并显示加载状态，而不是显示过时消息。
- 当某个技能加载失败时，[技能](/langsmith/fleet/skills)列表现在会优雅降级，因此剩余技能仍会显示。

</Update>

<Update label="June 8-12, 2026" rss={{ title: "2026-06-08 - Fleet 产品更新" }}>

## 新功能

- [模板](/langsmith/fleet/templates) 现在显示"Fleet 出品"并带有 Fleet 标志，因此精选模板符合 Fleet 品牌风格。

## 修复

- Fleet 列表线程端点现在返回 `items` 而不是 `threads`，因此响应结构与 API 其他部分一致。
- Fleet 线程请求现在在大型响应会触发 5xx 时返回更清晰的错误，使长列表优雅失败。

</Update>

<Update label="June 1-5, 2026" rss={{ title: "2026-06-01 - LangSmith Fleet 更新" }}>

## 新功能

- [技能](/langsmith/fleet/skills) 加载更快：技能列表先获取轻量级元数据，只在您打开技能时才加载文件内容。
- 智能体创建菜单新增[模板](/langsmith/fleet/templates)入口。
- [远程 MCP](/langsmith/fleet/remote-mcp-servers) 授权屏幕现在显示连接应用的名称、标志、主页、条款和隐私链接，而不是显示原始客户端 ID。
- [Slack 集成](/langsmith/fleet/slack-app) 现已在 AWS 和 APAC 区域可用。

## 修复

- 企业 Fleet 智能体的[计划（cron）执行](/langsmith/fleet/schedules)已恢复。
- 长时运行的智能体运行和智能体构建器生成不再在 60 秒后被截断。
- Gmail 读取邮件[工具](/langsmith/fleet/tools) 现在在您使用 `in:sent` 查询搜索已发送邮件时返回结果。
- 改进了智能体编辑器中长工具箱、技能和子智能体列表的滚动，webhook 对话框现在在视口内滚动。

</Update>

<Update label="March 16-20, 2026" rss={{ title: "2026-03-16 - Fleet 产品更新" }}>

## 新功能

- 智能体构建器现已更名为 [LangSmith Fleet](/langsmith/fleet)。新名称反映了 Fleet 专注于为整个团队构建和管理智能体：创建智能体、共享智能体、管理任务以及控制智能体访问和身份。所有现有智能体、配置、集成、计划、合同继续正常工作，无需您采取任何操作。

</Update>

<Update label="February 16-20, 2026" rss={{ title: "2026-02-16 - Fleet 产品更新" }}>

## 新功能

- 一个中央聊天智能体连接到您工作区的所有[工具](/langsmith/fleet/tools)，包括 Slack、Gmail、Linear 和 MCP 服务器，让您可以提问和执行操作而无需先设置专用智能体。
- 一键将有用对话转换为可重复使用的智能体，无需提示工程或条件逻辑。
- 直接在聊天中上传文件，包括 CSV、图片、文档和风格指南，让智能体立即处理。
- 中央工具注册表让工作区管理员连接[工具](/langsmith/fleet/tools)、管理身份验证并控制整个组织的访问权限。

</Update>

<Update label="October 27-31, 2025" rss={{ title: "2025-10-27 - Fleet 产品更新" }}>

## 新功能

- LangSmith 智能体构建器以私人预览形式推出，为非开发人员提供无代码构建智能体的方式，包括对话式设置、内置记忆、MCP 集成、自动触发器和子智能体支持。智能体构建器后来更名为 [LangSmith Fleet](/langsmith/fleet)。

</Update>
