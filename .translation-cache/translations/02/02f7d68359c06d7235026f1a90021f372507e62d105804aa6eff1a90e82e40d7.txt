---
title: "LangSmith 沙箱集成"
description: "使用 LangChain Python 与 LangSmithSandbox 类型集成。"
---

LangSmith 沙箱是由 LangChain 为您管理的沙箱环境，无需设置单独的提供商账户、计费或基础设施。它们是运行代理代码的零设置默认选项，而 AgentCore、Daytona、E2B、Modal、Runloop 和 Vercel 等第三方提供商在您想要使用自己的环境时仍然可用。由于它们是 LangChain 平台的一部分，您可以与其他 LangSmith 资源一起管理它们。有关设置、快照、服务 URL 和身份验证代理的信息，请参阅 [LangSmith 沙箱文档](/langsmith/sandboxes)。

## 安装

<CodeGroup>

</fragment>
```bash pip
pip install "langsmith[sandbox]" deepagents
```

```bash uv
uv add "langsmith[sandbox]" deepagents
```
</CodeGroup>

## 设置您的 API 密钥

```bash
export LANGSMITH_API_KEY="<your-api-key>"
```

`SandboxClient()` 会从环境变量中读取此密钥，因此无需进一步配置。

## 创建沙盒后端

使用 `SandboxClient` 创建 LangSmith 沙盒，然后使用 @[LangSmithSandbox] 将其包装为 [Deep Agents 后端](/oss/deepagents/backends) 运行。

```python
from deepagents.backends.langsmith import LangSmithSandbox
from langsmith.sandbox import SandboxClient

client = SandboxClient()
ls_sandbox = client.create_sandbox()
backend = LangSmithSandbox(sandbox=ls_sandbox)

result = backend.execute("echo hello")
print(result.output)

# Delete the sandbox when you are done (see Clean up)
client.delete_sandbox(ls_sandbox.name)
```

## 与 Deep Agents 配合使用

将后端与 @[create_deep_agent] 以及 @[ChatAnthropic] 等聊天模型一起传递，然后调用代理。`try`/`finally` 块会在 try 块执行完毕或发生错误时删除沙箱。

```python
from deepagents import create_deep_agent
from deepagents.backends.langsmith import LangSmithSandbox
from langchain_anthropic import ChatAnthropic
from langsmith.sandbox import SandboxClient

client = SandboxClient()
ls_sandbox = client.create_sandbox()
backend = LangSmithSandbox(sandbox=ls_sandbox)

agent = create_deep_agent(
    model=ChatAnthropic(model="claude-sonnet-4-6"),
    system_prompt="You are a coding assistant with sandbox access.",
    backend=backend,
)

try:
    result = agent.invoke(
        {
            "messages": [
                {"role": "user", "content": "Create a hello world Python script and run it"}
            ]
        }
    )
    print(result["messages"][-1].content_blocks)
finally:
    client.delete_sandbox(ls_sandbox.name)
```

## 清理

完成后请删除沙盒，因为它们在被删除前会持续消耗资源。

```python
client.delete_sandbox(ls_sandbox.name)
```

您还可以从 [LangSmith UI](https://smith.langchain.com) 的 **Sandboxes** 下管理沙箱资源。
