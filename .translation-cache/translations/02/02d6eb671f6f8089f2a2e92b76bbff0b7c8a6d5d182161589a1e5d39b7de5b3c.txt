title: "MindsDB 集成"

---

MindsDB 是一个从企业数据定制 AI 的平台。通过 MindsDB 及其近 200 个 [数据源](https://docs.mindsdb.com/integrations/data-overview) 和 [AI/ML frameworks](https://docs.mindsdb.com/integrations/ai-overview)的集成，任何开发者都可以使用他们的企业数据来更快、更安全地为自己的目的定制 AI。

With MindsDB, you can connect any data source to any AI/ML model to implement and automate AI-powered applications. Deploy, serve, and fine-tune models in real-time, utilizing data from databases, vector stores, or applications. Do all that using universal tools developers already know.

MindsDB 与 LangChain 集成，使用户能够：


- 在 MindsDB 中部署通过 LangChain 可用的模型，使它们可被众多数据源访问。
- 使用实时动态数据在 MindsDB 中微调通过 LangChain 可用的模型。
- 使用 LangChain 和 MindsDB 自动化 AI 工作流。

参阅 [我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain) 了解更多关于 MindsDB 与 LangChain 的集成信息并查看示例。
