本笔记本介绍如何使用 Google 搜索组件。

首先，您需要设置适当的 API 密钥和环境变量。要进行设置，请在 `GOOGLE_API_KEY` Google Cloud 凭据控制台中创建[console.cloud.google.com/apis/credentials](https://console.cloud.google.com/apis/credentials))，并获取一个 GOOGLE_CSE_ID，使用可编程搜索引擎 ([programmablesearchengine.google.com/controlpanel/create](https://programmablesearchengine.google.com/controlpanel/create))。接下来，最好按照 [可编程 Google 搜索说明](https://stackoverflow.com/questions/37083058/programmatically-searching-google-in-python-using-custom-search).

然后我们需要设置一些环境变量。

```python
pip install -qU  langchain-google-community
```

```python
import os

os.environ["GOOGLE_CSE_ID"] = ""
os.environ["GOOGLE_API_KEY"] = ""
```

```python
from langchain.tools import tool
from langchain_google_community import GoogleSearchAPIWrapper

search = GoogleSearchAPIWrapper()


@tool
def google_search(query: str) -> str:
    """Search Google for recent results."""
    return search.run(query)
```

```python
google_search.invoke("Obama's first name?")
```

## 结果数量

您可以使用 `k` 参数设置结果数量

```python
search = GoogleSearchAPIWrapper(k=1)


@tool
def im_feeling_lucky(query: str) -> str:
    """Search Google and return the first result."""
    return search.run(query)
```

```python
im_feeling_lucky.invoke("python")
```

```text
'The official home of the Python Programming Language.'
```

'Python 编程语言的官方网站。'

## 元数据结果

通过 GoogleSearch 运行查询并返回摘要、标题和链接元数据。

- 摘要：结果的描述。
- 标题：结果的标题。
- 链接：结果的链接。

```python
search = GoogleSearchAPIWrapper()


@tool
def google_search_snippets(query: str) -> str:
    """Search Google for recent results."""
    return search.results(query, 5)
```
