这将帮助您开始使用 `AzureAIOpenAIApiChatModel` [聊天模型](/oss/langchain/models)。

`AzureAIOpenAIApiChatModel` 类使用 Azure AI Foundry 中提供的 OpenAI 兼容 API。AI Foundry 提供多种聊天模型，包括 AzureOpenAI、Cohere、Llama、Phi-3/4 和 DeepSeek-R1 等。您可以在 [Azure 文档](https://learn.microsoft.com/azure/ai-studio/how-to/model-catalog-overview) 中找到其最新模型及其费用、上下文窗口和支持的输入类型的信息。

## 安装设置

要访问 `AzureAIOpenAIApiChatModel` 模型，您需要创建一个 [Azure 账户](https://azure.microsoft.com/pricing/purchase-options/azure-account)、获取 API 密钥，并安装 `langchain-azure-ai` 集成包。

### 凭证

前往 [Azure 文档](https://learn.microsoft.com/en-us/azure/ai-studio/how-to/develop/sdk-overview?tabs=sync&pivots=programming-language-python) 了解如何创建部署并生成 API 密钥。模型部署完成后，您可以在 AI Foundry 中点击"获取端点"按钮。这将显示您的端点和 API 密钥。完成此操作后，请设置环境变量：

```python
import getpass
import os

if not os.getenv("AZURE_AI_PROJECT_ENDPOINT"):
    os.environ["AZURE_AI_PROJECT_ENDPOINT"] = getpass.getpass(
        "Enter your Azure AI project endpoint: "
    )
```

如果你想让你的模型调用获得自动追踪，你也可以通过取消下面的注释来设置你的 [LangSmith](/langsmith/observability) API 密钥：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
```

### 安装

LangChain `AzureAIOpenAIApiChatModel` 集成位于 `langchain-azure-ai` 包中：

```python
pip install -qU langchain-azure-ai
```

## 实例化

现在我们可以实例化模型对象并生成聊天补全：

```python
from langchain_azure_ai.chat_models import AzureAIOpenAIApiChatModel
from azure.identity import DefaultAzureCredential

llm = AzureAIOpenAIApiChatModel(
    model="gpt-4o",
    credential=DefaultAzureCredential(),
    temperature=0,
    max_tokens=None,
    max_retries=2,
)
```

## 调用

```python
messages = [
    (
        "system",
        "You are a helpful assistant that translates English to French. Translate the user sentence.",
    ),
    ("human", "I love programming."),
]
ai_msg = llm.invoke(messages)
ai_msg
```
