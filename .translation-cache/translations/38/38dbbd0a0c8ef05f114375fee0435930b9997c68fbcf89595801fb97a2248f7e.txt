## 设置

要访问 Fireworks 嵌入模型，您需要创建一个 Fireworks 账户，获取 API 密钥，并安装 `@langchain/fireworks` 集成包。

### 凭证

前往 [fireworks.ai](https://fireworks.ai/) 注册 `Fireworks` 并生成 API 密钥。完成后设置 `FIREWORKS_API_KEY` 环境变量：

```bash
export FIREWORKS_API_KEY="your-api-key"
```

如果您想获取模型调用的自动追踪，您还可以设置您的 [LangSmith](/langsmith/observability) API 密钥（取消注释以下内容）：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain `FireworksEmbeddings` 集成位于 `@langchain/fireworks` package:

<CodeGroup>
```bash npm
npm install @langchain/fireworks @langchain/core
```
```bash yarn
yarn add @langchain/fireworks @langchain/core
```
```bash pnpm
pnpm add @langchain/fireworks @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化模型对象并嵌入文本：

```typescript
import { FireworksEmbeddings } from "@langchain/fireworks";

const embeddings = new FireworksEmbeddings({
  model: "nomic-ai/nomic-embed-text-v1.5",
});
```

## 索引和检索

嵌入模型常用于检索增强生成 (RAG) 流程中，包括索引数据和后续检索。如需更详细的说明，请参阅 [**学习** 标签页下的](/oss/learn/).

RAG 教程。下面请查看如何使用上面初始化的 `embeddings` 对象进行数据索引和检索。在此示例中，我们将使用演示 [`MemoryVectorStore`](/oss/integrations/vectorstores/memory).

```typescript
// Create a vector store with a sample text
import { MemoryVectorStore } from "@langchain/classic/vectorstores/memory";

const text = "LangChain is the framework for building context-aware reasoning applications";

const vectorstore = await MemoryVectorStore.fromDocuments(
  [{ pageContent: text, metadata: {} }],
  embeddings,
);

// Use the vector store as a retriever that returns a single document
const retriever = vectorstore.asRetriever(1);

// Retrieve the most similar text
const retrievedDocuments = await retriever.invoke("What is LangChain?");

retrievedDocuments[0].pageContent;
```

```text
"LangChain is the framework for building context-aware reasoning applications"
```

## 直接使用

在底层，向量存储和检索器的实现调用 `embeddings.embedDocument(...)` 和 `embeddings.embedQuery(...)` 来为 `fromDocuments` 和检索器的 `invoke` 操作创建嵌入。

您可以直接调用这些方法来获取用于自己用例的嵌入。

### 嵌入单个文本

您可以使用以下方式嵌入搜索查询 `embedQuery`。这会生成针对该查询的向量表示：

```typescript
const singleVector = await embeddings.embedQuery(text);

console.log(singleVector.slice(0, 100));
```

您可以使用以下方式嵌入多个文本进行索引 `embedDocuments`。此方法使用的内部实现可能（但不一定）与嵌入查询不同：

```typescript
const text2 = "LangGraph is a library for building stateful, multi-actor applications with LLMs";

const vectors = await embeddings.embedDocuments([text, text2]);

console.log(vectors[0].slice(0, 100));
console.log(vectors[1].slice(0, 100));
```
