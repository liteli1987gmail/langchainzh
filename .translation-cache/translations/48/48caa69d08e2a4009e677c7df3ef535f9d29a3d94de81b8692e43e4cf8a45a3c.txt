---
title: 如何审核评估器分数
sidebarTitle: 审核评估器分数
---

LLM-as-a-judge 评估器并不总是能正确判断。因此，让人工手动审核评估器留下的分数并在必要时进行纠正是非常有用的。LangSmith 允许您在 UI 或 SDK 中对评估器分数进行更正。

## 在对比视图中

在对比视图中，您可以点击任意反馈标签来打开反馈详情。然后点击右侧的"edit"图标打开更正视图。您可以在"Make correction"下的文本框中输入您期望的分数。如果需要，您还可以附加一条解释说明您的更正。这在您使用 [few-shot 评估器](/langsmith/create-few-shot-evaluators) 时非常有用，它会自动插入到您的 few-shot 示例中，替换 `few_shot_explanation` 提示变量。

![审核评估器对比视图](/langsmith/images/corrections-comparison-view.png)

## 在运行表中

在运行表中，找到"Feedback"列并点击反馈标签打开反馈详情。同样，点击右侧的"edit"图标打开更正视图。

![审核评估器运行表](/langsmith/images/corrections-runs-table.png)

## 在 SDK 中

可以通过 SDK 的 `update_feedback` 函数和 `correction` 字典进行更正。您必须指定一个 `score` 键，它对应一个数字，以便在 UI 中呈现。

<CodeGroup>


</CodeGroup>

```python Python
import langsmith

client = langsmith.Client()

client.update_feedback(
    my_feedback_id,
    correction={
        "score": 1,
    },
)
```

```typescript TypeScript
import { Client } from 'langsmith';

const client = new Client();

await client.updateFeedback(
    myFeedbackId,
    {
        correction: {
            score: 1,
        }
    }
)
```
