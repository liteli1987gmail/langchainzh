[Isaacus](https://isaacus.com/) 是一家基础性法律人工智能研究公司，致力于为法律科技生态系统构建人工智能模型、应用程序和工具。

Isaacus 的产品包括 [Kanon 2 Embedder](https://isaacus.com/blog/introducing-kanon-2-embedder)，这是世界上最好的法律嵌入模型（根据 [Massive Legal Embedding Benchmark](https://isaacus.com/blog/introducing-mleb) 衡量），以及 [法律零样本分类](https://docs.isaacus.com/models/introduction#universal-classification) 和 [法律抽取式问答模型](https://docs.isaacus.com/models/introduction#answer-extraction)。

Isaacus 为 LangChain 的嵌入接口提供一流支持，可通过 [`langchain-isaacus`](https://pypi.org/project/langchain-isaacus/) 集成包访问。

## 设置

要开始使用 LangChain 与 Isaacus 模型，请前往 [Isaacus Platform](https://platform.isaacus.com/accounts/signup/) 并创建一个新账户。

注册后，[添加付款方式](https://platform.isaacus.com/billing/)（从而获得您的[免费积分](https://docs.isaacus.com/pricing/credits)）并[生成 API 密钥](https://platform.isaacus.com/users/api-keys/)。

接下来，安装 [`langchain-isaacus`](https://pypi.org/project/langchain-isaacus/) 集成包：
<CodeGroup>

</fragment>
```bash pip
pip install langchain-isaacus
```

```bash uv
uv add langchain-isaacus
```
</CodeGroup>

然后你应该设置你的 `ISAACUS_API_KEY` 环境变量为你的 Isaacus API 密钥。
<CodeGroup>
```bash bash
export ISAACUS_API_KEY="your_api_key_here"
```
```powershell powershell
$env:ISAACUS_API_KEY="your_api_key_here"
```
</CodeGroup>

## 嵌入
以下代码片段演示了如何使用 Isaacus 的 Kanon 2 Embedder 模型结合 LangChain 来评估法律查询与法律文档之间的语义相似性。关于如何使用 Isaacus LangChain 集成生成嵌入的更详细教程，请参阅 [Isaacus 嵌入指南](/oss/integrations/embeddings/isaacus)。

```python
import numpy as np # NOTE you may need to `pip install numpy`.

from langchain_isaacus import IsaacusEmbeddings

# Create an Isaacus API client for Kanon 2 Embedder.
client = IsaacusEmbeddings(
    "kanon-2-embedder",
    # dimensions=1792, # You may optionally wish to specify a lower dimension.
)

# Embed a dummy document.
document_embedding = client.embed_documents(texts=["These are GitHub's billing policies."])[0]

# Embed our search queries.
relevant_query_embedding = client.embed_query(text="What are GitHub's billing policies?")
irrelevant_query_embedding = client.embed_query(text="What are Microsoft's billing policies?")

# Compute the similarity between the queries and the document.
relevant_similarity = np.dot(relevant_query_embedding, document_embedding)
irrelevant_similarity = np.dot(irrelevant_query_embedding, document_embedding)

# Log the results.
print(f"Similarity of relevant query to the document: {relevant_similarity * 100:.2f}")
print(f"Similarity of irrelevant query to the document: {irrelevant_similarity * 100:.2f}")
```
