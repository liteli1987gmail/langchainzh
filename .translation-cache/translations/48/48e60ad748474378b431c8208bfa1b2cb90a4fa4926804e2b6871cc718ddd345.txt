本页面介绍如何使用 [Yeager.ai](https://yeager.ai) 来生成 LangChain 工具和代理。

## 什么是 yeager.ai？
Yeager.ai 是一个旨在简化 AI 代理和工具创建过程的生态系统。

它具有 yAgents，这是一款无代码 LangChain 代理构建器，使用户能够轻松构建、测试和部署 AI 解决方案。yAgents 利用 LangChain 框架，允许与各种语言模型和资源无缝集成，适用于跨不同应用场景的开发者、研究人员和 AI 爱好者。

## yAgents
低代码生成式代理，旨在帮助您轻松构建、原型设计和部署 LangChain 工具。

### 如何使用？
<CodeGroup>
```bash pip
pip install yeagerai-agent
yeagerai-agent
```

```bash uv
uv add yeagerai-agent
yeagerai-agent
```
</CodeGroup>
Go to http://127.0.0.1:7860

这将安装必要的依赖项并在您的系统上设置 yAgents。首次运行后，yAgents 将创建一个 .env 文件，您可以在其中输入您的 OpenAI API 密钥。您也可以直接从 Gradio 界面的“设置”选项卡进行相同操作。

`OPENAI_API_KEY=<your_openai_api_key_here>`

我们建议使用 GPT-4。不过，如果问题分解得足够细致，该工具也可以与 GPT-3 配合使用。

### 使用 yAgents 创建和执行工具
yAgents 可以轻松创建和执行由 AI 驱动的工具。以下是流程概述：
1. 创建工具：要创建工具，请向 yAgents 提供自然语言提示。提示应清楚描述工具的用途和功能。例如：
`create a tool that returns the n-th prime number`

2. 将工具加载到工具包：要加载工具到 yAgents，只需向 yAgents 提供相应的命令即可。例如：
`load the tool that you just created it into your toolkit`

3. 执行工具：要运行工具或代理，只需向 yAgents 提供包含工具名称和任何必需参数的命令即可。例如：
`generate the 50th prime number`

您可以 [观看演示视频](https://www.youtube.com/watch?v=KA5hCM3RaWE) 了解其工作原理。

随着您对 yAgents 的日益熟悉，您可以创建更高级的工具和代理来自动化您的工作并提高工作效率。

更多信息，请参阅 [yAgents' GitHub](https://github.com/yeagerai/yeagerai-agent) 或我们的 [文档](https://yeagerai.gitbook.io/docs/general/welcome-to-yeager.ai)
