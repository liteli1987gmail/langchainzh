## 基本用法

### 保存文档

使用 `MemorystoreDocumentSaver.add_documents(<documents>)` 保存 LangChain 文档。要初始化 `MemorystoreDocumentSaver` 类，你需要提供两样东西：

1. `client` - 一个 `redis.Redis` 客户端对象。
1. `key_prefix` - 用于存储在 Redis 中的文档的键前缀。

文档将被存储到使用指定 `key_prefix` 前缀随机生成的键中。或者，你可以通过在 `add_documents` 方法中指定 `ids` 来设置键的后缀。

```python
import redis
from langchain_core.documents import Document
from langchain_google_memorystore_redis import MemorystoreDocumentSaver

test_docs = [
    Document(
        page_content="Apple Granny Smith 150 0.99 1",
        metadata={"fruit_id": 1},
    ),
    Document(
        page_content="Banana Cavendish 200 0.59 0",
        metadata={"fruit_id": 2},
    ),
    Document(
        page_content="Orange Navel 80 1.29 1",
        metadata={"fruit_id": 3},
    ),
]
doc_ids = [f"{i}" for i in range(len(test_docs))]

redis_client = redis.from_url(ENDPOINT)
saver = MemorystoreDocumentSaver(
    client=redis_client,
    key_prefix=KEY_PREFIX,
    content_field="page_content",
)
saver.add_documents(test_docs, ids=doc_ids)
```

### 加载文档

初始化一个加载器，加载 Memorystore for Redis 实例中存储的所有文档，并带有特定前缀。

使用 `MemorystoreDocumentLoader.load()` 或 `MemorystoreDocumentLoader.lazy_load()` 加载 LangChain 文档。`lazy_load` 返回一个生成器，仅在迭代时查询数据库。初始化 `MemorystoreDocumentLoader` 类需要提供：

1. `client` - 一个 `redis.Redis` 客户端对象。
1. `key_prefix` - 用于在 Redis 中存储文档的键的前缀。

```python
import redis
from langchain_google_memorystore_redis import MemorystoreDocumentLoader

redis_client = redis.from_url(ENDPOINT)
loader = MemorystoreDocumentLoader(
    client=redis_client,
    key_prefix=KEY_PREFIX,
    content_fields=set(["page_content"]),
)
for doc in loader.lazy_load():
    print("Loaded documents:", doc)
```

### 删除文档

使用 `MemorystoreDocumentSaver.delete()` 删除 Memorystore for Redis 实例中具有指定前缀的所有键。如果知道，还可以指定键的后缀。

```python
docs = loader.load()
print("Documents before delete:", docs)

saver.delete(ids=[0])
print("Documents after delete:", loader.load())

saver.delete()
print("Documents after delete all:", loader.load())
```

## 高级用法

### 自定义文档页面内容与元数据

当使用多个内容字段（超过1个）初始化加载器时，加载的文档的 `page_content` 将包含一个 JSON 编码的字符串，其顶级字段等于 `content_fields` 中指定的字段。

如果指定了 `metadata_fields`，加载的文档的 `metadata` 字段将只包含等于指定 `metadata_fields` 的顶级字段。如果元数据字段的任何值存储为 JSON 编码的字符串，它将在加载到元数据字段之前被解码。

```python
loader = MemorystoreDocumentLoader(
    client=redis_client,
    key_prefix=KEY_PREFIX,
    content_fields=set(["content_field_1", "content_field_2"]),
    metadata_fields=set(["title", "author"]),
)
```
