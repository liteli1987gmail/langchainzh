>[LocalAI](https://localai.io/) 是一个免费的、开源的 OpenAI 替代方案。
> `LocalAI` 充当一个 REST API 的直接替代品，与 OpenAI API 规范兼容，用于本地推理。
> 它允许您在本地或本地部署，使用消费级硬件运行 LLM、生成图像、音频（不仅如此），
> 支持多种模型系列和架构。

<Info>
`langchain-localai` 是 LocalAI 的第三方集成包。它提供了一种简单的方式在 LangChain 中使用 LocalAI 服务。
源代码可在 [GitHub](https://github.com/mkhludnev/langchain-localai) 获取
</Info>

## 安装和设置

<CodeGroup>

</CodeGroup>
```bash pip
pip install langchain-localai
```

```bash uv
uv add langchain-localai
```
</CodeGroup>
