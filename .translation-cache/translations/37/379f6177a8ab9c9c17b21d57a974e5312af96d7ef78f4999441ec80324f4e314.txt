>[LocalAI](https://localai.io/) 是免费的、开源的 OpenAI 替代方案。
> `LocalAI` 充当与 OpenAI API 兼容的即插即用 REST API
> 规范，用于本地推理。它允许您运行 LLM、生成图像，
> 音频（不仅限于此），在本地或企业内部使用消费级硬件运行，
> 支持多种模型系列和架构。

<Info>
`langchain-localai` 是 LocalAI 的第三方集成包。它提供了一种在 LangChain 中使用 LocalAI 服务的简单方法。
源代码可在以下位置获取 [GitHub](https://github.com/mkhludnev/langchain-localai)
</Info>

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-localai
```

```bash uv
uv add langchain-localai
```
</CodeGroup>
