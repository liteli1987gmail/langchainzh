## 安装

要访问 PyMuPDF4LLM 文档加载器，你需要安装 `langchain-pymupdf4llm` 集成包。

### 凭据

使用 PyMuPDF4LLMLoader 无需任何凭据。

要启用模型调用的自动追踪，请设置你的 [LangSmith](/langsmith/observability) API 密钥：

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

安装 **langchain-community** 和 **langchain-pymupdf4llm**。

```python
pip install -qU langchain-community langchain-pymupdf4llm
```

## 初始化

现在我们可以实例化模型对象并加载文档：

```python
from langchain_pymupdf4llm import PyMuPDF4LLMLoader

file_path = "./example_data/layout-parser-paper.pdf"
loader = PyMuPDF4LLMLoader(file_path)
```

## 分割模式及自定义页面分隔符

加载 PDF 文件时，可以采用两种不同的方式分割：

- 按页面
- 作为单一文本流

默认情况下，PyMuPDF4LLMLoader 将按页面分割 PDF。

### 按页面提取 PDF。每个页面作为 LangChain 文档对象被提取

```python
loader = PyMuPDF4LLMLoader(
    "./example_data/layout-parser-paper.pdf",
    mode="page",
)
docs = loader.load()

print(len(docs))
pprint.pp(docs[0].metadata)
```

从逻辑上讲，在此模式下，`page`（page_number）元数据会消失。以下是如何在文本流中清晰识别页面结束位置的方法：

### 添加自定义 *pages_delimiter* 来识别 *single* 模式下页面的结束位置

```python
loader = PyMuPDF4LLMLoader(
    "./example_data/layout-parser-paper.pdf",
    mode="single",
    pages_delimiter="\n-------THIS IS A CUSTOM END OF PAGE-------\n\n",
)
docs = loader.load()

part = docs[0].page_content[10663:11317]
print(part)
display(Markdown(part))
```

默认的 `pages_delimiter` 为 \n-----\n\n。
可以简单地设置为 \n，或 \f 来清楚地表示页面变化，或 \<!-- PAGE BREAK --> 以便在 Markdown 查看器中无缝注入而不产生视觉效果。

# 从 PDF 中提取图像

您可以从 PDF 中提取图像（以文本形式），有三种不同的解决方案可供选择：

- rapidOCR（轻量级光学字符识别工具）
- Tesseract（高精度 OCR 工具）
- 多模态语言模型

结果会插入到页面文本的末尾。

### 使用 rapidOCR 从 PDF 中提取图像

```python
pip install -qU rapidocr-onnxruntime pillow
```

<LangchainCommunityUnmaintained />

```python
from langchain_community.document_loaders.parsers import RapidOCRBlobParser

loader = PyMuPDF4LLMLoader(
    "./example_data/layout-parser-paper.pdf",
    mode="page",
    extract_images=True,
    images_parser=RapidOCRBlobParser(),
)
docs = loader.load()

part = docs[5].page_content[1863:]
print(part)
display(Markdown(part))
```

注意，RapidOCR 设计用于处理中文和英文，而非其他语言。

# 从 PDF 中提取表格

使用 PyMUPDF4LLM 可以从您的 PDF 中以 *markdown* 格式提取表格：

```python
loader = PyMuPDF4LLMLoader(
    "./example_data/layout-parser-paper.pdf",
    mode="page",
    # "lines_strict" is the default strategy and
    # is the most accurate for tables with column and row lines,
    # but may not work well with all documents.
    # "lines" is a less strict strategy that may work better with
    # some documents.
    # "text" is the least strict strategy and may work better
    # with documents that do not have tables with lines.
    table_strategy="lines",
)
docs = loader.load()

part = docs[4].page_content[3210:]
print(part)
display(Markdown(part))
```

## 使用文件

许多文档加载器涉及文件解析。这类加载器之间的差异通常源于文件的解析方式，而不是加载方式。例如，你可以使用 `open` 来读取 PDF 或 markdown 文件的二进制内容，但需要不同的解析逻辑将这些二进制数据转换为文本。

因此，将解析逻辑与加载逻辑解耦会很有帮助，这样可以使给定的解析器更容易复用，无论数据是如何加载的。
你可以使用这种策略来分析不同的文件，同时使用相同的解析参数。

<LangchainCommunityUnmaintained />

```python
from langchain_community.document_loaders import FileSystemBlobLoader
from langchain_community.document_loaders.generic import GenericLoader
from langchain_pymupdf4llm import PyMuPDF4LLMParser

loader = GenericLoader(
    blob_loader=FileSystemBlobLoader(
        path="./example_data/",
        glob="*.pdf",
    ),
    blob_parser=PyMuPDF4LLMParser(),
)
docs = loader.load()

part = docs[0].page_content[:562]
print(part)
display(Markdown(part))
```
