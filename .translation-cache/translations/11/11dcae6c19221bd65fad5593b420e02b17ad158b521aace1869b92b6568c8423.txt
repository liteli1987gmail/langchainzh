## 设置

要访问 Featherless AI 模型，您需要创建一个 Featherless AI 账户，获取 API 密钥，并安装 `langchain-featherless-ai` 集成包。

### 凭据

请访问 [featherless.ai/](https://featherless.ai/) 注册 FeatherlessAI 并生成 API 密钥。完成此操作后，设置 FEATHERLESSAI_API_KEY 环境变量：

```python
import getpass
import os

if not os.getenv("FEATHERLESSAI_API_KEY"):
    os.environ["FEATHERLESSAI_API_KEY"] = getpass.getpass(
        "Enter your FeatherlessAI API key: "
    )
```

如果您想获取模型调用的自动跟踪，您还可以通过取消下面的注释来设置您的 [LangSmith](/langsmith/observability) API 密钥：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
```

### 安装

LangChain FeatherlessAi 集成位于 `langchain-featherless-ai` 包中：

```python
pip install -qU langchain-featherless-ai
```

## 实例化

现在我们可以实例化模型对象并生成聊天补全：

```python
from langchain_featherless_ai import ChatFeatherlessAi

llm = ChatFeatherlessAi(
    model="featherless-ai/Qwerky-72B",
    temperature=0.9,
    max_tokens=None,
    timeout=None,
)
```
