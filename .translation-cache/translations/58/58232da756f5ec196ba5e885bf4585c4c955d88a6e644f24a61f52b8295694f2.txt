>[MongoDB](https://www.mongodb.com/) 是一种 NoSQL、面向文档的
> 数据库，支持带有动态模式的 JSON 类文档。

**NOTE:**
- 查看其他 `MongoDB` 集成，请访问 [MongoDB Atlas 页面](/oss/integrations/providers/mongodb_atlas).

## 安装和设置

安装 Python 包：

<CodeGroup>
```bash pip
pip install langchain-mongodb
```

```bash uv
uv add langchain-mongodb
```
</CodeGroup>
