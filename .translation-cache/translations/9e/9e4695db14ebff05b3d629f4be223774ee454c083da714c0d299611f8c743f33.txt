所有与 WRITER 相关的功能


>此页面介绍如何使用 [WRITER](https://writer.com/) LangChain 中的生态系统。更多信息请参阅 Writer [文档](https://dev.writer.com/home/introduction).
>[Palmyra](https://writer.com/blog/palmyra/) 是由 开发的大型语言模型 (LLM) `WRITER`.
>
>这 [Writer API](https://dev.writer.com/api-guides/introduction) 由多种具有不同能力和价格点的 Palmyra 子模型提供支持。

## 安装和设置

使用 安装集成包
<CodeGroup>
```bash pip
pip install langchain-writer
```

```bash uv
uv add langchain-writer
```
</CodeGroup>

获取 WRITER API 密钥并将其设置为环境变量 (`WRITER_API_KEY`)

## 聊天模型

```python
from langchain_writer import ChatWriter
```
参见 [详情](/oss/integrations/chat/writer).


## PDF 解析器

```python
from langchain_writer.pdf_parser import PDFParser
```
<Warning>
**弃用通知**：PDF 解析工具已弃用，将于 **2025 年 12 月 22 日**.

**迁移路径**：我们计划为聊天补全引入一个预构建的 PDF 解析工具，提供类似的功能。该工具将与其他预构建工具类似地工作。我们将在此替代方案可用时提供更多详细信息。
</Warning>

参见 [详情](/oss/integrations/document_loaders/parsers/writer_pdf_parser).
