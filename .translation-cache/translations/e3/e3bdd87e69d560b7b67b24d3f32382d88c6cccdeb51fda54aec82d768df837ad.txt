[You.com](https://you.com) 是一个 AI 生产力平台，提供专为 LLM 应用设计的实时网络搜索和内容提取 API。

## 安装和设置

<CodeGroup>

</CodeGroup>
    ```bash pip
    pip install langchain-youdotcom
    ```

    ```bash uv
    uv add langchain-youdotcom
    ```
</CodeGroup>

设置您的 You.com API 密钥。在 [you.com/platform](https://you.com/platform) 获取您的 API 密钥。

```python
import os

os.environ["YDC_API_KEY"] = "your-api-key"
```

## 检索器

查看[使用示例](/oss/integrations/retrievers/you-retriever)。

```python
from langchain_youdotcom import YouRetriever
```

## 工具

查看[使用示例](/oss/integrations/tools/you)。

from langchain_youdotcom import YouSearchTool, YouContentsTool
