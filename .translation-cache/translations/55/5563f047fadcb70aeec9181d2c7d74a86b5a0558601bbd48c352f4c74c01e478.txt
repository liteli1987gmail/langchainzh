[Vectara](https://vectara.com/) 是一个 RAG 平台，通过 API 暴露索引、检索和相关服务。
Vectara 无服务器 RAG 即服务通过 API 提供 RAG 的主要组件，包括：

1. 一种从文件（PDF、PPT、DOCX 等）中提取文本的方法
2. 基于 ML 的文本分块
3. The [Boomerang](https://vectara.com/how-boomerang-takes-retrieval-augmented-generation-to-the-next-level-via-grounded-generation/) 嵌入模型。
4. 其自带的内部向量数据库，用于存储文本块和嵌入向量。
5. 一个查询服务，自动将查询编码为嵌入，并检索最相关的文本片段，包括支持 [混合搜索](https://docs.vectara.com/docs/api-reference/search-apis/lexical-matching) 以及多种重新排序选项，例如 [多语言相关性重排序器](https://www.vectara.com/blog/deep-dive-into-vectara-multilingual-reranker-v1-state-of-the-art-reranker-across-100-languages), [MMR](https://vectara.com/get-diverse-results-and-comprehensive-summaries-with-vectaras-mmr-reranker/), [UDF 重排序器](https://www.vectara.com/blog/rag-with-user-defined-functions-based-reranking).
6. 一个用于创建的 LLM [生成式摘要](https://docs.vectara.com/docs/learn/grounded-generation/grounded-generation-overview)，基于检索到的文档（上下文），包括引用。

更多信息：

- [文档](https://docs.vectara.com/docs/)
- [API 演示平台](https://docs.vectara.com/docs/rest-api/)
- [快速入门](https://docs.vectara.com/docs/quickstart)

本笔记本展示了如何使用基本检索功能，当您仅将 Vectara 用作向量存储（不含摘要）时，包括： `similarity_search` 和 `similarity_search_with_score`，以及 LangChain `as_retriever` functionality.

# 开始使用

要开始使用，请按照以下步骤操作：

1. 如果您还没有账户， [注册](https://www.vectara.com/integrations/langchain) 免费 Vectara 试用版。
2. 在您的账户中，您可以创建一个或多个语料库。每个语料库代表一个存储区域，在从输入文档摄取时存储文本数据。要创建语料库，请使用 **"创建语料库"** 按钮。然后您为语料库提供名称和描述。您可以选择定义过滤属性并应用一些高级选项。如果您点击已创建的语料库，您可以在顶部看到其名称和语料库 ID。
3. 接下来您需要创建 API 密钥来访问语料库。点击语料库视图中的 **"访问控制"** 标签，然后点击 **"创建 API 密钥"** 按钮。为您的密钥命名，并选择您想要的是仅查询还是查询+索引密钥。点击"创建"，您现在就有了一个有效的 API 密钥。请妥善保管此密钥。

要将 LangChain 与 Vectara 结合使用，您需要提供这两个值： `corpus_key` 和 `api_key`.
您可以通过两种方式向 LangChain 提供 `VECTARA_API_KEY` ：

1. 在您的环境中包含这两个变量： `VECTARA_API_KEY`.

   例如，您可以使用 os.environ 和 getpass 设置这些变量，如下所示：

```python
import os
import getpass

os.environ["VECTARA_API_KEY"] = getpass.getpass("Vectara API Key:")
```

2. 将它们添加到 `Vectara` vectorstore 构造函数：

```python
vectara = Vectara(
    vectara_api_key=vectara_api_key
)
```

在本笔记本中，我们假设它们已在环境中提供。

```python
import os

os.environ["VECTARA_API_KEY"] = "<VECTARA_API_KEY>"
os.environ["VECTARA_CORPUS_KEY"] = "VECTARA_CORPUS_KEY"

from langchain_vectara import Vectara
from langchain_vectara.vectorstores import (
    ChainReranker,
    CorpusConfig,
    CustomerSpecificReranker,
    File,
    GenerationConfig,
    MmrReranker,
    SearchConfig,
    VectaraQueryConfig,
)

vectara = Vectara(vectara_api_key=os.getenv("VECTARA_API_KEY"))
```

首先，我们将国情咨文文本加载到 Vectara 中。

请注意，我们使用 add_文件接口，该接口不需要任何本地处理或分块——Vectara 接收文件内容并执行所有必要的预处理、分块和嵌入操作，将其存入知识库。

本例中使用的是 .txt 文件，但同样适用于许多其他 [文件类型](https://docs.vectara.com/docs/api-reference/indexing-apis/file-upload/file-upload-filetypes).

```python
corpus_key = os.getenv("VECTARA_CORPUS_KEY")
file_obj = File(
    file_path="../document_loaders/example_data/state_of_the_union.txt",
    metadata={"source": "text_file"},
)
vectara.add_files([file_obj], corpus_key)
```

## Vectara RAG（检索增强生成）

现在，我们创建一个 `VectaraQueryConfig` 对象来控制检索和摘要选项：
- 我们启用了摘要功能，指定希望 LLM 选择前 7 个匹配块并用英语回复

使用此配置，我们创建一个 LangChain `Runnable` 对象来封装完整的 Vectara RAG 管道，使用 `as_rag` method:

```python
generation_config = GenerationConfig(
    max_used_search_results=7,
    response_language="eng",
    generation_preset_name="vectara-summary-ext-24-05-med-omni",
    enable_factual_consistency_score=True,
)
search_config = SearchConfig(
    corpora=[CorpusConfig(corpus_key=corpus_key)],
    limit=25,
    reranker=ChainReranker(
        rerankers=[
            CustomerSpecificReranker(reranker_id="rnk_272725719", limit=100),
            MmrReranker(diversity_bias=0.2, limit=100),
        ]
    ),
)

config = VectaraQueryConfig(
    search=search_config,
    generation=generation_config,
)

query_str = "what did Biden say?"

rag = vectara.as_rag(config)
rag.invoke(query_str)["answer"]
```

He emphasized the importance of keeping schools open and noted that with a high vaccination rate and reduced hospitalizations, most Americans can safely return to normal activities without masks [1]. ```

我们也可以像这样使用流式接口：

```python
output = {}
curr_key = None
for chunk in rag.stream(query_str):
    for key in chunk:
        if key not in output:
            output[key] = chunk[key]
        else:
            output[key] += chunk[key]
        if key == "answer":
            print(chunk[key], end="", flush=True)
        curr_key = key
```

## 幻觉检测和事实一致性评分

Vectara 创建了 [HHEM](https://huggingface.co/vectara/hallucination_evaluation_model) ——一个可用于评估 RAG 响应事实一致性的开源模型。

作为 Vectara RAG 的一部分，API 提供了改进版开源 HHEM 的"事实一致性评分"（或 FCS），并会自动包含在 RAG 管道的输出中

```python
resp = rag.invoke(query_str)
print(resp["answer"])
print(f"Vectara FCS = {resp['fcs']}")
```

```text
President Biden discussed several key topics in his recent statements. He emphasized the importance of keeping schools open and noted that with a high vaccination rate and reduced hospitalizations, most Americans can safely return to normal activities without masks [1]. Additionally, he reaffirmed the need to protect women's rights, particularly the right to choose as affirmed in Roe v. Wade [5].
Vectara FCS = 0.61621094
```

## 使用 Vectara 进行高级 LangChain 查询预处理

Vectara 的"RAG 即服务"在创建问答或聊天机器人链时承担了大部分繁重工作。与 LangChain 的集成提供了使用额外功能（如查询预处理）的选项，例如 `SelfQueryRetriever` or `MultiQueryRetriever`。让我们看一个使用 [MultiQueryRetriever](https://python.langchain.com/docs/modules/data_connection/retrievers/MultiQueryRetriever).

由于 MQR 使用 LLM，我们必须进行设置——在这里我们选择 @[`ChatOpenAI`] :

```python
from langchain_classic.retrievers.multi_query import MultiQueryRetriever
from langchain_openai.chat_models import ChatOpenAI

llm = ChatOpenAI(temperature=0)
mqr = MultiQueryRetriever.from_llm(retriever=retriever, llm=llm)


def get_summary(documents):
    return documents[-1].page_content


(mqr | get_summary).invoke(query_str)
```

```text
'The remarks made by Biden include his emphasis on the importance of the Bipartisan Infrastructure Law, which he describes as the most significant investment to rebuild America in history. He highlights the bipartisan effort involved in passing this law and expresses gratitude to members of both parties for their collaboration. Biden also mentions the transition from "infrastructure weeks" to an "infrastructure decade" [1]. Additionally, he shares a personal story about his father having to leave their home in Scranton, Pennsylvania, to find work, which influenced his decision to fight for the American Rescue Plan to help those in need [2].'
```

```python

```
