>[ScrapeGraph AI](https://scrapegraphai.com) 是一个提供 AI 驱动网页抓取功能的服务。
>它提供使用自然语言提示提取结构化数据、将网页转换为 markdown 以及处理本地 HTML 内容的工具。

## 安装和设置

安装所需的包：

<CodeGroup>

</fragment>
```bash pip
pip install langchain-scrapegraph
```

```bash uv
uv add langchain-scrapegraph
```
</CodeGroup>

设置您的 API 密钥：

```bash
export SGAI_API_KEY="your-scrapegraph-api-key"
```

## 工具

查看[使用示例](/oss/integrations/tools/scrapegraph)。

有以下四个工具可用：

```python
from langchain_scrapegraph.tools import (
    SmartScraperTool,    # Extract structured data from websites
    SmartCrawlerTool,    # Extract data from multiple pages with crawling
    MarkdownifyTool,     # Convert webpages to markdown
    AgenticScraperTool,  # Extract specifying steps
    GetCreditsTool,      # Check remaining API credits
)
```

每个工具都有其特定的用途：

- `SmartScraperTool`：从网站中提取结构化数据，需要提供 URL、提示词和可选的输出模式
- `SmartCrawlerTool`：从多个页面提取数据，支持高级爬取选项，如深度控制、页面限制和域名限制
- `MarkdownifyTool`：将任何网页转换为干净的 markdown 格式
- `AgenticScraperTool`：通过指定步骤进行提取
- `GetCreditsTool`：查看您剩余的 ScrapeGraph AI 积分
