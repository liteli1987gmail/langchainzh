> `hkunlp/instructor-*` 系列引入了指令微调的句子嵌入。它们已被现代指令感知模型广泛取代，但原始模型仍在 Hugging Face 上可用，可通过遗留的 `HuggingFaceInstructEmbeddings` 类使用。

对于新项目，建议使用 [`langchain-huggingface`](/oss/integrations/embeddings/sentence_transformers) 中的 `HuggingFaceEmbeddings`，并配合当前指令感知模型，例如 `intfloat/e5-large-v2`、`Qwen/Qwen3-Embedding-0.6B` 或 `BAAI/bge-m3`。通过 `encode_kwargs` 和 `query_encode_kwargs` 传递查询和文档提示：

```python
from langchain_huggingface import HuggingFaceEmbeddings

embeddings = HuggingFaceEmbeddings(
    model_name="intfloat/e5-large-v2",
    encode_kwargs={"prompt": "passage: "},
    query_encode_kwargs={"prompt": "query: "},
)
```

## 旧版讲师使用

```shell
pip install -qU langchain-community InstructorEmbedding sentence-transformers
```

```python
from langchain_community.embeddings import HuggingFaceInstructEmbeddings

embeddings = HuggingFaceInstructEmbeddings(
    query_instruction="Represent the query for retrieval: "
)

query_result = embeddings.embed_query("This is a test document.")
```
