## 基本用法

### 设置 Cloud SQL 数据库值

在 [Cloud SQL Instances 页面](https://console.cloud.google.com/sql/instances) 中找到你的数据库变量。

```python
# @title Set Your Values Here { display-mode: "form" }
REGION = "us-central1"  # @param {type: "string"}
INSTANCE = "my-primary"  # @param {type: "string"}
DATABASE = "my-database"  # @param {type: "string"}
TABLE_NAME = "vector_store"  # @param {type: "string"}
```

### Cloud SQL 引擎

将 PostgreSQL 配置为文档加载器的必要条件之一是 `PostgresEngine` 对象。`PostgresEngine` 用于配置与 Cloud SQL for PostgreSQL 数据库的连接池，使应用程序能够成功建立连接并遵循行业最佳实践。

使用 `PostgresEngine.from_instance()` 创建 `PostgresEngine` 时，只需提供 4 个参数：

1. `project_id`：Cloud SQL 实例所在 Google Cloud Project 的项目 ID。
1. `region`：Cloud SQL 实例所在的区域。
1. `instance`：Cloud SQL 实例的名称。
1. `database`：Cloud SQL 实例上要连接的数据库名称。

默认情况下，将使用 [IAM 数据库身份验证](https://cloud.google.com/sql/docs/postgres/iam-authentication) 作为数据库身份验证方法。此库使用从环境中获取的[应用程序默认凭证 (ADC)](https://cloud.google.com/docs/authentication/application-default-credentials) 所关联的 IAM 主体。

也可以选择使用[内置数据库身份验证](https://cloud.google.com/sql/docs/postgres/users)（用户名和密码）来访问 Cloud SQL 数据库。只需向 `PostgresEngine.from_instance()` 提供可选的 `user` 和 `password` 参数：

* `user`：用于内置数据库身份验证和登录的数据库用户
* `password`：用于内置数据库身份验证和登录的数据库密码

**注意**：本教程演示的是异步接口。所有异步方法都有对应的同步方法。

```python
from langchain_google_cloud_sql_pg import PostgresEngine

engine = await PostgresEngine.afrom_instance(
    project_id=PROJECT_ID,
    region=REGION,
    instance=INSTANCE,
    database=DATABASE,
)
```

### 创建 PostgresLoader

```python
from langchain_google_cloud_sql_pg import PostgresLoader

# Creating a basic PostgreSQL object
loader = await PostgresLoader.create(engine, table_name=TABLE_NAME)
```

### 通过默认表加载文档

加载器从表中返回一个文档列表，其中第一列作为 page_content，其他所有列作为 metadata。默认表的第一列是 page_content，第二列是 metadata（JSON）。每一行成为一个文档。请注意，如果希望文档具有 id，则需要自行添加。

```python
from langchain_google_cloud_sql_pg import PostgresLoader

# Creating a basic PostgresLoader object
loader = await PostgresLoader.create(engine, table_name=TABLE_NAME)

docs = await loader.aload()
print(docs)
```

### 通过自定义表/元数据或自定义页面内容列加载文档

```python
loader = await PostgresLoader.create(
    engine,
    table_name=TABLE_NAME,
    content_columns=["product_name"],  # Optional
    metadata_columns=["id"],  # Optional
)
docs = await loader.aload()
print(docs)
```

### 设置页面内容格式

加载器返回文档列表，每个文档对应一行，页面内容以指定的字符串格式呈现，即文本（空格分隔拼接）、JSON、YAML、CSV 等。JSON 和 YAML 格式包含标题行，而文本和 CSV 不包含字段标题。

```python
loader = await PostgresLoader.create(
    engine,
    table_name="products",
    content_columns=["product_name", "description"],
    format="YAML",
)
docs = await loader.aload()
print(docs)
```
