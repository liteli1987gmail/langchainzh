在 [LangSmith UI](https://smith.langchain.com)，请确保您的 API 密钥已设置为 [工作区密钥](/langsmith/set-up-hierarchy#configure-workspace-settings).

1. 导航到 <Icon icon="settings" /> **设置** ，然后转到 **密钥** tab.
1. 选择 **添加密钥** ，然后输入密钥环境变量（例如，`OPENAI_API_KEY` or `ANTHROPIC_API_KEY`），并将您的 API 密钥作为 **值**.
1. 选择 **保存密钥**.

<Note> 在 LangSmith UI 中添加工作区密钥时，请确保密钥名称与模型提供商预期的环境变量名称相匹配。</Note>

<Note>如果您的提供商使用 OAuth2 进行身份验证 `client_credentials`，请改为在模型配置上配置凭据。在这种情况下，不需要工作区密钥。请参阅 [OAuth 客户端凭据](/langsmith/model-configurations#oauth-client-credentials).</Note>
