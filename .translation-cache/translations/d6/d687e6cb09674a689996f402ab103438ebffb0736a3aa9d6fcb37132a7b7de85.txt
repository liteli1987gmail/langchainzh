[Tavily](https://tavily.com/) 是一个专为 AI 代理（LLM）构建的搜索引擎，提供实时、准确和真实的结果，速度极快。Tavily 提供了一个[地图](https://docs.tavily.com/documentation/api-reference/endpoint/map)端点，可以遍历网站并返回发现的 URL 列表，而无需提取页面内容，这非常适合了解网站结构或在大型网站上定位特定页面。

## 设置

该集成位于 `@langchain/tavily` 包中，您可以按如下方式安装：

<CodeGroup>
```bash npm
npm install @langchain/tavily @langchain/core
```
```bash yarn
yarn add @langchain/tavily @langchain/core
```
```bash pnpm
pnpm add @langchain/tavily @langchain/core
```
</CodeGroup>

### 凭证

[设置 Tavily API 密钥](https://app.tavily.com) 并将其设置为名为 `TAVILY_API_KEY` 的环境变量。

```typescript
process.env.TAVILY_API_KEY = "YOUR_API_KEY"
```

这也有帮助（但非必需）设置 [LangSmith](https://smith.langchain.com) 以获得一流的可观测性：

```typescript
process.env.LANGSMITH_TRACING="true"
process.env.LANGSMITH_API_KEY="your-api-key"
```

## 实例化

该工具在实例化期间接受以下参数：

- `maxDepth`（可选，数字）：从起始 URL 开始的最大跳数。默认值为 `3`。
- `maxBreadth`（可选，数字）：每层返回的最大 URL 数量。默认值为 `50`。
- `limit`（可选，数字）：返回的最大 URL 总数。默认值为 `100`。
- `instructions`（可选，字符串）：引导地图遍历的自然语言指令。
- `selectPaths`（可选，字符串[]）：仅包含包含这些路径正则表达式的 URL。
- `selectDomains`（可选，字符串[]）：仅包含来自这些域名正则表达式的 URL。
- `excludePaths`（可选，字符串[]）：跳过包含这些路径正则表达式的 URL。
- `excludeDomains`（可选，字符串[]）：跳过来自这些域名正则表达式的 URL。
- `allowExternal`（可选，布尔值）：允许地图跟随外部链接。

有关可用参数的完整概述，请参阅 [Tavily Map API 文档](https://docs.tavily.com/documentation/api-reference/endpoint/map)。

```typescript
import { TavilyMap } from "@langchain/tavily";

const tool = new TavilyMap({
  maxDepth: 3,
  maxBreadth: 50,
  // limit: 100,
  // allowExternal: false,
});
```

## 调用

### [直接调用并传递参数](/oss/langchain/tools)

Tavily 地图工具在调用期间接受以下参数：

- `url`（必填）：用作映射起点的基础 URL。
- 以下参数也可以在调用期间设置：`instructions`、`selectPaths`、`selectDomains`、`excludePaths`、`excludeDomains`、`allowExternal`。

注意：可选参数可供智能体动态设置。如果您在实例化时设置了某个参数，然后使用不同的值调用该工具，该工具将使用您在调用期间传递的值。

```typescript
await tool.invoke({ url: "https://docs.tavily.com" });
```

```json
{
  "base_url": "https://docs.tavily.com",
  "results": [
    "https://docs.tavily.com/",
    "https://docs.tavily.com/changelog",
    "https://docs.tavily.com/welcome",
    "https://docs.tavily.com/documentation/mcp",
    "https://docs.tavily.com/documentation/about",
  ],
  "response_time": 0.07
}
```

### [使用 ToolCall 调用](/oss/langchain/tools)

我们也可以使用模型生成的 `ToolCall` 调用工具，在这种情况下将返回一个 @[`ToolMessage`]：

```typescript
// This is usually generated by a model, but we'll create a tool call directly for demo purposes.
const modelGeneratedToolCall = {
  args: {
    url: "https://docs.tavily.com",
    instructions: "Find API reference pages",
  },
  id: "1",
  name: tool.name,
  type: "tool_call",
};

const toolMsg = await tool.invoke(modelGeneratedToolCall);

console.log(toolMsg.content.slice(0, 400));
```

## 在代理中使用

我们可以直接通过将其传递给 `createAgent` 来将 map 工具与 LangChain 代理一起使用。代理可以动态设置 `url`、`instructions` 以及路径/域过滤器来发现其需要的 URL。

```typescript
// @lc-docs-hide-cell
import { ChatOpenAI } from "@langchain/openai";

const llm = new ChatOpenAI({
  model: "gpt-5.5",
  temperature: 0,
});
```

```typescript
import { TavilyMap } from "@langchain/tavily";
import { createAgent } from "langchain";

const tavilyMapTool = new TavilyMap({
  maxDepth: 2,
  maxBreadth: 20,
  limit: 30,
});

const agent = createAgent({
  model: llm,
  tools: [tavilyMapTool],
});

const userInput = "Map https://docs.tavily.com and list URLs that look like API reference pages.";

const stream = await agent.streamEvents(
  { messages: [["human", userInput]] },
  { version: "v3" },
);

for await (const snapshot of stream.values) {
  const lastMsg = snapshot.messages[snapshot.messages.length - 1];
  if (lastMsg.tool_calls?.length) {
    console.dir(lastMsg.tool_calls, { depth: null });
  } else if (lastMsg.content) {
    console.log(lastMsg.content);
  }
}
```
