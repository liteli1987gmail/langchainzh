# LangChain

LangChain 是一个开源框架，具有预构建的代理架构和对任何模型或工具的集成。用不到 10 行代码构建代理和 LLM 驱动的应用程序，集成支持 OpenAI、Anthropic、Google 以及数百种其他服务。

## 何时使用

在以下情况下使用 LangChain：
- 使用 `create_agent()` 和预构建的代理循环**构建工具调用代理**
- 通过 `init_chat_model()` **切换模型提供商**，无需更改应用程序代码
- **添加结构化输出**，将 LLM 响应解析为类型化对象
- 使用 LangChain 的提供商包**集成任何模型或工具**
- **使用中间件**处理跨领域关注点，如速率限制和缓存

## 何时不使用

- 对于具有自定义控制流的复杂多步骤工作流，请改用 [LangGraph](https://docs.langchain.com/oss/langgraph/overview)
- 对于包含规划、子代理和上下文管理的电池式代理，请改用 [Deep Agents](https://docs.langchain.com/oss/deepagents/overview)
- LangChain 提供**核心构建块**；LangGraph 增加编排；Deep Agents 在此基础上添加高级功能

## 安装

```bash
# Python
pip install -U langchain

# JavaScript/TypeScript
npm install langchain @langchain/core
```

安装提供商集成：

```bash
# Python
pip install -U langchain-openai       # or langchain-anthropic, langchain-google-genai

# JavaScript/TypeScript
npm install @langchain/openai         # or @langchain/anthropic, @langchain/google-genai
```

## 快速参考

### 创建智能体

```python
from langchain.agents import create_agent

def get_weather(city: str) -> str:
    """Get weather for a given city."""
    return f"It's always sunny in {city}!"

agent = create_agent(
    model="openai:gpt-5.5",
    tools=[get_weather],
    system_prompt="You are a helpful assistant",
)

result = agent.invoke(
    {"messages": [{"role": "user", "content": "What is the weather in SF?"}]}
)
```

### 初始化聊天模型

```python
from langchain.chat_models import init_chat_model

# Switch providers by changing the string
model = init_chat_model("openai:gpt-5.5")
model = init_chat_model("anthropic:claude-opus-4-8")
model = init_chat_model("google_genai:gemini-3.5-flash")
```

### 定义工具

```python
from langchain.tools import tool

@tool
def search(query: str) -> str:
    """Search the web for information."""
    return "search results"
```

## 注意事项

1. **蛇形命名工具名称**——工具函数名称必须是有效的 Python 标识符。请使用 `get_weather`，而不是 `get-weather`。
2. **保留参数**——不要将工具参数命名为 `type`、`name` 或 `description`，因为这些名称与工具 schema 冲突。
3. **提供商包**——模型位于单独的包中（例如 `langchain-openai`）。基础 `langchain` 包不包含提供商。
4. **模型字符串格式**——使用 `init_chat_model()` 时采用 `"provider:model-name"` 格式（例如 `"openai:gpt-5.5"`）。
