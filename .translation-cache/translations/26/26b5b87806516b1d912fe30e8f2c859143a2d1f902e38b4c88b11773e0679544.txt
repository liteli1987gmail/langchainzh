## 设置

要访问 (模块_NAME) models, you'll need to create a/an (MODULE_名称) 账户，获取 API 密钥，并安装 `langchain-(MODULE_NAME)` 集成包。

### 凭据

```python Set API key icon="key"
import getpass
import os

if "(MODULE_NAME)_API_KEY" not in os.environ:
    os.environ["(MODULE_NAME)_API_KEY"] = getpass.getpass("Enter your (MODULE_NAME) API key: ")
```

要启用自动 <Tooltip tip="Log each step of a model's execution to debug and improve it">追踪</Tooltip> 模型调用，请设置您的 [LangSmith](/langsmith/observability) API 密钥：

```python Enable tracing icon="flask"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```
