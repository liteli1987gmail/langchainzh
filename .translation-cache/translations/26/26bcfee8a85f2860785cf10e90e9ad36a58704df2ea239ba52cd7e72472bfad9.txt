在这个快速开始中，您将使用预定义的**邮件助手**[模板](/langsmith/fleet/templates)来组织和管理您的收件箱。
- 选择不同的模板。

<Callout icon="message" color="#8B5CF6" iconType="regular">
您将通过聊天与智能体进行交互，就像给一位乐于助人的助手发短信一样。
</Callout>

## 开始之前

您需要：
- 一个 LangSmith 账户（[在此注册](https://smith.langchain.com/agents?skipOnboarding=true)）。
- 一个 Gmail 账户。
- 一个 Google 日历。
- 一个 OpenAI 或 Anthropic API 密钥（第 1 步将向您展示如何获取）。

## 1. 获取模型 API 密钥

您的智能体需要 API 密钥才能连接到 AI 模型。AI 模型使您的智能体能够理解和响应您的请求。

<Tabs>
  <Tab title="OpenAI (ChatGPT)">
    1. 前往 [platform.openai.com/api-keys](https://platform.openai.com/api-keys)。
    1. 点击**创建新的密钥**。
    1. 为其命名，例如 "Fleet"。
    1. 复制密钥（它以 `sk-` 开头）。
    1. 将其保存在安全的地方，您在第 2 步中会需要它。
  </Tab>

  <Tab title="Anthropic (Claude)">
    1. 前往 [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys)。
    2. 点击**创建密钥**。
    3. 为其命名，例如 "Fleet"。
    4. 复制密钥（它以 `sk-ant-` 开头）。
    5. 将其保存在安全的地方，您在第 2 步中会需要它。
  </Tab>
</Tabs>

<Warning>
这两个服务都会根据使用量收费。
</Warning>

## 2. 将您的 API 密钥添加到 LangSmith

现在您将把 API 密钥添加到 LangSmith，以便您的智能体可以使用它：

<Steps>
  <Step title="打开设置">
    1. 前往 [smith.langchain.com](https://smith.langchain.com)。
    2. 点击左下角的 <Icon icon="settings" /> **设置**图标。
  </Step>

  <Step title="进入密钥">
    点击顶部的**密钥**选项卡。
  </Step>

  <Step title="添加您的密钥">
    1. 点击**添加密钥**。
    2. 对于**密钥**，输入：
       - `OPENAI_API_KEY`（如果使用 OpenAI）
       - `ANTHROPIC_API_KEY`（如果使用 Anthropic）
    3. 对于**值**，粘贴您在第 1 步中复制的 API 密钥。
    4. 点击**保存密钥**。
  </Step>
</Steps>

<Callout type="success" icon="check" color="#10B981" iconType="regular">
您的智能体现在可以访问 AI 模型来理解和响应您的请求。接下来，您将创建您的智能体。
</Callout>

## 3. 创建您的智能体

<Steps>
  <Step title="导航到 Fleet">
    1. 在 [LangSmith UI](https://smith.langchain.com) 中，点击左侧导航顶部的 <Icon icon="pointer"/> **切换到 Fleet**。
  </Step>

  <Step title="选择一个模板">
    1. 在左侧导航中选择**模板**。
    1. 选择**邮件助手**模板。
    1. 点击**使用此模板**。

    <Tip>
    如果您不想从模板开始，您还有其他两个选择。从 **+ 新建智能体**页面：
    - **聊天**：使用聊天界面描述您的智能体，它将逐步帮助您创建。
    - **手动**：选择**改为手动创建**以在配置页面上没有任何预填充响应的情况下构建您的智能体。
    </Tip>
  </Step>

  <Step title="授权账户">
    您的智能体会要求您连接您的 Google 账户：

    1. 点击**连接**。
    2. 使用您的 Google 账户登录。
    3. 查看权限并点击**允许**。
    4. 您将被重定向回 LangSmith，在那里您的智能体将被创建。
  </Step>

</Steps>

<Info>
您的智能体仅在处理您分配的任务时访问您的账户。您可以随时在 Google 账户设置中撤销访问权限。
</Info>

## 4. 查看智能体模板

<Steps>

  <Step title="查看和自定义模板">

    此时，您可以查看邮件助手的模板说明。如有需要，您可以调整说明。

    如果您做了任何更改，请点击**保存更改**。
  </Step>

  <Step title="开始测试聊天">
    1. 在配置页面的右侧面板中，选择**测试聊天**选项卡。
    2. 在聊天界面中尝试邮件助手，例如：

        > _为收到的需要我审核的电子邮件添加"审核"标签_
  </Step>

  <Step title="智能体开始工作">
    您的智能体将开始工作，并为每个需要您批准的操作提供**继续**选项。

    <img

    src="/langsmith/images/agent-builder-response.png"

    />

    <img

    />

    3. 在测试智能体时，您可以编辑说明，或添加可能需要的工具。当您对结果满意时，点击**保存更改**。

  </Step>

</Steps>

## 编辑您的智能体

您可能想要更新智能体的说明或包含更多工具。您可以直接与智能体聊天来请求更新，或者您可以：

1. 从左侧导航的**我的智能体**中，选择您想要编辑的智能体。
1. 选择 <Icon icon="pencil"/> **编辑智能体**。

从智能体的编辑页面，您可以：

- 使用 **+ 添加工具** 添加工具，以连接更多应用和服务，如 Slack、GitHub 或 Linear。
- 使用 **+ 添加子智能体** 添加更多助手，将复杂任务分解为专业化的子任务。
- 请求对现有工具进行审核暂停。
- 修改现有工具。
- 探索可以自动启动智能体的渠道。
