在本快速入门中，您将使用预定义的 **电子邮件助手** [模板](/langsmith/fleet/templates) 来为您整理和管理收件箱。
- 选择不同的模板。

<Callout icon="message" color="#8B5CF6" iconType="regular">
您可以通过聊天与您的代理进行交互，就像给一位有用的助手发短信一样。
</Callout>

## 开始之前

您需要：
- 一个 LangSmith 账户（[在此注册](https://smith.langchain.com/agents?skipOnboarding=true)).
- 一个 Gmail 账户。
- 一个 Google 日历。
- 一个 OpenAI 或 Anthropic API 密钥（第 1 步将向您展示如何获取）。

## 1. 获取您的模型 API 密钥

您的代理需要一个 API 密钥来连接 AI 模型。AI 模型使您的代理能够理解并响应您的请求。

<Tabs>
  <Tab title="OpenAI (ChatGPT)">
    1. Go to [platform.openai.com/api-keys](https://platform.openai.com/api-keys).
    1. 点击 **创建新的密钥**.
    1. 给它起一个类似 "Fleet" 的名称。
    1. 复制密钥（它以 `sk-`).
    1. 请妥善保管，您在第 2 步会用到它。
  </Tab>

  <Tab title="Anthropic (Claude)">
    1. Go to [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys).
    2. 点击 **创建密钥**.
    3. 给它起一个类似 "Fleet" 的名称。
    4. 复制密钥（它以 `sk-ant-`).
    5. 请妥善保管，您在第 2 步会用到它。
  </Tab>
</Tabs>

<Warning>
两项服务均按使用量收费。
</Warning>

## 2. 将您的 API 密钥添加到 LangSmith

现在您需要将 API 密钥添加到 LangSmith，以便您的代理可以使用它：

<Steps>
  <Step title="Open Settings">
    1. Go to [smith.langchain.com](https://smith.langchain.com).
    2. 点击左下角的 <Icon icon="settings" /> **设置** 图标。
  </Step>

  <Step title="Go to Secrets">
    点击 **密钥** 顶部的标签页。
  </Step>

  <Step title="Add your key">
    1. 点击 **添加密钥**.
    2. 对于 **密钥**，输入：
       - `OPENAI_API_KEY` （如果使用 OpenAI）
       - `ANTHROPIC_API_KEY` （如果使用 Anthropic）
    3. 对于 **值**，粘贴您在第 1 步复制的 API 密钥。
    4. 点击 **保存密钥**.
  </Step>
</Steps>

<Callout type="success" icon="check" color="#10B981" iconType="regular">
您的代理现在可以访问 AI 模型来理解并响应您的请求。接下来，您将创建您的代理。
</Callout>

## 3. 创建您的代理

<Steps>
  <Step title="Navigate to Fleet">
    1. 在 [LangSmith UI](https://smith.langchain.com)中，点击 <Icon icon="pointer"/> **切换到 Fleet** 左侧导航顶部的
  </Step>

  <Step title="Choose a template">
    1. 选择 **模板** 在左侧导航栏中。
    1. 选择 **电子邮件助手** template.
    1. 点击 **使用此模板**.

    <Tip>
    如果您不想从模板开始，还有另外两个选项。从 **+ 新建智能体** page:
    - **聊天**：使用聊天界面描述您的智能体，它将逐步帮助您创建它。
    - **手动**：选择 **改为手动创建** 以在配置页面上构建没有任何预填充回复的智能体。
    </Tip>
  </Step>

  <Step title="Authorize accounts">
    您的智能体会要求您连接您的 Google 账户：

    1. 点击 **连接**.
    2. 使用您的 Google 账户登录。
    3. 查看权限并点击 **允许**.
    4. 您将被重定向回 LangSmith，在那里创建您的智能体。
  </Step>

</Steps>

<Info>
您的智能体仅在处理您分配的任务时访问您的账户。您可以随时在 Google 账户设置中撤销访问权限。
</Info>

## 4. 查看智能体模板

<Steps>

  <Step title="View and customize the template">

    此时，您可以查看电子邮件助手的模板说明。如有需要，可以对说明进行调整。

    如果进行了任何更改，请点击 **保存更改**.
  </Step>

  <Step title="Start a test chat">
    1. 在配置页面的右侧面板中，选择 **测试聊天** tab.
    2. 在聊天界面中试用电子邮件助手，例如：

        > _对我收到的需要我审核的电子邮件应用"审核"标签_
  </Step>

  <Step title="Agent starts working">
    您的智能体将开始工作并提供 **继续** 选项供需要您批准的每个步骤使用。

    3. 在测试智能体时，您可以对说明进行编辑，或添加可能需要的工具。点击 **保存更改** 当您对结果满意时。

  </Step>

</Steps>

## 编辑您的智能体

您可能希望更新智能体的说明或包含更多工具。您可以直接与智能体聊天来请求更新，或者您可以：

1. 从 **我的智能体** 在左侧导航栏中，选择要编辑的智能体。
1. 选择 <Icon icon="pencil"/> **编辑智能体**.

从智能体的编辑页面，您可以：

- 使用以下方式添加工具 **+ 添加工具** 连接更多应用和服务，如 Slack、GitHub 或 Linear。
- 使用 **+ 添加子代理** 将复杂任务拆分为专门的子任务。
- 请求暂停以审核现有工具。
- 修改现有工具。
- 探索可自动启动您的代理的渠道。
