LangChain 托管 [托管深度智能体](/langsmith/managed-deep-agents-overview)，因此您可以创建智能体并流式传输响应而无需设置基础设施。本快速入门展示了如何使用 CLI 或 SDK 创建智能体，然后使用 Python SDK、TypeScript SDK 或 React 运行它。 `useStream`.

有关完整的部署工作流程和所有后端选项，请参阅 [部署智能体](/langsmith/managed-deep-agents-deploy)。有关包配置和 API 详情，请参阅 [托管深度智能体 SDK](/langsmith/managed-deep-agents-sdk).

<Note>
托管深度智能体处于 **私人测试版**，可在 [LangSmith Cloud](/langsmith/cloud) 仅限美国区域。 [加入候补名单](https://www.langchain.com/langsmith-managed-deep-agents-waitlist) 以请求访问权限。
</Note>

## 前提条件

在开始之前，请确保您已具备：

- 托管深度智能体 [私人测试版访问权限](https://www.langchain.com/langsmith-managed-deep-agents-waitlist).
- A [LangSmith API 密钥](/langsmith/create-account-api-key) 适用于具有私人测试版访问权限的工作空间。
- 一个客户端。Python SDK（`managed-deepagents`）或 TypeScript SDK（`@langchain/managed-deepagents`) 涵盖了整个快速入门。 `deepagents-cli>=0.2.2` 可以创建代理但无法运行它，因此运行代理需要 SDK。

## 创建并运行代理

<Steps>
  <Step title="Install a client" id="install-a-client">

为您的运行时选择客户端：

<CodeGroup>

```bash CLI
uv tool install "deepagents-cli>=0.2.2"

# Or with pip:
pip install -U "deepagents-cli>=0.2.2"
```

```bash Python
uv add managed-deepagents

# Or with pip:
pip install managed-deepagents
```

```bash TypeScript
npm install @langchain/managed-deepagents

# For React streaming:
npm install @langchain/react
```

</CodeGroup>

要升级现有的 CLI 安装，请运行 `uv tool upgrade deepagents-cli` (or `pip install -U "deepagents-cli"` （如果您使用 pip 安装）。

  </Step>

  <Step title="Set your API key">

为拥有私人测试版访问权限的工作区设置 LangSmith API 密钥：

```bash
export LANGSMITH_API_KEY="<LANGSMITH_API_KEY>"
```

<Note>
如果请求返回 401 或 403，请确认您的 API 密钥属于拥有私人测试版访问权限的工作区。
</Note>

  </Step>

  <Step title="Create the agent">

创建托管深度代理。保存返回的 `agent_id` 用于运行步骤。

<CodeGroup>

```bash CLI
deepagents init research-assistant
cd research-assistant

# Edit AGENTS.md to define the agent behavior, then deploy.
deepagents deploy
```

```python Python
from managed_deepagents import Client

with Client() as client:
    agent = client.agents.create(
        name="research-assistant",
        description="Research assistant that can search the web and summarize sources.",
        model="openai:gpt-5.5",
        backend={"type": "state"},
        instructions=(
            "You are a careful research assistant. Search for sources, "
            "keep notes, and return concise answers with citations."
        ),
    )

agent_id = agent["id"]
print(f"Agent ID: {agent_id}")
```

```ts TypeScript
import { Client } from "@langchain/managed-deepagents";

const client = new Client({
  apiKey: process.env.LANGSMITH_API_KEY,
});

const agent = await client.agents.create({
  name: "research-assistant",
  description: "Research assistant that can search the web and summarize sources.",
  model: "openai:gpt-5.5",
  backend: { type: "state" },
  instructions:
    "You are a careful research assistant. Search for sources, keep notes, and return concise answers with citations.",
});

const agentId = agent.id;
console.log(`Agent ID: ${agentId}`);
```

</CodeGroup>

CLI 会创建 `agent.json`, `AGENTS.md`, `.gitignore`、一个空的 `tools.json`、一个示例技能和一个示例子代理，然后再部署。SDK 示例直接创建托管代理。

这些示例使用 `state` 后端，以便代理可以在无需沙箱特定配置的情况下运行。切换到 `sandbox` 后端，当代理需要 [LangSmith 沙箱](/langsmith/sandboxes) 用于代码执行、文件系统操作或长时间运行的任务。如需了解选项，请参阅 [选择后端](/langsmith/managed-deep-agents-deploy#choose-a-backend).

如果代理调用 MCP 工具， [连接工具](/langsmith/managed-deep-agents-mcp) 后再创建或部署代理。

  </Step>

  <Step title="Run the agent">

从代理流式传输响应：

<CodeGroup>

```python Python
from managed_deepagents import Client

agent_id = "<agent_id>"

with Client() as client:
    thread = client.threads.create(
        agent_id=agent_id,
        options={
            "test_run": False,
            "skip_memory_write_protection": False,
        },
    )

    for event in client.threads.stream(
        thread["id"],
        agent_id=agent_id,
        messages=[
            {
                "role": "user",
                "content": "Research recent approaches to agent memory and summarize the main trade-offs.",
            }
        ],
        stream_mode=["values", "updates", "messages-tuple"],
        stream_subgraphs=True,
    ):
        print(event.event, event.data)
```

```ts TypeScript
import { Client } from "@langchain/managed-deepagents";

const agentId = "<agent_id>";
const client = new Client({
  apiKey: process.env.LANGSMITH_API_KEY,
});

const thread = await client.threads.create({
  agent_id: agentId,
  options: {
    test_run: false,
    skip_memory_write_protection: false,
  },
});

const langGraphClient = client.getLangGraphClient({ agentId });
const stream = langGraphClient.runs.stream(thread.id, agentId, {
  input: {
    messages: [
      {
        role: "user",
        content:
          "Research recent approaches to agent memory and summarize the main trade-offs.",
      },
    ],
  },
  streamMode: ["values", "updates", "messages-tuple"],
  streamSubgraphs: true,
});

for await (const event of stream) {
  console.log(event.event, event.data);
}
```

```tsx useStream
import { Client } from "@langchain/managed-deepagents";
import { useStream } from "@langchain/react";

const agentId = "<agent_id>";

const managedDeepAgents = new Client({
  // In browser apps, prefer passing a custom fetch that calls your backend.
  apiKey: process.env.LANGSMITH_API_KEY,
});

const client = managedDeepAgents.getLangGraphClient({ agentId });

export function ManagedDeepAgentStream() {
  const stream = useStream({
    client,
    assistantId: agentId
  });

  return (
    <section>
      <button
        type="button"
        disabled={stream.isLoading}
        onClick={() => {
          void stream.submit({
            messages: [
              {
                role: "user",
                content:
                  "Research recent approaches to agent memory and summarize the main trade-offs.",
              },
            ],
          });
        }}
      >
        Run agent
      </button>

      {stream.messages.map((message, index) => (
        <p key={message.id ?? index}>{String(message.content)}</p>
      ))}
    </section>
  );
}
```

</CodeGroup>

Python 流会生成来自运行的服务器发送事件。TypeScript SDK 和 React `useStream` 示例使用 LangGraph 流式预测来处理消息、值和输出状态。

  </Step>
</Steps>

<Note>
If `deepagents init`, `deploy`, `agents`, or `mcp-servers` 缺失或行为异常，请确认已安装的版本是 `0.2.2` 或更高版本，配合 `deepagents --version`。较旧的 `deepagents` 可能会遮蔽您当前版本 `PATH`.
</Note>
