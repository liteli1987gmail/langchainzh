[Kinetica](https://www.kinetica.com/) 是一个实时数据库，专为实现
时间序列和空间数据的分析和生成式 AI 而构建。

## 聊天模型

Kinetica LLM 包装器使用 [Kinetica SqlAssist
LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) to transform natural language into
SQL 来简化数据检索过程。

参见 [Kinetica Language To SQL Chat Model](/oss/integrations/chat/kinetica) 获取使用方法。


```python
from langchain_kinetica import ChatKinetica
```

## 向量存储

Kinetca 向量存储包装器利用 Kinetica 对 [vector 的原生支持
similarity search](https://docs.kinetica.com/7.2/vector_search/).

参见 [Kinetica Vectorstore API](/oss/integrations/vectorstores/kinetica) 获取使用方法。


```python
from langchain_kinetica import KineticaVectorstore
```

## 文档加载器

Kinetica 文档加载器可用于加载 LangChain [文档](https://reference.langchain.com/python/langchain-core/documents/base/Document) 从
[Kinetica](https://www.kinetica.com/) database.

参见 [Kinetica Document Loader](/oss/integrations/document_loaders/kinetica) 获取使用方法。


```python
from langchain_kinetica import KineticaLoader
```

## 检索器

Kinetica 检索器可根据非结构化查询返回文档。

参见 [Kinetica VectorStore based Retriever](/oss/integrations/retrievers/kinetica) 获取使用方法。
