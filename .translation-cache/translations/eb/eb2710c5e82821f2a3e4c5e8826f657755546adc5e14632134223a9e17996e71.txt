[Kinetica](https://www.kinetica.com/) 是一个专为时序和空间数据上的分析和生成式 AI 打造的实时数据库。

## 聊天模型

Kinetica LLM 包装器使用 [Kinetica SqlAssist
LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) 将自然语言转换为 SQL，以简化数据检索过程。

有关使用方法，请参阅 [Kinetica Language To SQL Chat Model](/oss/integrations/chat/kinetica)。


```python
from langchain_kinetica import ChatKinetica
```

## 向量存储

Kinetca 向量存储包装器利用 Kinetca 对[向量相似性搜索](https://docs.kinetica.com/7.2/vector_search/)的原生支持。

有关用法，请参阅 [Kinetica 向量存储 API](/oss/integrations/vectorstores/kinetica)。


```python
from langchain_kinetica import KineticaVectorstore
```

## 文档加载器

Kinetica 文档加载器可用于从 [Kinetica](https://www.kinetica.com/) 数据库加载 LangChain [文档](https://reference.langchain.com/python/langchain-core/documents/base/Document)。

有关用法，请参阅 [Kinetica 文档加载器](/oss/integrations/document_loaders/kinetica)。


```python
from langchain_kinetica import KineticaLoader
```

## Retriever

Kinetica Retriever 可以根据非结构化查询返回文档。

有关使用说明，请参阅[基于 Kinetica VectorStore 的 Retriever](/oss/integrations/retrievers/kinetica)。
