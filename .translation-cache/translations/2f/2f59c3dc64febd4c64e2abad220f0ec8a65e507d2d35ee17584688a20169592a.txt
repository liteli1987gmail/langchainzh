## 概述

在本教程中，您将学习如何使用 LangChain [代理](/oss/langchain/agents) 构建一个能够回答 SQL 数据库相关问题的智能体。

从高层次来看，智能体将：

1. 从数据库中获取可用的表和模式
2. 确定哪些表与问题相关
3. 获取相关表的模式
4. 根据问题以及模式信息生成查询
5. 使用 LLM 双重检查查询中的常见错误
6. 执行查询并返回结果
7. 修正数据库引擎暴露的错误，直到查询成功
8. 根据结果制定响应

<Warning>
构建 SQL 数据库的问答系统需要执行模型生成的 SQL 查询。这存在固有的风险。请确保您的数据库连接权限始终尽可能狭窄地限定在智能体需要的范围内。这将减轻（但不能消除）构建模型驱动系统的风险。
</Warning>

### 概念

以下教程涵盖以下概念：

- 用于从 SQL 数据库读取数据的[工具](/oss/langchain/tools)
- LangChain [代理](/oss/langchain/agents)
- [人机交互](/oss/langchain/human-in-the-loop)流程
