# LangSmith

LangSmith 是一个与框架无关的平台，用于构建、调试和部署 AI 智能体与 LLM 应用。在 [smith.langchain.com](https://smith.langchain.com) 一站式追踪请求、评估输出、测试提示词和管理部署。

## 何时使用

在以下场景使用 LangSmith：

- **追踪和调试** LLM 调用、智能体步骤、检索和工具使用
- **评估** LLM 输出，支持自动化或人工评分
- **工程化提示词**，提供可视化 playground 和版本控制
- **部署智能体** 到生产环境，使用基于 LangGraph 的智能体服务器
- **监控** 生产系统，提供 dashboard、告警和成本追踪

## 何时不使用

- 构建智能体逻辑或 LLM pipeline，请使用 [LangChain](https://docs.langchain.com/oss/langchain/overview)、[LangGraph](https://docs.langchain.com/oss/langgraph/overview) 或 [Deep Agents](https://docs.langchain.com/oss/deepagents/overview)
- LangSmith 是补充这些框架的**平台层**

## 快速设置

设置两个环境变量以启用对任何支持框架的追踪：

```bash
export LANGSMITH_TRACING=true
export LANGSMITH_API_KEY="your-api-key"  # from smith.langchain.com/settings
```

### 安装 SDK

```bash
# Python
pip install langsmith

# JavaScript/TypeScript
npm install langsmith
```

### 验证跟踪

```python
from langsmith import traceable

@traceable
def my_function(query: str) -> str:
    # Your LLM logic here—all calls inside are traced automatically
    return "result"
```

## 核心功能

| 功能 | 描述 |
|-----------|-------------|
| 可观测性 | 使用自动或手动插桩追踪 LLM 应用的每一步 |
| 评估 | 使用代码、LLM-as-judge 或复合评估器运行评估 |
| 提示词工程 | 在可视化 playground 中创建、版本控制和测试提示词 |
| 智能体部署 | 部署 LangGraph 智能体，支持流式输出、人机交互和持久执行 |
| 监控 | 生产工作负载的仪表盘、告警和成本追踪 |
