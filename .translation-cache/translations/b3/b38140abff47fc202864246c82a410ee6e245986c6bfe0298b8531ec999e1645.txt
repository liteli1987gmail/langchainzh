# LangSmith

LangSmith 是一个与框架无关的平台，用于构建、调试和部署 AI 智能体及 LLM 应用。在一个地方追踪请求、评估输出、测试提示词并管理部署，全部在一处完成，网址为 [smith.langchain.com](https://smith.langchain.com).

## 适用场景

在以下情况下使用 LangSmith：
- **追踪和调试** LLM 调用、智能体步骤、检索和工具使用
- **评估** 使用自动化或人工介入评分来评估 LLM 输出
- **工程化提示词** 使用可视化 playground 和版本控制
- **部署智能体** 使用基于 LangGraph 的智能体服务器部署到生产环境
- **监控** 使用仪表板、告警和成本追踪来监控生产系统

## 不适用场景

- 要构建智能体逻辑或 LLM 管道，请使用 [LangChain](https://docs.langchain.com/oss/langchain/overview), [LangGraph](https://docs.langchain.com/oss/langgraph/overview), or [Deep Agents](https://docs.langchain.com/oss/deepagents/overview) 代替
- LangSmith 是 **平台层** 对这些框架的补充

## 快速设置

设置两个环境变量以从任何支持的框架启用追踪：

```bash
export LANGSMITH_TRACING=true
export LANGSMITH_API_KEY="your-api-key"  # from smith.langchain.com/settings
```

### 安装 SDK

```bash
# Python
pip install langsmith

# JavaScript/TypeScript
npm install langsmith
```

### 验证追踪

```python
from langsmith import traceable

@traceable
def my_function(query: str) -> str:
    # Your LLM logic here—all calls inside are traced automatically
    return "result"
```

## 核心功能

| 功能 | 描述 |
|-----------|-------------|
| 可观测性 | 使用自动或手动插桩追踪 LLM 应用的每个步骤 |
| 评估 | 使用代码、LLM 即评判或组合评估器运行评估 |
| 提示词工程 | 在可视化 playground 中创建、版本控制和测试提示词 |
| 智能体部署 | 使用流式、人工介入和持久化执行部署 LangGraph 智能体 |
| 监控 | 生产工作负载的仪表板、告警和成本追踪 |
