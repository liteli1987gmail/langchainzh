[Oxylabs](https://oxylabs.io/) 是一个市场领先的网络智能采集平台，由最高标准的商业、
道德和合规标准驱动，使全球企业能够获得数据驱动的洞察。

[langchain-oxylabs](https://pypi.org/project/langchain-oxylabs/) 实现
使 LLMs 能够与 Oxylabs Web Scraper API 交互的工具。


## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-oxylabs
```

```bash uv
uv add langchain-oxylabs
```
</CodeGroup>
