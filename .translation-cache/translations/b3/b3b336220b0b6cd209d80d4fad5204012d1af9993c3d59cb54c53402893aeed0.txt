title: 数据集预构建 JSON Schema 类型
sidebarTitle: 数据集预构建 JSON Schema 类型
---

LangSmith 建议您在数据集模式的输入和输出上设置 Schema，以确保数据一致性，并确保您的示例采用正确的格式以便后续处理，例如运行评估。

为了更好地支持 LLM 工作流，LangSmith 支持多种预定义的预构建类型。这些 Schema 由 LangSmith API 公开托管，您可以在数据集模式中使用它们进行定义 [JSON Schema 引用](https://json-schema.org/understanding-json-schema/structuring#dollarref)。可用 Schema 的表格如下所示

| 类型 | JSON Schema 引用链接                                                                                                       | 用法                                                                                                                     |
| ------- | -------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| Message | [https://api.smith.langchain.com/public/schemas/v1/message.json](https://api.smith.langchain.com/public/schemas/v1/message.json) | 表示发送到聊天模型的消息，遵循 OpenAI 标准格式。                                           |
| Tool    | [https://api.smith.langchain.com/public/schemas/v1/tooldef.json](https://api.smith.langchain.com/public/schemas/v1/tooldef.json) | 聊天模型可用的工具定义，用于函数调用，采用 OpenAI JSON Schema 风格的函数格式定义。 |

LangSmith 允许您定义一系列转换，从您的 Trace 中收集上述预构建类型并将其添加到您的数据集中。有关可用转换的更多信息，请参阅我们的 [参考文档](/langsmith/dataset-transformations)
