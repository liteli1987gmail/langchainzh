本页介绍如何在 LangChain 中使用 [cloro](https://cloro.dev/) 监控 API。

## 安装和设置
- 使用以下命令安装依赖 `pip install -U langchain-cloro`
- 从获取 [cloro 仪表板](https://dashboard.cloro.dev/) 获取 API 密钥，并将其设置为环境变量 (`CLORO_API_KEY`)

## 工具

cloro 提供多种工具，用于抓取 AI 平台并提取结构化数据。使用这些工具：

```python
from langchain_cloro import CloroGoogleSearch, CloroChatGPT, CloroGemini, CloroPerplexity, CloroGrok, CloroCopilot
```
