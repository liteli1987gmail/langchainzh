本页面介绍如何在 LangChain 中使用 [cloro](https://cloro.dev/) 监控 API。

## 安装与设置

- 使用 `pip install -U langchain-cloro` 安装依赖
- 从 [cloro 仪表板](https://dashboard.cloro.dev/) 获取 cloro API 密钥，并将其设置为环境变量 (`CLORO_API_KEY`)

## 工具

cloro 提供多种工具用于抓取 AI 平台数据并提取结构化数据。使用这些工具的方法如下：

```python
from langchain_cloro import CloroGoogleSearch, CloroChatGPT, CloroGemini, CloroPerplexity, CloroGrok, CloroCopilot
```

详细文档和示例请参阅：
- [Google 搜索抓取器](/oss/integrations/tools/cloro#google-search-scraper)
- [ChatGPT 抓取器](/oss/integrations/tools/cloro#chatgpt-scraper)
- [Gemini 抓取器](/oss/integrations/tools/cloro#gemini-scraper)
- [Perplexity 抓取器](/oss/integrations/tools/cloro#perplexity-scraper)
- [Grok 抓取器](/oss/integrations/tools/cloro#grok-scraper)
