---
title: MODEL_AUTHENTICATION
---

<Note>
    目前仅用于 `langchainjs` (JavaScript/TypeScript).
</Note>

您的模型提供商拒绝访问其服务。

此错误通常发生在您的身份验证凭据或 API 密钥出现问题时。

## 故障排除

* 确认您的 API 密钥或身份验证凭据准确有效。
* 如果使用环境变量身份验证，请验证：
    - 变量名称拼写正确
    - 变量包含赋值
    - 第三方包（如 `dotenv` ）不会干扰加载
* 如果使用代理或非标准端点，请确保您的自定义提供程序不期望替代身份验证方案。
* 通过显式传递凭据来绕过环境变量问题：

:::python
```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(api_key="YOUR_KEY_HERE")
```
:::
:::js
```typescript
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({
  apiKey: "YOUR_KEY_HERE",
});
```
:::
