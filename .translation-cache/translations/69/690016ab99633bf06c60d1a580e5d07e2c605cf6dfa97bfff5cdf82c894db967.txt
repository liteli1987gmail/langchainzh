## 智能体框架（如 LangChain）

智能体框架提供抽象层，使基于 LLM 构建应用更加容易上手。

[LangChain](/oss/langchain/overview) 是一个提供结构化内容块、智能体循环和中间件等抽象的智能体框架。

LangChain 的抽象设计既易于上手，又能满足高级用例的灵活性需求。

虽然 LangChain 建立在 [LangGraph](/oss/langgraph/overview) 之上，但您无需了解 LangGraph 也能使用 LangChain。

其他智能体框架的示例包括 [Vercel 的 AI SDK](https://ai-sdk.dev/docs/introduction)、[CrewAI](https://www.crewai.com/)、[OpenAI Agents SDK](https://openai.github.io/openai-agents-python/)、[Google ADK](https://google.github.io/adk-docs/)、[LlamaIndex](https://www.llamaindex.ai/) 等等。

### 何时使用 LangChain

在以下情况下使用 LangChain：

- 您想快速构建智能体和自主应用。
- 您需要模型、工具和智能体循环的标准抽象。
- 您需要一个易于使用但仍提供灵活性的框架。
- 您正在构建没有复杂编排需求的简单智能体应用。

## 智能体运行时（如 LangGraph）

智能体运行时提供在生产环境中运行智能体的工具。
支持的工具可能包括：

- **持久执行**：智能体可以在失败时持续运行，并且可以长时间运行，从中断处恢复。
- **流式处理**：支持工作流和响应的流式处理。
- **人工介入**：通过检查和修改智能体状态来加入人工监督。
- **持久化存储**：用于状态管理的线程级和跨线程持久化。
- **底层控制**：直接控制智能体编排，不使用高层抽象。

[LangGraph](/oss/langgraph/overview) 是一个用于构建、管理和部署长时间运行、有状态智能体的底层编排框架和运行时。

智能体框架通常是更高层次的，运行在智能体运行时之上。
例如，LangChain 1.0 建立在 LangGraph 之上。

其他智能体运行时的示例包括 [Temporal](https://temporal.io/)、[Inngest](https://www.inngest.com/) 和其他持久执行引擎。

### 何时使用 LangGraph

在以下情况下使用 LangGraph：

- 您需要对智能体编排进行细粒度的底层控制。
- 您需要长时间运行、有状态智能体的持久执行。
- 您正在构建将确定性步骤和智能体步骤结合的复杂工作流。
- 您需要用于智能体部署的生产就绪基础设施。

## 智能体工具集（如 Deep Agents SDK）

智能体工具集是提供内置工具和功能的多功能、电池包含式框架，用于构建复杂的长时间运行智能体。
支持的工具可能包括：

- **规划能力**：使用待办事项列表跟踪多个任务。
- **任务委托**：委托工作并通过子智能体保持上下文清晰。
- **文件系统**：对不同可插拔存储后端的文件进行读写访问。
- **令牌管理**：对话历史摘要和大型工具结果驱逐。

[Deep Agents SDK](/oss/deepagents/overview) 在 LangGraph 之上构建，增加了规划能力、用于上下文管理的文件系统、生成子智能体的能力等。
Deep Agents 专为需要规划和任务分解的复杂多步骤任务而设计。

示例任务包括处理工作流中的搜索结果、脚本和其他产物。

其他智能体工具集的示例包括 [Claude Agent SDK](https://platform.claude.com/docs/en/agent-sdk/overview)、[Manus](https://manus.im/) 和其他编码 CLI。

### 何时使用 Deep Agents SDK

在以下情况下使用 [Deep Agents SDK](/oss/deepagents/overview)：

- 您正在构建长时间运行的智能体。
- 您正在构建需要处理复杂多步骤任务的智能体。
- 您想使用预定义工具，例如文件系统操作、bash 执行和自动化上下文工程。
- 您想使用预定义提示词和子智能体。

## 功能对比

虽然您可以使用 LangChain、LangGraph 和 Deep Agents 完成类似的任务，但它们的集成层次不同：

| 功能 | LangChain | LangGraph | Deep Agents |
| ------- | --------- | ----------- | ----------- |
| 短期记忆 | [短期记忆](/oss/langchain/short-term-memory) | [短期记忆](/oss/langgraph/add-memory#add-short-term-memory) | [`StateBackend`](/oss/deepagents/backends#statebackend) |
| 长期记忆 | [长期记忆](/oss/langchain/long-term-memory) | [长期记忆](/oss/langgraph/add-memory#add-long-term-memory) | [长期记忆](/oss/deepagents/memory) |
| 技能 | [多智能体技能](/oss/langchain/multi-agent/skills) | - | [技能](/oss/deepagents/skills) |
| 子智能体 | [多智能体子智能体](/oss/langchain/multi-agent/subagents) | [子图](/oss/langgraph/use-subgraphs) | [子智能体](/oss/deepagents/subagents) |
| 人工介入 | [人工介入中间件](/oss/langchain/human-in-the-loop) | [中断](/oss/langgraph/interrupts) | [`interrupt_on` 参数](/oss/deepagents/harness#human-in-the-loop) |
| 流式处理 | [智能体流式处理](/oss/langchain/event-streaming) | [流式处理](/oss/langgraph/streaming) | [流式处理](/oss/deepagents/event-streaming) |
