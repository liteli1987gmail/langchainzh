> [any-llm](https://github.com/mozilla-ai/any-llm) 是一个统一接口，用于调用 OpenAI、Anthropic、Google、本地模型（通过 Ollama/LocalAI）以及[更多](https://mozilla-ai.github.io/any-llm/providers/)。只需更改一个字符串即可切换提供商。

## 安装和设置

<CodeGroup>

</fragment>
```bash pip
pip install langchain-anyllm
```

```bash uv
uv add langchain-anyllm
```
</CodeGroup>

你需要为所选提供商获取相应的 API 密钥。API 密钥可以通过 `api_key` 参数传递，也可以设置为环境变量（例如 `OPENAI_API_KEY`、`ANTHROPIC_API_KEY`）。请参阅 [any-llm 文档](https://mozilla-ai.github.io/any-llm/providers/) 了解提供商的具体要求。

## 聊天模型

```python
from langchain_anyllm import ChatAnyLLM
```
