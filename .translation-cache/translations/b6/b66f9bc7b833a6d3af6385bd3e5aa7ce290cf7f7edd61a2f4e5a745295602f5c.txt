>[AI21 Labs](https://www.ai21.com/about) 是一家专注于自然
> 语言处理（NLP）的公司，开发人工智能系统
> 能够理解和生成自然语言。

本页面介绍如何在 `AI21` 生态系统内使用 `LangChain`.

## 安装与设置

- 获取 AI21 API 密钥并将其设置为环境变量（`AI21_API_KEY`)
- 安装 Python 包：

<CodeGroup>
```bash pip
pip install langchain-ai21
```

```bash uv
uv add langchain-ai21
```
</CodeGroup>

## 聊天模型

### AI21 聊天

查看 [使用示例](/oss/integrations/chat/ai21).

```python
from langchain_ai21 import ChatAI21
```
