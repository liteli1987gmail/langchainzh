## 配置追踪

要追踪 ADK 代理，请使用 LangSmith SDK 中的 `configure_google_adk()`。在创建任何 ADK 代理之前，在应用程序启动时调用此函数一次：

```python
from langsmith.integrations.google_adk import configure_google_adk

configure_google_adk(
    project_name="my-adk-project",  # Optional: defaults to LANGSMITH_PROJECT env var
)
```

函数接受以下可选参数：

- `project_name`：LangSmith 项目，用于发送追踪记录。默认为 `LANGSMITH_PROJECT` 环境变量。
- `name`：根追踪的名称。默认为 `"google_adk.session"`。
- `metadata`：键值对字典，用于添加额外的上下文信息。
- `tags`：字符串列表，用于对追踪记录进行分类。

## 在 LangSmith 中查看追踪记录

运行应用程序后，您可以在 [LangSmith UI](https://smith.langchain.com) 中查看包含以下内容的追踪记录：

- **智能体调用**：ADK 智能体的完整流程
- **工具调用**：智能体执行的单个函数调用
- **LLM 交互**：Gemini 模型的请求和响应
- **多智能体工作流**：顺序和并行智能体组合的追踪记录

## 自定义元数据和标签

配置追踪时添加元数据和标签，以便对追踪记录进行分类和筛选：

```python
from langsmith.integrations.google_adk import configure_google_adk

configure_google_adk(
    project_name="production-agents",
    metadata={
        "environment": "production",
        "team": "ml-platform",
    },
    tags=["adk", "weather", "v2"],
)
```
