## 配置追踪

要追踪 ADK agent，请使用 `configure_google_adk()` 来自 LangSmith SDK。在创建任何 ADK agent 之前，在应用程序开头调用此函数一次：

```python
from langsmith.integrations.google_adk import configure_google_adk

configure_google_adk(
    project_name="my-adk-project",  # Optional: defaults to LANGSMITH_PROJECT env var
)
```

该函数接受以下可选参数：

- `project_name`：LangSmith 项目，发送追踪的目标。默认为 `LANGSMITH_PROJECT` 环境变量。
- `name`：根追踪的名称。默认为 `"google_adk.session"`.
- `metadata`：用于额外上下文的键值对字典。
- `tags`：用于对追踪进行分类的字符串列表。

## 在 LangSmith 中查看追踪

运行应用程序后，您可以在 [LangSmith UI](https://smith.langchain.com) 中查看包含以下内容的追踪：

- **Agent 调用**：通过 ADK agent 的完整流程
- **工具调用**：agent 进行的单独函数调用
- **LLM 交互**：与 Gemini 模型的请求和响应
- **多 agent 工作流**：顺序和并行 agent 组合的追踪

## 自定义元数据和标签

在配置追踪时添加元数据和标签，以便对追踪进行分类和筛选：

```python
from langsmith.integrations.google_adk import configure_google_adk

configure_google_adk(
    project_name="production-agents",
    metadata={
        "environment": "production",
        "team": "ml-platform",
    },
    tags=["adk", "weather", "v2"],
)
```
