>[Perplexity](https://www.perplexity.ai/pro) 是搜索互联网最强大的方式
> 无限 Pro 搜索、升级的 AI 模型、无限文件上传、
> 图像生成和 API 额度。

## 安装和设置

安装 LangChain JavaScript 的 Perplexity 集成包：

<CodeGroup>
```bash npm
npm install @langchain/perplexity @langchain/core
```
```bash yarn
yarn add @langchain/perplexity @langchain/core
```
```bash pnpm
pnpm add @langchain/perplexity @langchain/core
```
</CodeGroup>

从以下位置获取您的 API 密钥 [Perplexity API 密钥仪表板](https://www.perplexity.ai/account/api/keys) 并将其设置为 `PERPLEXITY_API_KEY` 环境变量。请参阅 [Perplexity 快速入门指南](https://docs.perplexity.ai/docs/getting-started) 了解更多详情。

```bash
export PERPLEXITY_API_KEY="your-api-key"
```

## 聊天模型

请参阅 [使用示例](/oss/integrations/chat/perplexity).

```typescript
import { ChatPerplexity } from "@langchain/perplexity";
```

`ChatPerplexity` 也可以针对 [Perplexity Agent API](https://docs.perplexity.ai/api-reference/agent-api) 通过传递 `useResponsesApi: true` （或通过传递 `tools: [{ type: "web_search" }]`，这会自动启用它）。请参阅 [Agent API 支持](/oss/integrations/chat/perplexity#agent-api-support-useresponsesapi) 聊天页面上的详细信息和示例。

## 检索器

您可以使用 [`PerplexitySearchRetriever`](/oss/integrations/retrievers/perplexity_search) 从以下位置获取网络搜索结果 [Perplexity 搜索 API](https://docs.perplexity.ai/docs/search/quickstart) as `Document` 标准检索管道中的对象。

请参阅 [使用示例](/oss/integrations/retrievers/perplexity_search).

```typescript
import { PerplexitySearchRetriever } from "@langchain/perplexity";
```

## 工具

您可以将 Perplexity 用作代理工具，让您的代理访问 Perplexity 搜索 API。

请参阅 [使用示例](/oss/integrations/tools/perplexity_search).

### PerplexitySearchResults

一个查询 Perplexity 搜索 API 并返回结果 JSON 数组的工具（标题、URL、摘要、日期、最后更新）。

```typescript
import { PerplexitySearchResults } from "@langchain/perplexity";
```

## 组件参考

| 类 | 抽象 | 导入路径 | 描述 |
|-------|-------------|-------------|-------------|
| `ChatPerplexity` | 聊天模型 | `import { ChatPerplexity } from "@langchain/perplexity"` | 用于基础聊天完成的包装 Perplexity API 的聊天模型。 |
| `PerplexitySearchRetriever` | 检索器 | `import { PerplexitySearchRetriever } from "@langchain/perplexity"` | 返回 `Document` 来自 Perplexity 搜索 API 的对象的检索器。 |
| `PerplexitySearchResults` | 工具 | `import { PerplexitySearchResults } from "@langchain/perplexity"` | 为代理返回 Perplexity 搜索 API 结果作为 JSON 数组的工具。 |
