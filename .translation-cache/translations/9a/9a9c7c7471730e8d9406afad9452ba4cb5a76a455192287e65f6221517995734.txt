>[Perplexity](https://www.perplexity.ai/pro) 是最强大的互联网搜索方式，提供无限 Pro Search、升级的 AI 模型、无限文件上传、图片生成和 API 积分。

## 安装和设置

安装 LangChain JavaScript 的 Perplexity 集成包：

<CodeGroup>

</fragment>
```bash npm
npm install @langchain/perplexity @langchain/core
```
```bash yarn
yarn add @langchain/perplexity @langchain/core
```
```bash pnpm
pnpm add @langchain/perplexity @langchain/core
```
</CodeGroup>

从 [Perplexity API 密钥仪表板](https://www.perplexity.ai/account/api/keys) 获取您的 API 密钥，并将其设置为 `PERPLEXITY_API_KEY` 环境变量。欲了解更多详情，请参阅 [Perplexity 快速入门指南](https://docs.perplexity.ai/docs/getting-started)。

```bash
export PERPLEXITY_API_KEY="your-api-key"
```

## 聊天模型

参见[使用示例](/oss/integrations/chat/perplexity)。

```typescript
import { ChatPerplexity } from "@langchain/perplexity";
```

`ChatPerplexity` 还可以通过传递 `useResponsesApi: true` 来将 [Perplexity Agent API](https://docs.perplexity.ai/api-reference/agent-api) 作为目标（或者通过传递 `tools: [{ type: "web_search" }]`，这会自动启用它）。请参阅聊天页面上的 [Agent API 支持](/oss/integrations/chat/perplexity#agent-api-support-useresponsesapi) 获取详细信息和示例。

## 检索器

您可以使用 [`PerplexitySearchRetriever`](/oss/integrations/retrievers/perplexity_search) 从 [Perplexity Search API](https://docs.perplexity.ai/docs/search/quickstart) 获取网络搜索结果，并将其作为 `Document` 对象纳入标准检索管道。

请参阅[使用示例](/oss/integrations/retrievers/perplexity_search)。

```typescript
import { PerplexitySearchRetriever } from "@langchain/perplexity";
```

## 工具

您可以将 Perplexity 用作智能体工具，让您的智能体访问 Perplexity Search API。

请参阅[使用示例](/oss/integrations/tools/perplexity_search)。

### PerplexitySearchResults

该工具可查询 Perplexity Search API 并返回 JSON 格式的结果数组（包含 title、URL、snippet、date、last updated 字段）。

```typescript
import { PerplexitySearchResults } from "@langchain/perplexity";
```

## 组件参考

| 类 | 抽象 | 导入路径 | 描述 |
|-------|-------------|-------------|-------------|
| `ChatPerplexity` | 聊天模型 | `import { ChatPerplexity } from "@langchain/perplexity"` | 封装 Perplexity API 的聊天模型，用于生成式聊天补全。 |
| `PerplexitySearchRetriever` | 检索器 | `import { PerplexitySearchRetriever } from "@langchain/perplexity"` | 从 Perplexity Search API 返回 `Document` 对象的检索器。 |
| `PerplexitySearchResults` | 工具 | `import { PerplexitySearchResults } from "@langchain/perplexity"` | 工具，以 JSON 数组形式返回 Perplexity Search API 结果，供智能体使用。 |
