[Kinetica](https://www.kinetica.com/) 是一个内置向量相似性搜索功能的数据库。

它支持：

- 精确和近似最近邻搜索
- L2 距离、内积和余弦距离

本笔记本展示了如何使用 Kinetica 向量存储（`Kinetica`）。

这需要一个 Kinetica 实例，可以按照以下说明轻松设置——[安装说明](https://www.kinetica.com/developer-edition/)。

```python
# Pip install necessary package
pip install -qU langchain-kinetica
```

我们想要使用 `OpenAIEmbeddings`，所以我们必须获取 OpenAI API Key。


```python
import getpass
import os

from langchain_openai import OpenAIEmbeddings

if "OPENAI_API_KEY" not in os.environ:
    os.environ["OPENAI_API_KEY"] = getpass.getpass("OpenAI API Key:")

embeddings = OpenAIEmbeddings(model="text-embedding-3-large")
```

必须设置以下环境变量来配置数据库连接。如果使用虚拟环境，可以在项目的 `.env` 文件中设置这些变量：

* `KINETICA_URL`：数据库连接 URL（例如 `http://localhost:9191`）
* `KINETICA_USER`：数据库用户
* `KINETICA_PASSWD`：安全密码



## 使用欧几里得距离的相似性搜索（默认）

Kinetica 模块将尝试使用集合的名称创建表。
请确保集合名称唯一，且用户具有创建表的权限。


```python
COLLECTION_NAME = "langchain_example"

vectorstore = KineticaVectorstore(
    config=k_config,
    embedding_function=embeddings,
    collection_name=COLLECTION_NAME,
    pre_delete_collection=True,
)

vectorstore.add_documents(documents=documents, ids=uuids)

print()
print("Similarity Search")
results = vectorstore.similarity_search(
    "LangChain provides abstractions to make working with LLMs easy",
    k=2,
    filter={"source": "tweet"},
)
for res in results:
    print(f"* {res.page_content} [{res.metadata}]")

print()
print("Similarity search with score")
results = vectorstore.similarity_search_with_score(
    "Will it be hot tomorrow?", k=1, emb_filter={"source": "news"}
)
for res, score in results:
    print(f"* [SIM={score:3f}] {res.page_content} [{res.metadata}]")
```

## 使用向量存储

### 添加文档

上面，我们从头创建了一个向量存储。然而，通常我们希望使用现有的向量存储。为了做到这一点，我们可以直接初始化它。


```python
vectorstore = KineticaVectorstore(
    config=k_config,
    embedding_function=embeddings,
    collection_name=COLLECTION_NAME,
)

# We can add documents to the existing vectorstore.
vectorstore.add_documents([Document(page_content="foo")])

docs_with_score = vectorstore.similarity_search_with_score("foo")

print(f"First result: {docs_with_score[0]}")
print(f"Second result: {docs_with_score[1]}")
```

```text
First result: (Document(metadata={}, page_content='foo'), 0.0014664357295259833)
Second result: (Document(metadata={'source': 'tweet'}, page_content='Building an exciting new project with LangChain - come check it out!'), 1.260981559753418)
```

### 覆盖向量存储

如果你有一个已存在的集合，你可以通过执行 `from_documents` 并设置 `pre_delete_collection` = True 来覆盖它


```python
vectorstore = KineticaVectorstore.from_documents(
    documents=documents,
    embedding=embeddings,
    collection_name=COLLECTION_NAME,
    config=k_config,
    pre_delete_collection=True,
)

docs_with_score = vectorstore.similarity_search_with_score("foo")
docs_with_score[0]
```

```text
(Document(metadata={'source': 'tweet'}, page_content='Building an exciting new project with LangChain - come check it out!'),
    1.2609236240386963)
```
