>什么是 `FalkorDB`？

>- FalkorDB 是一个`开源数据库管理系统`，专注于图数据库技术。
>- FalkorDB 允许您以节点和边的形式表示和存储数据，非常适合处理关联数据和关系。
>- FalkorDB 支持 OpenCypher 查询语言，并带有专有扩展，让您轻松与图数据进行交互和查询。
>- 借助 FalkorDB，您可以实现高性能的`图遍历和查询`，适用于生产级系统。

>访问[他们的网站](https://docs.falkordb.com/)开始使用 FalkorDB。

## 安装和设置

- 使用 `pip install falkordb langchain-falkordb` 安装 Python SDK

## VectorStore

FalkorDB 向量索引用作向量存储，
无论是语义搜索还是示例选择。

```python
from langchain_falkordb.vectorstore import FalkorDBVector
```
