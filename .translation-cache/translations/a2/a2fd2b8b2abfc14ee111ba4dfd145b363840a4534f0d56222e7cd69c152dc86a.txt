>什么是 `FalkorDB`?

>- FalkorDB 是一个 `open-source database management system` 专门从事图数据库技术的
>- FalkorDB 允许您在节点和边中表示和存储数据，使其成为处理连接数据和关系的理想选择。
>- FalkorDB 支持带有专有扩展的 OpenCypher 查询语言，让您可以轻松地与图数据进行交互和查询。
>- 借助 FalkorDB，您可以实现高性能 `graph traversals and queries`，适用于生产级系统。

>开始使用 FalkorDB，请访问 [其网站](https://docs.falkordb.com/).

## 安装和设置

- 使用以下方式安装 Python SDK `pip install falkordb langchain-falkordb`

## 向量存储

FalkorDB 向量索引被用作向量存储，
无论是用于语义搜索还是示例选择。

```python
from langchain_falkordb.vectorstore import FalkorDBVector
```
