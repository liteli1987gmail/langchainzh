[Jenkins](https://www.jenkins.io/) 是一个开源自动化平台，能够实现
软件团队简化其开发工作流程。它在
DevOps 社区中被广泛采用，用于自动化
applications through CI/CD pipelines.


## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-jenkins
```

```bash uv
uv add langchain-jenkins
```
</CodeGroup>
