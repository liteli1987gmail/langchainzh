[Jenkins](https://www.jenkins.io/) 是一个开源自动化平台，使软件团队能够简化其开发工作流程。它在 DevOps 社区中被广泛采用，作为一种通过 CI/CD 管道自动化应用程序构建、测试和部署的工具。

## 安装和设置

<CodeGroup>

</fragment>
```bash pip
pip install langchain-jenkins
```

```bash uv
uv add langchain-jenkins
```
</CodeGroup>
