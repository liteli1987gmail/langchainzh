        ## UI 框架集成

        开箱即用的可嵌入式 UI 组件库，专为配合 `@langchain/react` 使用而构建。

        <CardGroup cols={1}>
        <Card title="ai-elements" icon="package" href="https://ui-patterns.langchain.com/ai-elements/" cta="Open preview" target="_blank">
            用于构建 AI 聊天界面的无头、可组合 React 组件。底层使用 @[`useStream`]，完全兼容 shadcn/ui。

            - 滚动粘性对话容器
            - 带语法高亮的流式代码块
            - 可折叠推理/思考块
            - 带参数和结果的工具调用显示
            - 微光加载状态和建议标签
        </Card>
        <Card title="assistant-ui" icon="package" href="https://ui-patterns.langchain.com/assistant-ui/" cta="Open preview" target="_blank">
            使用 `@assistant-ui/react` 原语构建的 Claude 风格助手 UI，通过 `useExternalStoreRuntime` 连接到 LangChain。

            - Claude 风格的暖羊皮纸主题
            - 思维链可折叠推理
            - 图片附件支持
            - 复制/赞成/反对操作栏
            - 完整深色模式支持
        </Card>
        <Card title="OpenUI" icon="package" href="https://ui-patterns.langchain.com/openui/" cta="Open preview" target="_blank">
            生成式 UI 聊天，LangChain 智能体在消息流中发送任意 React 组件——在浏览器中实时渲染。

            - 智能体驱动的动态组件渲染
            - 流式消息组合
            - 自定义组件注册表
            - 生成界面的实时预览
        </Card>
        </CardGroup>
