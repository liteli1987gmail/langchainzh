>什么是 `Timbr`？

>- Timbr 是一个 `语义 SQL 知识图谱平台`，专门通过本体驱动的语义层连接数据。
>- Timbr 允许您使用业务友好的语言来表示和查询数据，非常适合处理复杂的数据关系和业务逻辑。
>- Timbr 提供 `自然语言到 SQL` 的功能，让您可以轻松使用简单的英语查询与数据进行交互。
>- 使用 Timbr，您可以实现高性能的 `语义数据查询`，适用于生产级分析和商业智能。

>开始使用 Timbr，请访问[他们的网站](https://docs.timbr.ai/doc/docs/getting-started/intro-timbr/)。

## 安装和设置

- 使用 `pip install langchain-timbr` 安装 Python SDK

### 可选：安装选定的 LLM 提供商

选择一个：openai、anthropic、google、azure_openai、snowflake、databricks、vertex_ai（或 'all'）

```bash
pip install 'langchain-timbr[<your selected providers, separated by comma without spaces>]'
```

## 语义 SQL 查询

Timbr 提供了一个封装层，围绕其语义层构建，该层能够根据自然语言输入生成 SQL 语句，并从您的知识图谱中检索相关信息。

```python
from langchain_timbr import create_timbr_sql_agent, ExecuteTimbrQueryChain, GenerateTimbrSqlChain
```
