>什么是 `Timbr`?

>- Timbr 是一个 `semantic SQL knowledge graph platform` 专门通过本体驱动的语义层连接数据的平台。
>- Timbr 允许您使用业务友好的语言来表示和查询数据，非常适合处理复杂的数据关系和业务逻辑。
>- Timbr 提供 `natural language to SQL` 功能，让您使用简单的英语查询就能轻松与数据交互。
>- 借助 Timbr，您可以实现高性能 `semantic data querying`，适合生产级分析和高性能业务智能。

>开始使用 Timbr，请访问 [其网站](https://docs.timbr.ai/doc/docs/getting-started/intro-timbr/).

## 安装与设置

- 使用以下命令安装 Python SDK `pip install langchain-timbr`

### 可选：安装选定的 LLM 提供商

选择一个：openai、anthropic、google、azure_openai、snowflake、databricks、vertex_ai（或 'all'）

```bash
pip install 'langchain-timbr[<your selected providers, separated by comma without spaces>]'
```

## 语义 SQL 查询

Timbr 提供了一个围绕其语义层的包装器，可根据自然语言输入生成 SQL 语句，并从您的知识图谱中检索相关信息。

```python
from langchain_timbr import create_timbr_sql_agent, ExecuteTimbrQueryChain, GenerateTimbrSqlChain
```
