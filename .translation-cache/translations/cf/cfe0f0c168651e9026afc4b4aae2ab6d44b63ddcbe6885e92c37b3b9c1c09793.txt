---
title: 如何在单个评估器中返回多个分数
sidebarTitle: 在单个评估器中返回多个分数
---

有时，自定义评估器或摘要评估器返回多个指标会很有用。例如，如果您有多个由 LLM 评判生成的指标，您可以通过一次 LLM 调用生成多个指标来节省时间和成本，而不是进行多次 LLM 调用。

要使用 Python SDK 返回多个分数，只需返回以下格式的字典/对象列表：

```python
[
    # 'key' is the metric name
    # 'score' is the value of a numerical metric
    {"key": string, "score": number},
    # 'value' is the value of a categorical metric
    {"key": string, "value": string},
    ... # You may log as many as you wish
]
```

要使用 JS/TS SDK 做到这一点，请返回一个带有 'results' 键的对象，其中包含一个上述形式的列表。

```typescript
{results: [{ key: string, score: number }, ...]};
```

每个这些字典都可以包含任意或全部的 [feedback fields](/langsmith/feedback-data-format)；请参阅链接文档获取更多信息。

示例：

- Python：需要 `langsmith>=0.2.0`
- TypeScript：多个分数的支持在 `langsmith@0.1.32` 及更高版本中可用

<CodeGroup>

</CodeGroup>

```python Python
def multiple_scores(outputs: dict, reference_outputs: dict) -> list[dict]:
    # Replace with real evaluation logic.
    precision = 0.8
    recall = 0.9
    f1 = 0.85
    return [
        {"key": "precision", "score": precision},
        {"key": "recall", "score": recall},
        {"key": "f1", "score": f1},
    ]
```

```typescript TypeScript
import type { Run, Example } from "langsmith/schemas";

function multipleScores(rootRun: Run, example: Example) {
  // Your evaluation logic here
  return {
      results: [
          { key: "precision", score: 0.8 },
          { key: "recall", score: 0.9 },
          { key: "f1", score: 0.85 },
      ],
  };
}
```

</CodeGroup>

生成的实验行将显示每个分数。
