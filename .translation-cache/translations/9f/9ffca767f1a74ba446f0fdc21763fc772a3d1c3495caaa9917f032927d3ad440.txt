[Kinetica](https://www.kinetica.com/) 是一个集成了向量相似度搜索功能的数据库。

本笔记本介绍如何从 Kinetica 加载文档

```python
pip install -qU langchain-kinetica
```

您必须在以下环境变量中设置数据库连接。如果您使用虚拟环境，可以在项目的 `.env` 文件中设置它们：

* `KINETICA_URL`：数据库连接 URL（例如 `http://localhost:9191`）
* `KINETICA_USER`：数据库用户
* `KINETICA_PASSWD`：安全密码


```python
from gpudb import GPUdb

kdbc = GPUdb.get_connection()
```

```text

```

以下 `QUERY` 是一个示例，不会运行；需要将其替换为有效的 `QUERY` 以返回数据，且 `SCHEMA.TABLE` 组合必须存在于 Kinetica 中。

```python
from langchain_kinetica import KineticaLoader

QUERY = """select
    oid,
    object_name as text,
    creation_time
from ki_catalog.ki_objects
where schema_name = 'information_schema'
limit 10"""

kinetica_loader = KineticaLoader(
    kdbc=kdbc,
    query=QUERY,
    metadata_columns=["creation_time"],
)

kinetica_documents = kinetica_loader.load()
display(kinetica_documents)
```
