</Tip>

Oracle AI Database 支持 AI 工作负载，您可以通过**含义**（语义）而非仅通过关键词来查询数据。它将**非结构化内容的语义搜索**与**业务数据的关系过滤**结合在单一系统中——因此您可以构建检索工作流（如 RAG），而无需引入独立的向量数据库并将数据分散到多个平台。

本指南演示如何使用 Oracle AI Vector Search LangChain 集成中的 `OracleSummary` 生成文档摘要。

> **为什么要使用摘要？**
> 摘要是一种实用的方式，可以将长文档压缩为便于检索的内容（预览、元数据或精简上下文），同时在数据附近保持治理和运营保障。

如果您刚开始使用 Oracle Database，可以探索[免费 Oracle 26 AI](https://www.oracle.com/database/free/#resources)。有关用户管理的背景信息，请参阅官方 [Oracle 指南](https://docs.oracle.com/en/database/oracle/oracle-database/26/dbseg/configuring-privilege-and-role-authorization.html#GUID-16473474-7F47-4E40-A592-01836E7D911C)。

## 设置

要访问 OracleSummary，请安装 `@oracle/langchain-oracledb` 辅助工具（包含 `@langchain/core`），并确保您的环境满足 [Oracle Database 驱动程序](https://node-oracledb.readthedocs.io/en/latest/) 的前置条件。

### 凭证

设置拥有摘要配置之 Oracle 用户的凭证（或使用密钥管理器）：

```bash
export ORACLE_USER=testuser
export ORACLE_PASSWORD=testuser
export ORACLE_DSN="localhost:1521/free"
```

如果您计划调用第三方提供商（例如 OCI Generative AI 或 Hugging Face），请首先在 Oracle Database 中创建凭据（例如 `OCI_CRED`、`HF_CRED`），具体方法请参阅 [Oracle AI Vector Search guide](https://www.oracle.com/pls/topic/lookup?ctx=dblatest&id=GUID-EC9DDB58-6A15-4B36-BA66-ECBA20D2CE57) 中记录的 PL/SQL 助手。

## 实例化工具

```typescript
import oracledb from "oracledb";
import { OracleSummary } from "@oracle/langchain-oracledb";

const connection = await oracledb.getConnection({
  user: process.env.ORACLE_USER,
  password: process.env.ORACLE_PASSWORD,
  connectionString: process.env.ORACLE_DSN,
});

const summary = new OracleSummary(
  connection,
  {
    provider: "database",
    glevel: "S",
    numParagraphs: 1,
    language: "english",
  },
  process.env.HTTP_PROXY, // optional
);
```

第三个构造函数参数是一个可选的代理字符串。仅当出站请求必须通过 HTTP 代理时（例如，调用 Hugging Face 端点时）才需要提供。

## 使用数据库内模型进行文本摘要

直接在 Oracle Database 中运行 ONNX 摘要模型，使数据与事务工作负载保持在同一主机上。

```typescript
const result = await summary.getSummary(
  "Oracle Database combines relational and vector workloads so you can build secure RAG pipelines.",
);

console.log(result); // => concise summary text
```

## 使用托管提供商

切换 `provider` 参数，通过 [OCI Generative AI](https://docs.oracle.com/en-us/iaas/Content/generative-ai/home.htm) 或 Hugging Face 路由摘要请求。提供您在 Oracle Database 中注册的凭证名称，以及代理字符串（如有需要）。

```typescript
const ociSummary = new OracleSummary(
  connection,
  {
    provider: "ocigenai",
    credential_name: "OCI_CRED",
    url: "https://inference.generativeai.us-chicago-1.oci.oraclecloud.com/20231130/actions/summarizeText",
    model: "cohere.command",
  },
  process.env.HTTP_PROXY,
);

const hfSummary = new OracleSummary(connection, {
  provider: "huggingface",
  credential_name: "HF_CRED",
  url: "https://api-inference.huggingface.co/models/",
  model: "facebook/bart-large-cnn",
  wait_for_model: "true",
});
```

## 与 LangChain 工具链集成

您可以将 `OracleSummary` 包装在自定义 LangChain 工具或可运行对象中，以将其与代理工具调用集成。

```typescript
import { tool } from "@langchain/core/tools";
import { z } from "zod";

const summarizeTool = tool(
  async ({ text }: { text: string }) => summary.getSummary(text),
  {
    name: "oracle_summary",
    description: "Summarize Oracle-sourced documents.",
    schema: z.object({
      text: z.string().describe("Full text to summarize"),
    }),
  },
);

const toolSummary = await summarizeTool.invoke({ text });
```
