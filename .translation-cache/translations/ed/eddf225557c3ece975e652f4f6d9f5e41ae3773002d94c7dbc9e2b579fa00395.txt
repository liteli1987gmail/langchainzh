[LangSmith Cloud](/langsmith/cloud) 是一个**用于部署代理的托管平台**。LangChain 在 AWS 和 GCP 上托管和运营[控制平面](/langsmith/control-plane)、[数据平面](/langsmith/data-plane)、[Agent Server](/langsmith/agent-server)运行时及支持数据库。只需将代码推送到已连接的 GitHub 仓库或调用 `langgraph deploy` CLI，平台就会处理构建、配置、扩缩容和持续运营。

<Callout icon="clipboard-check" color="#4F46E5" iconType="regular">
在 Cloud 上运行的代理部署需要 [Plus 计划或更高版本](https://www.langchain.com/pricing)。在创建第一个代理部署之前，请使用 `langgraph dev` 验证您的应用程序在本地运行正常。请参阅[本地开发和测试](/langsmith/local-dev-testing)。
</Callout>
