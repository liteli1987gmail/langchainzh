### 功能

- 🔒 **安全的沙箱执行** - 在隔离环境中运行 Python 代码

- 📦 **包管理** - 在沙箱中动态安装 Python 包

## 设置

要访问 `DaytonaDataAnalysisTool`，您需要创建一个 Daytona [账户](https://app.daytona.io/)，获取一个 [API 密钥](https://app.daytona.io/dashboard/keys)，并安装 `langchain-daytona-data-analysis` 集成包。

### 凭证

您必须为 Daytona 配置凭证。有三种方式：

**1. 设置 `DAYTONA_API_KEY` 环境变量：**

```bash Set API key icon="key"
export DAYTONA_API_KEY="your-daytona-api-key"
```

**2. 将其添加到 `.env` 项目根目录的文件中：**

```env Set API key icon="key"
DAYTONA_API_KEY=your-daytona-api-key
```

**3. 在实例化时直接传递 API 密钥 `DaytonaDataAnalysisTool`:**

```python Set API key icon="key"
tool = DaytonaDataAnalysisTool(daytona_api_key="your-daytona-api-key")
```

It's also helpful (but not needed) to set up LangSmith for best-in-class observability/<Tooltip tip="Log each step of a model's execution to debug and improve it">追踪</Tooltip> 的工具调用追踪。要启用自动追踪，请设置您的 [LangSmith](/langsmith/observability) API 密钥：

```python Enable tracing icon="flask"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

## 实例化

导入并实例化工具：

```python Initialize tool instance icon="robot"
from langchain_daytona_data_analysis import DaytonaDataAnalysisTool
from daytona import ExecutionArtifacts

# Optionally, you can pass an on_result callback.
# This callback lets you apply custom logic to the data analysis result.
# For example, you can save outputs, display charts, or trigger other actions.
def process_data_analysis_result(result: ExecutionArtifacts):
    print(result)

tool = DaytonaDataAnalysisTool(
    daytona_api_key="your-daytona-api-key", # Only pass if not set as DAYTONA_API_KEY environment variable
    on_result=process_data_analysis_result
)
```


```python ToolCall icon="briefcase"
    "type": "tool_call",
```

## 附加功能

提供 `DaytonaDataAnalysisTool` 提供多种管理文件和沙箱环境的方法：

### 文件管理

**上传文件到沙箱：**

```python
with open("sales_data.csv", "rb") as f:
    uploaded = tool.upload_file(
        f,
        "CSV file containing sales data with columns: id, date, product, revenue"
    )
```

**从沙箱下载文件：**

```python
file_bytes = tool.download_file("/home/daytona/results.csv")
```

**删除已上传的文件：**

```python
tool.remove_uploaded_file(uploaded)
```

### 包管理

**在沙箱中安装 Python 包：**

```python
# Single package
tool.install_python_packages("pandas")

# Multiple packages
tool.install_python_packages(["numpy", "matplotlib", "seaborn"])
```

<Note>
有关预安装包的列表，请参阅 [Daytona 默认快照文档](https://www.daytona.io/docs/en/snapshots/#default-snapshot).
</Note>

### 沙箱管理

**访问沙箱实例：**

```python
sandbox = tool.get_sandbox()
```

**完成时关闭沙箱：**

```python
tool.close()  # Cleans up resources and deletes the sandbox
```

<Warning>
调用 `tool.close()` 完成所有数据分析任务后调用，以正确清理资源并避免不必要的使用。
</Warning>
