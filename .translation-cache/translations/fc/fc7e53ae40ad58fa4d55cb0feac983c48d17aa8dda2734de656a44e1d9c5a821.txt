---

[AgentPhone](https://agentphone.to) 是一个面向 AI 代理的电话平台，通过简单的 API 提供消息传递、语音通话、电话号码管理和联系人管理功能。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-agentphone
```

```bash uv
uv add langchain-agentphone
```
</CodeGroup>

从 [agentphone.to](https://agentphone.to) 获取您的 API 密钥，并将其设置为环境变量：

```bash
export AGENTPHONE_API_KEY="your-api-key"
```

## 工具

请参阅 [AgentPhone Toolkit](/oss/integrations/tools/agentphone) 指南，了解可用工具的详细信息，包括消息发送、语音通话、电话号码管理等。
