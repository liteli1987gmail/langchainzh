在本教程中，您将构建一个使用检索增强生成（RAG）的客户支持聊天机器人，并在开发的每个阶段添加 LangSmith 可观测性，从早期原型设计到生产环境。

学完本教程后，您将了解如何：

- 跟踪单个 LLM 调用和完整的应用程序管道。
- 收集和查询用户反馈。
- 记录元数据并将其用于过滤和 A/B 测试。
- 使用监控仪表板跟踪生产环境性能。

该应用程序将检索相关的文档片段，并用它们来回答用户问题。本教程中的检索器是模拟的；在实际应用中，您可以用向量搜索或类似技术替换它。

## 前提条件

在开始之前，确保您已具备：

- **LangSmith 账户**：在 [smith.langchain.com](https://smith.langchain.com) 注册或登录。
- **LangSmith API 密钥**：按照[创建 API 密钥](/langsmith/create-account-api-key)指南操作。
- **OpenAI API 密钥**：从 [OpenAI 仪表板](https://platform.openai.com/account/api-keys)生成。
- **LangSmith CLI**（可选）：安装后可在终端检查追踪记录。具体说明请参考 [LangSmith CLI](/langsmith/langsmith-cli)。

安装所需的软件包：

## 原型设计

从一开始设置可观测性可以让你更快地迭代。你可以看到发送到模型的内容、返回的内容以及时间花费在哪里，而无需添加打印语句或运行调试器。

### 设置环境

在 shell 中设置以下环境变量：

```bash
export LANGSMITH_TRACING=true
export LANGSMITH_API_KEY="<your-api-key>"
export OPENAI_API_KEY="<your-openai-api-key>"
```

<project />

<Note>
你可能会在其他地方看到这些变量被引用为 `LANGCHAIN_*`。两种方式都可以使用，但 `LANGSMITH_TRACING` 和 `LANGSMITH_API_KEY` 是推荐的名称。
</Note>

### 追踪 LLM 调用

首先追踪你的 OpenAI 调用，也就是模型实际被调用的地方。这样可以立即了解你的应用发送的提示词和模型返回的响应。

使用 @[`wrap_openai`] (Python) 或 [`wrapOpenAI`](https://reference.langchain.com/javascript/langsmith/wrappers/wrapOpenAI) (TypeScript) 包装 OpenAI 客户端。创建一个名为 `app.py`（或 `app.ts`）的文件，包含以下代码：

<CodeGroup>

</CodeGroup>

```python Python
from openai import OpenAI
from langsmith.wrappers import wrap_openai

client = wrap_openai(OpenAI())

docs = [
    "Acme Cloud supports unlimited users on Enterprise plans. Starter plans are limited to 5 users.",
    "To reset your password, click 'Forgot password' on the login page and follow the instructions sent to your email.",
    "API rate limits are 1,000 requests per hour on the Starter plan and 10,000 requests per hour on Enterprise.",
]

def retriever(query: str) -> list[str]:
    return docs

def support_bot(question: str) -> str:
    context = retriever(question)
    system_message = (
        "You are a helpful customer support agent. "
        "Answer using only the information provided below:\n\n"
        + "\n".join(context)
    )
    response = client.chat.completions.create(
        model="gpt-5.4-mini",
        messages=[
            {"role": "system", "content": system_message},
            {"role": "user", "content": question},
        ],
    )
    return response.choices[0].message.content

if __name__ == "__main__":
    print(support_bot("How many users can I have on the Starter plan?"))
```

```typescript TypeScript
import OpenAI from "openai";
import { wrapOpenAI } from "langsmith/wrappers";

const client = wrapOpenAI(new OpenAI());

const docs = [
    "Acme Cloud supports unlimited users on Enterprise plans. Starter plans are limited to 5 users.",
    "To reset your password, click 'Forgot password' on the login page and follow the instructions sent to your email.",
    "API rate limits are 1,000 requests per hour on the Starter plan and 10,000 requests per hour on Enterprise.",
];

function retriever(query: string): string[] {
    return docs;
}

async function supportBot(question: string): Promise<string> {
    const context = retriever(question);
    const systemMessage =
        "You are a helpful customer support agent. " +
        "Answer using only the information provided below:\n\n" +
        context.join("\n");
    const response = await client.chat.completions.create({
        model: "gpt-5.4-mini",
        messages: [
            { role: "system", content: systemMessage },
            { role: "user", content: question },
        ],
    });
    return response.choices[0].message?.content ?? "";
}

(async () => {
    console.log(await supportBot("How many users can I have on the Starter plan?"));
})();
```

调用 `support_bot("How many users can I have on the Starter plan?")` 会生成 OpenAI 调用的追踪记录。

### 追踪整个管道

追踪 LLM 调用很有用，但是追踪完整管道（包括检索）可以让你完整了解应用程序的行为。将 @[`@traceable`]（Python）或 [`traceable`](https://reference.langchain.com/javascript/langsmith/traceable)（TypeScript）添加到主函数：

```python Python
from openai import OpenAI
from langsmith import traceable
from langsmith.wrappers import wrap_openai

client = wrap_openai(OpenAI())

docs = [
    "Acme Cloud supports unlimited users on Enterprise plans. Starter plans are limited to 5 users.",
    "To reset your password, click 'Forgot password' on the login page and follow the instructions sent to your email.",
    "API rate limits are 1,000 requests per hour on the Starter plan and 10,000 requests per hour on Enterprise.",
]

def retriever(query: str) -> list[str]:
    return docs

@traceable  # [!code highlight]
def support_bot(question: str) -> str:
    context = retriever(question)
    system_message = (
        "You are a helpful customer support agent. "
        "Answer using only the information provided below:\n\n"
        + "\n".join(context)
    )
    response = client.chat.completions.create(
        model="gpt-5.4-mini",
        messages=[
            {"role": "system", "content": system_message},
            {"role": "user", "content": question},
        ],
    )
    return response.choices[0].message.content

if __name__ == "__main__":
    print(support_bot("How many users can I have on the Starter plan?"))
```

### 从终端检查跟踪记录

如果您已安装 [LangSmith CLI](/langsmith/langsmith-cli)，无需打开 UI 即可列出项目的最近跟踪记录：

```bash
langsmith trace list --project <your-project> --limit 5
```

要查看特定追踪的完整运行层级结构及输入/输出：

```bash
langsmith trace get <trace-id> --full
```

## Beta 测试

一旦您的应用在原型设计阶段运行良好，您就可以将其发布给一小部分真实用户。在此阶段，您通常无法确切知道用户将如何与应用交互，因此您需要更丰富的可观测性。您不仅想了解应用做了什么，还想了解用户的反应。

### 收集反馈

将[用户反馈](/langsmith/attach-user-feedback)链接到特定追踪，让您能够识别哪些回复有帮助或无帮助。从上一步更新 `app.py`（或 `app.ts`），为每次调用添加运行 ID，并在之后附加评分：

<CodeGroup>

</CodeGroup>

```python Python
from openai import OpenAI
from langsmith import traceable, Client, uuid7  # [!code highlight]
from langsmith.wrappers import wrap_openai

client = wrap_openai(OpenAI())

docs = [
    "Acme Cloud supports unlimited users on Enterprise plans. Starter plans are limited to 5 users.",
    "To reset your password, click 'Forgot password' on the login page and follow the instructions sent to your email.",
    "API rate limits are 1,000 requests per hour on the Starter plan and 10,000 requests per hour on Enterprise.",
]

def retriever(query: str) -> list[str]:
    return docs

@traceable
def support_bot(question: str) -> str:
    context = retriever(question)
    system_message = (
        "You are a helpful customer support agent. "
        "Answer using only the information provided below:\n\n"
        + "\n".join(context)
    )
    response = client.chat.completions.create(
        model="gpt-5.4-mini",
        messages=[
            {"role": "system", "content": system_message},
            {"role": "user", "content": question},
        ],
    )
    return response.choices[0].message.content

if __name__ == "__main__":
    run_id = str(uuid7())  # [!code highlight]
    support_bot(  # [!code highlight]
        "How many users can I have on the Starter plan?",  # [!code highlight]
        langsmith_extra={"run_id": run_id},  # [!code highlight]
    )  # [!code highlight]
    ls_client = Client()  # [!code highlight]
    ls_client.create_feedback(run_id, key="user-score", score=1.0)  # [!code highlight]
```

```typescript TypeScript
import OpenAI from "openai";
import { wrapOpenAI } from "langsmith/wrappers";
import { traceable, getCurrentRunTree } from "langsmith/traceable"; // [!code highlight]
import { Client } from "langsmith"; // [!code highlight]

const client = wrapOpenAI(new OpenAI());

const docs = [
    "Acme Cloud supports unlimited users on Enterprise plans. Starter plans are limited to 5 users.",
    "To reset your password, click 'Forgot password' on the login page and follow the instructions sent to your email.",
    "API rate limits are 1,000 requests per hour on the Starter plan and 10,000 requests per hour on Enterprise.",
];

function retriever(query: string): string[] {
    return docs;
}

let capturedRunId: string; // [!code highlight]

const supportBot = traceable(async function supportBot(question: string): Promise<string> {
    capturedRunId = getCurrentRunTree().id; // [!code highlight]
    const context = retriever(question);
    const systemMessage =
        "You are a helpful customer support agent. " +
        "Answer using only the information provided below:\n\n" +
        context.join("\n");
    const response = await client.chat.completions.create({
        model: "gpt-5.4-mini",
        messages: [
            { role: "system", content: systemMessage },
            { role: "user", content: question },
        ],
    });
    return response.choices[0].message?.content ?? "";
});

(async () => {
    await supportBot("How many users can I have on the Starter plan?"); // [!code highlight]
    const lsClient = new Client(); // [!code highlight]
    await lsClient.createFeedback(capturedRunId, "user-score", { score: 1.0 }); // [!code highlight]
    await lsClient.flush(); // [!code highlight]
})();
```

</CodeGroup>

<Note>
在生产环境中，这两个部分会位于不同的位置：`support_bot` 调用和 `run_id` 保留在您的应用中，而 `create_feedback` 移至接收用户反馈的端点（例如 `/feedback` API 路由）。`run_id` 在两者之间传递，以便将反馈链接到正确的追踪。
</Note>

反馈会显示在 UI 中检查运行时 **Feedback** 标签页中。然后您可以使用 **Runs** 表中的过滤控件按反馈分数筛选运行。

### 记录元数据

[元数据](/langsmith/add-metadata-tags) 允许您使用有助于筛选和比较的属性标记运行。例如，使用的模型版本或发出请求的用户。

以下示例追踪检索器（使用 `run_type="retriever"`）和主函数（带有模型名称的 `metadata` 属性）：

<CodeGroup>

</fragment>

```python Python
from openai import OpenAI
from langsmith import traceable
from langsmith.wrappers import wrap_openai

client = wrap_openai(OpenAI())

docs = [
    "Acme Cloud supports unlimited users on Enterprise plans. Starter plans are limited to 5 users.",
    "To reset your password, click 'Forgot password' on the login page and follow the instructions sent to your email.",
    "API rate limits are 1,000 requests per hour on the Starter plan and 10,000 requests per hour on Enterprise.",
]

@traceable(run_type="retriever")  # [!code highlight]
def retriever(query: str) -> list[str]:
    return docs

@traceable(metadata={"llm": "gpt-5.4-mini"})  # [!code highlight]
def support_bot(question: str) -> str:
    context = retriever(question)
    system_message = (
        "You are a helpful customer support agent. "
        "Answer using only the information provided below:\n\n"
        + "\n".join(context)
    )
    response = client.chat.completions.create(
        model="gpt-5.4-mini",
        messages=[
            {"role": "system", "content": system_message},
            {"role": "user", "content": question},
        ],
    )
    return response.choices[0].message.content

if __name__ == "__main__":
    support_bot("How many users can I have on the Starter plan?")
```

```typescript TypeScript
import OpenAI from "openai";
import { wrapOpenAI } from "langsmith/wrappers";
import { traceable } from "langsmith/traceable";

const client = wrapOpenAI(new OpenAI());

const docs = [
    "Acme Cloud supports unlimited users on Enterprise plans. Starter plans are limited to 5 users.",
    "To reset your password, click 'Forgot password' on the login page and follow the instructions sent to your email.",
    "API rate limits are 1,000 requests per hour on the Starter plan and 10,000 requests per hour on Enterprise.",
];

const retriever = traceable(  // [!code highlight]
    function retriever(query: string): string[] {  // [!code highlight]
        return docs;  // [!code highlight]
    },  // [!code highlight]
    { run_type: "retriever" }  // [!code highlight]
);  // [!code highlight]

const supportBot = traceable(
    async function supportBot(question: string): Promise<string> {
        const context = await retriever(question);
        const systemMessage =
            "You are a helpful customer support agent. " +
            "Answer using only the information provided below:\n\n" +
            context.join("\n");
        const response = await client.chat.completions.create({
            model: "gpt-5.4-mini",
            messages: [
                { role: "system", content: systemMessage },
                { role: "user", content: question },
            ],
        });
        return response.choices[0].message?.content ?? "";
    },
    { metadata: { llm: "gpt-5.4-mini" } }  // [!code highlight]
);

(async () => {
    await supportBot("How many users can I have on the Starter plan?");
})();
```

</CodeGroup>

两个元数据值都会显示在跟踪中。您可以使用 **Runs** 表中的筛选控件按元数据筛选运行。

## 生产环境

有了强大的可观测性保障，您可以自信地将应用发布到生产环境。在生产环境中，流量会大幅增加，无法逐一检查每条跟踪。LangSmith 提供监控工具，帮助您了解整体行为，并在出现异常时深入分析。

### 监控

在 UI 侧边栏中选择 **Monitoring**，然后从左上角的下拉菜单中选择一个跟踪项目。图表显示项目随时间变化的关键指标，包括跟踪数量、延迟、错误率、反馈分数和成本。有关可用指标和图表配置的更多信息，请参阅 [Dashboards](/langsmith/dashboards)。

### A/B 测试

<Note>
分组功能需要在给定的元数据键下至少有两个不同的值。</Note>

由于您已经记录了 `llm` 元数据属性，因此可以按该属性对监控图表进行分组，以比较模型随时间的性能表现。从 UI 侧边栏的 **Monitoring** 中，点击左上角的 **Group by**，从下拉菜单中选择 **Metadata**，然后选择 `llm`。图表将更新为按该属性分组显示结果。有关分组和自定义图表的更多信息，请参阅 [Dashboards](/langsmith/dashboards)。

### 深入分析

当监控图表显示异常情况时，点击数据点可以冻结工具提示，然后点击指标名称（例如 **Input**）可跳转到该时间窗口的筛选后运行表。有关搜索和筛选运行的更多信息，请参阅 [Filter traces](/langsmith/filter-traces-in-application)。
