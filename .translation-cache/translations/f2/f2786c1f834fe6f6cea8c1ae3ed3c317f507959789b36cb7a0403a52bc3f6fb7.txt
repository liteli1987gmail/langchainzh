# Moorcheh

[Moorcheh](https://www.moorcheh.ai/) 是一个超快速的语义搜索引擎和向量存储。与使用 L2 或 Cosine 等简单距离度量不同，Moorcheh 使用最大信息二值化（Maximally Informative Binarization, MIB）和信息论评分（Information-Theoretic Score, ITS）来检索准确的文档块。

以下教程将帮助您使用 Moorcheh 和 LangChain 上传和存储文本文档及向量嵌入，同时为您的所有查询检索相关块。

## 设置

首先，安装必要的包：

```bash
pip install langchain-moorcheh
```

## 初始化

开始使用 Moorcheh

1. 在 [Moorcheh 控制台](https://console.moorcheh.ai/) 注册或登录。
2. 进入 "API Keys" 标签页并生成一个 API 密钥。
3. 将该密钥保存为名为 `MOORCHEH_API_KEY` 的环境变量（你将在下面使用它）。
4. 创建一个用于存储数据的命名空间：
   - 在控制台中，打开 "Namespaces" 标签页并点击 "Create namespace"；或者
   - 使用下一节中的向量存储代码以编程方式初始化。
5. 使用你的 API 密钥创建命名空间、上传文档和检索答案。

有关 Moorcheh SDK 函数的更多信息，请参阅 [GitHub 仓库](https://github.com/moorcheh-ai/moorcheh-python-sdk)。

## 导入包

导入以下包：

```python
from langchain_moorcheh import MoorchehVectorStore
from moorcheh_sdk import MoorchehClient

import logging
import os
from uuid import uuid4
import asyncio
from typing import Any, List, Optional, Literal, Tuple, Type, TypeVar, Sequence
from langchain_core.documents import Document
from langchain_core.embeddings import Embeddings
from langchain_core.vectorstores import VectorStore
from google.colab import userdata
```

## 代码设置

在环境变量中设置您的 Moorcheh API Key：

```python
MOORCHEH_API_KEY = os.environ['MOORCHEH_API_KEY']
```

设置您的命名空间名称、类型，并创建向量存储：

```python
namespace = "your_namespace_name"
namespace_type = "text" # or vector
store = MoorchehVectorStore(
            api_key=MOORCHEH_API_KEY,
            namespace=namespace,
            namespace_type=namespace_type
        )
```

```python
store.delete(ids=["chunk_id_here"])
```

## 查询引擎

创建命名空间并向其上传文档后，您可以直接通过向量存储库对文档进行查询。设置查询内容和您希望回答查询的大语言模型。如需了解更多关于支持的 LLM 的信息，请访问我们的 [GitHub 页面](https://github.com/moorcheh-ai/moorcheh-python-sdk)。

```python
query = "Give me a brief summary of the provided documents"
answer = store.generative_answer(query, ai_model = "anthropic.claude-sonnet-4-6")
print(answer)
```
