当您准备将 LangGraph 代理部署到生产环境时，请选择适合您技术栈的托管模式。**[LangSmith Cloud](/langsmith/deploy-to-cloud)** 为有状态、长时间运行的代理提供完全托管的基础设施，支持持久状态和后台执行。

:::js
您还可以使用相同的 [代理流协议](https://github.com/langchain-ai/agent-protocol/tree/main/streaming)，在 **JavaScript 框架和平台**（如 Next.js、SvelteKit、Nuxt、Cloudflare Workers 和 Deno Deploy）上进行部署。

:::

<Tip>
LangSmith 除了 Cloud 之外还提供多种部署选项，包括[混合部署](/langsmith/hybrid)、[独立服务器](/langsmith/deploy-standalone-server)和[带控制平面的自托管](/langsmith/deploy-with-control-plane)。有关更多信息，请参阅 [LangSmith 部署概览](/langsmith/deployment)。
</Tip>

## LangSmith Cloud

本节将引导您完成从 GitHub 仓库将代理部署到 LangSmith Cloud 的过程。LangSmith 负责处理基础设施、扩展和运维相关的事务。

### 前置条件

在开始之前，请确保您满足以下条件：

- 一个 [GitHub 账户](https://github.com/)
- 一个 [LangSmith 账户](https://smith.langchain.com)（免费注册）

### 部署您的代理

#### 1. 在 GitHub 上创建仓库

您的应用程序代码必须位于 GitHub 仓库中才能部署到 LangSmith。公开和私有仓库均受支持。对于此快速入门，首先请确保您的应用与 LangGraph 兼容，方法遵循[本地服务器设置指南](/oss/langgraph/studio#set-up-local-agent-server)。然后，将您的代码推送到仓库。

#### 2. 部署到 LangSmith

<Steps>
<Step title="导航到 LangSmith 部署">
    登录 [LangSmith](https://smith.langchain.com)。在左侧边栏中，选择**部署**。
</Step>
<Step title="创建新部署">
    点击 **+ 新建部署** 按钮。将打开一个面板供您填写必填字段。
</Step>
<Step title="链接仓库">
    如果您是首次使用或添加尚未连接的私有仓库，请点击 **添加新账户** 按钮并按照说明连接您的 GitHub 账户。
</Step>
<Step title="部署仓库">
    选择您的应用程序仓库。点击**提交**开始部署。这可能需要约 15 分钟完成。您可以在**部署详情**视图中检查状态。
</Step>
</Steps>

#### 3. 在 Studio 中测试您的应用

一旦您的应用程序部署完成：

1. 选择您刚刚创建的部署以查看更多详情。
2. 点击右上角的 **Studio** 按钮。Studio 将打开并显示您的图。

#### 4. 获取部署的 API URL

1. 在 LangGraph 的**部署详情**视图中，点击 **API URL** 将其复制到剪贴板。
2. 点击该 `URL` 将其复制到剪贴板。

#### 5. 测试 API

您现在可以测试 API 了：

<Tabs>
:::python
<Tab title="Python">
1. 安装 LangGraph SDK：
```shell
pip install langgraph-sdk
```
2. 向代理发送消息：
```python
from langgraph_sdk import get_sync_client # or get_client for async

client = get_sync_client(url="your-deployment-url", api_key="your-langsmith-api-key")

for chunk in client.runs.stream(
    None,    # Threadless run
    "agent", # Name of agent. Defined in langgraph.json.
    input={
        "messages": [{
            "role": "human",
            "content": "What is LangGraph?",
        }],
    },
    stream_mode="updates",
):
    print(f"Receiving new event of type: {chunk.event}...")
    print(chunk.data)
    print("\n\n")
```
</Tab>
:::
:::js
<Tab title="TypeScript">
1. 安装 LangGraph SDK：

</fragment>
```shell
npm install @langchain/langgraph-sdk
```
2. 向代理发送消息：
```ts
import { Client } from "@langchain/langgraph-sdk";

const client = new Client({ apiUrl: "your-deployment-url", apiKey: "your-langsmith-api-key" });

const streamResponse = client.runs.stream(
  null,    // Threadless run
  "agent", // Name of agent. Defined in langgraph.json.
  {
    input: {
      "messages": [
        { "role": "user", "content": "What is LangGraph?"}
      ]
    },
    streamMode: "messages",
  }
);

for await (const chunk of streamResponse) {
  console.log(`Receiving new event of type: ${chunk.event}...`);
  console.log(JSON.stringify(chunk.data));
  console.log("\n\n");
}
```
</Tab>
