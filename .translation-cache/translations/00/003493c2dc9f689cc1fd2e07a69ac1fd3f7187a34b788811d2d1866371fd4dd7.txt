>[Docugami](https://docugami.com) 将商业文档转换为文档 XML 知识图谱，生成表示整个文档的 XML 语义树森林。这是一种丰富的表示形式，将文档中各种文本块的语义和结构特征表示为 XML 树。

## 安装和设置

<CodeGroup>

</CodeGroup>
```bash pip
pip install dgml-utils
pip install docugami-langchain
```

```bash uv
uv add dgml-utils
uv add docugami-langchain
```
</CodeGroup>

## 文档加载器

查看[使用示例](/oss/integrations/document_loaders/docugami)。

from docugami_langchain.document_loaders import DocugamiLoader
