>[Nomic](https://www.nomic.ai/) 构建工具，使每个人都能与 AI 规模的数据集交互，并在消费级电脑上运行 AI 模型。

## 安装

<CodeGroup>
```bash pip
pip install -U langchain-nomic
```

```bash uv
uv add langchain-nomic
```
</CodeGroup>

## 嵌入模型

### NomicEmbeddings

查看 [使用示例](/oss/integrations/embeddings/nomic).

```python
from langchain_nomic import NomicEmbeddings
```
