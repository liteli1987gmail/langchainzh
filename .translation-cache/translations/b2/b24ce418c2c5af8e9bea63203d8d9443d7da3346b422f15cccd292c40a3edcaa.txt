### 连接到您的智能体

智能体聊天界面可以连接到 [本地](/oss/langgraph/studio#set-up-local-agent-server) 和 [已部署的智能体](/oss/langgraph/deploy).

启动智能体聊天界面后，您需要配置它以连接到您的智能体：

1. **图 ID**：输入您的图名称（可在以下位置找到 `graphs` 中的 `langgraph.json` 文件）
2. **部署 URL**：您的智能体服务器端点（例如， `http://localhost:2024` 用于本地开发，或您已部署智能体的 URL）
3. **LangSmith API 密钥（可选）**：添加您的 LangSmith API 密钥（如果使用本地智能体服务器则不需要）

配置完成后，智能体聊天界面将自动获取并显示您智能体中的任何中断线程。

<Tip>
  智能体聊天界面支持开箱即用地渲染工具调用和工具结果消息。如需自定义显示的消息，请参阅 [隐藏聊天中的消息](https://github.com/langchain-ai/agent-chat-ui?tab=readme-ov-file#hiding-messages-in-the-chat).
</Tip>
