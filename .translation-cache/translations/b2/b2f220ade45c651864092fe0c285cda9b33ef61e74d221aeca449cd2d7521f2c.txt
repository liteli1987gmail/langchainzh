>[TruLens](https://trulens.org) 是一个[开源](https://github.com/truera/trulens)包，为基于大型语言模型（LLM）的应用提供工具和评估工具。

本页介绍如何使用 [TruLens](https://trulens.org) 来评估和追踪基于 langchain 构建的 LLM 应用。


## 安装和设置

安装 `trulens-eval` Python 包。

<CodeGroup>

</CodeGroup>
```bash pip
pip install trulens-eval
```

```bash uv
uv add trulens-eval
```
</CodeGroup>

## 快速开始

请参阅 [TruLens 文档](https://www.trulens.org/trulens_eval/getting_started/quickstarts/langchain_quickstart/) 中的集成详情。

### 跟踪

创建 LLM 链后，您可以使用 TruLens 进行评估和跟踪。
TruLens 提供多种[开箱即用的反馈函数](https://www.trulens.org/trulens_eval/evaluation/feedback_functions/)，
同时也是一个可扩展的 LLM 评估框架。

创建反馈函数：

```python
from trulens_eval.feedback import Feedback, Huggingface,

# Initialize HuggingFace-based feedback function collection class:
hugs = Huggingface()
openai = OpenAI()

# Define a language match feedback function using HuggingFace.
lang_match = Feedback(hugs.language_match).on_input_output()
# By default this will check language match on the main app input and main app
# output.

# Question/answer relevance between overall question and answer.
qa_relevance = Feedback(openai.relevance).on_input_output()
# By default this will evaluate feedback on main app input and main app output.

# Toxicity of input
toxicity = Feedback(openai.toxicity).on_input()
```

### Chains

在为您的 LLM 设置好评估反馈函数后，您可以使用 TruChain 包装您的应用程序，以获得对 LLM 应用程序的详细追踪、日志记录和评估。

注意：有关 `chain` 创建的代码，请参阅 [TruLens 文档](https://www.trulens.org/trulens_eval/getting_started/quickstarts/langchain_quickstart/)。

```python
from trulens_eval import TruChain

# wrap your chain with TruChain
truchain = TruChain(
    chain,
    app_id='Chain1_ChatApplication',
    feedbacks=[lang_match, qa_relevance, toxicity]
)
# Note: any `feedbacks` specified here will be evaluated and logged whenever the chain is used.
truchain("que hora es?")
```

### 评估

现在您可以开始探索您的基于 LLM 的应用程序了！

这样做将帮助您一目了然地了解 LLM 应用程序的运行状况。在您迭代 LLM 应用程序的新版本时，您可以比较它们在您设置的所有不同质量指标上的表现。您还能够按记录级别查看评估，并探索每条记录的链元数据。

```python
from trulens_eval import Tru

tru = Tru()
tru.run_dashboard() # open a Streamlit app to explore
```

欲了解更多关于 TruLens 的信息，请访问 [trulens.org](https://www.trulens.org/)
