Oracle AI 数据库专为 AI 工作负载设计，在这些工作负载中，您通过 **语义** （语义）而非关键词来查询数据。它将 **非结构化内容的语义搜索** 与 **业务数据的关系过滤** 结合在单一系统中——因此您可以构建 AI 检索工作流（如 RAG），而无需引入单独的向量数据库并导致数据分散在多个平台上。

### 为什么这很重要

- **AI + 业务数据的统一系统：** 向量和运营数据共存。
- **减少碎片化：** 比单独的向量存储更少的组件。
- **数据库级保证：** 将安全性、事务、扩展性和可用性应用于相同的 AI 工作负载。

## 前提条件

安装 `langchain-oracledb`。 `python-oracledb` 驱动程序将作为依赖项自动安装。

<CodeGroup>
```bash pip
pip install -qU langchain-oracledb
```
```bash uv
uv add langchain-oracledb
```
</CodeGroup>
