Oracle AI Database 专为 AI 工作负载而构建，在这些工作负载中，您通过**含义**（语义）而非仅通过关键词来查询数据。它将**非结构化内容的语义搜索**与**业务数据的关系过滤**结合在一个系统中，使您能够构建 AI 检索工作流（如 RAG），而无需引入单独的向量数据库，也不会将数据分散到多个平台上。

### 为什么它很重要

- **一个系统处理 AI + 业务数据：** 向量和操作数据共存于一处。
- **减少数据碎片化：** 与独立的向量存储相比，移动部件更少。
- **数据库级保障：** 将安全性、事务、扩展性和可用性应用于相同的 AI 工作负载。

## 前置条件

安装 `langchain-oracledb`。`python-oracledb` 驱动将作为依赖项自动安装。

<CodeGroup>

</CodeGroup>
```bash pip
pip install -qU langchain-oracledb
```
```bash uv
uv add langchain-oracledb
```
</CodeGroup>
