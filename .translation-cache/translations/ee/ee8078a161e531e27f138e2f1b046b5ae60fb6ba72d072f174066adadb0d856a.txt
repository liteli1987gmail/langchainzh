[Perplexity Search API](https://docs.perplexity.ai/docs/search/quickstart) 返回实时、可靠的网页搜索结果。`PerplexitySearchResults` 是 LangChain [工具](/oss/integrations/tools/) 包装器，允许代理查询 API 并接收搜索结果的 JSON 数组。

## 概览

| 类 | 包 | [Python 支持](https://python.langchain.com/docs/integrations/tools/perplexity_search/) | 版本 |
| :--- | :--- | :---: | :---: |
| [`PerplexitySearchResults`](https://reference.langchain.com/javascript/langchain-perplexity/PerplexitySearchResults) | [`@langchain/perplexity`](https://www.npmjs.com/package/@langchain/perplexity) | ✅ | ![NPM - 版本](https://img.shields.io/npm/v/@langchain/perplexity?style=flat-square&label=%20&) |

## 安装

该集成位于 `@langchain/perplexity` 包中：

<CodeGroup>

</fragment>
```bash npm
npm install @langchain/perplexity @langchain/core
```
```bash yarn
yarn add @langchain/perplexity @langchain/core
```
```bash pnpm
pnpm add @langchain/perplexity @langchain/core
```
</CodeGroup>

### 凭证

设置 [Perplexity API 密钥](https://www.perplexity.ai/account/api/keys) 并将其设置为名为 `PERPLEXITY_API_KEY` 的环境变量：

```typescript
process.env.PERPLEXITY_API_KEY = "your-api-key";
```

这也很有帮助（但不是必需的），可以设置 [LangSmith](https://smith.langchain.com/) 以获得一流的可观测性：

```typescript
process.env.LANGSMITH_TRACING = "true";
process.env.LANGSMITH_API_KEY = "your-api-key";
```

## 实例化

```typescript
import { PerplexitySearchResults } from "@langchain/perplexity";

const tool = new PerplexitySearchResults({
  maxResults: 5,
  // searchDomainFilter: ["wikipedia.org"],
  // searchRecencyFilter: "week",
});
```

构造函数选项包括 `maxResults`、`country`、`searchDomainFilter`、`searchRecencyFilter`、`searchAfterDate` 和 `searchBeforeDate`。详见 [Perplexity Search API 参数](https://docs.perplexity.ai/docs/search/quickstart)。

## 调用

### 直接调用

```typescript
const result = await tool.invoke("Who won the most recent FIFA World Cup?");
console.log(result);
```

该工具返回一个 JSON 编码的搜索结果数组，每个结果包含 `title`、`url`、`snippet`、`date` 和 `last_updated`。

### 与代理配合使用

```typescript
import { ChatAnthropic } from "@langchain/anthropic";
import { createReactAgent } from "@langchain/langgraph/prebuilt";

const llm = new ChatAnthropic({ model: "claude-3-5-haiku-latest" });

const agent = createReactAgent({
  llm,
  tools: [tool],
});

const response = await agent.invoke({
  messages: [
    { role: "user", content: "What's the latest news on the Mars Sample Return mission?" },
  ],
});
```
