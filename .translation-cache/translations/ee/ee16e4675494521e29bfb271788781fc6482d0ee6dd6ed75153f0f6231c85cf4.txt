>[Azure Blob Storage](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blobs-introduction) 是 Microsoft 的云对象存储解决方案。Blob Storage 专为存储大量非结构化数据而优化。非结构化数据是不遵循特定数据模型或定义的数据，例如文本或二进制数据。

`Azure Blob Storage` 适用于以下场景：

- 直接向浏览器提供图像或文档。
- 存储文件以供分布式访问。
- 流式传输视频和音频。
- 写入日志文件。
- 存储数据用于备份和还原、灾难恢复以及存档。
- 存储数据以供本地或 Azure 托管服务进行分析。

本笔记本介绍了如何从 `Azure Blob Storage` 上的容器加载文档对象。有关文档加载器的更多详细文档，请参阅 [Azure Blob Storage Loader API 参考](https://reference.langchain.com/python/integrations/langchain_azure/storage/)。

<Note>
建议使用这个新的加载器来替代之前来自 `langchain_community` 的 [`AzureBlobStorageFileLoader`](https://reference.langchain.com/python/langchain-community/document_loaders/azure_blob_storage_file/AzureBlobStorageFileLoader) 和 [`AzureBlobStorageContainerLoader`](https://reference.langchain.com/python/langchain-community/document_loaders/azure_blob_storage_container/AzureBlobStorageContainerLoader)。有关迁移到新加载器的详细说明，请参阅[迁移指南](https://github.com/langchain-ai/langchain-azure/blob/main/libs/azure-storage/README.md#migrating-from-langchain-community-azure-storage-document-loaders)。
</Note>

from langchain_azure_storage.document_loaders import AzureBlobStorageLoader

## 从容器加载

`AzureBlobStorageLoader` 从 Azure Blob Storage 中的给定容器加载所有 blob，需要提供[账户 URL 和容器名称](https://learn.microsoft.com/en-us/rest/api/storageservices/Naming-and-Referencing-Containers--Blobs--and-Metadata#resource-uri-syntax)。该加载器返回包含 blob 内容（默认为 UTF-8 编码）和元数据（包括 blob URL）的[`Document`](https://reference.langchain.com/python/langchain_core/documents/#langchain_core.documents.base.Document)对象，如下例所示。

无需显式配置凭据，因为它使用[`DefaultAzureCredential`](https://learn.microsoft.com/en-us/azure/developer/python/sdk/authentication/credential-chains?tabs=dac#defaultazurecredential-overview)，可根据当前环境自动检索 [Microsoft Entra ID 令牌](https://learn.microsoft.com/en-us/azure/storage/blobs/authorize-access-azure-active-directory)。

```python
loader = AzureBlobStorageLoader(
    "https://<storage-account-name>.blob.core.windows.net",
    "<container-name>",
)

for doc in loader.load():
    print(doc)
```

```python
page_content='Lorem ipsum dolor sit amet.' metadata={'source': 'https://<storage-account-name>.blob.core.windows.net/<container-name>/<blob-name>'}
```

你也可以指定一个前缀，只返回以该前缀开头的对象。

```python
loader = AzureBlobStorageLoader(
    "https://<storage-account-name>.blob.core.windows.net",
    "<container-name>",
    prefix="<prefix>",
)
```

## 按 blob 名称从容器加载

您可以从 blob 名称列表加载文档，这只会使用提供的 blob，而不会调用列出 blob 的 API。

```python
loader = AzureBlobStorageLoader(
    "https://<storage-account-name>.blob.core.windows.net",
    "<container-name>",
    blob_names=["blob-1", "blob-2", "blob-3"],
)
```

## 覆盖默认凭据

默认情况下，文档加载器使用 [`DefaultAzureCredential`](https://learn.microsoft.com/en-us/azure/developer/python/sdk/authentication/credential-chains?tabs=dac#defaultazurecredential-overview)。以下示例展示了如何覆盖此默认配置：

```python
from azure.core.credentials import AzureSasCredential
from azure.identity import ManagedIdentityCredential
from langchain_azure_storage.document_loaders import AzureBlobStorageLoader

# Override with SAS token
loader = AzureBlobStorageLoader(
    "https://<storage-account-name>.blob.core.windows.net",
    "<container-name>",
    credential=AzureSasCredential("<sas-token>")
)

# Override with more specific token credential than the entire
# default credential chain (e.g., system-assigned managed identity)
loader = AzureBlobStorageLoader(
    "https://<storage-account-name>.blob.core.windows.net",
    "<container-name>",
    credential=ManagedIdentityCredential()
)
```

## 自定义 blob 内容解析

目前，默认情况下，解析每个 blob 时会将其内容作为单个采用 UTF-8 编码的 @[`Document`] 对象返回，无论文件类型如何。对于需要特定解析的文件类型（例如 PDF、CSV 等），或者当您想要控制文档内容格式时，可以提供 `loader_factory` 参数来使用已有的文档加载器（例如 PyPDFLoader、CSVLoader 等）或自定义加载器。

其工作原理是将 blob 内容下载到临时文件。然后使用文件路径调用 `loader_factory`，以使用指定的文档加载器来加载/解析文件并返回 @[`Document`] 对象。

下面展示如何使用 [PyPDFLoader](https://reference.langchain.com/python/langchain-community/document_loaders/pdf/PyPDFLoader) 覆盖用于将 blob 解析为 PDF 的默认加载器：

<LangchainCommunityUnmaintained />

```python
from langchain_azure_storage.document_loaders import AzureBlobStorageLoader
from langchain_community.document_loaders import PyPDFLoader  # This example requires installing `langchain-community` and `pypdf`

loader = AzureBlobStorageLoader(
    "https://<storage-account-name>.blob.core.windows.net",
    "<container-name>",
    blob_names="<pdf-file.pdf>",
    loader_factory=PyPDFLoader,
)

for doc in loader.lazy_load():
    print(doc.page_content)  # Prints content of each page as a separate document
```

要提供额外的配置，你可以定义一个可调用对象，它返回一个实例化的文档加载器，如下所示：

```python
from langchain_azure_storage.document_loaders import AzureBlobStorageLoader
from langchain_community.document_loaders import PyPDFLoader  # This example requires installing `langchain-community` and `pypdf`

def loader_factory(file_path: str) -> PyPDFLoader:
    return PyPDFLoader(
        file_path,
        mode="single",  # To return the PDF as a single document instead of extracting documents by page
    )

loader = AzureBlobStorageLoader(
    "https://<storage-account-name>.blob.core.windows.net",
    "<container-name>",
    blob_names="<pdf-file.pdf>",
    loader_factory=loader_factory,
)

for doc in loader.lazy_load():
    print(doc.page_content)
```
