>Memgraph 是一个高性能的内存图数据库，针对实时查询和分析进行了优化。
>访问以下网站开始使用 Memgraph [其官网](https://memgraph.com/).

## 安装和设置

- 使用以下命令安装 Python SDK `pip install langchain-memgraph`

## MemgraphQAChain

存在一个围绕 Memgraph 数据库的包装器，允许您根据用户输入生成 Cypher 语句
并使用它们从数据库中检索相关信息。

```python
from langchain_memgraph.chains.graph_qa import MemgraphQAChain
from langchain_memgraph.graphs.memgraph import MemgraphLangChain
```

请参阅 [使用示例](/oss/integrations/graphs/memgraph)

## 从非结构化数据构建知识图谱

您可以使用该集成从非结构化数据构建知识图谱。

<LangchainExperimentalUnmaintained />

```python
from langchain_memgraph.graphs.memgraph import MemgraphLangChain
from langchain_experimental.graph_transformers import LLMGraphTransformer
```

请参阅 [使用示例](/oss/integrations/graphs/memgraph)

## Memgraph 工具和工具包

Memgraph 还提供了一个工具包，允许您与 Memgraph 数据库进行交互。
请参阅 [使用示例](/oss/integrations/tools/memgraph).

```python
from langchain_memgraph import MemgraphToolkit
```
