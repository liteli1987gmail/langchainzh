> Neo4j 是一个 `open-source database management system`，专门从事图数据库技术。
> Neo4j 允许您用节点和边来表示和存储数据，使其成为处理关联数据和关系的理想选择。
> Neo4j 提供了 `Cypher Query Language`，让您可以轻松地与图数据进行交互和查询。
> 使用 Neo4j，您可以实现高性能的 `graph traversals and queries`，适用于生产级系统。

> 通过访问 [他们的网站](https://neo4j.com/) 开始使用 Neo4j。

## 安装和设置

- 使用 `pip install neo4j langchain-neo4j` 安装 Python SDK

## VectorStore

Neo4j 向量索引被用作 vectorstore，无论是语义搜索还是示例选择。

```python
from langchain_neo4j import Neo4jVector
```

请参阅[使用示例](/oss/integrations/vectorstores/neo4jvector)。

## GraphCypherQAChain

Neo4j 图形数据库存在一个封装器，允许你根据用户输入生成 Cypher 语句，并使用这些语句从数据库中检索相关信息。

```python
from langchain_neo4j import GraphCypherQAChain, Neo4jGraph
```

查看[使用示例](/oss/integrations/graphs/neo4j_cypher)

## 从文本构建知识图谱

文本数据通常包含丰富的关​​系和洞察，可用于各种分析、推荐引擎或知识管理应用。
Diffbot 的 NLP API 可以从非结构化文本数据中提取实体、关系和语义信息。
通过将 Diffbot 的 NLP API 与图数据库 Neo4j 结合，您可以基于从文本中提取的信息创建强大的动态图结构。
这些图结构可完全查询，并可集成到各种应用中。

<LangchainExperimentalUnmaintained />

```python
from langchain_neo4j import Neo4jGraph
from langchain_experimental.graph_transformers.diffbot import DiffbotGraphTransformer
```

## Checkpoint 保存器

Neo4j 实现的 LangGraph Checkpoint 保存器，用于持久化 GitHub 代理内存，支持分支时间旅行。Checkpoint 保存器在每个 super-step 持久化图状态，支持会话记忆、人机协作工作流、时间旅行和容错。
适用于 LangGraph 图和 LangChain 代理：

```python
from langchain_neo4j import Neo4jSaver

with Neo4jSaver.from_conn_string(
    uri="bolt://localhost:7687",
    user="neo4j",
    password="password"
) as checkpointer:
    checkpointer.setup()  # Create indexes (run once)
    # Create agent with checkpointer
    agent = create_agent(
        model="google_genai:gemini-3.5-flash",
        tools=[get_weather],
        system_prompt="You are a helpful assistant",
        checkpointer=checkpointer
    )
```
