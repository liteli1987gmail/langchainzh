LangChain.js 支持调用 [YandexGPT](https://cloud.yandex.com/en/services/yandexgpt) LLMs.

## 设置

首先，您应该 [创建服务账户](https://cloud.yandex.com/en/docs/iam/operations/sa/create) 并 `ai.languageModels.user` role.

接下来，您有两种身份验证选项：

- [IAM 令牌](https://cloud.yandex.com/en/docs/iam/operations/iam-token/create-for-sa).
  您可以在构造函数参数中指定令牌 `iam_token` 或通过环境变量指定 `YC_IAM_TOKEN`.
- [API 密钥](https://cloud.yandex.com/en/docs/iam/operations/api-key/create)
  您可以在构造函数参数中指定密钥 `api_key` 或通过环境变量指定 `YC_API_KEY`.

## 用法

<Tip>
请参阅 [获取安装 LangChain 包的一般说明](/oss/langchain/install).
</Tip>

```bash npm
npm install @langchain/yandex @langchain/core
```

```typescript
import { YandexGPT } from "@langchain/yandex/llms";

const model = new YandexGPT();
const res = await model.invoke(['Translate "I love programming" into French.']);
console.log({ res });
```
