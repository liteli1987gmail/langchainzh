LangChain.js 支持调用 [YandexGPT](https://cloud.yandex.com/en/services/yandexgpt) LLM。

## 设置

首先，你需要[创建服务账号](https://cloud.yandex.com/en/docs/iam/operations/sa/create)并为其分配 `ai.languageModels.user` 角色。

接下来，你有两个身份验证选项：

- [IAM 令牌](https://cloud.yandex.com/en/docs/iam/operations/iam-token/create-for-sa)。
  你可以在构造函数参数 `iam_token` 或环境变量 `YC_IAM_TOKEN` 中指定令牌。
- [API 密钥](https://cloud.yandex.com/en/docs/iam/operations/api-key/create)
  你可以在构造函数参数 `api_key` 或环境变量 `YC_API_KEY` 中指定密钥。

## 用法

<Tip>
请参阅[此部分了解安装 LangChain 包的一般说明](/oss/langchain/install)。
</Tip>

```bash npm
npm install @langchain/yandex @langchain/core
```

```typescript
import { YandexGPT } from "@langchain/yandex/llms";

const model = new YandexGPT();
const res = await model.invoke(['Translate "I love programming" into French.']);
console.log({ res });
```
