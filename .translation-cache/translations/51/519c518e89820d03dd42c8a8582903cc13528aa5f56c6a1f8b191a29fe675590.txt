本页面介绍如何在 [MariaDB](https://github.com/mariadb/) 生态系统中使用它。
它分为两个部分：安装和设置，然后是特定 PGVector 包装器的参考。

## 安装
- Install c/c connector

在 Debian、Ubuntu 上
```bash
sudo apt install libmariadb3 libmariadb-dev
```

在 CentOS、RHEL、Rocky Linux 上
```bash
sudo yum install MariaDB-shared MariaDB-devel
```

- 使用以下命令安装 Python 连接器包 `pip install mariadb`


## 设置
1. 第一步是安装 MariaDB 11.7.1 或更高版本。

    Docker 镜像是最简单的入门方式。

## 包装器

### 向量存储

存在一个围绕 MariaDB 向量数据库的包装器，允许你将其用作向量存储，
无论是用于语义搜索还是示例选择。

导入此向量存储：
```python
from langchain_mariadb import MariaDBStore
```
