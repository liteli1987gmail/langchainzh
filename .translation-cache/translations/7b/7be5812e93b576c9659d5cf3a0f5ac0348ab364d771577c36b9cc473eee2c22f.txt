# Deep Agents

Deep Agents 是开始构建由 LLMs 驱动的智能体的最简单方式——具备任务规划、用于上下文管理的文件系统、子智能体委托和长期记忆的内置功能。它是一个基于 [LangChain](https://docs.langchain.com/oss/langchain/overview) 核心构建块和 [LangGraph](https://docs.langchain.com/oss/langgraph/overview) runtime.

## 何时使用

在需要以下情况时使用 Deep Agents：
- **快速构建智能体** 配备开箱即用的默认配置和最简配置
- **处理复杂的、多步骤任务** 受益于自动规划
- **管理上下文** 使用内置虚拟文件系统处理大型输入
- **委托子任务** 给专门的子智能体
- **安全运行代码** 在沙箱执行环境中
- **使用终端智能体** 通过 Deep Agents Code

## 何时不使用

- 对于没有规划或子智能体的简单工具调用智能体，使用 [LangChain](https://docs.langchain.com/oss/langchain/overview) 智能体更轻量
- 对于需要显式控制流的自定义基于图的编排，使用 [LangGraph](https://docs.langchain.com/oss/langgraph/overview) 直接进行
- Deep Agents 是 **最高层次的抽象**——它以灵活性换取便利性

## 安装

```bash
# Python
pip install deepagents

# JavaScript/TypeScript
npm install deepagents langchain @langchain/core
```

## 快速参考

### 创建 deep agent

```python
# pip install deepagents langchain-anthropic
from deepagents import create_deep_agent

def get_weather(city: str) -> str:
    """Get weather for a given city."""
    return f"It's always sunny in {city}!"

agent = create_deep_agent(
    model="anthropic:claude-sonnet-4-6",
    tools=[get_weather],
    system_prompt="You are a helpful assistant",
)

result = agent.invoke(
    {"messages": [{"role": "user", "content": "What is the weather in SF?"}]}
)
```

### 使用 Deep Agents Code

```bash
# Install Deep Agents Code
pip install deepagents-code

# Run an interactive terminal agent
deepagents
```

### 内置功能

| 功能 | 描述 |
|-----------|-------------|
| 规划 | 针对复杂请求的自动任务分解 |
| 文件系统 | 用于读取、写入和管理上下文的虚拟文件系统 |
| 子智能体 | 为并行子任务执行生成子智能体 |
| 上下文管理 | 针对长对话的自动上下文压缩 |
| 沙箱执行 | 在隔离环境中运行代码（Modal、Runloop、Daytona）|
| 协议 | 支持 ACP、MCP 和 A2A 以实现互操作性 |
