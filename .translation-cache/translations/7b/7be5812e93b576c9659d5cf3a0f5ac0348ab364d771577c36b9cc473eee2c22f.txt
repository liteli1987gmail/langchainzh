# Deep Agents

Deep Agents 是开始构建由 LLM 驱动的代理的最简单方式——具备任务规划、内置虚拟文件系统用于上下文管理、子代理委托和长期记忆等内置能力。它是一个基于 [LangChain](https://docs.langchain.com/oss/langchain/overview) 核心构建块和 [LangGraph](https://docs.langchain.com/oss/langgraph/overview) 运行时的“代理工具包”。

## 何时使用

在以下情况下使用 Deep Agents：
- **快速构建代理**，使用合理的默认配置和最少设置
- **处理复杂的多步骤任务**，受益于自动规划
- **管理上下文**，使用内置虚拟文件系统处理大型输入
- **将子任务委托**给专门的子代理
- **安全运行代码**，在沙箱执行环境中
- **使用终端代理**，通过 Deep Agents Code

## 何时不使用

- 对于没有规划或子代理的简单工具调用代理，请改用 [LangChain](https://docs.langchain.com/oss/langchain/overview) 代理——更轻量
- 对于需要显式控制流的自定义基于图的编排，请直接使用 [LangGraph](https://docs.langchain.com/oss/langgraph/overview)
- Deep Agents 是**最高级别的抽象**——它以灵活性换取便利性

## 安装

```bash
# Python
pip install deepagents

# JavaScript/TypeScript
npm install deepagents langchain @langchain/core
```

## 快速参考

### 创建深度代理

```python
# pip install deepagents langchain-anthropic
from deepagents import create_deep_agent

def get_weather(city: str) -> str:
    """Get weather for a given city."""
    return f"It's always sunny in {city}!"

agent = create_deep_agent(
    model="anthropic:claude-sonnet-4-6",
    tools=[get_weather],
    system_prompt="You are a helpful assistant",
)

result = agent.invoke(
    {"messages": [{"role": "user", "content": "What is the weather in SF?"}]}
)
```

### 使用深度代理代码

```bash
# Install Deep Agents Code
pip install deepagents-code

# Run an interactive terminal agent
deepagents
```

### 内置能力

| 能力 | 描述 |
|-----------|-------------|
| 规划 | 复杂请求的自动任务分解 |
| 文件系统 | 用于读取、写入和管理上下文的虚拟文件系统 |
| 子代理 | 生成子代理以并行执行子任务 |
| 上下文管理 | 长时间对话的自动上下文压缩 |
| 沙箱执行 | 在隔离环境中运行代码（Modal、Runloop、Daytona） |
| 协议 | 支持 ACP、MCP 和 A2A 以实现互操作性 |
