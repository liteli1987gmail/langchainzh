LangSmith Deployment 支持部署 [LangGraph](/oss/python/langgraph/overview) _graph_（图）。然而，图的 _node_（节点）实现可以包含任意代码。这意味着任何框架都可以在节点内实现并部署在 LangSmith Deployment 上。这让您无需使用其他 LangGraph OSS API 即可实现核心应用逻辑，同时仍可使用 LangSmith 进行 [deployment](/langsmith/deployment)、扩展和 [observability](/langsmith/observability)。有关详细信息，请参阅[将任何框架与 LangSmith Deployment 结合使用](/langsmith/application-structure#use-any-framework-with-langsmith-deployment)。
