LangSmith Deployment 支持部署 [LangGraph](/oss/python/langgraph/overview) _图_。然而，图的实现可以包含 _节点_ 的图实现可以包含任意代码。这意味着任何框架都可以在节点中实现，并部署在 LangSmith Deployment 上。这让您无需使用额外的 LangGraph 开源 API 即可实现核心应用逻辑，同时仍使用 LangSmith 进行 [部署](/langsmith/deployment)、扩缩容和 [可观测性](/langsmith/observability)。更多详细信息，请参阅 [在 LangSmith Deployment 中使用任意框架](/langsmith/application-structure#use-any-framework-with-langsmith-deployment).
