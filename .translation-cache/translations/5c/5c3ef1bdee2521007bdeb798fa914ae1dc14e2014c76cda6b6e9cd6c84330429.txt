本快速入门指南向您展示如何使用 [`langgraph deploy`](/langsmith/cli#deploy) command.

<Tip>
如需获取全面的云部署指南，包括基于 GitHub 的部署和所有配置选项，请参阅 [云部署设置指南](/langsmith/deploy-to-cloud).
</Tip>

<Note>
 `langgraph deploy` 命令处于 **[测试版](/langsmith/release-stages)**.
</Note>

## 前提条件

在开始之前，请确保您已具备：

- A [LangSmith 账户](https://smith.langchain.com) ，位于 [Plus 计划或更高级别](https://www.langchain.com/pricing) 以及一个 [API 密钥](/langsmith/create-account-api-key).
- [Docker](https://docs.docker.com/get-docker/) 已安装并正在运行。使用以下命令进行验证 `docker ps`.
- On Apple Silicon (M1/M2/M3): [Docker Buildx](https://docs.docker.com/build/install-buildx/) 用于交叉编译到 `linux/amd64`.
-  [LangGraph CLI](/langsmith/cli):

  ```shell
  uv tool install langgraph-cli
  ```

## 1. 创建 LangGraph 应用

从以下 [`new-langgraph-project-python` 模板](https://github.com/langchain-ai/new-langgraph-project):

```shell
langgraph new path/to/your/app --template new-langgraph-project-python
cd path/to/your/app
```

<Tip>
运行 `langgraph new` 不带 `--template` 即可获取可用模板的交互式菜单。
</Tip>

## 2. 设置您的 API 密钥

将您的 LangSmith API 密钥添加到项目根目录的 `.env` 文件中：

```shell
LANGSMITH_API_KEY=lsv2_...
```

 `langgraph deploy` 命令会自动读取此文件。另外，您也可以内联传递：

```shell
LANGSMITH_API_KEY=lsv2_... langgraph deploy
```

## 3. 部署
直接从 CLI 或通过 UI 进行部署。
<Tabs>
    <Tab title="Deploy from CLI">
    从项目目录运行部署命令：

    ```shell
    langgraph deploy
    ```

    这会创建一个默认以项目目录命名的 `dev` 部署。使用 `--name` or `--deployment-type prod` 进行覆盖。

    <Tip>
    要在进行代码更改后更新现有部署，请重新运行 `langgraph deploy`。它会按名称找到现有部署并就地更新。
    </Tip>

    您还可以使用 `langgraph deploy list` 查看所有部署， `langgraph deploy logs` 查看运行时日志，以及 `langgraph deploy delete <ID>` 删除部署。详情请参阅 [CLI 参考](/langsmith/cli#deploy).
    </Tab>
    <Tab title="Deploy from Studio">
    要从 Studio 部署：

    1. 启动 [本地开发服务器](/langsmith/local-dev-testing#langgraph-dev)。这将自动打开 [Studio](/langsmith/studio)，一个交互式代理 IDE。

    ```shell
    langgraph dev
    ```
    2. 点击 `deploy` button.
    ![从 Studio 部署](/langsmith/images/deploy-from-studio.gif)
    </Tab>
</Tabs>

## 4. 在 Studio 中测试

[Studio](/langsmith/studio) 是一个交互式智能体 IDE，直接连接到您的部署。使用它发送消息、检查每个节点的中间状态、在运行过程中编辑状态，以及从任何先前的检查点重放，无需编写代码。

部署就绪后：

1. Go to [LangSmith](https://smith.langchain.com) 并选择 **部署** 在左侧边栏中。
1. 选择您的部署以查看其详情。
1. 点击 **Studio** 在右上角打开 [Studio](/langsmith/studio).

## 5. 测试 API

复制 **API URL** 从部署详情视图，然后使用它来调用您的应用：

<Tabs>
    <Tab title="Python SDK (Async)">
    1. 安装 LangGraph Python SDK：
        ```shell
        pip install langgraph-sdk
        ```
    2. 向助手发送消息（无状态运行）：
        ```python
        from langgraph_sdk import get_client

        client = get_client(url="your-deployment-url", api_key="your-langsmith-api-key")

        async for chunk in client.runs.stream(
            None,  # Threadless run
            "agent", # Name of assistant. Defined in langgraph.json.
            input={
                "messages": [{
                    "role": "human",
                    "content": "What is LangGraph?",
                }],
            },
            stream_mode="updates",
        ):
            print(f"Receiving new event of type: {chunk.event}...")
            print(chunk.data)
            print("\n\n")
        ```
    </Tab>
    <Tab title="Python SDK (Sync)">
    1. 安装 LangGraph Python SDK：
        ```shell
        pip install langgraph-sdk
        ```
    2. 向助手发送消息（无线程运行）：
        ```python
        from langgraph_sdk import get_sync_client

        client = get_sync_client(url="your-deployment-url", api_key="your-langsmith-api-key")

        for chunk in client.runs.stream(
            None,  # Threadless run
            "agent", # Name of assistant. Defined in langgraph.json.
            input={
                "messages": [{
                    "role": "human",
                    "content": "What is LangGraph?",
                }],
            },
            stream_mode="updates",
        ):
            print(f"Receiving new event of type: {chunk.event}...")
            print(chunk.data)
            print("\n\n")
        ```
    </Tab>
    <Tab title="JavaScript SDK">
    1. 安装 LangGraph JS SDK：
        ```shell
        npm install @langchain/langgraph-sdk
        ```
    2. 向助手发送消息（无线程运行）：
        ```js
        const { Client } = await import("@langchain/langgraph-sdk");

        const client = new Client({ apiUrl: "your-deployment-url", apiKey: "your-langsmith-api-key" });

        const streamResponse = client.runs.stream(
            null, // Threadless run
            "agent", // Assistant ID
            {
                input: {
                    "messages": [
                        { "role": "user", "content": "What is LangGraph?"}
                    ]
                },
                streamMode: "messages",
            }
        );

        for await (const chunk of streamResponse) {
            console.log(`Receiving new event of type: ${chunk.event}...`);
            console.log(JSON.stringify(chunk.data));
            console.log("\n\n");
        }
        ```
    </Tab>
    <Tab title="Rest API">
    ```bash
    curl -s --request POST \
        --url <DEPLOYMENT_URL>/runs/stream \
        --header 'Content-Type: application/json' \
        --header "X-Api-Key: <LANGSMITH API KEY>" \
        --data "{
            \"assistant_id\": \"agent\",
            \"input\": {
                \"messages\": [
                    {
                        \"role\": \"human\",
                        \"content\": \"What is LangGraph?\"
                    }
                ]
            },
            \"stream_mode\": \"updates\"
        }"
    ```
    </Tab>
</Tabs>
