LangSmith 按以下方式将示例存储在数据集中：

| 字段名称          | 类型     | 描述                                                                                                       |
| ------------------- | -------- | ----------------------------------------------------------------------------------------------------------------- |
| **id**              | UUID     | 示例的唯一标识符。                                                                                |
| **name**            | string   | 示例的名称。                                                                                          |
| **created\_at**     | datetime | 创建此示例的时间                                                                                 |
| **modified\_at**    | datetime | 上次修改此示例的时间                                                                           |
| **inputs**          | object   | 示例的输入映射。                                                                                  |
| **outputs**         | object   | 运行生成的输出映射或集合。                                                                     |
| **dataset\_id**     | UUID     | 示例所属的数据集                                                                                |
| **source\_run\_id** | UUID     | 如果此示例是从 LangSmith [`Run`](/langsmith/run-data-format)创建的，则为该运行的 ID |
| **metadata**        | object   | 可存储在示例上的额外信息映射，由用户或 SDK 定义。                                                    |

要详细了解示例在评估中的使用方法，请阅读我们关于 [评估 LLM 应用程序](/langsmith/evaluate-llm-application).

<Tip>
的实用指南 `outputs` 字段还可以包含 [断言](/langsmith/assertions)，即评审员对正确答案所写的自由格式声明。离线评估器从 `reference_outputs["assertions"]`.
</Tip>
