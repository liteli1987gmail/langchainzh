LangSmith 将示例存储在数据集中的方式如下：

| 字段名称            | 类型     | 描述                                                                                                               |
| ------------------- | -------- | ----------------------------------------------------------------------------------------------------------------- |
| **id**              | UUID     | 示例的唯一标识符。                                                                                |
| **name**            | string   | 示例的名称。                                                                                          |
| **created\_at**     | datetime | 创建此示例的时间                                                                                 |
| **modified\_at**    | datetime | 上次修改此示例的时间                                                                           |
| **inputs**          | object   | 示例输入的映射。                                                                                  |
| **outputs**         | object   | 运行生成的输出映射或集合。                                                                     |
| **dataset\_id**     | UUID     | 示例所属的数据集                                                                                |
| **source\_run\_id** | UUID     | 如果此示例是从 LangSmith [`Run`](/langsmith/run-data-format) 创建的，则为所述运行的 ID |
| **metadata**        | object   | 可存储在示例上的用户或 SDK 定义的额外信息映射。                            |

要详细了解示例在评估中的用法，请阅读关于 [评估 LLM 应用](/langsmith/evaluate-llm-application) 的操作指南。

<Tip>
`outputs` 字段还可以包含 [断言](/langsmith/assertions)，即审阅者编写的关于正确答案的自由格式声明。离线评估器从 `reference_outputs["assertions"]` 中读取它们。
</Tip>
