>[Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建具有嵌入功能的 AI 应用的数据库。

## 安装和设置

<CodeGroup>

</CodeGroup>
```bash pip
pip install langchain-chroma
```

```bash uv
uv add langchain-chroma
```
</CodeGroup>


## VectorStore

存在一个围绕 Chroma 向量数据库的封装，允许你将其用作向量存储，无论是用于语义搜索还是示例选择。

</fragment>

```python
from langchain_chroma import Chroma
```

有关 Chroma 包装器的更详细演练，请参阅[此笔记本](/oss/integrations/vectorstores/chroma)。
