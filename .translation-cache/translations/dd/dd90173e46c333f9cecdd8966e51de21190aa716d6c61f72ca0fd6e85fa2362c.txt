>[Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建具有嵌入功能的 AI 应用的数据库。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-chroma
```

```bash uv
uv add langchain-chroma
```
</CodeGroup>


## 向量存储

存在一个 Chroma 向量数据库的封装，允许你将其用作向量存储，
无论用于语义搜索还是示例选择。

```python
from langchain_chroma import Chroma
```

有关 Chroma 封装的更详细说明，请参阅 [此笔记本](/oss/integrations/vectorstores/chroma).
