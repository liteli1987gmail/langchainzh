[OpenRouter](https://openrouter.ai/) 是一个统一 API，可通过单一端点访问多个提供商（OpenAI、Anthropic、Google、Meta 等）提供的模型，并提供提供商路由和多模型回退等功能。

## 安装

```bash
npm install @langchain/openrouter @langchain/core
```

在您的环境中设置 `OPENROUTER_API_KEY`。请参阅 [ChatOpenRouter](/oss/integrations/chat/openrouter) 页面了解设置详情。
