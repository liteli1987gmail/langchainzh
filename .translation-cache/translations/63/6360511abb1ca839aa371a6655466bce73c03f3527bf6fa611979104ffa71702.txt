[Fireworks AI](https://fireworks.ai/) 托管开源和专有语言模型，并提供快速推理服务。`langchain-fireworks` 包为 Fireworks API 实现了 LangChain 聊天和嵌入接口。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-fireworks
```

```bash uv
uv add langchain-fireworks
```
</CodeGroup>

从 [fireworks.ai](https://app.fireworks.ai/login) 获取 API 密钥，并设置 `FIREWORKS_API_KEY` 环境变量。
