
#### 2. 部署到 LangSmith

<Steps>
<Step title="Navigate to LangSmith Deployment">
    登录到 [LangSmith](https://smith.langchain.com)。在左侧边栏中，选择 **部署**.
</Step>
<Step title="Create new deployment">
    点击 **+ 新建部署** 按钮。将打开一个面板，您可以在其中填写必填字段。
</Step>
<Step title="Link repository">
    如果您是首次用户或添加之前未连接过的私有仓库，请点击 **添加新账户** 按钮，并按照说明连接您的 GitHub 账户。
</Step>
<Step title="Deploy repository">
    选择您应用程序的仓库。点击 **提交** 进行部署。这可能需要约 15 分钟才能完成。您可以在 **部署详情** view.
</Step>
</Steps>

#### 3. 在 Studio 中测试您的应用程序

您的应用程序部署完成后：

1. 选择您刚创建的部署以查看更多详情。
2. 点击 **Studio** 右上角的按钮。Studio 将打开并显示您的图。

#### 4. 获取您部署的 API URL

1. 在 **部署详情** 视图中，点击 **API URL** 将其复制到剪贴板。
2. 点击 `URL` 将其复制到剪贴板。

#### 5. 测试 API

您现在可以测试 API：

<Tabs>
<Tab title="Python">
1. 安装 LangGraph Python：
```shell
pip install langgraph-sdk
```
2. 向代理发送消息：
```python
from langgraph_sdk import get_sync_client # or get_client for async

client = get_sync_client(url="your-deployment-url", api_key="your-langsmith-api-key")

for chunk in client.runs.stream(
    None,    # Threadless run
    "agent", # Name of agent. Defined in langgraph.json.
    input={
        "messages": [{
            "role": "human",
            "content": "What is LangGraph?",
        }],
    },
    stream_mode="updates",
):
    print(f"Receiving new event of type: {chunk.event}...")
    print(chunk.data)
    print("\n\n")
```
</Tab>
<Tab title="Rest API">
```bash
curl -s --request POST \
    --url <DEPLOYMENT_URL>/runs/stream \
    --header 'Content-Type: application/json' \
    --header "X-Api-Key: <LANGSMITH API KEY> \
    --data "{
        \"assistant_id\": \"agent\", `# Name of agent. Defined in langgraph.json.`
        \"input\": {
            \"messages\": [
                {
                    \"role\": \"human\",
                    \"content\": \"What is LangGraph?\"
                }
            ]
        },
        \"stream_mode\": \"updates\"
    }"
```
</Tab>
</Tabs>
