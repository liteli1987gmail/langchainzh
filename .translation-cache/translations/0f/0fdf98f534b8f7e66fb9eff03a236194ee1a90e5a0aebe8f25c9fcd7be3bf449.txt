> [Kùzu](https://kuzudb.com/) 是一个可嵌入的、可扩展的、极快的图数据库。
> 它采用宽松的 MIT 许可证授权，你可以查看其源代码 [GitHub](https://github.com/kuzudb/kuzu).

> Kùzu 的主要特点：
>- 性能和可扩展性：实现现代化的图连接算法。
>- 可用性：非常容易设置和开始使用，因为没有服务器（嵌入式架构）。
>- 互操作性：可以方便地从外部列式格式、CSV、JSON 和关系数据库扫描和复制数据。
>- 结构化属性图模型：实现属性图模型，并增加了结构。
>- Cypher 支持：允许使用声明式查询语言 Cypher 方便地查询图。

> 开始使用 Kùzu，请访问其 [文档](https://docs.kuzudb.com/).


## 安装和设置

安装 Python SDK 如下：

<CodeGroup>
```bash pip
pip install -U langchain-kuzu
```

```bash uv
uv add langchain-kuzu
```
</CodeGroup>
