> [Kùzu](https://kuzudb.com/) 是一个可嵌入的、可扩展的、极快的图数据库。
> 它采用宽松的 MIT 许可证授权，您可以在 [GitHub](https://github.com/kuzudb/kuzu) 上查看其源代码。

> Kùzu 的主要特性：
> - 性能与可扩展性：采用现代最先进的图连接算法。
> - 易用性：非常容易设置和上手，因为没有服务器（嵌入式架构）。
> - 互操作性：可以方便地扫描和复制来自外部列式格式、CSV、JSON 和关系数据库的数据。
> - 结构化属性图模型：实现了属性图模型，并添加了额外的结构。
> - Cypher 支持：允许使用 Cypher（一种声明式查询语言）方便地查询图。

> 访问 Kùzu 的[文档](https://docs.kuzudb.com/)开始使用 Kùzu。


## 安装和设置

按如下方式安装 Python SDK：

<CodeGroup>

</CodeGroup>
```bash pip
pip install -U langchain-kuzu
```

```bash uv
uv add langchain-kuzu
```
</CodeGroup>
