---

[ScraperAPI](https://www.scraperapi.com/) 使您能够通过其网页抓取 API 从任何公共网站收集数据，无需担心代理、浏览器或 CAPTCHA 处理问题。[langchain-scraperapi](https://github.com/scraperapi/langchain-scraperapi) 封装了这项服务，使 AI 代理能够轻松浏览网页并从中抓取数据。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-scraperapi
```

```bash uv
uv add langchain-scraperapi
```
</CodeGroup>

从 [ScraperAPI](https://www.scraperapi.com/) 获取 API 密钥并将其设置为环境变量：

```python
import os

os.environ["SCRAPERAPI_API_KEY"] = "your-api-key"
```

## 工具

该包提供 3 个工具，分别用于抓取任意网站、获取结构化的 Google 搜索结果和获取结构化的 Amazon 搜索结果。

查看每个工具的详细文档：

- [ScraperAPITool](/oss/integrations/tools/scraperapi) - 浏览和抓取任意网站
- ScraperAPIGoogleSearchTool - 获取结构化的 Google 搜索 SERP 数据
- ScraperAPIAmazonSearchTool - 获取结构化的 Amazon 产品搜索数据

如需更详细的演示，请参阅[官方仓库](https://github.com/scraperapi/langchain-scraperapi)。
