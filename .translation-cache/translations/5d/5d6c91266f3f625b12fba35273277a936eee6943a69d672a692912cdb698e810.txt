[Nimble](https://www.nimbleway.com/) 通过搜索 API 提供实时网络数据访问，浏览真实网络而非依赖预构建索引。与传统搜索 API 不同，Nimble 使用无头浏览器实时导航网站——处理 JavaScript 渲染、动态内容、分页和复杂的多步导航流程。

**关键技术能力：**

- **实时网络浏览**：实时访问当前网络内容，而非缓存索引
- **JavaScript 渲染**：处理现代 SPA、懒加载和客户端渲染
- **灵活模式**：快速模式获取 SERP 数据，深度模式提取完整内容
- **AI 摘要**：可选的 LLM 生成答案与原始搜索结果一同提供
- **生产就绪**：为规模化而构建，支持异步和全面的错误处理

更多信息，请访问 [nimbleway.com](https://www.nimbleway.com/) 或探索 [官方 Nimble 文档](https://docs.nimbleway.com/)。

## 安装和设置

Nimble 集成存在于其独立的合作伙伴包中。您可以通过以下方式安装：

<CodeGroup>
```bash pip
pip install -U langchain-nimble
```
```bash uv
uv add langchain-nimble
```
</CodeGroup>

要使用此包，您需要将 `NIMBLE_API_KEY` 环境变量设置为您的 Nimble API 密钥。您可以通过在 [Nimble](https://www.nimbleway.com/) 注册来获取 API 密钥。
