[Nimble](https://www.nimbleway.com/) 通过搜索 API 提供实时网络数据访问，可浏览实时网络而非依赖预建索引。与传统搜索 API 不同，Nimble 使用无头浏览器实时导航网站——处理 JavaScript 渲染、动态内容、分页和复杂的多步导航流程。

**关键技术能力：**

- **实时网络浏览**：实时访问当前网络内容，而非缓存索引
- **JavaScript 渲染**：处理现代 SPA、延迟加载和客户端渲染
- **灵活的采集模式**：快速模式用于搜索结果数据采集，深度模式用于完整内容提取
- **AI 摘要**：可选的 LLM 生成答案，可与原始搜索结果一起返回
- **Production-ready**：为可扩展性而构建，支持异步和全面的错误处理

更多信息，请访问 [nimbleway.com](https://www.nimbleway.com/) 或查看 [Nimble 官方文档](https://docs.nimbleway.com/).

## 安装和设置

Nimble 集成位于其独立的合作伙伴包中。您可以通过以下方式安装：

<CodeGroup>
```bash pip
pip install -U langchain-nimble
```
```bash uv
uv add langchain-nimble
```
</CodeGroup>

要使用此包，您需要设置 `NIMBLE_API_KEY` 环境变量为您的 Nimble API 密钥。您可以在 [注册获取 API 密钥](https://www.nimbleway.com/).
