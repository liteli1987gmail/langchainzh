>[AI/ML API](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration) 提供单一 API 接口，可访问 300+ 托管基础模型（DeepSeek、Gemini、GPT 等），具备企业级正常运行时间和吞吐量。

## 安装和设置

- Install the AI/ML API integration package.

  ```bash
  pip install langchain-aimlapi
  ```

- 在以下网址创建账户 [aimlapi.com](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration) 并生成 API 密钥。
- 通过设置 `AIMLAPI_API_KEY` 环境变量进行身份验证。

```python
import os

os.environ["AIMLAPI_API_KEY"] = "aimlapi_..."
```
