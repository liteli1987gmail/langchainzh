[AgentQL](https://www.agentql.com/)的文档加载器使用 [AgentQL 查询](https://docs.agentql.com/agentql-query)从任何网页中提取结构化数据。AgentQL 可跨多种语言和网页使用，不会随时间和变化而失效。

## 概述

`AgentQLLoader` 需要以下两个参数：

- `url`：要从中提取数据的网页 URL。
- `query`：要执行的 AgentQL 查询。了解更多关于 [如何在文档中编写 AgentQL 查询](https://docs.agentql.com/agentql-query) 或在其中进行测试 [AgentQL Playground](https://dev.agentql.com/playground).

以下参数设置是可选的：

- `api_key`：来自您的 AgentQL API 密钥 [dev.agentql.com](https://dev.agentql.com). **`Optional`.**
- `timeout`：请求超时前的等待秒数。 **默认为 `900`.**
- `is_stealth_mode_enabled`：是否启用实验性反机器人规避策略。此功能可能并非在所有网站上都始终有效。启用此模式可能会延长数据提取时间。 **默认为 `False`.**
- `wait_for`：提取数据前等待页面加载的秒数。 **默认为 `0`.**
- `is_scroll_to_bottom_enabled`：提取数据前是否滚动到页面底部。 **默认为 `False`.**
- `mode`: `"standard"` 使用深度数据分析，而 `"fast"` 牺牲一些分析深度以换取速度，适合大多数用例。 [在此指南中了解更多关于模式的信息。](https://docs.agentql.com/accuracy/standard-mode) **默认为 `"fast"`.**
- `is_screenshot_enabled`：提取数据前是否截图。以 Base64 字符串形式返回在 'metadata' 中。 **默认为 `False`.**

AgentQLLoader 使用 AgentQL 的 [REST API](https://docs.agentql.com/rest-api/api-reference)

## 设置

要使用 AgentQL 文档加载器，您需要配置 `AGENTQL_API_KEY` 环境变量，或使用 `api_key` 参数。您可以从我们的 [开发者门户](https://dev.agentql.com).

### 安装

安装 **langchain-agentql**.

```python
pip install -qU langchain-agentql
```

### 设置凭证

```python
import os

os.environ["AGENTQL_API_KEY"] = "YOUR_AGENTQL_API_KEY"
```

## 初始化

接下来实例化您的模型对象：

```python
from langchain_agentql.document_loaders import AgentQLLoader

loader = AgentQLLoader(
    url="https://www.agentql.com/blog",
    query="""
    {
        posts[] {
            title
            url
            date
            author
        }
    }
    """,
    is_scroll_to_bottom_enabled=True,
)
```

## 加载

docs = loader.load()
docs[0]

## 懒加载

`AgentQLLoader` 目前仅加载一个 @[`Document`]。因此， `load()` 和 `lazy_load()` 行为相同：

```python
pages = [doc for doc in loader.lazy_load()]
pages
```
