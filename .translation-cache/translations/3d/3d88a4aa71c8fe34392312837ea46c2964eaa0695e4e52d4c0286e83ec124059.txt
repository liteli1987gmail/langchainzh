[Egnyte](https://www.egnyte.com) 是一个基于云的内容协作和文件共享平台，使组织能够安全地存储、共享和管理其文件。Egnyte 将云内容管理的强大功能与高级安全性、合规性和 AI 驱动的搜索功能相结合。

此包提供了与 Egnyte 混合搜索 API 的 LangChain 集成，该 API 结合了语义搜索和关键词搜索，帮助您从 Egnyte 实例中查找和检索文档。

# egnyte-langchain-connector

此包包含 Egnyte 与 LangChain 的集成。有关 Egnyte 的更多信息，请参阅 [Egnyte 开发者文档](https://developers.egnyte.com/)。

## 先决条件

要与 Egnyte 集成，您需要：

* 一个 Egnyte 账户 — 如果您不是当前的 Egnyte 客户或想在生产 Egnyte 实例之外进行测试，您可以使用[免费开发者账户](https://developers.egnyte.com/member/register)。
* 一个 Egnyte 应用 — 这在[开发者控制台](https://developers.egnyte.com/)中配置，并且必须启用相应的作用域。
* 该应用必须由管理员启用。

## 身份验证

`egnyte-langchain-connector` 包使用 Egnyte 用户令牌的 Bearer 令牌身份验证。

### 生成用户令牌

要生成用于身份验证的 Egnyte 用户令牌：

1. **注册开发者账户**
   - 访问 [https://developers.egnyte.com/member/register](https://developers.egnyte.com/member/register)
   - 创建您的免费开发者账户

2. **生成用户令牌**
   - 使用您的 API 密钥生成用户令牌，遵循 [Public API 身份验证指南](https://developers.egnyte.com/docs/read/Public_API_Authentication)
   - **重要**：生成令牌时使用作用域 `Egnyte.ai`，以确保对 AI 驱动的搜索功能的正确访问

### 使用令牌

您可以直接将令牌传递给检索器：

```python
import getpass

egnyte_user_token = getpass.getpass("Enter your Egnyte User Token: ")
domain = "company.egnyte.com"  # Your Egnyte domain (without https://)
```

或使用环境变量：

```python
import os
from dotenv import load_dotenv

load_dotenv()

egnyte_user_token = os.getenv("EGNYTE_USER_TOKEN")
domain = os.getenv("EGNYTE_DOMAIN")
```

## 检索器

### EgnyteRetriever

[查看使用示例](/oss/integrations/retrievers/egnyte)

```python
from langchain_egnyte import EgnyteRetriever

retriever = EgnyteRetriever(domain="company.egnyte.com")
```

## 工具

### EgnyteSearchOptions

配置高级搜索参数：

```python
from langchain_egnyte import EgnyteSearchOptions

search_options = EgnyteSearchOptions(
    limit=50,
    folderPath="/policies",
    excludeFolderPaths=["/temp", "/archive"],
    createdAfter=1640995200000,  # Unix timestamp in milliseconds
    createdBefore=1672531200000
)
```
