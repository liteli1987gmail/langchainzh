[Egnyte](https://www.egnyte.com) Egnyte是一个基于云的内容协作与文件共享平台，使组织能够安全地存储、共享和管理其文件。Egnyte整合了云内容管理的强大功能，并具备高级安全性、合规性和AI驱动的搜索功能。

此包提供了LangChain与Egnyte混合搜索API的集成，该API结合了语义搜索和关键词搜索，以帮助您从Egnyte实例中查找和检索文档。

# egnyte-langchain-connector

此包包含与Egnyte的LangChain集成。如需了解更多关于Egnyte的信息，请参阅 [Egnyte开发者文档](https://developers.egnyte.com/).

## 先决条件

要与Egnyte集成，您需要：

* Egnyte账户——如果您不是Egnyte现有客户或想在生产Egnyte实例之外进行测试，您可以使用 [免费开发者账户](https://developers.egnyte.com/member/register).
* Egnyte应用——可在 [开发者控制台](https://developers.egnyte.com/)中进行配置，且必须启用相应的权限范围。
* 该应用必须由管理员启用。

## 身份验证

该 `egnyte-langchain-connector` 包使用Bearer令牌身份验证与Egnyte用户令牌。

### 生成用户令牌

要生成用于身份验证的Egnyte用户令牌：

1. **注册开发者账户**
   - 访问 [https://developers.egnyte.com/member/register](https://developers.egnyte.com/member/register)
   - 创建免费开发者账户

2. **生成用户令牌**
   - 使用您的API密钥按照 [公共API身份验证指南](https://developers.egnyte.com/docs/read/Public_API_Authentication)
   - **重要提示**：生成令牌时使用 `Egnyte.ai` 权限范围，以确保正确访问AI驱动的搜索功能

### 使用令牌

您可以直接将令牌传递给检索器：

```python
import getpass

egnyte_user_token = getpass.getpass("Enter your Egnyte User Token: ")
domain = "company.egnyte.com"  # Your Egnyte domain (without https://)
```

或使用环境变量：

```python
import os
from dotenv import load_dotenv

load_dotenv()

egnyte_user_token = os.getenv("EGNYTE_USER_TOKEN")
domain = os.getenv("EGNYTE_DOMAIN")
```

## 检索器

### EgnyteRetriever

[查看使用示例](/oss/integrations/retrievers/egnyte)

```python
from langchain_egnyte import EgnyteRetriever

retriever = EgnyteRetriever(domain="company.egnyte.com")
```

## 工具

### EgnyteSearchOptions

配置高级搜索参数：

```python
from langchain_egnyte import EgnyteSearchOptions

search_options = EgnyteSearchOptions(
    limit=50,
    folderPath="/policies",
    excludeFolderPaths=["/temp", "/archive"],
    createdAfter=1640995200000,  # Unix timestamp in milliseconds
    createdBefore=1672531200000
)
```
