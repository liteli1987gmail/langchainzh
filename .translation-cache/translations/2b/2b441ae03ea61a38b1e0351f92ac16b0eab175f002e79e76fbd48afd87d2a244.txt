---
title: PyMuPDF4LLM 集成
description: 使用 LangChain Python 与 PyMuPDF4LLM 集成。
---

[PyMuPDF4LLM](https://pymupdf.readthedocs.io/en/latest/pymupdf4llm) 旨在更轻松地以 Markdown 格式提取 PDF 内容，满足 LLM 和 RAG 应用的需求。

[langchain-pymupdf4llm](https://github.com/lakinduboteju/langchain-pymupdf4llm) 将 PyMuPDF4LLM 集成到 LangChain 作为文档加载器。

```python
pip install -qU langchain-pymupdf4llm
```

```python
from langchain_pymupdf4llm import PyMuPDF4LLMLoader, PyMuPDF4LLMParser
```
