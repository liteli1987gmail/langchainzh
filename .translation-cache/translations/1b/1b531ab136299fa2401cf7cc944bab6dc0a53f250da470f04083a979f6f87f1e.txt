沙盒是隔离环境，允许代理安全地执行潜在风险操作（如运行任意代码或与文件系统交互），而不会影响您的主基础设施。

从 [LangSmith 主页](https://smith.langchain.com)，选择 **沙盒** 来管理您所有的沙盒资源。

![沙盒概览页面](/images/langsmith/sandboxes/sb-overview.png)

## 开始使用

### 1. 安装 SDK

<CodeGroup>

```bash Python
# uv
uv add "langsmith[sandbox]"

# pip
pip install "langsmith[sandbox]"
```

```bash TypeScript
npm install langsmith
```

</CodeGroup>

### 2. 设置您的 API 密钥

```bash
export LANGSMITH_API_KEY="<your-api-key>"
```

### 3. 创建并运行沙盒

<CodeGroup>

```python Python
from langsmith.sandbox import SandboxClient

client = SandboxClient()

with client.sandbox() as sb:
    result = sb.run("python -c 'print(2 + 2)'")
    print(result.stdout)  # "4\n"
```

```ts TypeScript
import { SandboxClient } from "langsmith/sandbox";

const client = new SandboxClient();
const sandbox = await client.createSandbox();
const result = await sandbox.run("node -e 'console.log(2 + 2)'");
console.log(result.stdout); // "4\n"
await sandbox.delete();
```

</CodeGroup>

<Tip>
  更喜欢命令行？ [沙盒 CLI](/langsmith/sandbox-cli) 让您无需编写任何代码即可创建沙盒、运行命令和打开交互式 shell。
</Tip>

### 4. 将沙盒与您的代理配合使用

要将沙盒接入代理代码，请参阅开源文档：

- **深度代理**: [使用 `LangSmithSandbox` 作为后端](/oss/integrations/sandboxes/langsmith)，涵盖安装、后端创建和清理。
- **沙盒作为代理后端**: [将任何沙盒配置为执行后端](/oss/deepagents/sandboxes) 为您的代理提供 `execute` 和文件系统工具。
- **LangChain / LangGraph integrations**：将 LangSmith 沙盒作为第一方选项，或 [连接第三方提供商](/oss/integrations/sandboxes) 如 AgentCore、Daytona、E2B、Modal、Runloop 和 Vercel。
