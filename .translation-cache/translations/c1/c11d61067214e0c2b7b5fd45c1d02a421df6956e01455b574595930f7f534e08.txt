>[Perplexity Search](https://docs.perplexity.ai/docs/search/quickstart) 是一个网络搜索 API，返回排名归因的结果，专为 LLMs 和代理设计。[Search API 端点](https://docs.perplexity.ai/api-reference/search-post) 返回支撑 Perplexity 答案引擎的底层网络结果。

我们可以将其用作[检索器](/oss/langchain/retrieval)。它将展示此集成特有的功能。阅读完本文后，查阅[相关用例页面](/oss/langchain/rag)了解如何将此检索器用作更大链的一部分可能会很有帮助。

## 设置

### 安装

该集成位于 `langchain-perplexity` 包中：

```python
pip install -qU langchain-perplexity
```

### 凭证

您需要一个 Perplexity API 密钥才能使用此集成。在 [Perplexity API 密钥仪表板](https://www.perplexity.ai/account/api/keys) 创建一个密钥，然后将其设置为 `PPLX_API_KEY` 环境变量（该集成也接受 `PERPLEXITY_API_KEY`）。

```python
import getpass
import os

if not os.environ.get("PPLX_API_KEY"):
    os.environ["PPLX_API_KEY"] = getpass.getpass("Perplexity API key:\n")
```

如果你想从单个查询获取自动追踪，你也可以通过取消下面注释的方式设置你的 [LangSmith](/langsmith/observability) API 密钥：

```python
# os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
# os.environ["LANGSMITH_TRACING"] = "true"
```

## 实例化

`PerplexitySearchRetriever` 接受以下构造函数参数：

- `k` — 最大结果数（1–20）。默认为 `10`。
- `max_tokens` — 所有结果的最大 token 总数。
- `max_tokens_per_page` — 每个结果页返回的最大 token 数。
- `country` — 用于偏好结果的 ISO 国家代码（例如 `"US"`）。
- `search_domain_filter` — 要包含或排除的域名列表（最多 20 个）。在域名前面加 `-` 可将其排除。请参阅[域名过滤器文档](https://docs.perplexity.ai/docs/search/filters/domain-filter)。
- `search_recency_filter` — 值为 `"day"`、`"week"`、`"month"` 或 `"year"` 之一。请参阅[日期和时间过滤器文档](https://docs.perplexity.ai/docs/search/filters/date-time-filters)。
- `search_after_date` / `search_before_date` — 日期字符串，格式为 `MM/DD/YYYY`。

```python
from langchain_perplexity import PerplexitySearchRetriever

retriever = PerplexitySearchRetriever(k=3)
```

## 用法

```python
query = "what year was breath of the wild released?"

retriever.invoke(query)
```

每个返回的 @[`Document`] 都将其结果 `snippet` 作为其 `page_content`，并在 `metadata` 中包含 `title`、`url`、`date` 和 `last_updated`。

### 筛选使用

您可以使用 Perplexity Search API 支持的相同筛选条件来约束搜索：

```python
from langchain_perplexity import PerplexitySearchRetriever

retriever = PerplexitySearchRetriever(
    k=5,
    country="US",
    search_recency_filter="week",
    search_domain_filter=["nytimes.com", "reuters.com", "-pinterest.com"],
)

retriever.invoke("US Federal Reserve interest rate decisions")
```

```python
from langchain_perplexity import PerplexitySearchRetriever

# Restrict results to a specific date range
retriever = PerplexitySearchRetriever(
    k=5,
    search_after_date="01/01/2025",
    search_before_date="06/30/2025",
)

retriever.invoke("US Federal Reserve interest rate decisions")
```

## 在链中使用

我们可以轻松地将此检索器组合到链中，以构建一个简单的 RAG 管道。

```python
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.runnables import RunnablePassthrough
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template(
    """Answer the question based only on the context provided.

Context: {context}

Question: {question}"""
)

llm = ChatOpenAI(model="gpt-5.5")


def format_docs(docs):
    return "\n\n".join(doc.page_content for doc in docs)


chain = (
    {"context": retriever | format_docs, "question": RunnablePassthrough()}
    | prompt
    | llm
    | StrOutputParser()
)

chain.invoke("What were the major outcomes of the most recent Federal Reserve meeting?")
```
