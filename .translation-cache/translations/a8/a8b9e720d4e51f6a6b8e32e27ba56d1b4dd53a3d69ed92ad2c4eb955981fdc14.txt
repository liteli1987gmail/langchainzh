Cognee 实现了可扩展的模块化 ECL（提取、认知、加载）管道，允许
您互联和检索过去的对话、文档和音频
转录文本，同时减少幻觉、开发工作量和成本。

Cognee 合并图数据库和向量数据库以发现隐藏的关系和新的
数据中的模式。您可以自动建模、加载和检索实体和
代表您业务领域的对象并分析它们的关系，发现
单独的向量存储或图存储都无法提供的洞察。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-cognee
```

```bash uv
uv add langchain-cognee
```
</CodeGroup>
