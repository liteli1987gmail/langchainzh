[Tavily](https://tavily.com/) 是一款专为 AI 智能体设计的搜索引擎，提供搜索、提取、爬取和映射 API，使开发者能够将应用程序连接到实时在线信息。Tavily 的主要使命是从可信来源提供真实可靠的信息，提高 AI 生成内容和推理的准确性和可靠性。

`@langchain/tavily` 包将 Tavily 的搜索、提取、爬取、映射和研究端点作为 LangChain 工具公开。

## 安装与设置

安装 LangChain JavaScript 的 Tavily 集成包：

<CodeGroup>

</fragment>
```bash npm
npm install @langchain/tavily @langchain/core
```
```bash yarn
yarn add @langchain/tavily @langchain/core
```
```bash pnpm
pnpm add @langchain/tavily @langchain/core
```
</CodeGroup>

[设置 Tavily API 密钥](https://app.tavily.com) 并将其设置为名为 `TAVILY_API_KEY` 的环境变量：

```bash
export TAVILY_API_KEY="your-api-key"
```

## 工具

### TavilySearch

一个从 Tavily 的 Search API 返回实时、可供 LLM 使用结果的搜索工具。

查看[使用示例](/oss/integrations/tools/tavily_search)。

```typescript
import { TavilySearch } from "@langchain/tavily";
```

### TavilyExtract

一个返回已清理、解析内容（来自一个或多个 URL）的工具。

查看[使用示例](/oss/integrations/tools/tavily_extract)。

```typescript
import { TavilyExtract } from "@langchain/tavily";
```

### TavilyCrawl

一种工具，用于从基础 URL 执行结构化网页遍历，支持可选的自然语言指令和路径/域名过滤器。

查看[使用示例](/oss/integrations/tools/tavily_crawl)。

```typescript
import { TavilyCrawl } from "@langchain/tavily";
```

### TavilyMap

一种发现站点 URL 结构但不提取页面内容的工具。

查看[使用示例](/oss/integrations/tools/tavily_map)。

```typescript
import { TavilyMap } from "@langchain/tavily";
```

### TavilyResearch

一个执行结构化研究任务并返回综合结果（可选流式传输）的工具。

```typescript
import { TavilyResearch } from "@langchain/tavily";
```

### TavilyGetResearch

一个工具，通过 `request_id` 从之前的 `TavilyResearch` 调用中检索研究输出。

```typescript
import { TavilyGetResearch } from "@langchain/tavily";
```

## 组件参考

| 类 | 抽象 | 导入路径 | 描述 |
|-------|-------------|-------------|-------------|
| `TavilySearch` | Tool | `import { TavilySearch } from "@langchain/tavily"` | 从 Tavily Search API 返回搜索结果。 |
| `TavilyExtract` | Tool | `import { TavilyExtract } from "@langchain/tavily"` | 从一个或多个 URL 中提取清理后的内容。 |
| `TavilyCrawl` | Tool | `import { TavilyCrawl } from "@langchain/tavily"` | 从基础 URL 开始爬取网站，支持深度和广度控制。 |
| `TavilyMap` | Tool | `import { TavilyMap } from "@langchain/tavily"` | 发现网站的 URL 结构，但不提取页面内容。 |
| `TavilyResearch` | Tool | `import { TavilyResearch } from "@langchain/tavily"` | 针对 Tavily Research API 运行结构化研究任务。 |
| `TavilyGetResearch` | Tool | `import { TavilyGetResearch } from "@langchain/tavily"` | 通过 `request_id` 获取研究结果。 |
