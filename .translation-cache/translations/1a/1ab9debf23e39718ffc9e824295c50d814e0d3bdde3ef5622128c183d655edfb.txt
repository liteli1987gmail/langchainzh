[Tavily](https://tavily.com/) 是一个专为 AI 智能体设计的搜索引擎，提供搜索、内容提取、爬取和映射 API，使开发者能够将应用程序连接到实时在线信息。Tavily 的主要使命是提供来自可信来源的真实可靠信息，增强 AI 生成内容和推理的准确性和可靠性。

该 `@langchain/tavily` 包将 Tavily Search、Extract、Crawl、Map 和 Research 端点作为 LangChain 工具公开。

## 安装与设置

安装 LangChain JavaScript 的 Tavily 集成包：

<CodeGroup>
```bash npm
npm install @langchain/tavily @langchain/core
```
```bash yarn
yarn add @langchain/tavily @langchain/core
```
```bash pnpm
pnpm add @langchain/tavily @langchain/core
```
</CodeGroup>

[设置 Tavily API 密钥](https://app.tavily.com) 并将其设置为名为 `TAVILY_API_KEY`:

```bash
export TAVILY_API_KEY="your-api-key"
```

## 工具

### TavilySearch

一个从 Tavily 搜索 API 返回实时、LLM 可用结果的搜索工具。

查看 [使用示例](/oss/integrations/tools/tavily_search).

```typescript
import { TavilySearch } from "@langchain/tavily";
```

### TavilyExtract

一个从一到多个 URL 返回清洗和解析内容的工具。

查看 [使用示例](/oss/integrations/tools/tavily_extract).

```typescript
import { TavilyExtract } from "@langchain/tavily";
```

### TavilyCrawl

A tool that performs a structured web traversal from a base URL, with optional natural-language instructions and path/domain filters.

查看 [使用示例](/oss/integrations/tools/tavily_crawl).

```typescript
import { TavilyCrawl } from "@langchain/tavily";
```

### TavilyMap

一个发现网站 URL 结构但不提取页面内容的工具。

查看 [使用示例](/oss/integrations/tools/tavily_map).

```typescript
import { TavilyMap } from "@langchain/tavily";
```

### TavilyResearch

一个运行结构化研究任务并返回综合结果（可选流式传输）的工具。

```typescript
import { TavilyResearch } from "@langchain/tavily";
```

### TavilyGetResearch

一个通过 ID 检索先前 `request_id` 的研究输出工具。 `TavilyResearch` invocation.

```typescript
import { TavilyGetResearch } from "@langchain/tavily";
```

## 组件参考

| 类 | 抽象 | 导入路径 | 描述 |
|-------|-------------|-------------|-------------|
| `TavilySearch` | 工具 | `import { TavilySearch } from "@langchain/tavily"` | 从 Tavily 搜索 API 返回搜索结果。 |
| `TavilyExtract` | 工具 | `import { TavilyExtract } from "@langchain/tavily"` | 从一个或多个 URL 提取清洗后的内容。 |
| `TavilyCrawl` | 工具 | `import { TavilyCrawl } from "@langchain/tavily"` | 从基础 URL 开始爬取网站，支持深度和广度控制。 |
| `TavilyMap` | 工具 | `import { TavilyMap } from "@langchain/tavily"` | 发现网站的 URL 结构但不提取页面内容。 |
| `TavilyResearch` | 工具 | `import { TavilyResearch } from "@langchain/tavily"` | 对 Tavily 研究 API 运行结构化研究任务。 |
| `TavilyGetResearch` | 工具 | `import { TavilyGetResearch } from "@langchain/tavily"` | 通过 ID 获取研究结果。 | `request_id`. |
