LangChain 通过 `@langchain/google` 包提供与 [Google AI Studio](https://aistudio.google.com/) 和 [Google Cloud Vertex AI](https://cloud.google.com/vertex-ai) 的集成。

<Info>
  正在寻找旧的 `@langchain/google-genai` 或 `@langchain/google-vertexai` 包？它们在[长期支持](#legacy-packages)下维护，但不再推荐用于新项目。
</Info>

## 聊天模型

[`ChatGoogle`](/oss/integrations/chat/google) 类是访问 Gemini 模型（如 `gemini-2.5-pro`、`gemini-2.5-flash` 和 `gemini-3.1-pro-preview`）以及 Gemma 等开放模型的方式。它在单一界面中同时支持 Google AI Studio 和 Vertex AI

<Tip>
有关安装 LangChain 包的通用说明，请参阅[此部分](/oss/langchain/install)。
</Tip>

<CodeGroup>

</CodeGroup>
```bash npm
npm install @langchain/google @langchain/core
```
```bash yarn
yarn add @langchain/google @langchain/core
```
```bash pnpm
pnpm add @langchain/google @langchain/core
```
</CodeGroup>

配置您的 API 密钥：

```bash
export GOOGLE_API_KEY=your-api-key
```

```typescript
import { ChatGoogle } from "@langchain/google";

const model = new ChatGoogle("gemini-2.5-flash");

const res = await model.invoke([
  ["human", "What would be a good company name for a company that makes colorful socks?"],
]);
```

`ChatGoogle` 支持工具调用、结构化输出、多模态输入（图像、音频、视频）、推理/思考、图像生成、文本转语音，以及 Gemini 特有的原生工具，如 Google Search 接地和代码执行。

### Vertex AI 上的第三方模型

[Anthropic](/oss/integrations/chat/anthropic) Claude 模型也可通过
[Vertex AI](https://cloud.google.com/vertex-ai/generative-ai/docs/partner-models/use-claude)
平台获取。详见[在 Vertex AI 上使用 Claude](https://cloud.google.com/vertex-ai/generative-ai/docs/partner-models/use-claude)，了解如何启用模型访问和可用的模型名称。

### Postgres 向量存储（Cloud SQL）

[`@langchain/google-cloud-sql-pg`](https://www.npmjs.com/package/@langchain/google-cloud-sql-pg) 包中的 [PostgresVectorStore](/oss/integrations/vectorstores/google_cloudsql_pg) 模块提供了使用 Cloud SQL for PostgreSQL 存储向量嵌入的方法。

```bash
npm install @langchain/google-cloud-sql-pg @langchain/core
```

## 传统包

以下软件包为现有用户提供长期支持。新项目应使用 `@langchain/google`。

### `@langchain/google-genai`

`@langchain/google-genai` 软件包提供 [`ChatGoogleGenerativeAI`](/oss/integrations/chat/google_generative_ai) 和 [`GoogleGenerativeAIEmbeddings`](/oss/integrations/embeddings/google_generative_ai)，用于通过 Google AI Studio 访问 Gemini 模型。此软件包基于已弃用的 Google SDK 构建，不会接收新功能。

```bash
npm install @langchain/google-genai @langchain/core
```

### `@langchain/google-vertexai`

`@langchain/google-vertexai` 包提供了 [`ChatVertexAI`](/oss/integrations/chat/google_vertex_ai)、[`VertexAIEmbeddings`](/oss/integrations/embeddings/google_vertex_ai) 和 [`VertexAI`](/oss/integrations/llms/google_vertex_ai)，用于 Node.js 上的 Vertex AI。它依赖于 [`@langchain/google-gauth`](#%40langchain%2Fgoogle-gauth) 进行身份验证。此包已被内置于 `@langchain/google` 中的 Vertex AI 支持所取代，用于聊天。

```bash
npm install @langchain/google-vertexai @langchain/core
```

### `@langchain/google-vertexai-web`

`@langchain/google-vertexai-web` 包提供相同的 Vertex AI 聊天、嵌入和 LLM 类，用于浏览器和 Edge 运行时环境。在 Web 环境中运行时，请安装此包（而非 `@langchain/google-vertexai`）。它依赖于 [`@langchain/google-webauth`](#%40langchain%2Fgoogle-webauth)。

```bash
npm install @langchain/google-vertexai-web @langchain/core
```

查看 [Vertex AI 聊天](/oss/integrations/chat/google_vertex_ai) 页面了解 `GOOGLE_WEB_CREDENTIALS` 和 Web 导入路径。

### `@langchain/google-webauth`

[`@langchain/google-webauth`](https://github.com/langchain-ai/langchainjs/tree/main/libs/providers/langchain-google-webauth) 包为旧版 Vertex AI 集成提供浏览器和 Edge 认证。它会随 `@langchain/google-vertexai-web` 自动安装——请勿将其与 `@langchain/google-gauth` 同时安装。

在 `GOOGLE_WEB_CREDENTIALS`（或已弃用的 `GOOGLE_VERTEX_AI_WEB_CREDENTIALS`）中设置服务账号 JSON。您也可以将 `apiKey` 或 `authOptions` 传递给模型构造函数，或设置 `API_KEY` 环境变量。

### `@langchain/google-gauth`

[`@langchain/google-gauth`](https://github.com/langchain-ai/langchainjs/tree/main/libs/providers/langchain-google-gauth) 包为基于 [`@langchain/google-common`](#%40langchain%2Fgoogle-common) 构建的旧版 Google 集成提供 Node.js 认证。添加 `@langchain/google-vertexai` 时会自动安装它——通常您**不需要**直接安装或导入 `@langchain/google-gauth`。

在 Node.js 上，凭证按以下顺序解析：

1. 传递给模型构造函数的 `apiKey`
2. 传递给模型构造函数的 `authOptions`
3. `API_KEY` 环境变量
4. `GOOGLE_APPLICATION_CREDENTIALS` 路径处的服务账号 JSON
5. 应用默认凭证（例如运行 `gcloud auth application-default login` 后，或在 Google Cloud 上）

请勿在同一项目中使用 `@langchain/google-gauth` 和 `@langchain/google-webauth`。

统一的 [`@langchain/google`](/oss/integrations/chat/google) 包直接使用 `google-auth-library`，不需要 `@langchain/google-gauth` 或 `@langchain/google-webauth`。

### `@langchain/google-cloud-sql-pg`

[`@langchain/google-cloud-sql-pg`](https://www.npmjs.com/package/@langchain/google-cloud-sql-pg) 包为 Cloud SQL for PostgreSQL 提供 [`PostgresVectorStore`](/oss/integrations/vectorstores/google_cloudsql_pg) 和 [`PostgresLoader`](/oss/integrations/document_loaders/web_loaders/google_cloudsql_pg)。它独立于上述 Gemini 聊天包。

### `@langchain/google-common`

[`@langchain/google-common`](https://github.com/langchain-ai/langchainjs/tree/main/libs/providers/langchain-google-common) 包为旧版集成（如 [`@langchain/google-vertexai`](/oss/integrations/chat/google_vertex_ai)）提供共享的 Gemini 客户端抽象。它不包含授权代码，**不是**独立包——请勿直接安装或导入它。

<Tip>
  要从 `@langchain/google-genai` 或 `@langchain/google-vertexai` 迁移到 `@langchain/google`，请参阅 [ChatGoogle](/oss/integrations/chat/google) 页面获取设置说明。`ChatGoogle` 类提供等效功能，可统一访问 Google AI Studio 和 Vertex AI。
</Tip>
