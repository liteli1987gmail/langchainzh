> [Snowflake](https://www.snowflake.com/) 是一个基于云的数据仓库平台
> 允许您存储和查询大量数据。

本页面介绍如何在 `Snowflake` 生态系统中使用 `LangChain`.

## 嵌入模型

Snowflake 提供其开源 `arctic` 系列嵌入模型免费使用
on [Hugging Face](https://huggingface.co/Snowflake/snowflake-arctic-embed-m-v1.5)。最新的模型 snowflake-arctic-embed-m-v1.5 具有 [嵌套嵌入](https://arxiv.org/abs/2205.13147) 功能，支持有效的向量截断。
您可以通过以下方式使用这些模型
[HuggingFaceEmbeddings](/oss/integrations/embeddings/huggingfacehub) connector:

<CodeGroup>
```bash pip
pip install sentence-transformers
```

```bash uv
uv add sentence-transformers
```
</CodeGroup>

```python
from langchain_huggingface import HuggingFaceEmbeddings

model = HuggingFaceEmbeddings(model_name="snowflake/arctic-embed-m-v1.5")
```
