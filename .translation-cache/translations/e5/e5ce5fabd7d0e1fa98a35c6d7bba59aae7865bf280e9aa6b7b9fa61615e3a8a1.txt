> [Snowflake](https://www.snowflake.com/) 是一个基于云的数据仓库平台，
> 可让您存储和查询大量数据。

本页面介绍如何在 `LangChain` 中使用 `Snowflake` 生态系统。

## 嵌入模型

Snowflake 在 [Hugging Face](https://huggingface.co/Snowflake/snowflake-arctic-embed-m-v1.5) 上免费提供其开源的 `arctic` 系列嵌入模型。最新模型 snowflake-arctic-embed-m-v1.5 具有 [matryoshka embedding](https://arxiv.org/abs/2205.13147) 功能，支持有效的向量截断。您可以通过
[HuggingFaceEmbeddings](/oss/integrations/embeddings/huggingfacehub) 连接器使用这些模型：

<CodeGroup>

</CodeGroup>
```bash pip
pip install sentence-transformers
```

```bash uv
uv add sentence-transformers
```
<CodeGroup>


</CodeGroup>

```python
from langchain_huggingface import HuggingFaceEmbeddings

model = HuggingFaceEmbeddings(model_name="snowflake/arctic-embed-m-v1.5")
```
