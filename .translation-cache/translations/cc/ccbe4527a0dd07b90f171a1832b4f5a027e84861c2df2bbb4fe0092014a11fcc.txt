> [Linkup](https://www.linkup.so/) 提供了一个 API，用于将 LLM 与网络和 Linkup Premium 合作伙伴来源连接。

这将帮助您开始使用 LinkupSearchRetriever [检索器](/oss/langchain/retrieval/)。

### 集成详情

| 检索器 | 来源 | 包 |
| :--- | :--- | :--- |
`LinkupSearchRetriever` | 网络和合作伙伴来源 | langchain-linkup |

## 设置

要使用 Linkup 提供商，您需要一个有效的 API 密钥，您可以通过 [注册 Linkup](https://app.linkup.so/sign-up) 获取。然后您可以将其设置为 `LINKUP_API_KEY` 环境变量。对于下面的链示例，您还需要将 OpenAI API 密钥设置为 `OPENAI_API_KEY` 环境变量，您也可以在此处完成设置：

```python
# import os
# os.environ["LINKUP_API_KEY"] = ""  # Fill with your API key
# os.environ["OPENAI_API_KEY"] = ""  # Fill with your API key
```

如果您想从单个查询获取自动跟踪，您也可以通过取消下面的注释来设置您的 [LangSmith](/langsmith/observability) API 密钥：

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

此检索器位于 `langchain-linkup` 包中：

```python
pip install -qU langchain-linkup
```

## 实例化

现在我们可以实例化我们的检索器：

```python
from langchain_linkup import LinkupSearchRetriever

retriever = LinkupSearchRetriever(
    depth="deep",  # "standard" or "deep"
    linkup_api_key=None,  # API key can be passed here or set as the LINKUP_API_KEY environment variable
)
```

## 用法

```python
query = "Who won the latest US presidential elections?"

retriever.invoke(query)
```
