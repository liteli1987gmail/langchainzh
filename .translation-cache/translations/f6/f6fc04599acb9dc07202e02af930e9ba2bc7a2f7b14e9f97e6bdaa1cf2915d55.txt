在 Cerebras，我们开发了世界上最大、最快的 AI 处理器——晶圆级引擎-3（WSE-3）。由 WSE-3 驱动的 Cerebras CS-3 系统代表了一类新型 AI 超级计算机，以无与伦比的性能和可扩展性为生成式 AI 训练和推理设立了新标准。

使用 Cerebras 作为您的推理提供商，您可以：
- 实现 AI 推理工作负载前所未有的速度
- 以高吞吐量进行商业化构建
- 通过我们的无缝集群技术轻松扩展您的 AI 工作负载

我们的 CS-3 系统可以快速、轻松地进行集群，创建世界上最大的 AI 超级计算机，让您轻松部署和运行最大的模型。领先的企业、研究机构和政府已经在使用 Cerebras 解决方案来开发专有模型和训练流行的开源模型。

想体验 Cerebras 的强大功能吗？查看我们的 [网站](https://cerebras.ai) 了解更多资源，并通过 Cerebras Cloud 或本地部署探索访问我们技术的选项！

了解更多关于 Cerebras Cloud 的信息，请访问 [cloud.cerebras.ai](https://cloud.cerebras.ai/)。我们的 API 参考文档可在 [inference-docs.cerebras.ai](https://inference-docs.cerebras.ai/).

## 安装和设置
安装集成包：

<CodeGroup>
```bash pip
pip install langchain-cerebras
```

```bash uv
uv add langchain-cerebras
```
</CodeGroup>

## API 密钥
从获取 API 密钥 [cloud.cerebras.ai](https://cloud.cerebras.ai/) 并将其添加到您的环境变量中：
```
export CEREBRAS_API_KEY="your-api-key-here"
```
