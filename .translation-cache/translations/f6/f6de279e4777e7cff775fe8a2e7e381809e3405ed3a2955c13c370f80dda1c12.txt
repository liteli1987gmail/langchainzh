[Salesforce](https://www.salesforce.com/) 是一家基于云的软件公司，
提供客户关系管理 (CRM) 解决方案和一套企业级
应用，专注于销售、客户服务、营销自动化和分析。

[langchain-salesforce](https://pypi.org/project/langchain-salesforce/) 实现
使 LLMs 能够与 Salesforce 数据交互的工具。


## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-salesforce
```

```bash uv
uv add langchain-salesforce
```
</CodeGroup>

有关可用工具的详细信息，请参阅 [Salesforce 工具文档](/oss/integrations/tools/salesforce/).
