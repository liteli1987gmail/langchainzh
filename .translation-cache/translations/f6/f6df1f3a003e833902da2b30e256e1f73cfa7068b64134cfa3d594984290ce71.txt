> [Polaris AI DataInsight](https://datainsight.polarisoffice.com/playground) 是一个文档解析器，
> 可以从各种文件格式中提取文档元素（文本、图像、复杂表格、图表等）到结构化的 JSON 中，
> 便于集成到 RAG 系统中。

## 安装和设置

```bash
pip install langchain-polaris-ai-datainsight
```

## 文档加载器

查看[使用示例](/oss/integrations/document_loaders/polaris_ai_datainsight)。

```
from langchain_polaris_ai_datainsight import PolarisAIDataInsightLoader
```
