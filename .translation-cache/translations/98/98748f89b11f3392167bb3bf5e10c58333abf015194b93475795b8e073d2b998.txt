[Anchor](https://anchorbrowser.io?utm=langchain) 是 AI Agentic 浏览器自动化平台，解决了自动化缺乏 API 或 API 覆盖有限的 Web 应用程序工作流程的挑战。它简化了基于浏览器的自动化创建、部署和管理，将复杂的 Web 交互转换为简单的 API 端点。

`langchain-anchorbrowser` 提供 3 个主要工具：
- `AnchorContentTool` - 用于网页内容提取，支持 Markdown 或 HTML 格式。
- `AnchorScreenshotTool` - 用于网页截图。
- `AnchorWebTaskTools` - 用于执行网页任务。

## 快速开始

### 安装

安装包：

<CodeGroup>
```bash pip
pip install langchain-anchorbrowser
```

```bash uv
uv add langchain-anchorbrowser
```
</CodeGroup>

### 用法

导入并使用所需的工具。完整的 Anchor 浏览器可用工具列表请参阅 **工具特性** 中的 [Anchor 浏览器工具页面](/oss/integrations/tools/anchor_browser)

```python
from langchain_anchorbrowser import AnchorContentTool

# Get Markdown Content for https://www.anchorbrowser.io
AnchorContentTool().invoke(
    {"url": "https://www.anchorbrowser.io", "format": "markdown"}
)
```
