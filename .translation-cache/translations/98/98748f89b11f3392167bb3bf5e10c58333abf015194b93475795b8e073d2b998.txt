[Anchor](https://anchorbrowser.io?utm=langchain) 是一个用于 AI 智能体浏览器自动化的平台，它解决了自动化缺乏 API 或 API 覆盖有限的 Web 应用程序工作流程的挑战。它简化了基于浏览器的自动化创建、部署和管理，将复杂的 Web 交互转换为简单的 API 端点。

`langchain-anchorbrowser` 提供 3 个主要工具：
- `AnchorContentTool` - 用于以 Markdown 或 HTML 格式提取 Web 内容。
- `AnchorScreenshotTool` - 用于截取网页截图。
- `AnchorWebTaskTools` - 用于执行 Web 任务。

## 快速入门

### 安装

安装该包：

<CodeGroup>
```bash pip
pip install langchain-anchorbrowser
```

```bash uv
uv add langchain-anchorbrowser
```
</CodeGroup>

### 用法

导入并使用您想要的工具。完整的 Anchor Browser 可用工具列表，请参阅 [Anchor Browser 工具页面](/oss/integrations/tools/anchor_browser) 中的 **工具特性** 表

```python
from langchain_anchorbrowser import AnchorContentTool

# Get Markdown Content for https://www.anchorbrowser.io
AnchorContentTool().invoke(
    {"url": "https://www.anchorbrowser.io", "format": "markdown"}
)
```
