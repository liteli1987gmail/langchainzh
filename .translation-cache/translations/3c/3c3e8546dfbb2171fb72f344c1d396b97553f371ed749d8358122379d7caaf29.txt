这个 [MCP Toolbox](https://googleapis.github.io/genai-toolbox/getting-started/introduction/) LangChain 中的 MCP Toolbox 允许您为代理配备一组工具。当代理收到查询时，它可以智能地选择并使用 MCP Toolbox 提供的最合适的工具来满足请求。

## 这是什么？

MCP Toolbox 本质上是您工具的容器。把它想象成代理的多工具设备；它可以容纳您创建的任何工具。然后代理根据用户的输入决定使用哪个特定工具。

当您有一个需要执行各种需要不同能力的任务的代理时，这特别有用。

## 安装

要开始使用，您需要安装必要的包：

<CodeGroup>
```bash pip
pip install toolbox-langchain
```

```bash uv
uv add toolbox-langchain
```
</CodeGroup>
