LangChain 中的 [MCP Toolbox](https://googleapis.github.io/genai-toolbox/getting-started/introduction/) 允许您为智能体配备一组工具。当智能体收到查询时，它可以智能地选择并使用 MCP Toolbox 提供的最合适的工具来满足请求。

## 它是什么？

MCP Toolbox 本质上是您工具的容器。把它想象成您的智能体的多功能设备；它可以容纳您创建的任何工具。然后智能体根据用户输入决定使用哪个特定工具。

当您的智能体需要执行需要不同能力的各种任务时，这尤其有用。

## 安装

要开始使用，您需要安装必要的包：

<CodeGroup>
```bash pip
pip install toolbox-langchain
```

```bash uv
uv add toolbox-langchain
```
</CodeGroup>
