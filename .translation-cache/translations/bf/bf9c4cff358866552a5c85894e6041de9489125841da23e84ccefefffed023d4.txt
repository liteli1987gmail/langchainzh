sidebarTitle: LangGraph v1

**LangGraph v1 是一个专注于稳定性的 agent 运行时版本。** 它保持核心图 API 和执行模型不变，同时改进类型安全、文档和开发者体验。

它旨在与 [LangChain v1](/oss/releases/langchain-v1)（其 `create_agent` 基于 LangGraph 构建）紧密配合，让你可以从高级开始，在需要时深入到细粒度控制。

<CardGroup cols={1}>
    <Card title="稳定的核心 API" icon="sitemap">
        图原语（状态、节点、边）和执行/运行时模型保持不变，使升级简单直接。
    </Card>
    <Card title="默认可靠性" icon="database">
        带检查点、持久化、流式处理和人机交互的持久执行继续作为一流特性。
    </Card>
    <Card title="与 LangChain v1 无缝衔接" icon="link">
        LangChain 的 `create_agent` 在 LangGraph 上运行。使用 LangChain 快速上手；需要自定义编排时深入到 LangGraph。
    </Card>
</CardGroup>

升级方法：

<CodeGroup>

</fragment>
```bash pip
pip install -U langgraph
```
```bash uv
uv add langgraph
```
</CodeGroup>

## `create_react_agent` 的弃用

LangGraph @[`create_react_agent`] 预构建已被弃用，转而使用 LangChain 的 @[`create_agent`]。它提供了更简单的接口，并通过引入中间件提供了更大的自定义潜力。

* 有关新的 @[`create_agent`] API 的信息，请参阅 [LangChain v1 发布说明](/oss/releases/langchain-v1#create_agent)。
* 有关从 @[`create_react_agent`] 迁移到 @[`create_agent`] 的信息，请参阅 [LangChain v1 迁移指南](/oss/migrate/langchain-v1#migrate-to-create_agent)。
