[Tavily](https://tavily.com) 是一个专为 AI 代理设计的搜索引擎，提供搜索、提取、爬取和映射 API，使开发者能够将应用程序连接到实时在线信息。Tavily 的主要使命是从可信来源提供准确可靠的信息，提升 AI 生成内容和推理的准确性与可靠性。

`langchain-tavily` 包将 Tavily 的搜索、提取、爬取和映射端点作为 LangChain 工具暴露出来。

## 安装和设置

安装 LangChain Python 的 Tavily 集成包：

<CodeGroup>

</fragment>
```bash pip
pip install langchain-tavily
```

```bash uv
uv add langchain-tavily
```
</CodeGroup>

[设置 Tavily API 密钥](https://app.tavily.com) 并将其设置为名为 `TAVILY_API_KEY` 的环境变量：

```bash
export TAVILY_API_KEY="your-api-key"
```

## 工具

### TavilySearch

一个从 Tavily 搜索 API 返回实时、LLM 可用结果的搜索工具。

查看[使用示例](/oss/integrations/tools/tavily_search)。

```python
from langchain_tavily import TavilySearch
```

### TavilyExtract

一个返回一到多个 URL 的清洗和解析内容的工具。

查看[使用示例](/oss/integrations/tools/tavily_extract)。

```python
from langchain_tavily import TavilyExtract
```

### TavilyCrawl

一个从基础 URL 执行结构化网络遍历的工具，支持可选的自然语言指令和路径/域名过滤。

查看[使用示例](/oss/integrations/tools/tavily_crawl)。

```python
from langchain_tavily import TavilyCrawl
```

### TavilyMap

一种在不解压页面内容的情况下发现网站 URL 结构的工具。

查看[使用示例](/oss/integrations/tools/tavily_map)。

```python
from langchain_tavily import TavilyMap
```
