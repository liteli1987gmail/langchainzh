## 设置

要创建一个 `AstraDBByteStore` 字节存储，你需要 [创建一个 DataStax 账户](https://www.datastax.com/products/datastax-astra).

### 凭证

注册后，设置以下凭证：

```python
from getpass import getpass

ASTRA_DB_API_ENDPOINT = getpass("ASTRA_DB_API_ENDPOINT = ")
ASTRA_DB_APPLICATION_TOKEN = getpass("ASTRA_DB_APPLICATION_TOKEN = ")
```

### 安装

LangChain AstraDB 集成位于 `langchain-astradb` package:

```python
pip install -qU langchain-astradb
```

## 实例化

现在我们可以实例化字节存储：

```python
from langchain_astradb import AstraDBByteStore

kv_store = AstraDBByteStore(
    api_endpoint=ASTRA_DB_API_ENDPOINT,
    token=ASTRA_DB_APPLICATION_TOKEN,
    collection_name="my_store",
)
```

## 用法

你可以使用以下方式在键下设置数据 `mset` method:

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

你可以使用以下方式删除数据 `mdelete` method:

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```

你可以在任何使用其他 `AstraDBByteStore` 的地方使用它。
