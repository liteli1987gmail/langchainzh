## 设置

要访问 Together AI 嵌入模型，请创建 Together 账户，获取 API 密钥，并安装 `@langchain/together-ai` 集成包。

### 凭据

您可以 [注册 Together 账户并创建 API 密钥](https://api.together.xyz/)。设置 `TOGETHER_AI_API_KEY` 环境变量：

```bash
export TOGETHER_AI_API_KEY="your-api-key"
```

如果您想获取模型调用的自动追踪，还可以设置您的 [LangSmith](/langsmith/observability) API 密钥，请取消下方注释：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain TogetherAIEmbeddings 集成位于 `@langchain/together-ai` package:

<CodeGroup>
```bash npm
npm install @langchain/together-ai @langchain/core
```
```bash yarn
yarn add @langchain/together-ai @langchain/core
```
```bash pnpm
pnpm add @langchain/together-ai @langchain/core
```
</CodeGroup>

## 实例化

实例化嵌入模型：

```typescript
import { TogetherAIEmbeddings } from "@langchain/together-ai";

const embeddings = new TogetherAIEmbeddings({
  model: "togethercomputer/m2-bert-80M-8k-retrieval", // Default value
});
```

## 索引和检索

嵌入模型常用于检索增强生成 (RAG) 流程中，既可以作为索引数据的一部分，也可以在后续检索中使用。如需更详细的说明，请参阅 [**学习** 选项卡下的](/oss/learn/).

请参见下方，了解如何使用 `embeddings` 对象。在本例中，我们将使用演示 [`MemoryVectorStore`](/oss/integrations/vectorstores/memory).

```typescript
// Create a vector store with a sample text
import { MemoryVectorStore } from "@langchain/classic/vectorstores/memory";

const text = "LangChain is the framework for building context-aware reasoning applications";

const vectorstore = await MemoryVectorStore.fromDocuments(
  [{ pageContent: text, metadata: {} }],
  embeddings,
);

// Use the vector store as a retriever that returns a single document
const retriever = vectorstore.asRetriever(1);

// Retrieve the most similar text
const retrievedDocuments = await retriever.invoke("What is LangChain?");

retrievedDocuments[0].pageContent;
```

```text
LangChain is the framework for building context-aware reasoning applications
```

## 直接使用

在底层，向量存储和检索器的实现正在调用 `embeddings.embedDocument(...)` 和 `embeddings.embedQuery(...)` 来创建 `fromDocuments` 以及检索器的 `invoke` 操作中使用的文本嵌入。

您可以直接调用这些方法来获取用于您自己用例的嵌入。

### 嵌入单个文本

您可以使用 `embedQuery`嵌入搜索查询。这会生成特定于查询的向量表示：

```typescript
const singleVector = await embeddings.embedQuery(text);

console.log(singleVector.slice(0, 100));
```

### 嵌入多个文本

您可以使用 `embedDocuments`嵌入多个文本进行索引。此方法使用的内部实现可能（但不一定）与嵌入查询不同：

```typescript
const text2 = "LangGraph is a library for building stateful, multi-actor applications with LLMs";

const vectors = await embeddings.embedDocuments([text, text2]);

console.log(vectors[0].slice(0, 100));
console.log(vectors[1].slice(0, 100));
```
