[Together AI](https://www.together.ai/) 提供了一个 API，只需几行代码即可查询[50 多个领先的开源模型](https://docs.together.ai/docs/inference-models)。

本文档将帮助您开始使用 LangChain 进行 Together AI 文本补全模型（LLM）。有关 `TogetherAI` 功能和配置选项的详细文档，请参阅 [API 参考](https://reference.langchain.com/javascript/langchain-together-ai/TogetherAI)。

## 设置

要访问旧的 `TogetherAI` 补全类，您需要创建一个 Together 账户、[获取 API 密钥](https://api.together.xyz/)，并安装 `@langchain/together-ai` 集成包。对于聊天和指导模型，建议使用 [`ChatTogetherAI`](/oss/integrations/chat/togetherai)。

### 凭据

访问 [api.together.ai](https://api.together.ai/) 注册 Together AI 并生成 API 密钥。设置 `TOGETHER_AI_API_KEY` 环境变量：

```bash
export TOGETHER_AI_API_KEY="your-api-key"
```

如果您想获取模型调用的自动追踪，您也可以通过取消下面的注释来设置您的 [LangSmith](/langsmith/observability) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain TogetherAI 集成位于 `@langchain/together-ai` 包中：

<CodeGroup>
```bash npm
npm install @langchain/together-ai @langchain/core
```
```bash yarn
yarn add @langchain/together-ai @langchain/core
```
```bash pnpm
pnpm add @langchain/together-ai @langchain/core
```
</CodeGroup>

## 实例化

实例化补全模型：

```typescript
import { TogetherAI } from "@langchain/together-ai";

const llm = new TogetherAI({
  model: "meta-llama/Meta-Llama-3.1-8B-Instruct-Turbo",
  maxTokens: 256,
});
```
