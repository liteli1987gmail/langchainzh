[Nia](https://trynia.ai) 是一个搜索和索引 API，持续从文档、研究论文、数据集、代码库等来源提供上下文，使代理永远不会依赖过时数据。可扩展，成本降低 5 倍，且可靠。

## 安装和设置

<CodeGroup>
```bash pip
pip install langchain-nia
```

```bash uv
uv add langchain-nia
```
</CodeGroup>

设置您的 API 密钥：

```bash
export NIA_API_KEY="nk_..."
```

## 工具

此 `langchain-nia` 包提供了 20 个工具，组织成一个可配置的 toolkit。

请参阅 [Nia Toolkit 集成指南](/oss/integrations/tools/nia) 获取完整详情、使用示例和可用工具的完整列表。

```python
from langchain_nia import NiaToolkit

toolkit = NiaToolkit()
tools = toolkit.get_tools()  # 20 tools
```
