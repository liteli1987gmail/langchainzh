[ClickSend](https://www.clicksend.com/) 是一个基于云的通信平台，使开发者能够通过统一的 API 发送短信、电子邮件、语音、传真和邮寄消息。

## 概述

### 集成详情

| 类 | 包 | 可序列化 | [Python 支持](/oss/integrations/tools/clicksend) | 版本 |
|:---|:---|:---:|:---:|:---:|
| `ClickSendTools` | [@clicksend/langchain-clicksend-mcp](https://www.npmjs.com/package/@clicksend/langchain-clicksend-mcp) | ❌ | ❌ | ![npm 版本](https://img.shields.io/npm/v/@clicksend/langchain-clicksend-mcp?style=flat-square&label=%20) |

### 工具功能

- **短信**：向全球手机号码发送文本消息
- **电子邮件**：发送交易和营销电子邮件
- **语音**：发送自动语音消息和通话
- **MCP 集成**：通过模型上下文协议连接，实现无缝的代理集成

## 设置

该集成位于 `@clicksend/langchain-clicksend-mcp` package.

```bash
npm install @clicksend/langchain-clicksend-mcp
```

### 凭证

您需要一个 ClickSend 账户和 API 凭证。请在以下网址注册 [clicksend.com](https://www.clicksend.com/) 获取您的用户名和 API 密钥。

```typescript Set credentials
import dotenv from 'dotenv';
dotenv.config();
```

## 实例化

通过连接 ClickSend MCP 服务器来初始化 ClickSend 工具：

```typescript Initialize ClickSend
import { ClickSendTools } from "@clicksend/langchain-clicksend-mcp";

const tools = await ClickSendTools.fromHTTP({
  url: "https://mcp.clicksend.com",
  username: process.env.CLICKSEND_USERNAME,
  apiKey: process.env.CLICKSEND_API_KEY,
});

console.log(`Loaded ${tools.length} ClickSend tools`);
```

## 调用

在您的 LangChain 代理中使用这些工具发送通信：

```typescript Use with an agent
import { ChatOpenAI } from "@langchain/openai";
import { createAgent } from "langchain";

// Create agent with tools
const agent = await createAgent({
  model: new ChatOpenAI({ 
    model: "gpt-4o-mini",
    temperature: 0 
  }),
  tools,
});

// Use the agent
const result = await agent.invoke({
  messages: [
    { role: "user", content: "Send an SMS to +1234567890 saying 'Hello from LangChain!'" },
  ],
});

console.log(result);
```
