>[Hugging Face 上的 BGE 模型](https://huggingface.co/BAAI) 是由[北京人工智能研究院 (BAAI)](https://en.wikipedia.org/wiki/Beijing_Academy_of_Artificial_Intelligence)发布的开源嵌入和重排序模型系列。BGE 在 2023 和 2024 年是最领先的开源嵌入模型家族之一，虽然 [MTEB 排行榜](https://huggingface.co/spaces/mteb/leaderboard) 上的更新模型在原始检索分数上已超越它们，但 BGE（尤其是 `BAAI/bge-m3`）仍然是多语言检索中广泛使用、均衡的默认选择。

LangChain 提供了两种使用 BGE 模型的方式：

- `langchain-huggingface` 中的 `HuggingFaceEmbeddings`：通用的 Sentence Transformers 类。覆盖所有 BGE 变体，是新项目的**推荐**选择。

## `BAAI/bge-m3` 及更新版本

```shell
pip install -qU langchain-huggingface
```

```python
from langchain_huggingface import HuggingFaceEmbeddings

embeddings = HuggingFaceEmbeddings(
    model_name="BAAI/bge-m3",
    encode_kwargs={"normalize_embeddings": True},
)
```

`BAAI/bge-m3` 使用无查询提示的方式训练，因此无需额外配置。对于余弦相似度，推荐设置 `normalize_embeddings=True`，模型作者也建议这样做。

## `BAAI/bge-*-en-v1.5`（快速路径）

```python
from langchain_huggingface import HuggingFaceEmbeddings

embeddings = HuggingFaceEmbeddings(
    model_name="BAAI/bge-large-en-v1.5",
    encode_kwargs={"normalize_embeddings": True},
    query_encode_kwargs={
        "prompt": "Represent this sentence for searching relevant passages: ",
        "normalize_embeddings": True,
    },
)
```

不同的 BGE 变体使用不同的提示词；请查看 Hugging Face 上各模型卡获取确切的字符串。

## 选择 BGE 模型

| 模型 | 规模 | 备注 |
|---|---|---|
| `BAAI/bge-small-en-v1.5` | 33M | 最小英文模型，适合 CPU |
| `BAAI/bge-large-en-v1.5` | 335M | 更强的英文能力，广泛使用 |
| `BAAI/bge-m3` | 570M | 多语言；dense、sparse 和 multi-vector 于一身 |

有关重排序（不是嵌入），请通过 [Cross Encoder Reranker 指南](/oss/integrations/document_transformers/cross_encoder_reranker) 查看 [`BAAI/bge-reranker-v2-m3`](https://huggingface.co/BAAI/bge-reranker-v2-m3)。
