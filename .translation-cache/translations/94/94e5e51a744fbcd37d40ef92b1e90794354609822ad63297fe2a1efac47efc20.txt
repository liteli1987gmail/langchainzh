代理技能是可重用的按需功能，将指令与可选的辅助脚本捆绑在一起。本页总结了你可添加到兼容编码代理的 LangSmith 定向技能，用于查询追踪、生成数据集和定义评估器。若要直接从终端处理相同的 LangSmith 数据，请使用 [LangSmith CLI](/langsmith/langsmith-cli).

<Note>
这些技能遵循代理技能规范，并维护在 [`langsmith-skills` GitHub 仓库](https://github.com/langchain-ai/langsmith-skills)中。你可以复制 `SKILL.md` 及任何引用的 `scripts/` 到你的代理技能目录中。下面的安装程序仅安装 LangSmith 技能（追踪、数据集、评估器）。
</Note>

## 快速安装

使用 `npx skills`:

<CodeGroup>
```bash Local (current project)
npx skills add langchain-ai/langsmith-skills --skill '*' --yes
```

```bash Global (all projects)
npx skills add langchain-ai/langsmith-skills --skill '*' --yes --global
```

```bash Link to a specific agent (e.g., Claude Code)
npx skills add langchain-ai/langsmith-skills --agent claude-code --skill '*' --yes --global
```
</CodeGroup>

<Tip>
仅安装 LangSmith 技能（追踪、数据集、评估器）。要更新，请重新运行命令。如果目标技能文件夹已存在，请先删除它们（例如 `rm -rf ~/.claude/skills/langsmith-*`).
</Tip>

## 配置环境变量

安装技能后，设置所有 LangSmith 技能、辅助脚本和 [LangSmith CLI](/langsmith/langsmith-cli):

```bash
export LANGSMITH_API_KEY=<your-key>
# Optional defaults
export LANGSMITH_PROJECT=<default-project>
# Advanced: multi-workspace or certain self-hosted setups only
# export LANGSMITH_WORKSPACE_ID=<workspace-id>
```

## 使用的环境变量

- [追踪](/langsmith/observability-concepts#traces)：向应用程序添加追踪；列出、筛选、检查和导出追踪以进行调试和分析。
- [数据集](/langsmith/evaluation-concepts#datasets)：将追踪转换为评估数据集（最终_响应、单_步、轨迹、RAG），并可选择上传到 LangSmith。
- [评估器](/langsmith/evaluation-concepts#evaluators)：定义代码或 LLM 即评判评估器，并将其附加到数据集（离线）或项目（在线）。

每个技能目录都附带一个 `SKILL.md` 以及可选的 `scripts/` 辅助脚本，你可以运行或改编。这些技能设计为插入兼容的编码代理（如 Claude Code 或 Deep Agents Code），如果你不想配置完整的代理，也可以直接重用辅助脚本。对于更重的查询、导出或自动化，你可以将这些技能与 [LangSmith CLI](/langsmith/langsmith-cli) 配对，从终端脚本化处理相同的项目、数据集和评估器。

## 手动安装

克隆仓库并运行安装脚本以获取更多选项：

```bash
git clone https://github.com/langchain-ai/langsmith-skills.git
cd langsmith-skills

# Install for Claude Code in current directory (default)
./install.sh

# Install for Claude Code globally
./install.sh --global

# Install for Deep Agents Code in current directory
./install.sh --deepagents

# Install for Deep Agents Code globally (includes agent persona)
./install.sh --deepagents --global

# Install only LangSmith skills (any target)
./install.sh --langsmith
```

如果你只想要复制特定的技能，请将所需目录从 `config/skills/` 复制到你的代理技能文件夹中。

包含的 LangSmith 技能：
- langsmith-trace — Traces (query/export)
- langsmith-dataset — Datasets (generate/upload)
- langsmith-evaluator — Evaluators (create/attach)
