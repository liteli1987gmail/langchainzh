[OceanBase 数据库](https://github.com/oceanbase/oceanbase) 是一个分布式关系型数据库。
它完全由蚂蚁集团开发。OceanBase 数据库构建在普通的服务器集群之上。
基于 Paxos 协议及其分布式架构，OceanBase 数据库提供高可用性和线性扩展能力。

OceanBase 目前具备向量存储能力。用户可以通过 SQL 轻松执行以下操作：

- 创建包含向量类型字段的表；
- 基于 HNSW 算法创建向量索引表；
- 执行向量近似最近邻查询；
- ...

## 安装

<CodeGroup>
```bash pip
pip install -U langchain-oceanbase
```

```bash uv
uv add langchain-oceanbase
```
</CodeGroup>

我们推荐使用 Docker 部署 OceanBase：

```shell
docker run --name=ob433 -e MODE=slim -p 2881:2881 -d oceanbase/oceanbase-ce:4.3.3.0-100000132024100711
```

[更多 OceanBase 集群部署方法](https://github.com/oceanbase/oceanbase-doc/blob/V4.3.1/en-US/400.deploy/500.deploy-oceanbase-database-community-edition/100.deployment-overview.md)
