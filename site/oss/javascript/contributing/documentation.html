快速开始 - 本地开发

要运行文档的本地预览：

git clone https://github.com/langchain-ai/docs.git
cd docs
make install
make dev

这将启动一个热重载开发服务器，访问地址为 http://localhost:3000。编辑 src/ 中的文件即可即时查看更改。

建议

**使用 AI 编码代理？** 安装 LangChain Skills 来提升代理在 LangChain 生态任务中的表现，然后点击页面右上角的"复制页面"按钮，将原始内容粘贴到你的代理中，让它自动为你配置环境。

建议

如果本地预览遇到问题，请尝试运行 mint update 以确保你使用的是最新版本的 Mintlify。

前提条件

**必需：** - Python 3.13+ - uv - Python 包管理器 - Node.js 和 npm - Make - Git

**可选：** - markdownlint-cli - npm install -g markdownlint-cli - Mintlify MDX VSCode 扩展 </section>

操作指南

面向任务的指导，面向已知晓目标的用户。

概念指南

提供更深入理解和见解的解释性内容。

API 和实现细节的技术描述。

教程

操作指南

技巧

### 将 Python 和 JavaScript/TypeScript 内容放在一起

所有文档在可能的情况下都必须用 Python 和 JavaScript/TypeScript 两种语言编写。为此，我们使用一种自定义的内联语法来区分应单独或同时出现在不同语言中的部分：
mdx :::python Python-specific content. In real docs, the preceding backslash (before `python`) is omitted. :::

:::js JavaScript/TypeScript-specific content. In real docs, the preceding backslash (before js) is omitted. :::

Content for both languages (not wrapped)

这将生成两个输出（每个语言各一个），分别位于 `/oss/python/concepts/foo.mdx` 和 `/oss/javascript/concepts/foo.mdx`。每个输出的页面都需要添加到 `/src/docs.json` 文件中以包含在导航中。


<aside class="callout"><strong>提示</strong>

    我们不希望缺乏对等性而阻止贡献。如果某个功能仅在一种语言中可用，则可以仅在该语言中提供文档，直到另一种语言跟进。在这种情况下，请包含一个说明，指出该功能在另一种语言中尚不可用。

    如果您在 Python 和 JavaScript/TypeScript 之间翻译内容时需要帮助，请在 [社区 Slack](https://www.langchain.com/join-community) 中提问，或在您的 PR 中标记维护者。

### 交叉引用

使用一致的交叉引用来连接文档和 API 参考文档。

**从文档到 API 参考：**

使用 `@[]` 语法链接到 API 参考页面：
mdx See @[`ChatAnthropic`] for all configuration options.

The @[bind_tools][ChatAnthropic.bind_tools] method accepts... ```

构建管道会根据当前的语言范围（Python 或 JavaScript）将这些转换为正确的 Markdown 链接。例如，@[ChatAnthropic] 会变成指向 Python 或 JS API 参考页面的链接，具体取决于正在构建的文档版本，**但前提是 link_map.py 文件中存在相应的条目！** 详见下文。

自动链接的工作原理

@[] 语法由 handle_auto_links.py 处理。它会在 link_map.py 中查找链接键，其中包含 Python 和 JavaScript 范围的字典映射。

**支持的格式：**

语法结果
@[ChatAnthropic]显示文本为 "ChatAnthropic" 的链接
` @[ChatAnthropic] `显示文本为 ` ChatAnthropic ` (代码格式) 的链接
@[text][ChatAnthropic]显示文本为 "text"，链接键为 ChatAnthropic 的链接
\@[ChatAnthropic]转义：渲染为字面量 @[ChatAnthropic]（无链接——本页使用的就是这个！）

**添加新链接：**

如果链接在映射中未找到，它将保持不变输出到最终结果中。添加新的自动链接：

  1. 1. 打开 pipeline/preprocessors/link_map.py
  2. 2. 在 LINK_MAPS 中向相应的范围（python 或 js）添加条目
  3. 3. 键是 @[key] 或 @[text][key] 中使用的链接名称，值是相对于参考主机的路径

本地化

当某个功能在两个 SDK 中都存在时，请将 Python 和 JavaScript/TypeScript 的文档放在一起。如果目前仅支持一种语言，请确保该功能及其引用仅对该语言可见。
