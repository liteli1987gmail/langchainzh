本页面涵盖 LangChain 与 Microsoft Azure 及 Microsoft 其他产品的所有集成。

建议

**推荐：Azure OpenAI**

我们推荐在聊天模型、LLMs和嵌入模型中使用 Azure OpenAI。通过 v1 API（2025 年 8 月正式发布），您可以直接使用 Azure 端点和 API 密钥，通过 langchain-openai 包调用部署在 Microsoft Foundry 中的任何模型（包括 OpenAI、Llama、DeepSeek、Mistral 和 Phi），通过单一界面即可完成。您还可以获得对 Microsoft Entra ID 身份验证的原生支持，以及访问最新功能，包括 Responses API 和推理模型。从这里开始。

**示例和教程：** - Azure-Samples/langchain-azure-openai-starter：从生产就绪的 LangChain 和 Azure OpenAI 应用模板开始，只需一个 azd 命令即可直接部署到 Azure。 - microsoft/langchain-for-beginners：一门介绍 LangChain 与 Azure OpenAI 配合使用的手把手课程。 - Azure-Samples/langchain-agent-python：在 Azure 上构建和部署 LangChain 智能体。

Microsoft Foundry 还提供对所有 Anthropic Claude 模型的访问，包括 Opus、Sonnet 和 Haiku。Claude 模型通过专用的 Anthropic 原生端点提供服务，而非 Azure OpenAI v1 API。使用指向 Foundry Anthropic 端点的 langchain-anthropic。

聊天模型

Microsoft 提供三种主要方式通过 Azure 访问聊天模型：

1. **Azure OpenAI**（推荐）— 通过单一界面访问部署在 Microsoft Foundry 中的任何模型（包括 OpenAI、Llama、DeepSeek、Mistral 和 Phi），并提供企业级功能，例如通过 Microsoft Entra ID 实现无密钥身份验证、区域数据驻留和私有网络。使用 v1 API 上的 ChatOpenAI，或使用传统部署的 AzureChatOpenAI。

Azure OpenAI 还支持 Responses API，让您可以直接从聊天模型访问服务器端工具，如代码解释器、图像生成和文件搜索。 2. **Azure AI** — 推荐用于在聊天模型旁边访问更广泛 Azure 生态系统中的工具、存储和自定义中间件。 3. **Azure ML** — 允许通过 Azure Machine Learning 部署和管理自定义或微调的开源模型。

Azure OpenAI

要开始使用 Azure OpenAI，请创建 Azure 部署并安装 langchain-openai 包：

    pip install -U langchain-openai
    
    uv add langchain-openai

在 v1 API 上，直接使用 ChatOpenAI 访问你的 Azure 端点——无需 api_version：

Entra ID（推荐）

        pip install azure-identity

        llm = ChatOpenAI(
            model="gpt-5.4-mini",  # your Azure deployment name
            base_url="https://YOUR-RESOURCE-NAME.openai.azure.com/openai/v1/",

Responses API

Azure OpenAI 支持 Responses API，它提供有状态对话、内置工具（网络搜索、文件搜索、代码解释器）以及结构化推理摘要。当你设置 reasoning 参数时，[ChatOpenAI] 会自动路由到 Responses API，或者你可以使用 use_responses_api=True 显式选择加入：

Entra ID（推荐）

        from azure.identity import DefaultAzureCredential, get_bearer_token_provider
        from langchain_openai import ChatOpenAI

        token_provider = get_bearer_token_provider(
            DefaultAzureCredential(),
            "https://cognitiveservices.azure.com/.default",
        )

        llm = ChatOpenAI(
            model="gpt-5.4-mini",
            base_url="https://YOUR-RESOURCE-NAME.openai.azure.com/openai/v1/",
            api_key=token_provider,
            use_responses_api=True,
        )

        response = llm.invoke("Summarize the bitter lesson.")
        print(response.text)

>Azure Blob 存储 是微软为云提供的对象存储解决方案。Blob 存储专为存储海量非结构化数据而优化。非结构化数据是不遵循特定数据模型或定义的数据，例如文本或二进制数据。

    pip install -U "langchain-azure-ai[tools]"
    
    uv add "langchain-azure-ai[tools]"

请参阅 Azure Logic Apps 集成指南。

from langchain_azure_ai.tools import AzureLogicAppTool

工具包

Microsoft Foundry 项目工具箱

通过模型上下文协议 (MCP) 从 Azure AI Foundry 工具箱动态加载工具。

使用 tools extra 安装包：

    pip install -U "langchain-azure-ai[tools]" langchain-mcp-adapters httpx
    
    uv add "langchain-azure-ai[tools]" langchain-mcp-adapters httpx

请参阅 Azure AI Foundry 工具箱指南。

from langchain_azure_ai.tools import AzureAIProjectToolbox

Microsoft Foundry 工具（原 Azure AI Services）

安装集成包：

    pip install -U "langchain-azure-ai[tools]"
    
    uv add "langchain-azure-ai[tools]"

请参阅使用示例。

from langchain_azure_ai.tools import AzureAIServicesToolkit

AzureAIServicesToolkit 工具包包含以下工具：

  • - 图像分析：AzureAIImageAnalysisTool
  • - 文档智能：AzureAIDocumentIntelligenceTool
  • - 语音转文本：AzureAISpeechToTextTool
  • - 文本转语音：AzureAITextToSpeechTool
  • - 健康文本分析：AzureAITextAnalyticsHealthTool
