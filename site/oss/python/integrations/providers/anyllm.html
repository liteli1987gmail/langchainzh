> any-llm 是一个统一接口，用于调用 OpenAI、Anthropic、Google、本地模型（通过 Ollama/LocalAI）以及更多。只需更改一个字符串即可切换提供商。

安装和设置

pip install langchain-anyllm
uv add langchain-anyllm

你需要为所选提供商获取相应的 API 密钥。API 密钥可以通过 api_key 参数传递，也可以设置为环境变量（例如 OPENAI_API_KEY、ANTHROPIC_API_KEY）。请参阅 any-llm 文档 了解提供商的具体要求。

聊天模型

from langchain_anyllm import ChatAnyLLM
