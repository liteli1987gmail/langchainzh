LangSmith Cloud 和 LangSmith Fleet 的每周更新。

信息

如果您使用自托管的 LangSmith，请参阅自托管更新日志获取更新。

LangSmith Cloud

可观测性与评估

自动化

  • 自动化规则 现在允许您按操作控制跟踪保留，因此被规则匹配的跟踪可以保持基础保留级别，而不会被升级。

Engine

  • 当 GitHub 未连接时，Engine 问题板现在会显示"连接 GitHub"操作，这样您就可以在不离开问题板的情况下设置拉取请求创建。
  • Engine 现在有一个统一的启用屏幕，包含访问请求，且组织设置在一个位置整合了 Engine 使用情况和限制。
  • 组织管理员现在会在支出超过每个配置的阈值时收到 Engine 支出邮件，且暂停或禁用 Engine 现在需要确认。

数据集与实验

  • 实验 现在在标题和进度列中显示实时加载进度，因此您可以实时跟踪已完成的运行和已评估的运行。
  • 评估器 现在在高级选项中包含跟踪保留切换，因此评分后的跟踪在适合您的工作流程时可以保持基础保留级别。
  • 评估器 提示编辑现在提供高级模式，可直接编辑 Mustache 模板并使用独立的变量映射。
  • 您现在可以在创建数据集时应用资源标签，包括从头创建、通过文件上传或克隆。
  • 自动附加的断言评估器 现在从参考输出中读取断言，因此实验分数反映实际的通过和失败结果。

提示词与 Playground

  • OAuth 客户端凭证 现在支持在模型配置上进行每个工作区独立设置，因此工作区管理员可以自助管理保存的提示词和模型的 OAuth。
  • Playground 现在为 OpenAI 推理模型在 Responses API 上公开推理摘要选项。
  • 模型下拉菜单不再为 OpenAI 兼容端点建议 OpenAI 模型，因此您可以输入自己的自定义模型名称。

跟踪

  • 跟踪查询语法 现在有完整的操作符参考、字段表和快速示例，因此 API 过滤更容易发现。
  • OpenTelemetry 指南 现在说明了如何将跨度链接到现有 LangSmith SDK 跟踪，以及当父跨度永远不到达时会发生什么，因此跨进程跟踪更容易调试。

监控与告警

  • 仪表板 现在包含图表构建器，具有图表模板、创建和编辑窗格，以及时间序列图表上的刷选和系列控制。
  • 您现在可以将告警作为原生通知目标发送到 Slack，并可从 UI 连接或断开 Slack 应用。

部署

  • 预览部署 现在为预览提交构建镜像，而不是重用父部署的镜像。

沙箱

  • 沙箱认证代理 现在记录了 GCP 规则和服务账户处理，因此通过代理访问 Google API 更加清晰。
  • 沙箱 现在将 AWS US SaaS 可用性标记为正式发布，因此区域表反映了当前的推广进度。
  • 沙箱 现在支持 Git 挂载和 Google Cloud Storage 存储桶挂载。

管理员与计费

管理

  • 组织设置 现在明确了 SSO/SCIM 组名可以省略空格，因此不允许空格的企业 IdP 仍能正常工作。
  • Vanta MCP 集成现在对所有工作区正式发布。
  • 在创建数据集、提示词和项目时应用标签现在由专门的创建时标签权限控制。

LLM Gateway

  • LLM gateway 现在支持 Vertex AI 的原生 Gemini 路由和 OpenAI embeddings 端点。
  • Gateway guard 策略现在接受粒度 PII 配置和可配置的的超时操作。

使用与计费

  • 粒度计费使用量 现在明确了组织范围，因此您可以更准确地解读使用量总计。
