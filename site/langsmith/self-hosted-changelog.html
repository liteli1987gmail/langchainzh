  • - 隐藏沙箱文件浏览器控件，同时允许显式沙箱摘要下载。
  • - 引擎现在支持可选的月度 LCU 消费限额（由财务、计划或组织管理员设置），一旦达到该限额，新引擎运行将暂停。
  • - Chat-input file uploads in agent builder/fleet now reached the sandbox filesystem at /tmp/uploads/ 当沙箱被启用时。
  • - “Fleet Default”现在在符合条件的计划的模型选择器中排在首位。
  • - 修复了项目统计侧边栏跟踪计数标签和标题布局。
  • - 数据平面无访问权限屏幕上的工作区切换器仅列出当前组织的工作区。
  • - 恢复了自 2026 年 3 月初以来静默失败的企业 Fleet 代理的 cron 执行。
  • - 运行规则 webhook 载荷现在包含 trace_url 每个运行的深度链接。

  • - 沙箱现在允许非 root 用户使用基于密码的 SSH，同时保持 root SSH 登录仅限密钥认证。

重大变更

  • - 已弃用 agent-bootstrap 脚本。LangSmith 代理现在是独立服务，通过 Helm chart 部署，而不是通过 LangSmith Deployment 控制平面。如果您之前在使用 Fleet 通过此脚本进行部署，可能需要迁移。请联系支持团队以协助完成迁移流程。

  • - POST /workspaces/current/members 现在需要 role_id 用于 RBAC启用的组织。如果没有该设置，请求将返回 400 而不是默认返回 WORKSPACE_ADMIN.
  • - 已弃用 USAGE_EXPORT_ADMIN_EMAILS 环境变量。请使用 INSTANCE_ADMIN_EMAILS instead.
  • - 已替换 projects:update-retention 权限替换为 projects:increase-trace-tier 和 projects:decrease-trace-tier 用于分别控制提高和降低追踪保留期。权限已回填到现有角色，因此现有角色无需更改。新角色应使用新权限。请参阅 RBAC 权限.
  • - 添加了 fleet-admin:read 控制新 Fleet Admin 部分的权限。现有租户的管理员需要授予此权限。权限已回填到现有角色，因此现有角色无需更改。新角色应使用新权限。请参阅 RBAC 权限.

基础设施变更

  • - **因 Fleet 重命名导致的章节重命名** —— 作为 Agent Builder 到 Fleet 重命名的一部分（请参阅 重大变更）。您可能需要更新服务账号或调整配置中的值。

管理员变更
