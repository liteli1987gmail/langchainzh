langsmith-0.15.0


LangSmith 自托管 v0.15 引入了**可复用评估器和包含 30+ 评估器模板的库**，集中管理整个工作区的评估；推出了**每个示例的断言**功能，可与标注队列中的参考输出一起使用；支持下载 **Insights 报告**为 PDF 用于离线分析；并推出 **Context Hub**，用于对 agent 指令和工具进行版本控制和环境感知管理。在升级前有几个破坏性变更值得注意：`agent-bootstrap` 脚本已弃用，Agent Builder 更名为 [Fleet](/langsmith/fleet/) 可能需要更新 workload-identity 服务账号，`projects:update-retention` 权限拆分为 `projects:increase-trace-tier` 和 `projects:decrease-trace-tier`。

请按照[升级说明](/langsmith/self-host-upgrades.html)获取所有功能。如需预约 LangChain 支持团队协助升级，请通过[支持门户](https://support.langchain.com)联系团队。

### 破坏性变更

- 弃用了 `agent-bootstrap` 脚本。LangSmith agents 现在是独立服务，通过 Helm chart 部署，而非通过 LangSmith Deployment 控制平面。如果您之前通过此脚本使用 [Fleet](/langsmith/fleet/)，可能需要进行迁移。请联系支持团队以获取迁移指导。
- 将 Agent Builder 重命名为 [Fleet](/langsmith/fleet/)。如果您使用 workload identity，可能需要更新任何服务账号。
- `POST /workspaces/current/members` 现在要求[启用 RBAC](/langsmith/rbac.html)的组织提供 `role_id`。不带此参数的请求将返回 `400`，而非默认为 `WORKSPACE_ADMIN`。
- 弃用了 `USAGE_EXPORT_ADMIN_EMAILS` 环境变量。请改用 `INSTANCE_ADMIN_EMAILS`。
- 将 `projects:update-retention` 权限替换为 `projects:increase-trace-tier` 和 `projects:decrease-trace-tier`，以分别控制提高和降低 trace 保留期限。权限已回填到现有角色，因此现有角色无需更改。新角色应使用新权限。请参阅 [RBAC 权限](/langsmith/rbac.html)。
- 新增了 `fleet-admin:read` 权限，用于访问新的 Fleet Admin 部分。现有租户的管理员需要授予此权限。权限已回填到现有角色，因此现有角色无需更改。新角色应使用新权限。请参阅 [RBAC 权限](/langsmith/rbac.html)。

### 基础设施变更

- **由于 Fleet 重命名而导致的板块重命名** — 作为 Agent Builder 更名为 [Fleet](/langsmith/fleet/) 的一部分（请参阅[破坏性变更](#breaking-changes)），多个板块已重命名。您可能需要更新服务账号或调整配置中的值。

### 新功能

- **Context Hub** — 对 agent 指令和工具进行版本控制和环境感知管理。创建和管理版本化的 [skill 和 agent 仓库](/langsmith/context-engineering-concepts.html)，将 commit 提升到 `staging` 或 `production` 环境，并在运行时按环境标签解析上下文。请参阅[使用 Context Hub](/langsmith/use-the-context-hub.html)和[使用 SDK 管理 contexts](/langsmith/manage-contexts-sdk.html) 开始使用。
- **可复用评估器和评估器模板** — 全新的[评估器](/langsmith/evaluators.html)标签页集中管理您工作区中的所有评估器，提供 30+ 模板覆盖安全性、响应质量、轨迹、用户行为和多模态评估。只需几秒即可将现有评估器附加到新的 tracing 项目中，无需维护重复副本。
- **每个示例的断言** — 在编辑[标注队列](/langsmith/annotation-queues.html)中的示例时，编写[断言](/langsmith/assertions.html)来替代或配合参考输出使用。
- **可下载的 Insights 报告** — 从报告详情页面将 [Insights](/langsmith/insights.html) 报告下载为 PDF 以进行离线分析。

### 管理变更

- **扩展的 ABAC 覆盖范围** — [ABAC](/langsmith/abac.html) 现在适用于 `POST /runs` 和 `POST /runs/batch` 上的 `runs:create`，以及剩余的 `/sessions/{session_id}/` 端点。
- **修复 SCIM 邮箱大小写不匹配问题** — 发送不同邮箱大小写的身份提供商不再被拒绝为邮箱更改尝试。

- **自托管聊天** — 用于理解跟踪、运行和评估器反馈的产品内聊天功能现已可在自托管中使用。
